## スクレイピング戦略

### Step 1: 会議録トップページの取得

`http://www.town.mihama.wakayama.jp/bunya/gikai_kaigiroku/` にアクセスし、年度別の会議録リンクを収集する。

- HTTP のみ対応（HTTPS にリダイレクトされない可能性あり）
- アクセスにタイムアウトが発生しやすいため、タイムアウト値を長め（30 秒以上）に設定する

### Step 2: 年度別一覧から PDF リンクの抽出

各年度ページから会議録 PDF へのリンクを抽出する。

### Step 3: PDF のダウンロード

抽出したリンクから PDF をダウンロードする。

---

## 注意事項

- **HTTP のみ**: サイトが HTTPS に対応していないため、HTTP で接続する必要がある
- **タイムアウト対策**: 調査時にアクセスタイムアウトが複数回発生した。接続タイムアウトを長めに設定し、リトライ処理を実装する
- **詳細 URL 構造**: サイトへのアクセスが安定しないため、URL 構造・PDF 命名規則・年度範囲は実装時に再確認が必要
- テキスト検索システムは存在しないため（備考より）、全 PDF のダウンロードが必要

---

## 推奨アプローチ

1. **HTTP 接続**: `http://` プロトコルを使用し、HTTPS へのアップグレードを行わない
2. **タイムアウト・リトライ**: タイムアウトを 30〜60 秒に設定し、失敗時は 3 回程度リトライする
3. **実装前の再調査**: サイトの安定性を確認した上で、詳細な URL 構造・年度範囲・PDF 命名規則を調査する
4. **レート制限**: アクセスが不安定なサイトのため、リクエスト間に 2〜3 秒の待機時間を設ける
