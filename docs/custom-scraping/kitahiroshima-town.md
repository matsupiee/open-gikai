## ページ構造

### 会議録トップページ（list98.html）

- 年度別のリンクが並ぶシンプルなリスト
- 各年度のリンクは `list98-{ID}.html` 形式の一覧ページへ遷移
- バックナンバーへのリンクも含む

### 年度別一覧ページ（list98-{ID}.html）

- 各年度につき1件のリンクのみが表示される
- リンク先は会議録詳細ページ（`{ページID}.html`）

### 会議録詳細ページ（{ページID}.html）

- 1年度分の全会議録 PDF リンクが1ページにまとめて掲載
- 会議種別ごとにセクション分け（定例会・臨時会）
- 各セクション内に開催日ごとの PDF リンクがリスト形式で並ぶ
- リンクテキスト形式: `令和X年X月X日　会議録 [PDFファイル／XXXkb]`
- PDF URL: `/uploaded/attachment/{数字ID}.pdf`

---

## 会議種別

各年度のページには以下の会議種別が含まれる:

- **定例会**: 年4回（第1回〜第4回）、各回2〜4日程
- **臨時会**: 年2〜4回程度、各回1〜2日程

委員会の会議録は公開されていない。

## スクレイピング戦略

### Step 1: 会議録詳細ページ URL の収集

1. トップページ `list98.html` から年度別一覧ページ（`list98-{ID}.html`）へのリンクを取得
2. バックナンバーページ `list98-325.html` からも各年度の詳細ページ URL を取得
3. 各年度別一覧ページにアクセスし、会議録詳細ページ（`{ページID}.html`）への URL を取得

### Step 2: PDF リンクの収集

各会議録詳細ページ（1年度分が1ページにまとまっている）から全 PDF リンクを抽出する。

```typescript
// PDF リンクの抽出
const pdfLinks = $('a[href$=".pdf"]');
// URL パターン: /uploaded/attachment/{数字ID}.pdf
```

### Step 3: メタ情報の取得

PDF リンクのテキストおよびページの見出し構造から以下を抽出:

- **会議種別**: 定例会 / 臨時会（セクション見出しから判定）
- **回次**: 第X回（セクション見出しから判定）
- **開催日**: リンクテキスト内の `令和X年X月X日` から抽出
- **年度**: ページタイトルまたは URL から判定

```typescript
// 開催日の抽出（リンクテキストから）
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;

// 会議種別の判定（セクション見出しから）
// 例: "第1回定例会", "第2回臨時会"
const sessionPattern = /第(\d+)回(定例会|臨時会)/;
```

### Step 4: PDF ダウンロードとテキスト抽出

- 各 PDF をダウンロードし、テキストを抽出する
- PDF ファイルサイズは 145KB〜580KB 程度

## 注意事項

- **北海道北広島市との混同に注意**: `ssp.kaigiroku.net/tenant/kitahiroshima/` は北海道北広島市の会議録検索システムであり、広島県北広島町のものではない
- **ページネーションなし**: 各年度の会議録詳細ページに全 PDF リンクが1ページにまとまっているため、ページネーション処理は不要
- **委員会の会議録は非公開**: 本会議（定例会・臨時会）のみが対象
- **年度別ページ構造が2層**: トップ → 年度別一覧 → 会議録詳細ページと2段階のリンクを辿る必要がある
- **バックナンバーのURL構造が異なる**: 令和4年以降は `list98-{ID}.html` 経由だが、バックナンバー（平成27年〜令和3年）は `list98-325.html` から直接詳細ページにリンクされている
- PDF ファイル名（数字ID）には規則性がなく、ファイル名からメタ情報を推定することは困難

---

## 推奨アプローチ

1. **トップページ + バックナンバーから全詳細ページ URL を収集**: `list98.html` と `list98-325.html` の2ページを起点に、全年度の詳細ページ URL を網羅的に取得する
2. **詳細ページから PDF URL を一括取得**: 各年度の詳細ページ内の `a[href$=".pdf"]` を全件抽出する
3. **セクション見出しとリンクテキストからメタ情報を構造化**: PDF リンクの前にあるセクション見出し（定例会/臨時会の区分・回次）とリンクテキスト（開催日）を紐づける
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
5. **差分更新**: 既取得済みの PDF ID（attachment ID）と比較して新規分のみ取得する
