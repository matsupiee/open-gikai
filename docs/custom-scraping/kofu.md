## 会議録の提供形式

### 定例会

- 年度別一覧ページから会議別のサブページへリンク
- サブページに日付別の PDF リンクが並ぶ（1会議あたり複数日分）

### 臨時会

- 年度別一覧ページから PDF への直リンク（サブページなし）
- 1回の臨時会につき PDF 1ファイル

### PDF の URL パターン

新しい年度（概ね平成28年以降）:
```
/user/filer_public/{UUID}/{ローマ字ファイル名}.pdf
例: /user/filer_public/05/09/05094fd1-7ae5-4748-b03f-e38d635880e0/jiang-fu-ting-di-07hui-ding-li-_r06nian-12yue-09ri.pdf
```

古い年度（概ね平成27年以前）:
```
/system/site/upload/live/{ID}/atc_{タイムスタンプ}.pdf
例: /system/site/upload/live/540/atc_1323084387.pdf
```

## PDF 内部構造

### メタ情報（冒頭部分）

```
江府町告示第６１号
令和６年１１月２６日
江府町長　白石　祐治

第７回江府町議会１２月定例会を下記のとおり招集する。
記
１、期日　令和６年１２月９日
２、場所　江府町役場議場
```

- 招集告示、応招議員一覧が冒頭に記載
- 続いて議事日程、出席議員、説明出席者の一覧

### 発言者パターン

```
○議長（三好　晋也君）
○町長（白石　祐治君）
○総務課長（生田　志保君）
○教育長（富田　敦司君）
○副町長（八幡　徳弘君）
```

- `○` + 役職名 + `（` + 氏名 + `君）` の形式
- 氏名は姓と名の間に全角スペースが入る場合がある
- 議員番号付きのパターン: `○１番（加藤　邦樹君）`

### 発言の構造

```
○議長（三好　晋也君）　日程第１、会議録署名議員の指名を行います。
```

- 発言者行の直後（同行または次行）に発言内容が続く
- 議事進行の合間に `〔「異議なし」と呼ぶ者あり〕` のような記述あり

### ページ番号

- 各ページの末尾に `－{ページ番号}－` の形式でページ番号が入る

## スクレイピング戦略

### Step 1: 年度別一覧ページから会議リンクを収集

会議録トップページ `/2/1/10/2/` にアクセスし、全年度のリンクを取得する。

**収集方法:**

1. トップページから年度別リンク（16年度分）を取得
2. 各年度ページにアクセスし、会議ごとのリンクを収集
3. 定例会のサブページリンクと臨時会の PDF 直リンクを区別して保存

### Step 2: 定例会サブページから日付別 PDF リンクを収集

定例会のサブページにアクセスし、日付別の PDF リンクを取得する。

**収集方法:**

1. 定例会サブページの HTML から PDF リンク（`.pdf`）を抽出
2. リンクテキストから開催日を抽出

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

**抽出方法:**

1. PDF をダウンロード
2. `pdftotext` 等のツールでテキストを抽出
3. PDF 由来のレイアウト崩れ（氏名の文字間スペース等）を正規化

### Step 4: 会議録のパース

#### メタ情報の抽出

```typescript
// 開催日の抽出
const datePattern = /(?:令和|平成)[０-９\d]+年[０-９\d]+月[０-９\d]+日/;

// 会議名の抽出
const meetingPattern = /第[０-９\d]+回江府町議会[０-９\d]+月(定例会|臨時会)/;
```

#### 発言者の抽出

```typescript
// 発言者パターン（役職 + 氏名）
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○議長（三好　晋也君） → role="議長", name="三好　晋也"
// 例: ○１番（加藤　邦樹君） → role="１番", name="加藤　邦樹"

// 議場内の反応
const reactionPattern = /^〔(.+?)〕/;
// 例: 〔「異議なし」と呼ぶ者あり〕
```

#### PDF テキストの正規化

```typescript
// ページ番号の除去
const pageNumberPattern = /－\d+－/g;

// 全角数字 → 半角数字
const normalizeNumbers = (text: string) =>
  text.replace(/[０-９]/g, (c) => String.fromCharCode(c.charCodeAt(0) - 0xFEE0));

// 氏名の文字間スペース除去（PDF 由来）
// "白　石　祐　治" → "白石祐治"
```

## 注意事項

- 会議録はすべて PDF 形式で提供されており、HTML 本文ページは存在しない
- PDF のテキスト抽出時、表組みや段組みでレイアウトが崩れる可能性がある
- 古い年度と新しい年度で PDF の URL パターンが異なる（`/system/site/upload/` vs `/user/filer_public/`）
- 臨時会は年度一覧ページから PDF への直リンクであり、サブページを経由しない
- 年度 ID にはランダムな英数字が使われており、規則性がないため、トップページからの動的取得が必須
- PDF ファイル名がローマ字（ヘボン式）で会議情報をエンコードしている場合がある（例: `jiang-fu-ting-di-07hui-ding-li-`）

---

## 推奨アプローチ

1. **3段階クロール**: トップ → 年度一覧 → 会議サブページ（定例会のみ）の順で PDF リンクを収集
2. **PDF テキスト抽出**: `pdftotext`（poppler）や `pdf-parse` 等を使用してテキスト化
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
4. **差分更新**: 年度一覧ページの会議数を前回取得時と比較し、新規追加分のみダウンロード
5. **文字正規化**: PDF 抽出テキストの全角数字・文字間スペースを正規化してからパースする
