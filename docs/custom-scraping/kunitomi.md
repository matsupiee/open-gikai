# 国富町議会 カスタムスクレイピング方針

## 概要

- サイト: http://www.town.kunitomi.miyazaki.jp/main/administration/town_council/page000865.html
- 分類: 自治体 CMS による PDF 公開（年度ごとに階層的に掲載）
- 文字コード: UTF-8
- 特記: 会議録は全て PDF ファイルで公開。検索システムなし。1 ページに全年度分を掲載。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧（全年度） | `http://www.town.kunitomi.miyazaki.jp/main/administration/town_council/page000865.html` |
| PDF ファイル | `http://www.town.kunitomi.miyazaki.jp/main/administration/{ハッシュ値}.pdf` |

### PDF URL の特徴

- ファイル名は英数字のハッシュ値（例: `441bef3d8c9e0bf11e53e468548b16fa.pdf`）
- 一部のファイルにはサフィックス `_N` が付く（例: `de6c1e37aad8beb3bdf3b180ba316661_1.pdf`）
- ファイル名からは内容を推測できないため、リンクテキストからメタ情報を取得する必要がある

## ページ構造

### HTML の階層

```
<h2>令和N年会議録</h2>           ← 年度の区切り
  <h3>令和N年第X回定例会</h3>    ← 会議の区切り
    <p><a href="...pdf">令和N年第X回定例会（M月　目次）.pdf</a></p>   ← 目次 PDF
    <p><a href="...pdf">令和N年第X回定例会（M月D日）.pdf</a></p>     ← 日別会議録 PDF
    ...
  <h3>令和N年第X回臨時会</h3>
    <p><a href="...pdf">...</a></p>
    ...
```

### 会議種別

- **定例会**: 年 4 回（第 1 回〜第 4 回）
- **臨時会**: 不定期

### 公開範囲

- 令和 7 年（2025 年）〜 平成 30 年（2018 年）の会議録が掲載
- 平成 30 年第 3 回定例会以前はリンクが空（PDF 未掲載の可能性あり）

---

## リンクテキストのパターン

### 目次 PDF

```
令和N年第X回定例会（M月　目次）.pdf
令和N年第X回臨時会（M月　目次）.pdf
```

### 日別会議録 PDF

```
令和N年第X回定例会（M月D日）.pdf
令和N年第X回臨時会（M月D日）.pdf
```

- 月日は全角数字で表記（例: `９月５日`、`１２月１６日`）
- 平成 30 年のみ半角数字が混在（例: `12月7日`）

---

## スクレイピング戦略

### Step 1: 会議録一覧ページの取得

`page000865.html` を取得し、全ての PDF リンクを抽出する。

- 単一ページに全年度分が掲載されているため、ページネーション不要
- `<a href="...pdf">` タグからリンク URL とリンクテキストを収集

### Step 2: メタ情報の抽出

リンクテキストからメタ情報をパースする。

**パース用正規表現（案）:**

```typescript
// リンクテキストから会議情報を抽出
const sessionPattern = /^(令和|平成)([\d３１]+)年第([\d１２３４]+)回(定例会|臨時会)/;

// 日付の抽出（目次以外）
const datePattern = /（(\d+|[０-９]+)月(\d+|[０-９]+)日）/;

// 目次の判定
const tocPattern = /目次/;
```

**抽出するフィールド:**

| フィールド | 例 | 備考 |
| --- | --- | --- |
| 年号 | 令和 / 平成 | |
| 年 | 7 / 30 | 全角・半角混在あり |
| 回 | 1〜4 | |
| 会議種別 | 定例会 / 臨時会 | |
| 月 | 9 | |
| 日 | 5 | 目次 PDF の場合は null |
| PDF URL | `/main/administration/xxx.pdf` | 相対パス→絶対パスに変換 |

### Step 3: PDF のダウンロードとテキスト抽出

- 各 PDF をダウンロードし、PDF パーサー（pdf-parse 等）でテキストを抽出
- 目次 PDF はスキップまたは別途処理（本文ではないため）

---

## 注意事項

- 全角・半角数字が混在しているため、パース前に正規化が必要
- HTML の `<h2>` / `<h3>` タグの一部に不正な属性（`<a h2="">` `<a h3="">`）が含まれるが、リンクテキストの抽出には影響しない
- 一部の `<a>` タグが分割されている（例: リンクテキストが 2 つの `<a>` タグにまたがる）ため、隣接する同一 href の `<a>` タグを結合する処理が必要
- PDF ファイルのサイズは不明。大量ダウンロード時はディスク容量に注意

---

## 推奨アプローチ

1. **単一ページから全量取得**: 1 ページに全リンクが掲載されているため、クロール負荷は極めて低い
2. **目次 PDF を除外**: `目次` を含むリンクテキストの PDF は会議録本文ではないため除外する
3. **レート制限**: PDF ダウンロード時はリクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 前回取得済みの PDF URL リストと比較し、新規追加分のみダウンロードする
5. **全角→半角変換**: 数字の正規化処理を事前に行い、日付パースの精度を向上させる
