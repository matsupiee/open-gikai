## スクレイピング戦略

### Step 1: 一覧ページから PDF リンクを収集

一覧ページ `336.html` を取得し、`<a href="...">` タグから `.pdf` へのリンクをすべて抽出する。

- ページネーションなし（全件が単一ページに掲載）
- リンクテキストに「会議録」を含むものを会議録として選別し、「議決結果」を含むものは除外する
- href は `//www.vill.mitsue.nara.jp/...` 形式（プロトコル相対 URL）のため、`https:` を補完する

```typescript
// href の正規化例
const href = anchor.attr("href") ?? "";
const url = href.startsWith("//") ? `https:${href}` : href;
```

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF URL に対してダウンロードし、PDF パーサー（`pdf-parse` 等）でテキストを抽出する。

- PDF は画像 PDF ではなくテキスト埋め込み PDF であることを確認済み
- ファイル名から日付・会議種別を推測することは命名規則の不統一により困難なため、PDF 本文から抽出する

### Step 3: メタ情報のパース

PDF のテキスト冒頭から以下を抽出する。

#### 開催日・会議名

```
平成29年5月26日　御杖村議会臨時会会議録
令和7年12月定例会会議録
```

- 開催日: `(?:令和|平成|昭和)(\d+)年(\d+)月(\d+)日` パターン
- 会議種別: `定例会` / `臨時会`

#### 発言の構造

```
○議長（田中○○）　...
○村長（山本○○）　...
○○番（氏名）　...
```

- `○` で始まり、役職または議席番号 + 括弧内に氏名
- 発言内容は同行または次行以降に続く

#### パース用正規表現（案）

```typescript
// 開催日の抽出
const datePattern = /(?:令和|平成|昭和)(\d+)年(\d+)月(\d+)日/;

// 発言者の抽出
const speakerPattern = /^○(.+?)(?:（(.+?)）)?/;
// 例: ○議長（田中一郎） → role="議長", name="田中一郎"
// 例: ○３番（山本花子） → role="３番", name="山本花子"
```

## 注意事項

- **ファイル名に命名規則の統一性なし**: `kaigiroku`、`kaigijiroku`、`ka` など表記が年度によって異なる。一覧ページの href を唯一の正とする。
- **group 番号は非連続**: `group/1`、`group/3`、`group/16` に分散しており、URL を構築して列挙する方法は使えない。
- **議決結果は除外**: 「議決結果」ファイルは会議の発言録ではないため取得対象外とする。
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける。

---

## 推奨アプローチ

1. **一覧ページを起点とする**: `336.html` から PDF リンクを全件取得し、リンクテキストで会議録と議決結果を振り分ける。
2. **差分更新**: 取得済み URL をリストで管理し、新規追加分のみダウンロードする。一覧ページは更新のたびに新しいリンクが追記される構造のため、前回取得時との差分を比較する。
3. **PDF テキスト抽出でメタ情報を補完**: ファイル名に依存せず、PDF 本文から開催日・会議種別を抽出する。
