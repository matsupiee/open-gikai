## HTML 構造

### 年度一覧ページ（list156.html）

```html
<ul>
  <li><a href="/site/gikai/16630.html">令和８年</a></li>
  <li><a href="/site/gikai/14776.html">令和７年</a></li>
  <li><a href="/site/gikai/13204.html">令和６年</a></li>
  ...
</ul>
```

- 各年度へのリンクが `<li><a>` 形式で列挙されている
- リンクテキストは「令和X年」形式（西暦は含まれない）

### 年度別会議録一覧ページ（{ID}.html）

```html
<h2>令和８年　会議録一覧</h2>

<h3>定例会</h3>
<ul>
  <li>
    12月定例会（12月X日）
    <a href="/uploaded/attachment/6900.pdf">本会議（１日目） [PDFファイル／520KB]</a>
  </li>
  ...
</ul>

<h3>臨時会</h3>
<ul>
  <li>
    10月臨時会（10月14日）
    <a href="/uploaded/attachment/6831.pdf">本会議 [PDFファイル／250KB]</a>
  </li>
  ...
</ul>
```

- `<h2>` で年度見出し（例: 「令和X年　会議録一覧」）
- `<h3>` で会議種別（「定例会」「臨時会」）
- PDF リンクは `<a href="/uploaded/attachment/{ID}.pdf">` 形式
- リンクテキストは「本会議（X日目） [PDFファイル／XXXKB]」形式
- 複数日にわたる定例会は日ごとに PDF が分割される（「1日目」「10日目」「16日目」等）
- 1 日のみの会議は「本会議」または「本会議（１日目）」

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

トップページ `https://www.town.matsuno.ehime.jp/site/gikai/list156.html` から年度別ページへのリンクを抽出する。

- `a[href^="/site/gikai/"]` セレクタで全年度ページへのリンクを抽出
- `list156` を含むリンクは除外（トップページ自身へのリンク）
- 年度ページの ID は非連番のため、必ずトップページから動的に取得する

### Step 2: PDF リンクの収集

各年度別ページ `{ID}.html` から PDF リンクを直接抽出する。

- `a[href*="/uploaded/attachment/"][href$=".pdf"]` セレクタで PDF リンクを抽出
- リンクのテキストから会議名（「本会議（X日目）」）を取得
- `<h3>` タグの直前テキストから会議種別（「定例会」「臨時会」）を取得
- 見出し前後のテキスト（「X月定例会（X月X日）」等）から開催月・日を取得

### Step 3: PDF のダウンロード

収集した URL から PDF をダウンロードする。

- 総 PDF 数は約 50〜80 件（8 年分 × 年 6〜8 件程度）
- 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

### Step 4: PDF からのテキスト抽出・パース

PDF からテキストを抽出し、構造化データに変換する。

- PDF パーサー（pdf-parse 等）でテキストを抽出
- 1 つの PDF に 1 日分の会議の全発言が含まれる

#### メタ情報の抽出

年度別ページの HTML から以下を取得:

```
令和7年　会議録一覧
  定例会
    12月定例会（12月X日）
      本会議（１日目） → /uploaded/attachment/6900.pdf
  臨時会
    10月臨時会（10月14日）
      本会議 → /uploaded/attachment/6831.pdf
```

- 開催年度: `<h2>` の「令和X年」から抽出
- 開催月・日: PDF リンク前後のテキスト「X月定例会（X月X日）」から抽出
- 会議種別: 直前の `<h3>` テキスト（「定例会」「臨時会」）から抽出
- 日次区分: リンクテキスト「本会議（X日目）」の X 日目部分から抽出

#### 発言の構造（PDF 内テキスト）

PDF 内のテキスト構造は実際のファイルをダウンロードして確認する必要がある。一般的な議会会議録 PDF では以下のパターンが想定される:

```
○議長（氏名）
  発言内容...

○X番（氏名）
  発言内容...

○町長（氏名）
  発言内容...
```

※ 実際のフォーマットは PDF ダウンロード後に確認・調整が必要

## 注意事項

- **平成31年 / 令和元年の扱い**: 令和元年のページ（`10863.html`）には「平成31年3月定例会」として掲載されている会議録が含まれる。年度情報をパースする際は「平成31年」も令和元年度として処理すること。
- **年度別ページ URL の非連番**: `{ID}` 部分は非連番（例: 10861, 10863, 10864, 10865 が令和2〜4年に対応）なため、URL を固定値でハードコードせずトップページから動的に取得する。
- **複数 PDF の統合**: 1 つの定例会が複数の PDF に分割されている場合（1日目・10日目・16日目等）、同一会議の連続する日程として扱う。
- **PDF 品質の確認**: テキスト PDF かスキャン PDF かを事前に数件ダウンロードして確認すること。スキャン PDF の場合は OCR が必要になる。

---

## 推奨アプローチ

1. **2段階のクロール**: トップページ → 年度別一覧ページ → PDF という 2 段階の階層構造をたどる（詳細ページは存在しない）
2. **動的な URL 収集**: 年度ページの ID は非連番のため、必ずトップページから URL を動的に取得する
3. **H3 タグによる種別判定**: `<h3>` タグを手がかりに「定例会」「臨時会」を判定し、各 PDF に会議種別を付与する
4. **PDF テキスト品質の事前確認**: 数件の PDF をダウンロードしてテキスト抽出の品質を確認し、スキャン PDF かテキスト PDF かを判定する
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
