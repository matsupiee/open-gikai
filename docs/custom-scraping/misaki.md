# 岬町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.misaki.osaka.jp/soshiki/gikai/chogikai/gijiroku/index.html
- 分類: 自治体公式サイト上で年度別に PDF ファイルとして会議録を公開（外部の会議録検索システムは未導入）
- 文字コード: UTF-8
- 特記: 会議録は全て PDF 形式で提供。HTML 本文ページは存在しない

## PDF ファイル命名規則

### 基本パターン

`{元号コード}{年2桁}{月日4桁}{会議種別}.pdf`

### 元号コード

| 元号 | コード | 例 |
| --- | --- | --- |
| 令和 | `r` | `r061224tei3.pdf` |
| 平成 | なし（年月日のみ） | `180307tei.pdf` |

### 会議種別コード

| コード | 会議種別 |
| --- | --- |
| `tei` / `tei1` / `tei2` / `tei3` 等 | 定例会（数字は日程の日数目） |
| `rinji` / `rin` | 臨時会 |
| `soumu` / `soumubunkyo` | 総務文教委員会 |
| `kousei` | 厚生委員会 |
| `jigyo` / `jigyou` / `jigyominsei` / `jigyouminsei` | 事業委員会（旧：事業民生委員会） |

### ファイル名の例

```
r061224tei3.pdf       → 令和6年12月24日 定例会3日目
r061211soumu.pdf      → 令和6年12月11日 総務文教委員会
r060508rinji.pdf      → 令和6年5月8日 臨時会
180307tei.pdf         → 平成18年3月7日 定例会
180509rin.pdf         → 平成18年5月9日 臨時会
180309soumu.pdf       → 平成18年3月9日 総務文教委員会
```

※ ファイル名の表記揺れ（`jigyo` / `jigyou`、`rinji` / `rin` 等）があるため、ファイル名からの会議種別推定は補助的に使い、ページ上のテキストから正確な会議名を取得する。

## 会議種別

### 本会議

- **定例会**: 年4回（3月・6月・9月・12月）。複数日にわたる場合あり（1日目〜5日目）
- **臨時会**: 不定期開催

### 常任委員会（3種）

- **総務文教委員会**
- **厚生委員会**
- **事業委員会**（旧：事業民生委員会）

### その他

- **子ども議会**（平成27年度のみ確認）

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

トップページ `/gijiroku/index.html` から年度別ページへのリンクを収集する。

- `<ul><li>` 構造のリストで年度名とリンクが表示されている
- リンクは `/gijiroku/{ページID}.html` 形式
- ページネーションなし（全年度が1ページに表示）

### Step 2: 各年度ページから PDF リンクの収集

各年度ページ `/gijiroku/{ページID}.html` にアクセスし、PDF リンクを収集する。

- PDF リンクは `<ul><li>` 構造のリスト内に表示
- URL パターン: `//www.town.misaki.osaka.jp/material/files/group/29/{ファイル名}.pdf`
- 各リンクのテキストに会議名と日付が含まれる（例: 「令和6年12月24日 第4回定例会3日目」）
- ページネーションなし（1年度分の全会議録が1ページに表示）
- リンクテキストからメタ情報（日付、会議種別、回数、日数目）を抽出する

### Step 3: PDF のダウンロードとテキスト抽出

- PDF をダウンロードし、テキスト抽出ライブラリ（pdf-parse 等）で本文を取得する
- PDF 内のテキストから発言者・発言内容を構造化する

### Step 4: メタ情報のパース

年度ページ上のリンクテキストから以下を抽出:

```
令和6年12月24日 第4回定例会3日目
```

- 開催日: `令和X年XX月XX日` 形式
- 会議種別: 定例会 / 臨時会 / 常任委員会名
- 回次: `第X回`
- 日程: `X日目`（定例会のみ）

#### パース用正規表現（案）

```typescript
// リンクテキストから日付を抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;

// 会議種別の抽出
const sessionPattern = /第(\d+)回(定例会|臨時会)(?:(\d+)日目)?/;

// 常任委員会の判定
const committeePattern = /(総務文教|厚生|事業(?:民生)?)委員会/;
```

---

## 注意事項

- 会議録は全て PDF 形式のため、HTML パースではなく PDF テキスト抽出が必要
- PDF ファイル名の命名規則に表記揺れがあるため、ファイル名だけでなくリンクテキストからメタ情報を取得する
- 平成18年〜令和7年の約20年分、推定200件以上の PDF が対象
- PDF 内の発言者パターンは PDF の内容次第であり、ダウンロード後に別途分析が必要

---

## 推奨アプローチ

1. **2段階クロール**: トップページ → 年度ページ → PDF URL の順にリンクを収集
2. **リンクテキストからメタ情報抽出**: PDF をダウンロードする前にリンクテキストから日付・会議種別を取得し、DB に登録
3. **PDF テキスト抽出**: pdf-parse 等のライブラリで PDF からテキストを抽出
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度ページの更新日表示を活用し、更新があった年度のみ再クロールする
