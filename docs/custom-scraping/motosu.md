## スクレイピング戦略

### Step 1: 年度別カテゴリページから会議録詳細ページ URL を収集

年度別カテゴリページ `category/6-3-{年度ID}-0-0-0-0-0-0-0.html` の `ul/li` リストから、各会議録詳細ページへのリンク（`/{7桁ページID}.html` 形式）を抽出する。

- ページネーションは存在しない（1 ページに全件掲載）
- 各年度 ID（16 〜 23、1 〜 2）を順番にクロールする

### Step 2: 会議録詳細ページから PDF URL を収集

各会議録詳細ページに列挙されている PDF リンクを抽出する。

- PDF URL は相対パス `./cmsfiles/contents/{上位ページID}/{7桁ページID}/{ファイル名}.pdf` で記述されている
- 会議開催日別に 1 日 1 PDF が対応する（1 定例会あたり複数 PDF が存在する）
- 「付された案件」等の補足 PDF も含まれるため、会議録本文 PDF を判別する必要がある

**会議録本文 PDF の判別:**
- リンクテキストが「N月N日（曜日）」形式のものが本文 PDF
- 「第N回定例会に付された案件」等のテキストは案件一覧 PDF のため除外する

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、`pdftotext` 等でテキストを抽出する。

- PDF は FlateDecode 圧縮済みテキスト PDF（スキャン画像ではない）
- フォント: MS-Mincho、MS-Gothic（日本語対応）
- 平成 16 年（2004 年）創立時の PDF から令和の PDF まで同一フォーマットで抽出可能

## 会議録のフォーマット

### メタ情報

PDF 冒頭の議事日程部分から抽出:

```
令和６年第４回本巣市議会定例会議事日程（第１号）
令和６年11月25日（月曜日）午前９時 開議
```

- 会議名: `{元号}{年}年第{回}回本巣市議会{種別}会議事日程（第{号}号）`
- 開催日: `{元号}{年}年{月}月{日}日（{曜日}）`

### 発言の構造

発言者は `○` で始まる行で表記され、括弧内に氏名が含まれる:

```
○議長（大西德三郎君）
○１０番（今枝和子君）
○市長（藤原 勉君）
○副市長（谷口博文君）
○総務部長（村澤 勲君）
○市民部長（加納正康君）
○議会だより編集特別委員会委員長（寺町 茂君）
```

- 役職と議席番号が混在する
- 氏名末尾は `君` または `さん` が付く（「君」が大多数）
- 発言内容は発言者行の直後の段落に続く

### ト書き・補足表記

```
〔「異議なし」と呼ぶ者あり〕
〔挙手する者あり〕
〔賛成者起立〕
```

`〔` 〜 `〕` で囲まれた表記は動作・反応の説明。

### 区切り線

```
―――――――――――――――――――――――――――――――――――――――
```

日程間の区切りとして使用されている。

### ページ番号

```
－ １－
－ ２－
```

全角数字で `－ N－` 形式。テキスト抽出後はノイズとして除去する。

### 休憩・再開・散会

```
午前10時17分 休憩
午前11時20分 休憩
午前11時27分 再開
午前11時49分 散会
```

## パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君|(.+?)さん）/;
// 例: ○議長（大西德三郎君） → role="議長", name="大西德三郎"
// 例: ○１０番（今枝和子さん） → role="１０番", name="今枝和子"

// 議席番号の正規化
const seatPattern = /^([０-９]+)番$/;

// ト書きの検出
const stageDirectionPattern = /^〔.+〕$/;

// 区切り線の検出
const separatorPattern = /^－{10,}$/;

// ページ番号の検出（除去対象）
const pageNumberPattern = /^－\s[０-９]+－$/;

// 開催日の抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;

// 時刻の検出
const timePattern = /^(?:午前|午後)\d+時\d+分\s(?:開会|休憩|再開|散会)/;
```

## 注意事項

- 会議録は PDF のみで提供されており、HTML 形式の全文は存在しない
- 委員会録は公開されていない（本会議のみ）
- 平成 16 年（2004 年）の創立時の臨時会から現在まで同一の発言者表記パターンが使われている
- 1 定例会・臨時会は複数 PDF（開催日ごと）に分かれており、全 PDF を取得する必要がある
- PDF ファイル名は年度・回・号を含む命名規則（例: `R6_dai4_1gou.pdf`, `H16-1_02-12.pdf`）だが、ファイル名のパターンは年度によって異なる場合があるため、詳細ページから動的に取得する

## 推奨アプローチ

1. **年度 ID の全量クロール**: 年度 ID 1〜23 の各カテゴリページを順番にクロールし、会議録詳細ページの URL リストを作成する
2. **詳細ページから PDF URL を収集**: 各詳細ページのリンクテキストを確認し、「N月N日（曜日）」形式のものだけを本文 PDF として収集する
3. **PDF を日別に取得**: 1 定例会あたり複数ページに分かれているため、全 PDF をダウンロードする
4. **pdftotext でテキスト抽出**: テキスト PDF のため OCR 不要。`pdftotext` コマンドで直接テキストを取得できる
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
