# 大潟村議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.vill.ogata.akita.jp/genre/parliament/minutes
- 分類: 自治体 CMS による PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: Google Tag Manager（GTM-WKT6CKM）使用。会議録は PDF 形式で提供。令和4年～令和7年の定例会・臨時会が掲載されている。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧 | `https://www.vill.ogata.akita.jp/genre/parliament/minutes` |
| 会議録詳細（個別会議） | `https://www.vill.ogata.akita.jp/archive/p{タイムスタンプ}` |
| 会議録詳細（旧形式） | `https://www.vill.ogata.akita.jp/archive/contents-{番号}` |
| PDF ファイル | `https://www.vill.ogata.akita.jp/uploads/public/archive_{コンテンツ番号}_00/{ファイル名}.pdf` |

### URL パターンの詳細

- 詳細ページの URL は `/archive/p{YYYYMMDDHHmmss}` 形式（例: `p20250310144400`）
- 令和4年の一部は `/archive/contents-{番号}` 形式（例: `contents-168`, `contents-169`）
- PDF の URL は `/uploads/public/archive_{7桁のゼロ埋めコンテンツ番号}_00/{ファイル名}.pdf` 形式

## サイト構造

### 一覧ページ

会議録一覧ページ（`/genre/parliament/minutes`）に全会議録が年度別にリスト表示される。

- ページネーションなし（単一ページに全件表示）
- 検索機能なし（サイト全体検索のみ）
- 年度ごとにセクション分け（令和7年、令和6年、令和5年、令和4年）
- 各会議が `<a>` タグで詳細ページにリンク

### 詳細ページ

各会議の詳細ページには PDF ファイルへのリンクが掲載される。

- **本会議セクション**: 目次 PDF + 会議録本文 PDF
- **委員会セクション**: 総務福祉教育委員会、生活産業委員会、決算特別委員会等の PDF（定例会のみ）
- 臨時会の場合は本会議の PDF のみ

## スクレイピング戦略

### Step 1: 会議録一覧の取得

一覧ページ `https://www.vill.ogata.akita.jp/genre/parliament/minutes` から全会議のリンクを取得する。

- 単一ページに全件表示されるため、ページネーション処理は不要
- 各会議のリンクテキストから年度・回次・月・種別（定例会/臨時会）を抽出
- リンク先 URL（詳細ページ）を収集

**リンクテキストのパターン:**

```
令和{年}年第{回}回（{月}月）{定例会|臨時会}　会議録
```

### Step 2: 詳細ページから PDF URL の収集

各詳細ページにアクセスし、PDF ファイルへのリンクを収集する。

- `<a>` タグの `href` 属性から `/uploads/public/archive_` で始まる PDF リンクを抽出
- 本会議・委員会の区分を `<h2>` や `<h3>` 等の見出し要素から判定
- ファイル名から会議種別（本会議/委員会名）を特定

### Step 3: PDF のダウンロードとテキスト抽出

PDF ファイルをダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）でテキスト抽出
- 目次 PDF はスキップ可能（本文のみ必要な場合）

### Step 4: テキストのパース

抽出したテキストから発言者と発言内容を構造化する。

#### メタ情報

PDF のファイル名およびリンクテキストから以下を抽出:

- 開催年: 令和X年 → 西暦変換
- 開催月: （X月）から抽出
- 会議種別: 定例会 / 臨時会
- 回次: 第X回

#### 発言の構造（PDF テキスト依存）

PDF からのテキスト抽出後、発言者パターンを解析する必要がある。具体的なパターンは PDF の内容に依存するため、実際の PDF テキストを確認して正規表現を調整する。

想定される発言者パターン:

```typescript
// 発言者の抽出（PDF テキスト内容に応じて調整）
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○議長（山田太郎） → role="議長", name="山田太郎"

// 開催情報の抽出
const sessionPattern = /令和(\d+)年第(\d+)回（(\d+)月）(定例会|臨時会)/;
```

## 注意事項

- 会議録は PDF 形式のため、HTML パースではなく PDF テキスト抽出が必要
- PDF のファイル名に日本語が含まれるため、URL エンコーディングに注意
- ファイル名の命名規則に年度による揺れがある（`目次_R0612定例.pdf` vs `目次_令和4年第4回（9月）定例会.pdf`）
- 委員会の PDF は定例会のみに掲載され、臨時会には含まれない
- 令和4年の一部ページは旧 URL 形式（`/archive/contents-{番号}`）を使用

---

## 推奨アプローチ

1. **一覧ページから一括取得**: 単一ページに全件表示されるため、一覧ページ1回のアクセスで全会議のリンクを取得可能
2. **詳細ページ経由で PDF URL を収集**: 各詳細ページから PDF リンクを抽出し、ダウンロード対象リストを作成
3. **PDF テキスト抽出**: pdf-parse 等のライブラリで PDF からテキストを抽出
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 一覧ページの会議リストを前回取得分と比較し、新規追加分のみを処理する
