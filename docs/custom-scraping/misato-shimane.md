## ページ構造

### 会議録トップページ

会議録トップ (`/gikai/1910/`) には年別のリンクが `<ul>` リストで並ぶ。各リンクは年別会議録一覧ページへ遷移する。

### 年別会議録一覧ページ

年別一覧ページ（例: `/gikai/1910/6161`）に、当該年の全会議録 PDF へのリンクが掲載される。

- `<h2>` タグで年（例: `令和７年`）が表示される
- `<h3>` タグで各会議録エントリが表示される
- h3 のテキストに会議種別・日目・日付・「議事録」の語が含まれる
- h3 内に PDF への `<a>` リンクが含まれる
- ページネーションはない（1 ページに当該年の全会議録が一覧表示される）

**h3 テキストの例:**

```
第４回定例会（４日目）R7.12.9議事録
第4回定例会（1日目）R6.11.29議事録
第２回臨時会 R6.4.23議事録
第１回定例会（５日目）R6.3.13議事録
```

## PDF ダウンロード URL の特徴

PDF の URL は `/files/original/{タイムスタンプ+ハッシュ値}.pdf` 形式。

- ファイル名はタイムスタンプ（アップロード日時）+ ハッシュ値で構成される
- 例: `/files/original/20250129141859688326c17a6.pdf`
- URL は推測不可能なため、**必ず HTML の `<a href>` から収集する**

## スクレイピング戦略

### Step 1: 年別会議録一覧ページの URL を収集

会議録トップ (`/gikai/1910/`) を取得し、年別一覧ページへのリンクを抽出する。

```typescript
// 年別ページリンクの抽出例
const yearLinks = $('a[href*="/gikai/1910/"]')
  .filter((_, el) => /\/gikai\/1910\/\d+$/.test($(el).attr('href')!))
  .map((_, el) => new URL($(el).attr('href')!, BASE_URL).toString())
  .toArray();
```

### Step 2: 各年別ページから PDF URL と会議メタ情報を収集

各年別一覧ページ（例: `/gikai/1910/6161`）を取得し、`/files/original/` を含む `<a href>` をすべて抽出する。

```typescript
// PDF リンクの抽出例
const pdfEntries = $('h3')
  .map((_, el) => {
    const text = $(el).text().trim();
    const link = $(el).find('a[href*="/files/original/"]').attr('href');
    return link ? { label: text, url: new URL(link, BASE_URL).toString() } : null;
  })
  .toArray()
  .filter(Boolean);
```

### Step 3: 会議メタ情報の抽出

h3 テキストから会議種別・回次・日目・開催日を抽出する。

**テキスト例:**

- `第４回定例会（４日目）R7.12.9議事録`
- `第２回臨時会 R6.4.23議事録`

```typescript
// 会議メタ情報の抽出
const sessionPattern = /第[０-９\d]+回(定例会|臨時会)(?:（([０-９\d]+)日目）)?/;
// 例: "第４回定例会（４日目）" → type="定例会", day="４"

// 日付の抽出（和暦略記）
const datePattern = /[RH]\d+\.\d+\.\d+/;
// 例: "R7.12.9" → 令和7年12月9日, "H26.3.5" → 平成26年3月5日
```

### Step 4: 各 PDF を取得・テキスト抽出

PDF ファイルを取得し、テキスト抽出ツール（pdf-parse 等）でテキスト化する。

## 注意事項

- **PDF URL の構造**: `/files/original/` 配下のファイル名はタイムスタンプ+ハッシュ値で構成されており、URL からメタ情報を推測できない。必ず HTML から収集する
- **2 階層構造**: トップ → 年別会議録一覧 の 2 段階のナビゲーションで全 PDF を取得できる
- **日付表記の揺れ**: 全角数字と半角数字が混在する（例: `第４回` と `第4回`）。正規表現では両方に対応する必要がある
- **和暦略記**: 日付は `R7.12.9`（令和）や `H26.3.5`（平成）の略記形式。R = 令和、H = 平成
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **全年度を網羅**: トップページから全年別ページの URL を収集し、各ページへアクセスする
2. **h3 テキストからメタ情報取得**: 会議種別・回次・日目・日付は h3 のテキストに含まれるため、正規表現で抽出する
3. **PDF リンクは HTML から直接収集**: URL を推測・構築せず、`<a href>` から取得する
4. **差分更新**: 既取得の PDF URL リストと比較し、新規 URL のみ取得する
5. **レート制限**: 各リクエスト間に 1〜2 秒の待機を設ける
