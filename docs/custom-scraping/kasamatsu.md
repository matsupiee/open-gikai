## PDF ファイル名パターン

ファイル名は年度・会議種別・号数・開催日の組み合わせで構成される。命名規則は年度によって異なる場合がある。

### 命名規則（令和5年以降の主パターン）

```
{元号略称}{年}-{回数}{種別}-{号数}-{元号略称}{年}{月日}.pdf
```

例:
- `R05-4teireikai-1gou-R051205.pdf` → 令和5年第4回定例会第1号（令和5年12月5日）
- `R05-3rinnjikai-1gou-R050725.pdf` → 令和5年第3回臨時会第1号（令和5年7月25日）
- `R06_4teireikai-1gou-R061206.pdf` → 令和6年第4回定例会第1号（令和6年12月6日）

### 令和7年の一部ファイル（タイムスタンプ形式）

一部のファイルはタイムスタンプ形式の名前が使用されている:

```
file_{タイムスタンプ}_{連番}.pdf
```

例: `file_2026216110227_1.pdf`

## スクレイピング戦略

### Step 1: 年度別ページの URL 収集

会議録トップページ `https://www.town.kasamatsu.gifu.jp/category/bunya/chouno_jouhou/gikai/gikaikaigiroku/` から各年度のページリンクを取得する。

- `<a>` タグの `href` が `/docs/{ページID}/` 形式のリンクを抽出
- テキストが「○○年議会会議録」のリンクが対象

### Step 2: PDF リンクの収集

各年度ページから会議録 PDF へのリンクを取得する。

- `<a>` タグの `href` が `file_contents/` で始まる、または `.pdf` で終わるリンクを抽出
- リンクテキストから会議名・号数・開催日を取得する
- ベース URL は `https://www.town.kasamatsu.gifu.jp/` に `href` を結合する

**リンクテキストのパターン:**

```
第4回定例会(第1号)　令和5年12月5日(PDF形式406KB)
第4回臨時会（第1号）　令和7年12月22日(PDF形式304KBytes)
```

- 会議名・号数: `第{N}回{定例会|臨時会}(第{M}号)` または `（第{M}号）`
- 開催日: `令和{N}年{M}月{D}日` または `平成{N}年{M}月{D}日`

### Step 3: PDF のダウンロードとテキスト抽出

会議録本文は PDF 形式のみで提供される。HTML 形式の本文は存在しない。

- PDF をダウンロードして pdftotext 等でテキスト抽出する
- 発言者情報・発言内容は PDF のテキストレイヤーから取得する

---

## 注意事項

- **PDF のみ提供**: HTML 形式の会議録本文は存在しない。全ての会議録が PDF で配布されている
- **ファイル名の命名規則が年度によって不統一**: 令和7年度の一部ファイルはタイムスタンプ形式（`file_{タイムスタンプ}_{連番}.pdf`）が使われており、ファイル名からのメタ情報抽出には依存しない設計が必要
- **ページネーションなし**: 各年度ページの会議録は 1 ページに全件表示される
- **年度ページ ID はランダム**: ページ ID（例: `2025030500045`）はランダムな数値のため、トップページからのリンク収集が必要

---

## 推奨アプローチ

1. **トップページから年度リスト取得**: 会議録トップページをクロールして年度別ページ URL を動的に収集する（ハードコードしても可だが、将来の年度追加に対応するため動的収集が望ましい）
2. **PDF リンクはリンクテキストからメタ情報を抽出**: ファイル名の命名規則が年度によって異なるため、ファイル名ではなくリンクテキストから会議名・開催日を取得する
3. **PDF テキスト抽出**: pdftotext 等を使用して PDF からテキストを抽出し、発言者・発言内容をパースする
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
