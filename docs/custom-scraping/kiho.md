# 紀宝町議会 カスタムスクレイピング方針

## 概要

- サイト: https://kiho-town.stream.jfit.co.jp/
- 分類: JFIT（株式会社ジェイフィット）提供のインターネット議会映像配信システム
- データ範囲: 令和3年（2021）〜現在
- 特記: **テキスト形式の会議録は提供されていない**。映像配信専用システムであり、質問項目の概要テキストのみ取得可能。

---

## ステータス

**テキスト会議録なし（映像配信のみ）**

全期間にわたってテキスト形式の会議録は提供されていない。
各発言ページには質問項目（数行程度の概要）は含まれるが、全文会議録の代替にはならない。
現時点ではテキストスクレイピングの対象外とする。

---

## システム構成

JFIT が提供する映像配信専用 SPA（Single Page Application）。
JavaScript が必須であり、JavaScript 無効時は主要機能が動作しない。

### フロントエンド

| ページ | URL パターン |
| --- | --- |
| トップ | `https://kiho-town.stream.jfit.co.jp/` |
| 会議名一覧 | `/?tpl=gikai_list` |
| 議員名一覧 | `/?tpl=speaker_list` |
| 会議日程一覧 | `/?tpl=gikai_days_list&gikai_id={gikai_id}` |
| 発言一覧（会議日ごと） | `/?tpl=gikai_result&gikai_day_id={id}&category_id={id}&inquiry_id={id}` |
| 議員別発言一覧 | `/?tpl=speaker_result&speaker_id={speaker_id}` |
| 映像再生 | `/?tpl=play_vod&inquiry_id={inquiry_id}` |

### gikai_day_id / category_id / inquiry_id

`/?tpl=gikai_days_list&gikai_id={gikai_id}` ページから取得。
各日程のリンクに `gikai_day_id`・`category_id`・`inquiry_id` の3つが埋め込まれている。

- `gikai_day_id`: 会議の日ごとの ID（例: 285, 286）
- `category_id`: 日程カテゴリ ID（例: 1, 2）
- `inquiry_id`: 発言・映像単位の ID（例: 723〜731）

## 取得可能なデータ

| データ種別 | 取得可否 | 備考 |
| --- | --- | --- |
| 会議名・日付 | ○ | `gikai_list` ページから取得可能 |
| 日程名 | ○ | `gikai_days_list` ページから取得可能 |
| 発言者氏名 | △ | 一般質問のみ。開会・閉会・採決等は議員名なし |
| 質問概要 | △ | `gikai_result` ページに数行程度の項目名のみ |
| 全文テキスト会議録 | ✕ | 全期間で未提供 |
| 動画 URL | ○ | `play_vod` ページにストリーミング URL あり（Base64 エンコードされた設定値） |
| 字幕・VTT | 不明 | ページ上では確認できず |

## 発言ページの構造（`gikai_result`）

`/?tpl=gikai_result&gikai_day_id={id}&category_id={id}&inquiry_id={id}` ページには以下が含まれる：

- 各発言者の顔写真・所属・氏名
- 質問項目（番号付きリストで数行程度）
- 「映像を再生します」ボタン（`/?tpl=play_vod&inquiry_id={id}` へのリンク）

**質問項目テキスト取得例（令和7年第4回定例会 12月10日）:**

```
奥峪康之議員:
  1. ツキノワグマ対策について
  2. 公用車へのドライブレコーダー設置について

平野美津子議員:
  1. 保育所の休所問題について
```

---

## 検索機能

サイト内にキーワード検索機能が提供されているが、検索対象は「会議名、議員名、質問項目」に限定されており、全文テキスト検索ではない。API エンドポイントは未確認（JavaScript により動的に処理される可能性あり）。

---

## 注意事項

- 「配信される映像・音声は公式記録ではない」旨が明記されている
- 無断転載・リンク禁止の記載あり
- JavaScript が必須のため、静的 HTML のみでは一部情報を取得できない可能性がある
- `play_vod` ページの動画設定は Base64 エンコードされており、直接的な動画 URL の取得には追加の解析が必要

---

## 推奨アプローチ

テキスト会議録が存在しないため、**現時点ではスクレイピング対象外**とする。

質問項目の概要テキストのみを収集する場合の実装方針（参考）:

1. `/?tpl=gikai_list` から `gikai_id` の全量リストを取得
2. 各 `gikai_id` に対して `/?tpl=gikai_days_list&gikai_id={id}` を取得し、`gikai_day_id`・`category_id`・`inquiry_id` を抽出
3. 各 `inquiry_id` に対して `/?tpl=gikai_result` ページをスクレイピングし、発言者名と質問項目テキストを取得
4. メタ情報（日付・会議名）は `gikai_days_list` ページから抽出

将来的に公式テキスト会議録が追加された場合は、あらためて調査が必要。
