## HTML 構造

会議録一覧ページは以下の構造を持つ:

- `<h2>` で年度別の見出し（例: `令和8年分会議録`、`令和7年分会議録`）
- 各見出しの下に `<p class="filelink">` で PDF リンクが並ぶ
- PDF リンクは `<a class="pdf" href="gikai-kaigiroku.files/{ファイル名}.pdf">` 形式

```html
<div class="h2bg"><div><h2>令和8年分会議録</h2></div></div>
<p class="filelink">
  <img src="/images/spacer.gif" alt="ファイルダウンロード　新規ウインドウで開きます。" width="1" height="1" class="img-guidance" />
  <a class="pdf" href="gikai-kaigiroku.files/0801-ri.pdf" target="_blank">
    令和8年第1回臨時会本会議（令和8年2月12日）（PDF：417KB）
  </a>
</p>
```

### リンクテキストのフォーマット

```
令和{N}年第{X}回{種別}（令和{N}年{M}月{D}日[から{M}月{D}日[まで]]）（PDF：{サイズ}）
```

例:
- `令和8年第1回臨時会本会議（令和8年2月12日）（PDF：417KB）`
- `令和7年第4回定例会本会議（令和7年12月9日から12月12日）（PDF：1,249KB）`
- `令和7年第1回定例会本会議（令和7年2月28日から3月10日まで）（PDF：1,624KB）`

## スクレイピング戦略

### Step 1: 会議録一覧ページの取得

単一ページ `gikai-kaigiroku.html` を取得する。

### Step 2: PDF リンクの収集

`a.pdf[href$=".pdf"]` セレクタで全 PDF リンクを収集する。

```typescript
const pdfLinks = $('a.pdf[href$=".pdf"]');
```

`href` は相対パス（`gikai-kaigiroku.files/0801-ri.pdf`）なので、ベース URL と結合して絶対 URL を生成する。

### Step 3: メタ情報の抽出

リンクテキストから以下の情報を抽出する:

```typescript
// 会議情報の抽出
const meetingPattern = /令和(\d+)年第(\d+)回(.+?)（令和\d+年(\d+)月(\d+)日(?:から(\d+)月?(\d+)日)?(?:まで)?）/;
// 例: 令和7年第4回定例会本会議（令和7年12月9日から12月12日）
// → year=7, number=4, type="定例会本会議", month=12, day=9
```

親の `<h2>` から年度情報も取得できる:

```typescript
const yearHeading = $('div.h2bg h2');
// 例: "令和8年分会議録" → 令和8年
```

### Step 4: PDF のダウンロードとテキスト抽出

1. PDF をダウンロード
2. pdf-parse 等でテキスト抽出
3. 抽出したテキストから発言者・発言内容をパースする

---

## 注意事項

- **単一ページ構成**: 全会議録が1ページにまとまっているため、スクレイピングは非常にシンプル。1リクエストで全 PDF の URL を取得できる
- **相対パス**: PDF の `href` は相対パス（`gikai-kaigiroku.files/xxxx.pdf`）で記述されている。URL 結合時に注意が必要
- **ファイル名の規則性**: ファイル名に一定の規則性があるが、URL 推測ではなく必ず HTML からリンクを収集すること
- **掲載タイミング**: 閉会から約1か月後に掲載されるため、最新の会議録が未掲載の場合がある

---

## 推奨アプローチ

1. **一覧ページを起点にする**: `gikai-kaigiroku.html` の1リクエストで全 PDF の URL を収集できる
2. **PDF テキスト抽出**: 全会議録が PDF 公開のため、PDF パーサーの実装のみで対応可能（HTML パーサーは不要）
3. **レート制限**: 自治体サイトのため、PDF ダウンロード間に1〜2秒の待機時間を設ける
4. **差分更新**: 既取得の PDF URL リストと比較し、新規 URL のみをダウンロードする
