## 詳細ページ構造

詳細ページ（`/q/aview/282/{記事ID}.html`）には以下が含まれる:

- タイトル（会議名と会期）
- PDF へのダウンロードリンク（`/dl?q={ファイルID}.pdf` 形式、ファイルサイズ表示あり）
- 関連記事へのリンク（過去の会議録 10 件程度）

### PDF リンクのパターン

```
/dl?q={数値ID}_filelib_{ハッシュ}.pdf
```

例:
```
/dl?q=39813_filelib_918ab7be7640c1103a74bf1ca673331a.pdf
```

リンクテキストには会議名とファイルサイズが記載される（例: `令和7年4月(12月)会議録(PDF 約785KB)`）。

### 1 詳細ページ内の PDF 数

通常 1 件だが、2 件含まれるページも確認されている。全リンクを抽出する必要がある。

## スクレイピング戦略

### Step 1: 会議録一覧の取得

`https://www.town.kumamoto-kashima.lg.jp/q/list/282.html` を取得し、`/q/aview/282/` へのリンクを全件抽出する。

- ページネーションなし（全件が 1 ページに表示）
- 新規公開分の検出は、取得済みの記事 ID リストとの差分で行う

### Step 2: 詳細ページの取得

各詳細ページにアクセスし、PDF ダウンロードリンクを抽出する。

- `/dl?q=` で始まるリンクを全件抽出
- リンクテキストからファイルサイズや会議名を補助的に取得

### Step 3: PDF のダウンロードと処理

抽出した PDF URL から PDF をダウンロードし、テキストを抽出する。

- ファイルサイズは数百 KB〜1 MB 程度
- 会議録テキストは PDF 内に格納されており、HTML ページには本文テキストは含まれない

---

## 注意事項

- 専用の検索システムがないため、全件把握は一覧ページのクロールのみで完結する
- 一覧ページに掲載される件数が少ない（16 件、2023 年以降）ため、それ以前の会議録はサイト上に存在しない可能性がある
- 詳細ページに複数 PDF が含まれる場合があるため、`/dl?q=` リンクを全件取得すること
- 問い合わせ先: 議会事務局（096-237-5525）

---

## 推奨アプローチ

1. **一覧ページを定期クロール**: `/q/list/282.html` を定期的に取得し、新規追加の記事 ID を検出
2. **差分更新**: 取得済みの記事 ID（`/q/aview/282/{ID}.html`）を管理し、未取得分のみ詳細ページをクロール
3. **PDF リンクの全件取得**: 詳細ページ内の `/dl?q=` リンクを Cheerio 等で全件抽出
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
