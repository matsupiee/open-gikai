# 周防大島町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.suo-oshima.lg.jp/site/gikai/
- 分類: 町公式サイト内での直接公開（会議録検索システム未導入）
- 文字コード: UTF-8
- 特記: 会議録は PDF 形式で提供。公式サイト上で HTML の個別ページとして会議録一覧を公開し、各ページから PDF ファイルへのリンクを提供している。Google カスタム検索で検索可能。

## ページ構造

### 議事録一覧ページ（list18-56.html）

- 約120件の会議録リンクが時系列（新→旧）で単一ページに全件表示される
- ページネーションなし
- 各リンクのテキストは `{元号}{年} 本会議 {定例会/臨時会名}会議録` 形式
- 掲載期間: 平成16年（2004年）〜現在
- `<ul>/<li>` のリスト構造

### 会議録詳細ページ

- 1会議 = 1ページ
- ページタイトルに会議名を含む（例: `令和7年 本会議　第4回定例会会議録`）
- ページ上部にページID・更新日・所属（議会事務局 議事課）を表示
- `<table>` 形式で PDF ファイルへのリンクを列挙
- 各行に日付とファイルサイズ情報を含む

### PDF ファイル構成（1会議あたり）

| 種別 | 説明 |
| --- | --- |
| 目次 | 会議録の目次（数十KB〜） |
| 第1日〜第N日 | 各日程の会議録本文（数百KB〜1MB超） |

PDF ファイルの URL パターン: `/uploaded/attachment/{数値ID}.pdf`

- ファイル ID は連番（例: 22500, 22501, 22502, 22503）
- 1会議の PDF は連続した ID が割り当てられる傾向がある

## スクレイピング戦略

### Step 1: 会議録詳細ページ URL の収集

1. 議事録一覧ページ `list18-56.html` を取得
2. ページ内の全リンクから `/site/gikai/{ページID}.html` パターンに一致する URL を抽出
3. 一覧ページにページネーションはないため、1回の取得で全件収集可能

```typescript
// 詳細ページ URL の抽出パターン
const detailLinks = $('a[href*="/site/gikai/"]')
  .filter((_, el) => /\/site\/gikai\/\d+\.html$/.test($(el).attr('href') || ''));
```

### Step 2: 各詳細ページからメタ情報と PDF リンクの取得

各詳細ページにアクセスし、以下を抽出:

1. **会議名**: ページタイトルまたは見出しから取得
2. **PDF リンク**: `<a href="/uploaded/attachment/{ID}.pdf">` を抽出
3. **更新日**: ページ上部の更新日表示から取得

```typescript
// PDF リンクの抽出
const pdfLinks = $('a[href$=".pdf"]');

// メタ情報の抽出（タイトルから）
const titlePattern = /(令和|平成)(\d+)年\s*本会議\s*(第\d+回(?:定例会|臨時会))/;
```

### Step 3: PDF のダウンロードとテキスト抽出

- 各 PDF をダウンロードし、テキスト抽出ライブラリ（pdf-parse 等）で本文を取得
- 目次 PDF はスキップし、第1日〜第N日の本文 PDF のみを処理対象とする

## メタ情報の抽出

### ページタイトルからの抽出

```
令和7年 本会議　第4回定例会会議録
令和7年 本会議第1回臨時会
平成16年 本会議 第1回臨時会
```

- 元号・年: `(令和|平成)(\d+)年`
- 会議種別: すべて `本会議`
- 回数・種類: `第(\d+)回(定例会|臨時会)`

```typescript
const metaPattern = /(令和|平成)(\d+)年\s*本会議\s*第(\d+)回(定例会|臨時会)/;
// 例: "令和7年 本会議　第4回定例会会議録"
// → era="令和", year="7", number="4", type="定例会"
```

## 注意事項

- 会議録は PDF 形式のみで提供されており、HTML での本文表示はない。テキスト抽出には PDF パーサーが必要
- PDF ファイル名は数値 ID のみで、ファイル名から会議種別や日付を推定することはできない。必ず詳細ページのタイトルからメタ情報を取得すること
- 一覧ページに全件が表示されるため、ページネーション対応は不要
- Google カスタム検索はサイト内検索用であり、スクレイピングでは使用しない
- リクエスト間には適切な待機時間（1〜2秒）を設けること
- 問い合わせ先: 議会事務局 議事課（Tel: 0820-74-1003、Fax: 0820-74-1030）

## 推奨アプローチ

1. **一覧ページの全件取得**: `list18-56.html` から全詳細ページ URL を一括収集（ページネーション不要）
2. **詳細ページからメタ情報と PDF URL を取得**: 各詳細ページのタイトルから会議名・年度・種別を抽出し、PDF リンクを収集
3. **PDF のダウンロードとテキスト抽出**: 目次以外の PDF をダウンロードし、pdf-parse 等でテキストを抽出
4. **差分更新**: ページ ID は数値のため、前回取得済みの最大ページ ID 以降のみを取得する差分更新が可能
