## スクレイピング戦略

### Step 1: 会議録一覧ページから PDF リンクを収集

`https://www.town.taiji.wakayama.jp/gikai/kaigiroku.html` を取得し、`files/` 配下の PDF リンクを全て抽出する。

- リンク形式: `<a href="files/{ファイル名}.pdf">` または `<a href="../tyousei/{...}">`
- `.pdf` リンクを全件取得し、相対 URL をフル URL に変換

### Step 2: PDF のダウンロード

収集した URL（`https://www.town.taiji.wakayama.jp/gikai/files/{ファイル名}.pdf`）からダウンロードする。

---

## 注意事項

- PDF ファイル名に体系的な命名規則がなく、年度・回次によって命名が大きく異なる
- 目次ファイル（`mokuzi`、`mokugi` を含む）は議事録本文を含まないため、取得対象から除外するか区別して管理する
- 一部ファイルが `.docx` 形式（例: `50509121.docx`）で提供されている場合がある
- 会議録一覧ページは静的 HTML のため、JavaScript 不要で取得可能
- ページネーションなし（全年度分が 1 ページに掲載）
- レート制限: リクエスト間に 1〜2 秒の待機時間を設ける

---

## 推奨アプローチ

1. `kaigiroku.html` の PDF リンク（`a[href$=".pdf"]`、`a[href$=".docx"]`）を Cheerio で全件取得
2. 相対 URL（`files/...`）を絶対 URL（`https://www.town.taiji.wakayama.jp/gikai/files/...`）に変換
3. ファイル名に `mokuzi`・`mokugi` を含むものを目次として区別
4. 全本文 PDF（および `.docx`）をダウンロードして処理
