## スクレイピング戦略

### Step 1: PDF リンクの収集

一覧ページ `index.html` から全 PDF リンクを抽出する。

- ページネーションなし（単一ページに全件掲載）
- `<a>` タグの `href` 属性で `.pdf` に一致するリンクを収集
- アンカーテキストから年度・回次・会議種別（定例会/臨時会）を抽出

**収集方法:**

1. `index.html` を取得
2. Cheerio 等で `a[href$=".pdf"]` を抽出
3. アンカーテキストをパースしてメタ情報を取得

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、`pdftotext` 等でテキストを抽出する。

- PDF はテキスト埋め込み型（OCR 不要）
- 1つの PDF に複数日の会議が含まれる場合がある（例: 定例会は初日と最終日）

### Step 3: 会議録のパース

#### PDF 内部の構造

1. **表紙**: 会議名、開催期間
2. **目次**: 各号の日付ごとの議事日程一覧
3. **招集告示**: 日時・場所・議題一覧
4. **各日の会議録本文**: 日付ごとにセクション分け

#### ページ番号

ページ番号は `－{数字}－` の形式で埋め込まれている（例: `－1－`, `－10－`）。テキスト抽出時に除去が必要。

#### セクション区切り

- 罫線: `─────────────────────────────────────────────────` で区切り
- セクション見出し: `◎` で始まる（例: `◎開会及び開議の宣告`, `◎一般質問`）

#### メタ情報

アンカーテキストから抽出:

```
令和{N}年第{M}回草津町議会{定例会|臨時会}会議録
```

PDF 内の本文冒頭からも抽出可能:

```
令和６年３月４日（月曜日）
```

- 開催日: `令和X年X月X日（X曜日）` 形式
- 開会時刻: `開会 午前１０時００分` 形式

#### 出席者情報

各日の冒頭に以下の情報が記載:

- 議事日程
- 会議に付した事件
- 出席議員（番号・氏名の一覧）
- 欠席議員
- 説明のため出席した者（役職・氏名の一覧）
- 事務局職員出席者

#### 発言の構造

発言者パターン:

```
○議長（宮﨑謹一君）
○町長（黒岩信忠君）
○総務課長（石坂恒久君）
○７番（金丸勝利君）
○２番（安齋 努君）
○総務観光常任委員長（黒岩 卓君）
○温泉温水対策特別委員長（...君）
○企画創造課長（田中 浩君）
○福祉課長（中澤一夫君）
○こどもみらい課長（高井洋一君）
```

- `○` で始まり、役職（または議員番号）+ 括弧内に氏名 + `君`
- 議員番号は全角数字（例: `７番`, `１０番`）
- 氏名中のスペースは全角（例: `安齋 努`）
- 発言内容は発言者行の後に続くテキスト

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○議長（宮﨑謹一君） → role="議長", name="宮﨑謹一"

// セクション見出しの抽出
const sectionPattern = /^◎(.+)/;
// 例: ◎一般質問 → section="一般質問"

// 開催日の抽出（PDF 本文内）
const datePattern = /^令和(\d+)年(\d+)月(\d+)日（(.+?)曜日）/;

// ページ番号の除去
const pageNumberPattern = /^－\d+－$/;

// アンカーテキストからのメタ情報抽出
const anchorPattern = /令和(\d+)年第(\d+)回草津町議会(定例会|臨時会)会議録/;
```

## 注意事項

- PDF ファイル名に統一的な命名規則がないため、一覧ページのリンクから動的に収集する必要がある
- 1つの PDF に複数日の会議録が含まれるため、日付ごとに分割して処理する必要がある
- pdftotext によるテキスト抽出時、表形式の部分（出席者一覧、議事日程等）はレイアウトが崩れる可能性がある
- 全角数字・全角スペースが多用されているため、正規表現のマッチングに注意
- 小規模議会（定数11名程度）のため、データ量は比較的少ない

---

## 推奨アプローチ

1. **一覧ページから全 PDF リンクを収集**: 単一ページなのでクロール負荷は最小限
2. **PDF ダウンロード → テキスト抽出**: `pdftotext` でプレーンテキスト化し、OCR は不要
3. **日付単位で分割**: PDF 内の `令和X年X月X日（X曜日）` を境界として複数日の会議を分割
4. **発言者パターンで発言を切り出し**: `○{役職}（{氏名}君）` パターンで発言区間を特定
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
6. **差分更新**: 一覧ページの PDF リンク数を前回と比較し、新規追加分のみダウンロード
