## 年度ページの構造

### 見出しと会議種別

各年度ページは `<h2>` タグで月別・会議種別ごとにセクション分けされている。

例（令和7年）:
- `令和７年１２月臨時会`
- `令和７年１２月定例会`
- `令和７年９月臨時会`
- `令和７年９月定例会`
- `令和７年７月臨時会`
- `令和７年６月定例会`
- `令和７年３月定例会`

### 会議種別

| 種別 | 説明 |
| --- | --- |
| 定例会 | 3月、6月、9月、12月に開催される定例議会 |
| 臨時会 | 必要に応じて開催される臨時議会 |
| 予算特別委員会 | 3月定例会に付随して開催 |
| 決算特別委員会 | 9月定例会に付随して開催 |

### PDF リンクの構造

各 `<h2>` セクション配下に、日程ごとの PDF リンクが列挙される。

- リンクテキスト例: `第１日`、`第２日`、`予算特別委員会（第１日）`
- ページネーションなし（1ページに全会議を表示）

## PDF ファイルの URL パターン

### 命名規則

```
/uploads/user/gikaijimu/File/{年}議会/{会議名称}.pdf
```

デコード後の例:
```
/uploads/user/gikaijimu/File/2025議会/令和７年１２月臨時会会議録（第１日）.pdf
/uploads/user/gikaijimu/File/2025議会/令和７年１２月定例会会議録（第１日）.pdf
/uploads/user/gikaijimu/File/2024/令和６年１２月臨時会会議録（第１日）.pdf
```

- ファイル名は全角文字で構成（令和、月、会議種別、日数）
- URL エンコードされた状態でリンクされている
- 年ディレクトリは `{西暦}議会` または `{西暦}` の形式（年度により異なる）

## PDF 会議録の構造

### ファイル構成

1. **タイトル**: `令和X年XX月羽後町議会{臨時会|定例会}会議録　（第X日）`
2. **開催日・場所**: `１、令和X年XX月XX日羽後町役場議場で会議を開いた。`
3. **出席議員一覧**: `１、本日の会議に応招し出席した議員は次のとおりである。` + 番号・氏名のリスト（2列構成）
4. **欠席議員**: `１、本日の会議に欠席した議員は次のとおりである。`（`なし` の場合あり）
5. **説明員一覧**: `１、議事説明のため会議に出席した者の職氏名は次のとおりである。` + 役職・氏名のリスト
6. **事務局職員**: `１、職務のため議場に出席した事務局職員の職氏名は次のとおりである。`
7. **議事日程**: `１、本日の議事日程は次のとおりである。` + 日程一覧
8. **本文**: 発言の記録

### 発言者パターン

```
●議長（中村房司）
●町長（佐々木康寛）
●企画財政課長（佐々木正志）
```

- `●` で始まり、役職 + 括弧内に氏名（敬称なし）
- 発言内容は発言者行の後に続くテキスト

### 登壇表記

```
【町長（佐々木康寛）登壇】
【企画財政課長（佐々木正志）登壇】
```

- `【` で始まり `登壇】` で終わる
- 役職 + 括弧内に氏名 + `登壇`

### その他の記録

```
【「異議なし」と呼ぶ者あり】
午前１０時００分　開　議
```

### 議員番号

議員は番号で管理（1番〜12番）。出席議員一覧に番号と氏名が記載される。

## スクレイピング戦略

### Step 1: 年度ページの URL 一覧を構築

過去分目次ページ `detail.html?id=2314&category_id=51` と最新ページ `detail.html?id=1247&category_id=51` から、全年度ページへのリンクを取得する。

- 年度ページ ID は固定値のため、ハードコードまたは目次ページから動的に取得
- 公開範囲: 平成22年度（2010年）〜令和7年（2025年）

### Step 2: 各年度ページから PDF リンクを収集

各年度ページの HTML をパースし、PDF リンクを収集する。

**収集方法:**

1. 年度ページの `<a>` タグから `href` を抽出
2. `.pdf` で終わるリンクをフィルタリング
3. 直前の `<h2>` 見出しから会議種別（定例会 / 臨時会）と開催月を抽出
4. リンクテキストから日程（第X日）を抽出

### Step 3: PDF のダウンロードとテキスト抽出

1. PDF をダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. テキストからメタ情報と発言を分離

### Step 4: 会議録のパース

#### メタ情報の抽出

```
令和７年１２月羽後町議会臨時会会議録　（第１日）
１、令和７年１２月２５日羽後町役場議場で会議を開いた。
```

- 会議名: タイトル行から抽出
- 開催日: `令和X年XX月XX日` 形式（全角数字）
- 開催場所: `羽後町役場議場`

#### 発言の分割

```typescript
// 発言者の抽出
const speakerPattern = /^●(.+?)（(.+?)）/;
// 例: ●議長（中村房司） → role="議長", name="中村房司"
// 例: ●町長（佐々木康寛） → role="町長", name="佐々木康寛"

// 登壇の検出
const podiumPattern = /^【(.+?)登壇】/;

// 開催日の抽出
const datePattern = /(?:令和|平成)([\d０-９]+)年([\d０-９]+)月([\d０-９]+)日/;

// 休憩・再開の検出
const breakPattern = /(午前|午後)([\d０-９]+)時([\d０-９]+)分\s*(開\s*議|休\s*憩|再\s*開|散\s*会)/;
```

## 注意事項

- 全会議録が PDF 形式のみで公開されており、HTML 版は存在しない
- PDF のテキスト抽出精度に依存するため、レイアウトによっては文字化けや順序の乱れが発生する可能性がある
- PDF ファイル名は全角文字を含むため、URL エンコーディングの取り扱いに注意が必要
- 年ディレクトリ名が年度により異なる（`2025議会` vs `2024` など）ため、PDF の URL はページから動的に取得する必要がある
- 発言者マーカーは `●`（黒丸）であり、他の自治体でよく使われる `○`（白丸）とは異なる
- 議員氏名に敬称（「君」等）は付かない
- 議員番号・数字は全角で表記されている
- 検索機能がないため、全ページを順次クロールする必要がある
- 令和5年のみ `category_id=46` で他と異なる

---

## 推奨アプローチ

1. **目次ページから全年度 URL を取得**: 過去分目次ページと最新ページから年度別ページのリンクを動的に収集する
2. **PDF リンクの動的収集**: 各年度ページから PDF の `<a>` タグを抽出し、URL を収集する（ファイル名のパターンに依存しない）
3. **PDF テキスト抽出**: pdf-parse 等のライブラリで PDF からテキストを抽出する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度ページの PDF リンク数を前回取得時と比較し、新規追加分のみをダウンロードする
