## PDF ファイルの命名規則

### 臨時会（トップページから直リンク）

`r{和暦2桁}-{月2桁}r-{回数2桁}-{月日4桁}.pdf`

例: `r07-02r-01-0203.pdf`（令和7年2月臨時会 第1回 2月3日）

### 定例会（個別ページ内のリンク）

複数パターンが混在:

- 議事日程: `teireikainittei_{年月4桁}-{号}.pdf`（例: `teireikainittei_2506-1.pdf`）
- 一般質問要旨: `youshi_{年月4桁}.pdf`（例: `youshi_2506.pdf`）
- 会議録本体: `{連番2桁}-r{和暦2桁}-{月2桁}t-{号2桁}-{月日4桁}.pdf`（例: `07-r07-06t-01-0610.pdf`）

## スクレイピング戦略

### Step 1: 一覧ページから年度別ページ URL を収集

トップページ `260.html` をパースし、各年度の定例会ページ URL と臨時会 PDF の直リンクを収集する。

- 臨時会: `<a>` タグから PDF URL を直接取得
- 定例会: 個別ページ `/R{年}/{ページID}.html` への URL を取得

### Step 2: 定例会ページから PDF URL を収集

各定例会ページをパースし、`material/files/group/38/` 配下の PDF リンクを全て取得する。

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- PDF は `material/files/group/38/` ディレクトリに集約されている
- 定例会は年4回（3月・6月・9月・12月）
- 代表質問数・一般質問数が定例会ページに表示される
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
