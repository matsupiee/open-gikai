## PDF ファイル命名規則

各一覧ページ内の `file_contents/` 以下に PDF が格納されている。

### 令和期（R）

| ファイル名パターン | 会議種別 | 例 |
| --- | --- | --- |
| `R{年}-T{月}.pdf` | 定例会 | `R07-T09.pdf`（令和7年9月定例会） |
| `R{年}-YK{月}.pdf` | 予算決算常任委員会 | `R07-YK09.pdf`（令和7年9月予算決算常任委員会） |
| `R{年}-R{回}.pdf` | 臨時会 | `R07-R02.pdf`（令和7年第2回臨時会） |

### 平成期（H）

| ファイル名パターン | 会議種別 | 例 |
| --- | --- | --- |
| `H{年}-T{月}.pdf` | 定例会 | `H31-T03.pdf`（平成31年3月定例会） |
| `H{年}-R{回}.pdf` | 臨時会 | `H30-R03.pdf`（平成30年第3回臨時会） |

**月の値**: 定例会は `03`、`06`、`09`、`12`（3月・6月・9月・12月）

## 会議録リスト構造

各一覧ページは年度別のセクションで構成されており、ページネーションはない。

- **令和元年～現在**: 令和7年～令和元年（新→旧順）
- **平成21年～31年**: 平成31年～平成21年（新→旧順）
- **平成11年～20年**: 平成20年～平成11年（新→旧順）

各年度内に以下が含まれる：
- 定例会（3月・6月・9月・12月）
- 予算決算常任委員会（令和期のみ確認）
- 臨時会（開催回数は年度により異なる）

アンカーテキストの形式：
```
{元号}年{月}月南伊豆町議会定例会.pdf[PDF：{サイズ}MB]
{元号}年第{N}回南伊豆町議会臨時会.pdf[PDF：{サイズ}MB]
```

## スクレイピング戦略

### Step 1: 各期間の一覧ページを取得

以下の3つの固定 URL から HTML を取得する。

1. `https://www.town.minamiizu.shizuoka.jp/docs/2022012000012/`（令和元年～）
2. `https://www.town.minamiizu.shizuoka.jp/docs/2021122800062/`（平成21年～31年）
3. `https://www.town.minamiizu.shizuoka.jp/docs/2021122800055/`（平成11年～20年）

### Step 2: PDF リンクの抽出

各一覧ページから `file_contents/*.pdf` を含むアンカー要素を Cheerio で抽出する。

```typescript
// 一覧ページから PDF リンクを抽出
const links = $('a[href*="file_contents/"]')
  .toArray()
  .map((el) => ({
    href: $(el).attr("href"),
    text: $(el).text().trim(),
  }));
```

PDF の絶対 URL は `https://www.town.minamiizu.shizuoka.jp/docs/{ページID}/{href}` で構成される。

### Step 3: ファイル名からメタ情報を抽出

PDF ファイル名の命名規則に基づき、メタ情報を解析する。

```typescript
// ファイル名パターン: [R/H]{年}-[T/YK/R]{月or回}.pdf
const filenamePattern = /^([RH])(\d+)-([A-Z]+)(\d+)\.pdf$/;

const eraMap = { R: "令和", H: "平成" };
const meetingTypeMap = {
  T: "定例会",
  YK: "予算決算常任委員会",
  R: "臨時会",
};
```

アンカーテキストからも以下の正規表現で日付・会議種別を取得できる：

```typescript
// 定例会: "令和6年12月南伊豆町議会定例会"
const teirei = /(?:令和|平成)(\d+)年(\d+)月南伊豆町議会定例会/;

// 臨時会: "令和7年第2回南伊豆町議会臨時会"
const rinji = /(?:令和|平成)(\d+)年第(\d+)回南伊豆町議会臨時会/;
```

### Step 4: PDF の取得とテキスト抽出

PDF を直接ダウンロードし、`pdf-parse` 等のライブラリでテキストを抽出する。

## 注意事項

- ページネーションなし。各一覧ページは単一ページに全データを掲載
- 一覧ページの URL（`/docs/2022012000012/` 等）は固定のため、変更された場合は手動で更新が必要
- PDF は会議録全体が1ファイルにまとまっている形式（発言者ごとの分割なし）
- 令和期のみ予算決算常任委員会（`YK`）が確認されているが、平成期では確認されていない（`R` のみ）
- レート制限: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける

---

## 推奨アプローチ

1. **3ページ固定アクセス**: 一覧 URL が固定のため、3ページを順番に取得するシンプルな実装で全量収集が可能
2. **ファイル名ベースの管理**: PDF ファイル名（例: `R06-T12.pdf`）が一意のキーとして機能するため、差分更新の識別子として利用できる
3. **アンカーテキストの優先利用**: メタ情報の抽出はファイル名よりもアンカーテキストの方が日本語で明確なため、両方を参照して補完する
