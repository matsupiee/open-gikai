# 松茂町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.matsushige.tokushima.jp/docs/2015112600039/
- 分類: 公式ウェブサイト上で年度別・定例会別に PDF を直接公開（専用の会議録検索システムなし）
- 文字コード: UTF-8
- 対象期間: 平成25年（2013年）〜現在

## URL 構造

| ページ | URL |
| --- | --- |
| 会議録一覧（単一ページ） | `https://www.town.matsushige.tokushima.jp/docs/2015112600039/` |
| PDF ファイル | `https://www.town.matsushige.tokushima.jp/file_contents/{ファイル名}.pdf` |

## HTML 構造

```
<h2>令和７年</h2>
  <h3>第４回定例会</h3>
    <a href="file_contents/202512.pdf">令和７年第４回定例会会議録目次[PDF：○○KB]</a>
    <a href="file_contents/20251204.pdf">令和７年第４回定例会　１２月４日[PDF：○○KB]</a>
    <a href="file_contents/20251208.pdf">令和７年第４回定例会　１２月８日[PDF：○○KB]</a>
    <a href="file_contents/20251219.pdf">令和７年第４回定例会　１２月１９日[PDF：○○KB]</a>
  <h3>第２回臨時会</h3>
    ...
```

- `<h2>` タグ: 年度（例: `令和７年`、`令和６年`、`平成２５年`）
- `<h3>` タグ: 会議種別（例: `第４回定例会`、`第２回臨時会`）
- `<a>` タグ: PDF へのリンク（相対パス）
- ファイルサイズ情報はリンクテキスト内に `[PDF：○○KB]` 形式で記載

## PDF ファイル名のパターン

年度によってファイル名の命名規則が異なる（統一されていない）:

| 年度 | ファイル名パターン | 例 |
| --- | --- | --- |
| 令和元年以降 | `YYYYMMDD.pdf`（日付形式） | `20251204.pdf` |
| 令和元年以降（目次） | `YYYYMM.pdf`（年月形式） | `202512.pdf` |
| 平成26〜28年頃 | 連番 ID | `1409.pdf`、`1410.pdf` |
| 平成27年一部 | 独自略称 | `274mokuzi.pdf`、`274124.pdf` |

ファイル名からのメタ情報の復元は困難なため、一覧ページの HTML から日付・会議名を取得する。

## スクレイピング戦略

### Step 1: PDF リンク一覧の取得

`https://www.town.matsushige.tokushima.jp/docs/2015112600039/` を取得し、全 `<a>` タグの href から PDF リンクを収集する。

同時に HTML 構造から以下のメタ情報を抽出する:

- **年度**: 直近の `<h2>` テキスト（例: `令和７年`）
- **会議種別**: 直近の `<h3>` テキスト（例: `第４回定例会`）
- **開催日・種別**: `<a>` タグのリンクテキスト（例: `令和７年第４回定例会　１２月４日[PDF：○○KB]`）
- **目次 PDF の判別**: リンクテキストに `会議録目次` を含む場合は目次

### Step 2: PDF の取得

各 PDF を `https://www.town.matsushige.tokushima.jp/file_contents/{ファイル名}.pdf` でダウンロードする。

- PDF の内容はテキスト抽出ツール（`pdf-parse` 等）で処理する
- 目次 PDF（`会議録目次`）は会議録本文ではないため、取得対象から除外してよい

### Step 3: メタ情報のパース

リンクテキストから開催日と会議名を抽出する:

```typescript
// リンクテキスト例: "令和７年第４回定例会　１２月４日[PDF：○○KB]"
// ファイルサイズ部分を除去
const text = linkText.replace(/\[PDF：.+?\]/, "").trim();

// 開催日の抽出（全角数字）
const datePattern = /(\d+)月(\d+)日/;

// 会議名の抽出
const sessionPattern = /第(\d+)回(定例会|臨時会)/;

// 年度の抽出（h2 から）
const yearPattern = /(?:令和|平成)(\d+)年/;
```

## 注意事項

- **PDF のみの公開**: HTML 形式の会議録は存在せず、すべての会議録が PDF で提供される
- **ファイル名の非統一**: 古い年度（平成25〜27年頃）のファイル名は連番 ID や独自略称であり、ファイル名から日付・会議情報を復元できない。一覧ページの HTML を正として扱う
- **一覧ページの単一性**: 全データが 1 URL にまとまっているため、差分更新には `<h2>` の年度や PDF の href を前回取得分と比較して新規分を特定する
- **相対パス**: `<a>` タグの href は `file_contents/xxx.pdf` 形式の相対パスのため、ベース URL `https://www.town.matsushige.tokushima.jp/` を補完して完全 URL にする

---

## 推奨アプローチ

1. **一覧ページを起点にする**: 全 PDF リンクと関連メタ情報を 1 リクエストで取得できる
2. **目次 PDF を除外**: `会議録目次` を含むリンクは索引のため、本文取得対象から外す
3. **メタ情報は HTML から復元**: PDF ファイル名に依存せず、一覧ページの `<h2>`・`<h3>`・リンクテキストを組み合わせてメタ情報を構築する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 一覧ページを定期取得し、前回取得済みの href と比較して未取得の PDF のみをダウンロードする
