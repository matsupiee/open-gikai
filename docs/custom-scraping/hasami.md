## スクレイピング戦略

### Step 1: 一覧ページから PDF リンクの収集

会議録一覧ページ `4358.html` を取得し、すべての PDF リンクを抽出する。

- `<a>` タグの `href` 属性から `/kiji003879/` を含むリンクを収集
- リンクテキストから会議名（定例会/臨時会、回数、月）を抽出
- 年度の見出し要素から年度情報を取得し、各 PDF に紐付ける

**収集方法:**

1. HTML を Cheerio でパースし、年度見出しと PDF リンクの対応関係を構築
2. リンクテキストから正規表現で会議種別・回数・月を抽出
3. PDF URL とメタ情報（年度、会議種別、回数）をセットで保存

### Step 2: PDF のダウンロード

収集した URL から PDF ファイルをダウンロードする。

- ベース URL: `https://www.town.hasami.lg.jp`
- リンクが相対パスの場合はベース URL と結合

### Step 3: PDF からテキストの抽出

ダウンロードした PDF から本文テキストを抽出する。

- `pdf-parse` 等のライブラリを使用して PDF をテキスト化
- PDF のレイアウト（段組み等）に注意してテキスト順序を維持

### Step 4: 会議録のパース

#### メタ情報

リンクテキストおよび年度見出しから以下を抽出:

- 年度: 見出しの年度情報（例: 令和5年）
- 会議種別: 定例会 or 臨時会
- 回数: 第N回
- 開催月: リンクテキストに含まれる場合のみ

#### パース用正規表現（案）

```typescript
// リンクテキストから会議情報を抽出
const meetingPattern = /第(\d+)回(定例会|臨時会)(?:\s*(\d+)月)?/;
// 例: "第1回臨時会 2月" → 回数="1", 種別="臨時会", 月="2"
// 例: "第3回定例会" → 回数="3", 種別="定例会", 月=undefined

// PDF リンクの抽出
const pdfLinkPattern = /href="([^"]*kiji003879[^"]*\.pdf)"/g;

// 年度見出しの抽出
const fiscalYearPattern = /(令和|平成)(\d+)年/;
```

## 注意事項

- PDF ファイル名に統一的な命名規則がないため、ファイル名からのメタ情報抽出は信頼性が低い。リンクテキストと年度見出しからメタ情報を取得すべき
- 平成30年の第1回定例会は日ごとに分割された複数の PDF で提供されている。同一会議として統合するかどうかは要検討
- 現時点で掲載されている年度は平成30年〜令和5年に限定されている。今後、令和6年以降や平成29年以前が追加される可能性がある
- ページ自体にはページネーションがないため、単一ページのスクレイピングで全データを取得可能

---

## 推奨アプローチ

1. **単一ページ取得**: 一覧ページは1ページのみのため、1回の HTTP リクエストで全 PDF リンクを収集できる
2. **リンクテキストベースのメタ情報抽出**: PDF ファイル名ではなく、HTML のリンクテキストと年度見出しからメタ情報を構築する
3. **PDF テキスト抽出**: `pdf-parse` や `pdfjs-dist` を使用して PDF 本文をテキスト化する
4. **レート制限**: 自治体サイトのため、PDF ダウンロード間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 前回取得済みの PDF URL リストと比較し、新規追加分のみをダウンロードする
