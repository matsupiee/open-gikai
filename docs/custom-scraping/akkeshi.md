# 厚岸町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.akkeshi-town.jp/chogikai/minutes/
- 分類: 年度別ディレクトリ形式で PDF 会議録を公開（既存アダプターでは対応不可）
- 文字コード: UTF-8（推定）
- 特記: 会議録は PDF ファイルで提供。HTML ベースの会議録検索システムは無い

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録トップ | `https://www.akkeshi-town.jp/chogikai/minutes/` |
| 年度別ページ（令和） | `https://www.akkeshi-town.jp/chogikai/minutes/r{年}/` |
| 年度別ページ（平成14〜30年） | `https://www.akkeshi-town.jp/chogikai/minutes/h{年}/` |
| 年度別ページ（平成10〜13年） | `https://www.akkeshi-town.jp/chogikai/minutes/h10_13/` |
| PDF ファイル | `https://www.akkeshi-town.jp/file/contents/{ID1}/{ID2}/{ファイル名}.pdf` |

### 年度別ページ一覧

令和8年(`r8`)〜令和元年(`r1`)、平成30年(`h30`)〜平成14年(`h14`)、平成10〜13年(`h10_13`) の計26ページ。

---

## PDF ファイル名パターン

PDF ファイル名には複数の命名規則が混在している:

| パターン | 例 | 説明 |
| --- | --- | --- |
| `r{YY}{MMDD}-{種別}.pdf` | `r060306-honnkaigi.pdf` | 令和年度・日付・種別 |
| `{種別}_{YYMMDD}.pdf` | `honkaigi_060910.pdf` | 種別・日付（アンダースコア区切り） |
| `R{YY}-{回}{種別}{MMDD}.pdf` | `R06-4honkaigi1211.pdf` | 令和年度・定例会回数・種別・日付 |
| `{MM}_{DD}.pdf` | `12_11.pdf` | 月日のみ（令和元年以前に多い） |
| `h{YY}-{回}{種別}{MMDD}.pdf` | `h13-4honkaigi1212.pdf` | 平成年度・回数・種別・日付 |
| `h{YY}rinzi{回}-1.pdf` | `h13rinzi5-1.pdf` | 平成年度・臨時会回数 |
| `rinji_{YYMMDD}.pdf` | `rinji_060426.pdf` | 臨時会・日付 |
| `{MMDD}.pdf` | `0116.pdf` | 日付のみ（最新年度） |

## HTML 構造

### 会議録トップページ (`/chogikai/minutes/`)

- 年度別リンクがリスト形式で配置
- 各リンクは `/chogikai/minutes/{年度コード}/` 形式

### 年度別ページ (`/chogikai/minutes/r6/` 等)

- 見出しタグ (`<h3>`〜`<h5>`) で定例会・臨時会ごとに区分
- 各会議種別（本会議、予算審査特別委員会等）がサブ見出しで分類
- PDF リンクはリスト要素 (`<ul>`, `<li>`) で配置
- リンクテキストは日付のみ（例: 「3月6日」）、括弧内にファイル形式とサイズ
- PDF の URL は `/file/contents/{ID1}/{ID2}/{ファイル名}.pdf` 形式

---

## ページネーション

なし。各年度ページに当該年度の全会議録 PDF リンクが一括掲載されている。

---

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録トップページ `https://www.akkeshi-town.jp/chogikai/minutes/` から年度別ページへのリンクを収集する。

- リンクは `/chogikai/minutes/{年度コード}/` 形式
- 全26年度分のリンクを取得

### Step 2: 各年度ページから PDF リンクの収集

各年度ページの HTML をパースし、PDF ファイルへのリンクを抽出する。

**収集方法:**

1. 各年度ページの HTML を取得
2. `<a>` タグから `/file/contents/` を含む PDF リンクを Cheerio で抽出
3. 見出しタグからメタ情報（定例会回数、会議種別）を取得
4. リンクテキスト（日付）と見出し情報を組み合わせて、各 PDF の会議情報を構造化

```typescript
// PDF リンクの抽出
const pdfLinks = $('a[href*="/file/contents/"]').map((_, el) => ({
  url: $(el).attr("href"),
  text: $(el).text().trim(),
})).get();
```

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）を使用してテキストを抽出
- PDF ファイルのため、HTML パースは不要

### Step 4: メタ情報の構造化

見出し構造と PDF ファイル名からメタ情報を構築する。

#### 見出しからの情報抽出

```
令和6年第1回臨時会（1月26日）
令和6年第1回定例会（3月6日～3月13日）
```

- 開催年: 見出しから `令和X年` or `平成X年` を抽出
- 定例会/臨時会: 「第N回定例会」「第N回臨時会」
- 会期: 括弧内の日付

#### ファイル名からの情報抽出

```typescript
// ファイル名から種別を判定
const categoryPattern = /(?:honnkaigi|honkaigi|yosan|hosei|hoseiyosan|kessan|keikaku|rinji|rinzi)/;

// ファイル名から日付を抽出（複数パターン対応）
const datePatterns = [
  /r(\d{2})(\d{2})(\d{2})/,       // r060306
  /(\d{2})(\d{2})(\d{2})/,         // 060910
  /(\d{1,2})_(\d{1,2})/,           // 12_11
  /(\d{4})/,                        // 0116
];
```

---

## 注意事項

- PDF ファイル名の命名規則が年度によって異なるため、複数のパターンに対応する必要がある
- 「honnkaigi」と「honkaigi」のように同じ意味で表記揺れがあるファイル名キーワードに注意
- 平成10〜13年は1ページにまとめられており、他の年度とは異なる構造
- PDF のテキスト抽出品質は元の PDF の作成方法に依存する（スキャン PDF の場合は OCR が必要）
- 古い年度（平成10〜13年）は臨時会のみ掲載の場合がある

---

## 推奨アプローチ

1. **2段階クロール**: まずトップページから年度別 URL を収集し、次に各年度ページから PDF リンクを収集する
2. **見出し構造の活用**: HTML の見出しタグ階層から定例会回数・会議種別を判定し、PDF ファイル名だけに頼らない
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 新しい年度ページのみを再クロールし、既存の PDF URL リストと差分を取る
5. **PDF テキスト抽出**: pdf-parse 等のライブラリで PDF からテキストを抽出し、発言者・発言内容をパースする
