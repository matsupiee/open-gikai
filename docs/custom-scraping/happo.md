## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録トップページ `/genre/kurashi/ghosei/gikai/gijiroku` から年度別ページへのリンクを収集する。

- 全年度のリンクが単一ページに列挙されている（ページネーションなし）
- リンクは `/archive/p{タイムスタンプ}` または `/archive/contents-{ID}` の2パターン
- Cheerio で `a[href]` を抽出し、`/archive/` を含むリンクをフィルタリング

### Step 2: PDF リンクの収集

各年度別ページから PDF ファイルへのリンクを収集する。

- 各ページに定例会・臨時会の会議録 PDF リンクが掲載されている
- PDF の URL パターン: `/uploads/public/archive_{ID}_00/{ファイル名}.pdf`
- ファイル名は日本語または英数字（命名規則は年度により異なる）
- 近年のページでは「全ページ版」「日程別」「一般質問抜粋」など複数の PDF に分かれている
- スクレイピング対象は「全ページ版」を優先する

**収集方法:**

1. 各年度別ページの HTML を取得
2. Cheerio で `a[href$=".pdf"]` を抽出
3. ファイル名からメタ情報（会議種別、開催年月）を推定

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

- PDF は OCR 不要のテキスト埋め込み型（コピー＆ペースト可能）
- `pdf-parse` 等のライブラリでテキスト抽出が可能
- ファイルサイズは 100KB〜1MB 程度

### Step 4: 会議録のパース

#### メタ情報

PDF ファイル名および本文冒頭から以下を抽出:

- 開催年度: ファイル名またはページタイトルから取得（例: `令和６年第１回八峰町議会臨時会会議録`）
- 会議種別: `定例会` / `臨時会`
- 開催日: PDF 本文内に記載

#### ファイル名パターン

```
# 新しい年度のファイル名（日本語）
令和６年第１回八峰町議会臨時会会議録.pdf
令和７年９月八峰町議会定例会会議録.pdf

# 古い年度のファイル名（英数字）
001_kaigiroku2006-12.pdf
002_kaigirokurinji2006-10.pdf
```

#### パース用正規表現（案）

```typescript
// ファイル名から会議情報を抽出（日本語ファイル名）
const japaneseFilePattern = /(?:令和|平成)([\d０-９]+)年.*?(定例会|臨時会)会議録/;

// ファイル名から会議情報を抽出（英数字ファイル名）
const alphanumericFilePattern = /kaigiroku(rinji)?(\d{4}[-]?\d{1,2})/;
// rinji があれば臨時会、なければ定例会
```

## 注意事項

- 検索機能がないため、全年度ページを巡回して PDF リンクを収集する必要がある
- PDF ファイル名の命名規則が年度によって異なる（日本語 / 英数字の混在）
- 近年（2023年〜）は日程別・一般質問別に PDF が分割されており、重複取得に注意が必要
- 「全ページ版」が存在する場合はそれを優先し、日程別・抜粋版は取得しない
- 年度別ページの URL パターンが新旧2種類ある（`/archive/p{タイムスタンプ}` と `/archive/contents-{ID}`）

---

## 推奨アプローチ

1. **2段階クロール**: トップページ → 年度別ページ → PDF リンクの順に収集
2. **全ページ版を優先**: 日程別・抜粋版との重複を避けるため、「全ページ」を含むリンクを優先取得
3. **PDF テキスト抽出**: `pdf-parse` 等でテキストを抽出し、発言者・発言内容をパース
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別ページの URL が固定のため、新しい年度ページのみを監視して差分更新が可能
