## 会議録の提供形式

PDF 直接リンク形式。各定例会・臨時会ごとに会期別の詳細ページが設けられ、その中に複数の PDF ファイルがリンクされる。

**PDF URL 形式:**

```
https://www.city.ureshino.lg.jp/var/rev0/{グループID}/{ファイルID}/{タイムスタンプ}.pdf
```

**令和7年第1回定例会（2025年2月28日〜3月21日）の例:**

| 文書種別 | URL パターン |
| --- | --- |
| 目次 | `/var/rev0/0047/2590/12585154739.pdf` |
| 議決一覧表 | `/var/rev0/0047/2591/12585154810.pdf` |
| 1日目会議録 | `/var/rev0/0047/2592/1258610641.pdf` |
| 2〜8日目会議録 | `/var/rev0/0047/2593〜2599/...pdf` |

- グループ ID（`0047` 等）はページ単位で共通
- ファイル ID（`2590`〜`2599` 等）は連番で振られる
- タイムスタンプはランダム文字列

## スクレイピング戦略

### Step 1: 年度ページ URL の収集

トップページ（`/gikai/hokoku/394.html`）から全年度リンクを抽出する。

- リンク形式: `./394/_[ID].html`
- 平成18年（2006年）〜現在まで約20年分

### Step 2: 年度別ページから会議別ページ URL を収集

各年度ページ（例: `394/_32067.html`）から当年の会議リンクを抽出する。

- リンク形式: `./394/_32067/_[ID].html`
- 1年度あたり4〜6件（定例会4回 + 臨時会0〜2回）

### Step 3: 会議別ページから PDF リンクを収集

各会議別ページ（例: `394/_32067/_32074.html`）から PDF リンクを抽出する。

- `<a href="/var/rev0/...pdf">` パターンで抽出
- 1会議あたり「目次」「議決一覧表」「日別会議録（複数）」の PDF が含まれる
- 会議録本文のみを取得する場合は「目次」「議決一覧表」を除外する

### Step 4: PDF のダウンロードと解析

- 会期情報（開会日・閉会日・会期日数）は会議別ページの本文に記載
- 日別会議録は連番で管理されているため順序通りに処理する

---

## 注意事項

- PDF URL のタイムスタンプ部分は予測不可能なため、必ずページから直接抽出する
- 委員会報告は別 URL（`/gikai/hokoku/_20466.html`）で管理されており、本会議会議録とは別途クロールが必要
- 平成18年（2006年）から掲載あり
