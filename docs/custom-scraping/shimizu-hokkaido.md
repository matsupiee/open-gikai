# 清水町議会（北海道） カスタムスクレイピング方針

## 概要

- サイト: https://www.town.shimizu.hokkaido.jp/gikai/proceeding/
- 分類: 町独自 CMS による HTML 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録は PDF ではなく HTML で直接公開。新サイト（平成29年以降）と旧サイト（平成17年〜平成28年）でテンプレートが異なる。委員会会議録は別セクション (`/gikai/committee/`) で管理。

## サイト構造

### トップページ (`/gikai/proceeding/`)

- 最新年度の会議録リンクが `ul.title-list` 内に `<li><a href="...">` として列挙される
- 年別一覧リンクが `ul.title-list-2columns` 内に列挙される
- ページネーションなし（全件が1ページに表示）

### 年別一覧ページ (`/gikai/proceeding/{年}/`)

- その年の全会議（定例会・臨時会・予算審査特別委員会）のリンクが `ul.title-list` に列挙
- 各リンクは議事日程表ページ（詳細ページ）を指す
- ページネーションなし

### 議事日程表ページ（詳細ページ）

- `.page-title` に会議名（例: `令和7年第5回清水町議会定例会（9月4日～9月22日） 議事日程表`）
- 日付ごとにテーブル（`table.table-gray`）で議事日程を表示
- 各議件から個別の会議録ページ（`/gikai/proceeding/details/{ID}.html`）へリンク
- 「当日の全会議録へ」リンクで、その日の全発言を含むページへ遷移可能

### 会議録ページ（本文）

- `.page-title` に会議名（例: `令和7年第5回定例会会議録（9月4日）`）
- 本文は `<p>` タグの連続で構成
- 発言者は `○{役職}（{氏名}）` 形式で記載
- 議事区切りは `◇・・・・・・・・・・・・・・・・・・・・・・◇`
- HTML のみ。PDF は使用されていない

## スクレイピング戦略

### Step 1: 年別一覧 URL の生成

トップページから年別リンクを取得する、または既知の年数パターンから URL を生成する。

**新サイト（平成29年〜）:**
- `/gikai/proceeding/29/` 〜 `/gikai/proceeding/7/`（令和の年数は1〜7）

**旧サイト（平成17年〜平成28年）:**
- `/gikai/past/kaigiroku/17/index.html` 〜 `/gikai/past/kaigiroku/28/index.html`

### Step 2: 議事日程表ページ URL の収集

各年別一覧ページの `ul.title-list` 内の `<a>` タグから `href` を抽出する。

```
/gikai/proceeding/details/7594922.html
/gikai/proceeding/details/74612620.html
...
```

### Step 3: 会議録ページ URL の収集

議事日程表ページ内のリンクから、会議録本文ページの URL を抽出する。

2種類のリンクがある:
1. **「当日の全会議録へ」リンク**: その日の全発言をまとめた1ページ（効率的）
2. **個別議件リンク**: 議件ごとに分割されたページ

全量取得には「当日の全会議録へ」リンクの使用を推奨。

**収集方法:**

```typescript
// 議事日程表ページから「当日の全会議録へ」リンクを抽出
// パターン: <a href="/gikai/proceeding/details/{ID}.html">当日の全会議録へ</a>
const fullRecordPattern = /href="(\/gikai\/proceeding\/details\/[^"]+\.html)"[^>]*>当日の全会議録へ/g;
```

### Step 4: 会議録本文の取得とパース

#### メタ情報

`.page-title` から抽出:

```
令和7年第5回定例会会議録（9月4日）
```

- 開催年: `令和{N}年` or `平成{N}年`
- 会議種別: `第{N}回定例会` / `第{N}回臨時会` / `予算審査特別委員会`
- 開催日: 括弧内の日付

#### 発言の構造

発言者パターン:

```
○議長（山下清美）
○総務課長（藤田哲也）
○４番（川上　均）
○事務局長（大尾　智）
○総務産業常任委員長（川上　均）
○厚生文教常任委員長（田村幸紀）
```

- `○` で始まり、役職 + 括弧内に氏名
- 議員の場合は番号表記（例: `○４番（川上　均）`）
- 括弧内の氏名には全角スペースが含まれる場合がある

議事区切り:

```
◇・・・・・・・・・・・・・・・・・・・・・・◇
```

傍聴者等の記載:

```
（「異議なし」と呼ぶ者あり）
（「なし」と呼ぶ者あり）
```

時刻表記:

```
（午前10時45分）
```

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○４番（川上　均） → role="４番", name="川上　均"
// 例: ○議長（山下清美） → role="議長", name="山下清美"

// 議事区切りの検出
const delimiterPattern = /^◇・+◇$/;

// 会議録タイトルの解析
const titlePattern = /^(令和|平成)(\d+)年(第\d+回)?(.*?)(会議録|会議日程)（(.+?)）$/;

// 開催日の抽出
const datePattern = /(\d+)月(\d+)日/;
```

## 旧サイト（平成17年〜平成28年）の差異

- テンプレートが異なる（XHTML 1.0 Transitional）
- コンテンツ領域は `#contents_in` 内
- 年別一覧のリンク先は `../details/{ID}.html` の相対パス形式
- 会議録本文の形式は新サイトと同様（`<p>` タグの連続、`○` で始まる発言者表記）

---

## 注意事項

- ページネーションなし: 年別一覧・トップページともに全件が1ページに表示されるため、ページ送り処理は不要
- PDF は使用されていない: 会議録は全て HTML で公開されている
- 議事日程表ページと会議録ページは別: 議事日程表ページは日程一覧のみで、会議録本文は別ページにある
- 旧サイトと新サイトのテンプレートが異なる: セレクタを分けて対応する必要がある
- 氏名の全角スペース: `川上　均` のように全角スペースが含まれる場合がある
- Incapsula（WAF）が導入されている: スクレイピング時にブロックされる可能性があるため、適切なリクエスト間隔を設ける

---

## 推奨アプローチ

1. **「当日の全会議録へ」リンクを優先**: 議件ごとの分割ページではなく、1日分の全発言をまとめたページを取得することで、リクエスト数を大幅に削減できる
2. **年別一覧から網羅的に収集**: トップページではなく年別一覧ページを起点にすることで、全年度の会議録を漏れなく取得できる
3. **レート制限**: Incapsula（WAF）が導入されているため、リクエスト間に適切な待機時間（2〜3 秒）を設ける
4. **新旧サイトの統一処理**: 本文の `<p>` タグ構造と発言者パターンは新旧サイトで共通のため、パース処理は共通化できる
5. **差分更新**: 年別一覧の最新年のみを定期的にクロールし、新しい会議録リンクの有無を確認する
