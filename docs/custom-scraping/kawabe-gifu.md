## PDF ファイルの命名規則

ファイル名は統一された規則がなく、アップロード時の入力内容に依存する。以下のパターンが混在している:

| パターン | 例 |
| --- | --- |
| 年号＋回数＋種別 | `令和７年第４回定例会.pdf` |
| 回数＋種別＋開催日（括弧付き） | `第３回定例会（9.9-19）HP用.pdf` |
| 回数＋種別＋開催日（括弧付き）＋注記 | `第２回定例会（6.4-14）訂正後（閲覧用）.pdf` |
| 末尾に連番 | `令和３年第４回定例会-3.pdf`、`令和３年第２回定例会-1.pdf` |

ファイル名単体では年号が判明しない場合があるため、ページ上のリンクテキストや格納ディレクトリのパス（アップロード年月）を合わせて参照すること。

## スクレイピング戦略

検索システムが存在しないため、インデックスページから PDF リンクを直接収集する。

### Step 1: インデックスページから PDF リンクを収集

1ページに全 PDF リンクが列挙されているため、インデックスページを1回取得するだけで全リンクが得られる。

```
GET https://www.kawabe-gifu.jp/?page_id=48191
```

- `wp-content/uploads/` を含む `<a>` タグの `href` 属性を抽出する
- リンクテキスト（例: `令和７年第４回定例会`）も合わせて取得し、年号・回数・種別を記録する
- リンクテキストに年号が含まれない場合は、URLパス中のアップロード年月（例: `2025/11/`）を補助情報として活用する

### Step 2: PDF のダウンロードとテキスト抽出

収集した PDF URL から PDF をダウンロードし、テキストを抽出する。

- 1つの PDF ファイルに1回分の会議録が収録されている
- スキャン画像形式の PDF が含まれる可能性があるため、OCR 処理への対応を考慮すること

## 注意事項

- **差分更新**: インデックスページは会議録が追加されるたびに更新される。新しい会議録の追加を検知するには、インデックスページの PDF リンク一覧を前回取得時と比較する。
- **ファイル名の揺れ**: 命名規則が統一されていないため、PDF URL 単体での年号・回数の解析は不確実。リンクテキストを優先的に参照すること。
- **令和３年第１回定例会が未掲載**: 調査時点（2026年3月）では令和３年の掲載は第２回〜第４回の3件のみ。第１回（3月定例会）は掲載されていない可能性がある。
- **PDF のみ**: テキスト形式での提供はなく、全データが PDF 形式。
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける。
