# 八女市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.yame.fukuoka.jp/shisei/12/7/index.html
- 分類: 独自 CMS による HTML ページ公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 専用の全文検索システムは存在しない。各会議（定例会・臨時会）ごとに個別 HTML ページが設けられており、会議録本文は PDF として提供される。

## 各会議詳細ページの構成

会議詳細ページ（`/{年号}/{記事ID}.html`）には以下が含まれる：

- 会議名・開催日（テキスト形式）
- 会期日程 PDF リンク
- 議決結果一覧表 PDF リンク

## スクレイピング戦略

### Step 1: トップページから年度一覧を収集

`index.html` を取得し、以下のリンクを抽出する：

- `shisei/12/7/{年号}/index.html`（最新2年程度）
- `shisei/12/7/kako-kaigikekka/{年号}/index.html`（過去年度）

### Step 2: 年度インデックスから会議詳細ページ ID を収集

各年度インデックス（`/{年号}/index.html`）から会議詳細ページへのリンク（`/{年号}/{記事ID}.html`）を抽出する。

### Step 3: 各会議詳細ページから PDF リンクを収集

会議詳細ページ内の PDF リンクを抽出する。

- 会議録 PDF は開催日ごとに個別ファイルで提供される場合がある
- 会期日程・議決結果一覧・会議録本文が別ファイルとして区別されている

### Step 4: PDF の取得・解析

各会議録 PDF を取得してテキストを抽出する。

---

## 注意事項

- 会議録の全文テキストは HTML ページには含まれず、すべて PDF 内にある
- 最新年度と過去年度でディレクトリ構造が異なる（`kako-kaigikekka/` の有無）
- 年号ディレクトリの命名規則が統一されておらず、`R6_1`、`reiwa4`、`H31` 等が混在している
- インターネット中継リンクは外部サービス（`www.yame-city.stream.jfit.co.jp`）に繋がっており、テキスト取得不可

---

## 推奨アプローチ

1. **トップページから動的収集**: `index.html` の年度リンクを起点として、ハードコードを避ける
2. **ディレクトリ名の正規化**: `R6_1` → 令和6年 のような変換テーブルを保持する
3. **PDF テキスト抽出**: pdfjs や pdfminer 相当のライブラリで PDF を解析する
4. **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒の待機時間を設ける
