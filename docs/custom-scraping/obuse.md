# 小布施町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.obuse.nagano.jp/diet/minutes/
- 分類: 自治体公式サイトで年度別に PDF 形式で会議録を公開
- 文字コード: UTF-8
- 特記: 会議録本文は PDF のみで提供（HTML 上にはテキストなし）

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録トップ | `https://www.town.obuse.nagano.jp/diet/minutes/` |
| 年度別一覧 | `https://www.town.obuse.nagano.jp/diet/minutes/{年度}/` |
| 会議詳細ページ | `https://www.town.obuse.nagano.jp/docs/{ID}.html` |
| 会議録 PDF | `https://www.town.obuse.nagano.jp/fs/{数字パス}/_/{ファイル名}.pdf` |

### 年度別一覧の範囲

## 会議の構造

小布施町議会は「定例会」ではなく月単位の「会議」形式で開催される（通年議会制）。

### 会議名の命名規則

- 通常: `令和X年Y月会議`
- 同月に複数回開催: `令和X年Y月第Z回会議`

### 会議詳細ページの ID パターン

ID に統一的な規則はなく、以下の2パターンが混在する:

- 記号ベース: `/docs/r7_3.html`（令和7年3月会議）、`/docs/r6-12.html`（令和6年12月会議）
- 数値ベース: `/docs/327358.html`、`/docs/27327.html`

---

## 会議詳細ページの構造

各会議詳細ページには以下のセクションが含まれる:

1. **会議録** - 日付別の PDF リンク（複数号に分割）
2. **審議内容** - 議案一覧（テーブル形式）
3. **一般質問** - 議員別の質問（MP3 音声ファイル付き）
4. **審議日程** - 日程表

### PDF リンクの構造

会議録 PDF は日付別・号数別に分割されている。

例（令和7年3月会議）:

| 日付 | ファイル名 | サイズ |
| --- | --- | --- |
| R7.3.3 | 令和7年3月3日会議録第1号.pdf | 294KB |
| R7.3.6 | 令和7年3月6日会議録第2号.pdf | 569KB |
| R7.3.7 | 令和7年3月7日会議録第3号.pdf | 372KB |
| R7.3.21 | 令和7年3月21日会議録第4号.pdf | 301KB |

PDF URL パターン:

```
https://www.town.obuse.nagano.jp/fs/{数字}/{数字}/{数字}/{数字}/{数字}/{数字}/_/{URLエンコードされたファイル名}.pdf
```

- `/fs/` 配下の数字パスはファイルごとに異なる（ファイル管理システムの内部 ID）
- ファイル名は URL エンコードされた日本語

## スクレイピング戦略

### Step 1: 年度別一覧から会議詳細ページ URL を収集

トップページ `/diet/minutes/` から年度リンクを取得し、各年度ページから会議詳細ページの URL を収集する。

**収集方法:**

1. `/diet/minutes/` をフェッチし、`/diet/minutes/{年度}/` のリンクを抽出
2. 各年度ページをフェッチし、`/docs/{ID}.html` のリンクを抽出
3. 会議名と掲載日も併せて取得

### Step 2: 会議詳細ページから PDF URL を収集

各会議詳細ページをフェッチし、会議録 PDF へのリンクを抽出する。

**収集方法:**

1. 各 `/docs/{ID}.html` をフェッチ
2. `.pdf` で終わるリンクを Cheerio で抽出
3. ファイル名から日付・号数のメタ情報を取得

**PDF リンク抽出の注意点:**

- 会議録 PDF 以外に委員会報告書 PDF も含まれる場合がある
- ファイル名に「会議録」を含むものを対象とする

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

**テキスト抽出:**

- `pdf-parse` 等のライブラリで PDF からテキストを抽出
- PDF のファイル名から開催日・号数を取得

### Step 4: 会議録のパース

#### メタ情報

PDF ファイル名から以下を抽出:

```
令和7年3月3日会議録第1号.pdf
```

- 開催日: `令和X年Y月Z日` 形式
- 号数: `第N号`
- 会議名: 年度ページの会議名リンクテキストから取得（例: `令和7年3月会議`）

#### ファイル名パース用正規表現（案）

```typescript
// PDF ファイル名から日付と号数を抽出
const pdfNamePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日会議録第(\d+)号/;
// 例: 令和7年3月3日会議録第1号 → era_year=7, month=3, day=3, number=1
```

---

## 注意事項

- 会議録はすべて PDF 形式のため、HTML パースではなく PDF テキスト抽出が必要
- PDF の内部構造（フォント、レイアウト）によってはテキスト抽出の精度が異なる可能性がある
- 会議詳細ページの ID に統一規則がないため、年度ページからのリンク収集が必須
- 委員会報告書 PDF も同じページに掲載されるため、ファイル名でフィルタリングが必要
- ページネーションは存在しない（各年度ページに全会議が一覧表示される）

---

## 推奨アプローチ

1. **2段階クロール**: まず年度ページ → 会議詳細ページ → PDF URL の順に URL を収集
2. **ファイル名ベースのフィルタ**: 「会議録」を含む PDF のみを対象とし、委員会報告書等を除外
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度ページの会議一覧と既取得データを比較し、新規追加分のみ取得する
5. **PDF テキスト品質の検証**: 初回取得時に PDF テキスト抽出の品質を確認し、必要に応じて OCR 等の代替手段を検討
