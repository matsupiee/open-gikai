# 遠軽町議会 カスタムスクレイピング方針

## 概要

- サイト: https://engaru.jp/life/page.php?id=398
- 分類: 自治体 CMS による PDF ファイル一覧（検索機能なし）
- 文字コード: UTF-8
- 特記: 全会議録が単一ページに掲載。PDF 形式のみ。平成28年第1回臨時会以降を掲載。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧（全件） | `https://engaru.jp/life/page.php?id=398` |
| PDF ファイル | `https://engaru.jp/common/img/content/content_{YYYYMMDD}_{HHMMSS}.pdf` |

- 会議録一覧は単一ページに全件が掲載されており、ページネーションはない
- PDF リンクは相対パス `../common/img/content/content_{YYYYMMDD}_{HHMMSS}.pdf` で記載されている
- ベース URL `https://engaru.jp/life/` からの相対パスを解決すると `https://engaru.jp/common/img/content/` になる

---

## ページ構造

### 年度セクション

会議録は年度別の見出し（h3/h5）で区切られ、新しい年度が上に表示される。

- 令和7年（2025年）〜 平成28年（2016年）

### 各エントリの形式

各会議録エントリは箇条書きで以下の情報を含む:

- 会議種別（第N回定例会 / 第N回臨時会）
- 開催日（M月D日）
- ファイル形式（PDF）
- ファイルサイズ（KB または MB）
- PDF ダウンロードリンク

---

## スクレイピング戦略

### Step 1: 一覧ページの取得

`https://engaru.jp/life/page.php?id=398` から HTML を取得する。

- 全件が単一ページに掲載されているため、ページネーション処理は不要
- 1回の HTTP リクエストで全会議録の情報を取得可能

### Step 2: PDF リンクの抽出

HTML をパースし、各 PDF リンクとそのメタ情報を抽出する。

**抽出対象:**

1. 年度（セクション見出しから）
2. 会議種別（定例会 / 臨時会、回数）
3. 開催日
4. PDF の URL

**パース方法:**

- Cheerio で HTML をパース
- 年度セクションの見出し（h3/h5）を基準に年度を判定
- 各セクション内のリンク要素から PDF URL を抽出
- リンクテキストから会議種別と開催日を抽出

### Step 3: PDF のダウンロードと本文抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）でテキスト抽出
- 会議録の本文をテキストとして保存

### パース用正規表現（案）

```typescript
// リンクテキストから会議種別と開催日を抽出
// 例: "12月9日開催　PDFファイル　779KB"
const entryPattern = /(\d+)月(\d+)日開催/;

// セクション見出しから年度を抽出
// 例: "令和７年" or "平成２８年"
const yearPattern = /(?:令和|平成)[０-９]+年/;

// 会議種別の抽出
// 例: "第６回定例会" or "第１回臨時会"
const sessionPattern = /第[０-９\d]+回(定例会|臨時会)/;

// PDF URLのパターン
const pdfUrlPattern = /\.\.\/common\/img\/content\/content_\d{8}_\d{6}\.pdf/;
```

---

## 注意事項

- PDF ファイルのみの提供であり、HTML の会議録本文は存在しない
- PDF の URL にはタイムスタンプ（アップロード日時）が含まれており、会議の開催日とは異なる
- 全角数字が使用されている箇所がある（年度、回数など）
- 平成28年第1回臨時会より前の会議録は掲載されていない

---

## 推奨アプローチ

1. **単一ページ取得**: 一覧ページ1回のリクエストで全 PDF リンクを収集可能
2. **PDF テキスト抽出**: HTML 本文がないため、PDF からのテキスト抽出が必須
3. **レート制限**: PDF ダウンロード時はリクエスト間に適切な待機時間（1〜2秒）を設ける
4. **差分更新**: 取得済み PDF の URL リストを保持し、新規追加分のみダウンロードする
