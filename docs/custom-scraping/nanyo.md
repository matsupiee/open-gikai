## 概要

- サイト: http://www.city.nanyo.yamagata.jp/gikaikaigiroku/
- 分類: PDF 公開（自治体公式サイト）
- 文字コード: UTF-8

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧（トップ） | `http://www.city.nanyo.yamagata.jp/gikaikaigiroku/` |
| 年度別ページ | `http://www.city.nanyo.yamagata.jp/gikaikaigiroku/{ページID}` |
| PDF ファイル | `http://www.city.nanyo.yamagata.jp/up/files/giyousei/sigikai/gikaikaigiroku/{ファイル名}.pdf` |

## PDF ファイルの命名規則

ファイル名に統一的な規則はなく、複数パターンが混在:

| パターン | 例 | 説明 |
| --- | --- | --- |
| 連番 | `419.pdf` | 3月定例会会議録 |
| 日本語ファイル名 | `南陽市議会令和7年12月定例会.pdf` | 定例会会議録 |
| 日本語（詳細） | `【背表紙・白紙頁削除ver_】南陽市議会第1回臨時会・令和7年6月定例会(本会議・予算特別委員会合冊).pdf` | 臨時会+定例会合冊 |
| 発議 | `発議第３号　日米地位協定の見直しを求める意見書の提出について.pdf` | 発議書 |

## スクレイピング戦略

### Step 1: 年度別ページ URL を収集

トップページから `<a>` タグで年度別ページへのリンクを収集する。URL は `/gikaikaigiroku/{数字ID}` 形式。

### Step 2: 年度別ページから PDF URL を収集

各年度ページをパースし、`/up/files/giyousei/sigikai/gikaikaigiroku/` 配下の PDF リンクを全て取得する。

- ファイル名が日本語の場合は URL エンコードされている
- `<small>` タグ内にファイルサイズ情報あり

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- ファイル名が日本語のため URL エンコード処理が必要
- 定例会と臨時会が合冊で1つの PDF になっている場合がある
- PDF ファイルサイズが大きい（最大 36MB 超）ため、ダウンロード時のタイムアウトに注意
- 発議書は個別の小さな PDF として掲載される
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
