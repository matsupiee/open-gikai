# 鋸南町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.kyonan.chiba.jp/site/machigikai/list23.html
- 分類: 町公式ホームページで年度別に PDF 形式で会議録を公開
- 文字コード: UTF-8（HTML ページ）/ PDF は MS-Mincho / MS-Gothic フォント使用
- 特記: 会議録は定例会・臨時会ごとに 1 つの PDF ファイルにまとめて公開。HTML 上の検索システムは無く、年度別一覧ページから PDF をダウンロードする形式。

## 会議の種類

各年度ページには以下の種類の会議録が掲載される:

- **定例会**: 年 3〜4 回程度（第1回〜第N回定例会）
- **臨時会**: 必要に応じて開催（第N回臨時会）

例（令和7年）:
- 令和7年第1回定例会
- 令和7年第2回臨時会
- 令和7年第3回定例会
- 令和7年第4回臨時会

## PDF の構造

### ページ構成

1. **表紙**: 会議名、開会日・閉会日、「鋸南町議会」の表記
2. **議案一覧表**: 議案番号と件名の一覧
3. **目次**: 各議事の項目名とページ番号
4. **招集告示**: 招集告示番号、日時、場所、付議事件
5. **議事日程**: 日程番号と議案の対応
6. **出席議員**: 番号、氏名（11名程度）
7. **欠席議員**: 番号、氏名
8. **説明出席者**: 町長、副町長、教育長、各課長等の職氏名
9. **事務局職員**: 事務局長、書記
10. **本文（発言録）**: 発言者と発言内容

### 発言者パターン

発言者は `○` で始まり、**役職（氏名）** の形式で太字表記される:

```
○議長（青木悦子）
○町長（白石治和）
○議会運営委員会委員長（鈴木辰也）
○税務住民課長（菊間寛之）
```

パターン:
- `○議長（氏名）` — 議長の発言
- `○町長（氏名）` — 町長の発言
- `○副町長（氏名）` — 副町長の発言
- `○教育長（氏名）` — 教育長の発言
- `○{委員会名}委員長（氏名）` — 委員長の発言
- `○{課名}課長（氏名）` — 課長の発言
- `○{番号}番（氏名）` — 議員の発言（番号付き）

### その他の記法

- 登壇表記: `〔{役職} {氏名}{役職名} 登壇〕`（例: `〔白石治和町長 登壇〕`、`〔議会運営委員会 鈴木辰也委員長 登壇〕`）
- 議場の声: `［「なし」の声あり］`、`［「異議なし」の声あり］`
- 休憩・再開: `……………休憩・午前１０時０７分 ……………` / `……………再開・午前１０時０８分 ……………`
- 議事見出し: `◎{見出し}` 形式（例: `◎開会の宣言`、`◎会議録署名議員の指名`、`◎議案第１号の上程、説明、質疑、討論、採決`）
- ページフッター: `- {ページ番号} -` の後に `{会議名}（{月}月）`

### 開催日の抽出

表紙から抽出:

```
令和７年５月８日　開会
令和７年５月８日　閉会
```

招集告示から抽出:

```
１　日　　時　令和７年５月８日（金）　午前１０時
```

## スクレイピング戦略

### Step 1: PDF URL の収集

1. 一覧ページ `list23.html` から各年度ページの URL を取得
2. 各年度ページ（例: `0012731.html`）にアクセスし、PDF リンク（`/uploaded/attachment/{ID}.pdf`）を抽出
3. リンクテキストから会議名（例: `令和７年第１回定例会会議録`）を取得

**HTML 構造**: 各 PDF リンクは `<a>` タグで `href="/uploaded/attachment/{ID}.pdf"` の形式。リンクテキストに会議名とファイルサイズが含まれる。

### Step 2: PDF のダウンロードとテキスト抽出

- PDF を `pdf-parse` や `pdfjs-dist` 等のライブラリでテキスト化
- PDF は FlateDecode 圧縮、CIDFont（MS-Mincho, MS-Gothic）使用のため、テキスト抽出ライブラリの互換性を事前に確認する
- ファイルサイズは 300KB〜2.5MB 程度

### Step 3: 会議録のパース

#### メタ情報の抽出

表紙ページから:
- 会議名: `令和{X}年第{N}回{定例会|臨時会}`
- 開会日: `令和{X}年{M}月{D}日　開会`
- 閉会日: `令和{X}年{M}月{D}日　閉会`

#### 発言の抽出

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○議長（青木悦子） → role="議長", name="青木悦子"
// 例: ○税務住民課長（菊間寛之） → role="税務住民課長", name="菊間寛之"

// 議事見出しの抽出
const agendaPattern = /^◎(.+)/;
// 例: ◎開会の宣言 → agenda="開会の宣言"

// 登壇の検出
const podiumPattern = /^〔(.+?)登壇〕/;

// 休憩・再開の検出
const breakPattern = /休憩・(.+)/;
const resumePattern = /再開・(.+)/;

// 開催日の抽出
const datePattern = /(?:令和|平成)([\d０-９]+)年([\d０-９]+)月([\d０-９]+)日/;
```

## 注意事項

- PDF からのテキスト抽出時、全角スペースによる字間調整が多用されているため、正規化処理が必要（例: `令 和 ７ 年` → `令和７年`）
- 議員番号は全角数字で表記（例: `１番`、`１２番`）
- 1 つの PDF に複数日の会議が含まれる場合がある（定例会は数日間にわたることがある）
- ページ ID は連番ではないため、一覧ページからの動的取得が必須
- PDF ファイル ID（`/uploaded/attachment/{ID}.pdf`）も連番ではない
- ページネーションは無し（各年度ページに当該年度の全会議録が掲載）

---

## 推奨アプローチ

1. **2段階クロール**: まず一覧ページ → 年度ページの順で全 PDF URL を収集し、次に PDF をダウンロード・パースする
2. **PDF テキスト抽出**: `pdfjs-dist` または `pdf-parse` を使用。CIDFont 対応を確認すること
3. **全角正規化**: テキスト抽出後、全角スペース除去・全角数字→半角数字変換を行ってからパースする
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度ページの更新日（`更新日：YYYY年M月D日更新`）を監視し、更新があった年度のみ再取得する
