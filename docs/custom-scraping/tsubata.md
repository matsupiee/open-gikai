# 津幡町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.tsubata.lg.jp/page/1738.html
- 分類: 津幡町公式ウェブサイト上での月別 PDF 直接公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録検索システムは採用されておらず、1 ページに全会議録 PDF リンクを列挙する構成。discussnet_ssp / dbsearch / kensakusystem / gijiroku_com はいずれも使用していない。

---

## URL 構造

| ページ | URL |
| --- | --- |
| 会議録一覧（唯一のページ） | `https://www.town.tsubata.lg.jp/page/1738.html` |
| PDF ファイル | `https://www.town.tsubata.lg.jp/uploaded/attachment/{数値}.pdf` |

ページネーションはなく、全会議録が 1 ページに掲載される。

## ページ構造

### 年度見出し

```html
<h2>令和○年津幡町議会　会議録</h2>
```

年度の新しい順（令和8年 → 令和元年（平成31年） → 平成30年 → … → 平成21年）に並ぶ。

### PDF リンク

```html
<a href="/uploaded/attachment/{数値}.pdf">{会議名} [PDFファイル／{ファイルサイズ}]</a>（{開催日程}）
```

例:

```html
<a href="/uploaded/attachment/6442.pdf">津幡町議会12月会議 [PDFファイル／1.82MB]</a>（令和7年12月4日～12月11日）
```

- リンクテキストに会議名とファイルサイズが含まれる
- リンク直後の括弧内テキストに開催日程が記載される（単日または「月日〜月日」形式）

### 速報版

ページ冒頭に「議会会議録　速報版」セクションがあり、正式版作成前の仮掲載 PDF が含まれる。正式版公開後に削除されるため、取得時は速報版と正式版を区別する必要がある。

会議の命名規則は時期によって異なる。

| 期間 | 命名規則 | 例 |
| --- | --- | --- |
| 令和6年以降 | `津幡町議会{月}月会議` | 津幡町議会12月会議 |
| 令和5年（一部） | `第2回津幡町議会{月}月会議` | 第2回津幡町議会12月会議 |
| 令和元年〜令和4年 | `津幡町議会{月}月会議`（一部 `第2回` 付き） | 津幡町議会9月会議 |
| 平成28年〜令和元年 | `津幡町議会{月}月会議`（一部 `第2回` 付き） | 津幡町議会3月会議 |
| 平成24〜28年 | `第{N}回津幡町議会定例会{月}月会議` / `第{N}回津幡町議会臨時会` | 第2回津幡町議会定例会12月会議 |

## 開催日程のパターン

```
（令和7年12月4日～12月11日）  # 複数日にまたがる場合
（令和8年1月21日）             # 単日の場合
```

- 開始日と終了日が異なる場合は「〜」で接続
- 元号は「令和」または「平成」

## スクレイピング戦略

### Step 1: 一覧ページの取得

`https://www.town.tsubata.lg.jp/page/1738.html` を 1 回取得する。

- ページネーションなし（全データが 1 ページ）
- 文字コード: UTF-8

### Step 2: 年度見出しと PDF リンクの抽出

Cheerio を使い、`h2` 見出しと直後のリスト（`li > a[href$=".pdf"]`）を年度ごとに関連付けながら抽出する。

**抽出する情報:**

| 項目 | 取得元 |
| --- | --- |
| 年度 | `h2` テキストから「令和○年」「平成○年」を抽出 |
| 会議名 | `a` テキストから `[PDFファイル／...]` 部分を除去 |
| 開催日程 | `a` の次のテキストノード（括弧内） |
| PDF URL | `a[href]` に `https://www.town.tsubata.lg.jp` を補完 |

**パース用正規表現（案）:**

```typescript
// 年度の抽出（h2 テキストから）
const yearPattern = /(?:令和|平成)(\d+)年/;

// 会議名のクリーニング（ファイルサイズ表記を除去）
const titleCleanPattern = /\s*\[PDFファイル／[^\]]+\]/;

// 開催日程の抽出（リンク直後のテキストノードから）
const datePattern = /（((?:令和|平成)\d+年\d+月\d+日(?:～\d+月\d+日)?）/;

// 開催日の開始日・終了日の分離
const dateRangePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日(?:～(\d+)月(\d+)日)?/;
```

### Step 3: 速報版の識別

ページ冒頭の「議会会議録　速報版」セクション内のリンクは速報版として扱う。正式版公開後に同リンクが削除されるため、差分取得時は URL の存在確認が必要。

---

## 注意事項

- PDF URL の数値部分（attachment ID）は連番ではなく、古い年度のものが後から追加されたケースがある（例: 平成21年のファイルが 5672〜5679 番台）。そのため ID の大小で新旧を判断することはできない。
- 会議録の本文テキストは PDF 内にあり、HTML からはメタ情報（会議名・日付・PDF URL）のみ取得可能。
- リクエストは 1 回で全データを取得できるため、レート制限の問題は小さいが、PDF ダウンロード時には 1〜2 秒の待機を設ける。

---

## 推奨アプローチ

1. **単一ページ取得**: 一覧ページへの GET リクエスト 1 回で全会議録のメタ情報を取得できる
2. **Cheerio でのパース**: `h2` を年度の区切りとして、直後の `ul > li` 内のリンクを関連付ける
3. **速報版の除外または別管理**: 速報版セクション（ページ先頭の「速報版」見出し配下）を識別し、正式版と区別する
4. **PDF ダウンロード**: 本文抽出が必要な場合は PDF を個別ダウンロードし、pdf-parse 等でテキスト抽出する
5. **差分更新**: 前回取得時のリンク一覧と比較し、新規追加された PDF のみを処理する
