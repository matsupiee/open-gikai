# 外ヶ浜町議会（青森県） カスタムスクレイピング方針

## 概要

- サイト: http://www.town.sotogahama.lg.jp/gyosei/gikai/
- 分類: 町公式サイトで議会だよりを PDF 公開（会議録検索システムなし）
- 文字コード: UTF-8
- 特記: 会議録の全文公開は確認できず、議会だより PDF と一般質問通告表 PDF のみ掲載

## 議会だより

### PDF ファイル命名規則

```
files/YYYYMM_soto_gikaidayori_XX.pdf
```

- `YYYYMM`: 発行年月（例: `202602`）
- `XX`: 号数（例: `83`）

### ページ構造

## 一般質問通告表

### PDF ファイル命名規則

```
files/YYYYMM_ippan_situmon.pdf
```

- `YYYYMM`: 定例会の年月（例: `202603`）

### 掲載範囲

令和6年度（2024年9月〜）以降の定例会分が確認できる。回数表記あり（例: 第164回、第162回）。

## スクレイピング方針

### 会議録について

外ヶ浜町議会では会議録検索システムの導入が確認できず、会議録全文のオンライン公開も行われていない。そのため、**会議録の直接的なスクレイピングは不可**。

### 取得可能なコンテンツ

1. **議会だより PDF**: 議会の審議内容の要約、一般質問と答弁の概要が掲載されている
2. **一般質問通告表 PDF**: 定例会の日程と一般質問の通告内容

### 推奨アプローチ

1. **議会だより PDF の取得・テキスト抽出**を主軸とする
   - `gikai_dayori.html` から全 PDF リンクを収集
   - PDF をダウンロードし、テキスト抽出ツール（pdf-parse 等）でテキスト化
   - 号数・発行年月をファイル名から抽出
2. **一般質問通告表 PDF** を補助的に取得
   - `ippan_situmon.html` から PDF リンクを収集
   - 質問者名・質問内容の抽出

### 制約事項

- 議会だよりは会議録の要約であり、発言の全文は含まれない
- PDF のレイアウトが号によって異なる可能性がある（段組み、画像配置等）
- テキスト抽出の精度は PDF の作成方法に依存する

---

## 注意事項

- **HTTP のみ**: サイトは `http://` で提供されている（HTTPS 非対応の可能性あり）
- **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒の待機時間を設ける
- **差分更新**: 既取得ファイル名のリストと比較し、新規 PDF のみを取得する
- **会議録全文の公開状況は定期的に確認する**: 将来的に会議録検索システムが導入される可能性がある
