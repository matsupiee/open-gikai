# 新地町議会 カスタムスクレイピング方針

## 概要

- サイト: http://shinchi-k.k-quick.net/index.html
- 分類: k-quick.net ドメインの静的 HTML + PDF 公開型（福島県内の複数自治体が使用する議事録システム）
- 文字コード: Shift_JIS（HTML ページ）/ PDF 形式（会議録本文）
- 特記: 会議録本文は PDF ファイルで公開されており、HTML ベースの検索・閲覧システムではない

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| トップページ | `http://shinchi-k.k-quick.net/index.html` |
| 年度別一覧 | `http://shinchi-k.k-quick.net/{年度コード}.html` |
| 会議録 PDF | `http://shinchi-k.k-quick.net/pdf/{年度コード}/{ファイル名}.pdf` |
| 利用マニュアル | `http://shinchi-k.k-quick.net/pdf/manual.pdf` |

### 年度コードの規則

| 年度コード | 年度 |
| --- | --- |
| `R0700` | 令和7年度 |
| `R0600` | 令和6年度 |
| `R0500` | 令和5年度 |
| `R0400` | 令和4年度 |
| `R0300` | 令和3年度 |
| `R0200` | 令和2年度 |
| `R0100` | 令和元年度 |
| `H3100` | 平成31年度 |
| `H3000` | 平成30年度 |
| `H2900` | 平成29年度 |
| `H2800` | 平成28年度 |
| `H2700` | 平成27年度 |

- 令和は `R` + 2桁年 + `00`、平成は `H` + 2桁年 + `00`
- 確認できる最古の年度は平成27年度（H2700）

### PDF ファイル名の規則

```
{年度プレフィックス}.{連番}{会議種別}.pdf
```

例:
- `R06.12T.pdf` → 令和6年度、12月、定例会（T）
- `R06.07R.pdf` → 令和6年度、7月、臨時会（R）
- `H27.01R.pdf` → 平成27年度、第1回、臨時会（R）

| サフィックス | 意味 |
| --- | --- |
| `T` | 定例会 |
| `R` | 臨時会 |

※ 数字部分は月を表す場合（R06 以降）と連番の場合（H27 等）がある。

---

## サイト構成の特徴

- **検索機能なし**: フリーワード検索や発言者検索の機能は存在しない
- **フレーム・フォームなし**: 静的 HTML ページに PDF リンクを並べた簡素な構成
- **ページネーションなし**: 年度ごとに1ページで全会議録 PDF を一覧表示
- **会議録は PDF 形式のみ**: HTML での会議録本文公開はない

---

## スクレイピング戦略

### Step 1: PDF URL の収集

年度別一覧ページ（`{年度コード}.html`）およびトップページ（`index.html`）から PDF リンクを収集する。

**収集方法:**

1. トップページから最新年度の PDF リンクを抽出（トップページには最新年度の PDF が掲載）
2. 全年度コード（R0700〜H2700）の一覧ページ URL を生成
3. 各ページから `pdf/` で始まるリンクを Cheerio で抽出
4. PDF の URL を正規化して重複を除去

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF パーサー（`pdf-parse` 等）を使用してテキスト化
- 1つの PDF に定例会/臨時会の全日程分の会議録が含まれる（例: 第7回定例会は第1号〜第4号の4日分）

### Step 3: 会議録のパース

#### PDF 内の構造

1つの PDF は以下の構成:

```
表紙（会議名、開会・閉会日）
目次
招集告示
応招・不応招議員
第1号（第N日目の会議録）
  - 議事日程
  - 出席議員
  - 欠席議員
  - 説明出席者
  - 職務出席者
  - 本文（発言録）
第2号...
...
```

#### メタ情報の抽出

表紙から:
```
令和６年　第７回定例会
新 地 町 議 会 会 議 録
令和６年１２月６日　開会
令和６年１２月１１日　閉会
```

各号の議事日程から:
```
令和６年第７回新地町議会定例会
議 事 日 程　（第１号）
令和６年１２月６日（金曜日）午前１０時開会
```

#### 発言者パターン

発言者は `○` で始まり、役職/番号 + 氏名の形式:

```
○遠藤　満議長
○大堀　武町長
○佐藤武志事務局長
○１番大内広行議員
○大堀　武町長
```

パターン:
- `○{姓}　{名}{役職}` — 議長パターン（例: `○遠藤　満議長`）
- `○{姓}　{名}{役職}` — 執行部パターン（例: `○大堀　武町長`）
- `○{番号}番{姓}{名}議員` — 議員パターン（例: `○１番大内広行議員`）
- `○{姓}{名}{役職}` — 職員パターン（例: `○佐藤武志事務局長`）

#### 登壇表記

```
〔大堀　武町長登壇〕
〔１番　大内広行議員登壇〕（拍手）
```

#### 議場の反応

```
〔「異議なし」と言う人あり〕
```

#### パース用正規表現（案）

```typescript
// 発言者の抽出（議長・町長・職員など役職付き）
const speakerWithTitlePattern = /^○(.+)/;

// 議員パターン（番号付き）
const memberPattern = /^○(\d+)番(.+?)議員/;

// 役職者パターン
const officialPattern = /^○(.+?)(議長|町長|副町長|事務局長|課長)/;

// 登壇の検出
const podiumPattern = /^〔(.+?)登壇〕/;

// 開催日の抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;

// 会議種別の抽出
const sessionPattern = /第(\d+)回.*?(定例会|臨時会)/;

// 号数の抽出
const issuePattern = /第(\d+)号/;
```

## 注意事項

- 文字コードが Shift_JIS のため HTML ページの取得時にエンコーディング変換が必要
- PDF からのテキスト抽出では、全角スペースが発言者名の姓と名の間に入る場合がある（例: `遠藤　満議長`）
- 1つの PDF に複数日の会議録が含まれるため、「第N号」の区切りで分割する必要がある
- 議員定数は12名（令和6年時点）
- 町長は大堀武（令和6年時点）
- PDF のページ番号は通し番号で付与される（例: 109ページ）

---

## 推奨アプローチ

1. **PDF 中心の取得**: HTML ページは PDF リンクの収集のみに使用し、本文は PDF から抽出する
2. **年度単位で処理**: 1つの PDF が1つの定例会/臨時会の全日程分を含むため、PDF 単位で処理する
3. **テキスト抽出ライブラリ**: `pdf-parse` または `pdfjs-dist` を使用して PDF からテキストを抽出
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別ページの PDF リンク数を前回と比較し、新規 PDF のみをダウンロードする
