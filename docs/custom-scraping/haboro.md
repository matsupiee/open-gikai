## 会議の種類

年度別ページでは以下の会議種別が表形式で分類されている。

| 分類 | 会議名 |
| --- | --- |
| 定例会・臨時会 | 第N回定例会、第N回臨時会 |
| 特別委員会 | 予算特別委員会、決算特別委員会 |
| 常任委員会 | 文教厚生常任委員会、総務産業常任委員会 |
| その他特別委員会 | 公共事業等調査研究特別委員会、広報広聴常任委員会（年度により異なる） |

## PDF ファイル名のパターン（平成25年以降）

ファイル名の命名規則は統一されておらず、年度ごとにばらつきがある。

```
# 定例会・臨時会の例
060312T.pdf     → 令和6年3月12日 定例会
R070128R.pdf    → 令和7年1月28日 臨時会
070311.pdf      → 令和7年3月11日
teirei1-6-13.pdf → 定例会 6月13日

# 常任委員会の例
0207b2.pdf      → 2月7日 文教厚生(b)常任委員会
0424s.pdf       → 4月24日 総務産業(s)常任委員会
R070127B.pdf    → 令和7年1月27日 文教厚生(B)

# 予算・決算の例
060313Y.pdf     → 令和6年3月13日 予算(Y)
0609kessan.pdf  → 9月 決算
```

主な略語:
- `T` / `teirei`: 定例会
- `R` / `r`: 臨時会
- `Y` / `y` / `yosan`: 予算特別委員会
- `K` / `kessan`: 決算特別委員会
- `B` / `b` / `bunkyo`: 文教厚生常任委員会
- `S` / `s` / `soumu`: 総務産業常任委員会

## HTML 会議録の構造（平成18年〜24年）

### ファイル名パターン

```
h18-01-0307.html       → 平成18年 第1回 3月7日
h18-yosan-0308.html    → 平成18年 予算委員会 3月8日
h18-kessan-1124.html   → 平成18年 決算委員会 11月24日
```

### 本文構造

- HTML の `<p>` タグと改行で構成（テーブル不使用）
- 議事進行の見出しは `◎` で表記: `◎開議の宣告`、`◎町長あいさつ` など

### メタ情報

ページ冒頭に以下の情報が記載:

- 出席議員: `〇出席議員（N名）` に続いて議員名を列記
- 欠席議員: `〇欠席議員（N名）`
- 説明出席者: `〇地方自治法第121条の規定により説明のため出席した人`

### 発言者パターン

```
〇議長（森　淳君）
〇町長（舟橋泰博君）
〇11番（磯野直君）
```

- `〇` で始まり、役職 or 番号 + 括弧内に氏名（敬称「君」付き）
- 発言内容は発言者行の後に続くテキスト

### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^〇(.+?)（(.+?)）/;
// 例: 〇11番（磯野直君） → role="11番", name="磯野直君"

// 議事進行の見出し
const agendaPattern = /^◎(.+)/;
// 例: ◎開議の宣告 → agenda="開議の宣告"

// 開催日の抽出（ページタイトルから）
const datePattern = /(?:令和|平成)(\d+)年.+?(\d+)月(\d+)日/;
```

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

議事録目次ページ（トップ）から年度別ページへのリンクを抽出する。

- `<ul class="indexlist">` 内の `<a>` タグからリンクを収集
- 各年度ページの相対パスを絶対 URL に変換

### Step 2: 各年度ページから会議録リンクの収集

年度別ページの HTML をパースし、会議録（PDF or HTML）へのリンクを抽出する。

- 近年（平成25年以降）: `files/*.pdf` へのリンクを収集
- 古い年度（平成18年〜24年）: `*.html` へのリンクを収集
- 各リンクの前後のテキストから会議名・日付を抽出

### Step 3: 会議録本文の取得

#### PDF の場合（平成25年以降）

1. PDF をダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 発言者パターンでテキストを分割

#### HTML の場合（平成18年〜24年）

1. HTML を取得
2. Cheerio でテキストを抽出
3. `〇` で始まる発言者行を検出し、発言単位に分割

---

## 注意事項

- PDF ファイル名の命名規則が年度ごとに統一されていないため、ファイル名からのメタ情報抽出は信頼性が低い。年度ページの HTML から会議名・日付を取得するほうが確実
- 年度ページの HTML 構造も年度によって若干異なる（表形式 vs リスト形式）
- ページネーションは存在しない（年度ごとに1ページで全件表示）
- サイトは Incapsula（CDN/WAF）を使用しているため、アクセス制限に注意

---

## 推奨アプローチ

1. **2段階クロール**: まず目次ページ → 年度ページ → 個別会議録の順でリンクを収集
2. **PDF 優先**: 平成25年以降の PDF を優先的に取得（データ量が多い）
3. **HTML は補完的に**: 平成18年〜24年の HTML 会議録は構造化しやすいが、古いデータのため優先度は下げてもよい
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **メタ情報は年度ページから取得**: PDF ファイル名ではなく、年度ページの表・リストの記載テキストから会議名・日付を抽出する
