# 丸森町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.marumori.miyagi.jp/town/detail.php?content=451
- 分類: 独自 CMS による議決結果 PDF 公開（会議録本文のテキスト公開なし）
- 文字コード: UTF-8（推定）
- 特記: 会議録本文（発言録）のオンライン公開はなく、議決結果 PDF と議会中継動画（MP4）のみ提供

## 公開コンテンツの分析

### 1. 議決結果一覧（`detail.php?content=1031`）

- 令和2年〜令和8年の議決結果を年度別に掲載
- 全データが 1 ページに表示（ページネーションなし）
- 各会議の議決結果は PDF リンクで提供
- `<h5>` タグで年度見出し、`<ul>` リストで各会議を列挙
- リンクテキスト形式: `令和X年第Y回丸森町議会{定例会|臨時会}（M月D日～D日）`
- PDF URL は相対パス `../../common/img/content/content_{YYYYMMDD}_{HHMMSS}.pdf` 形式
- PDF 内容は議案名・議決結果の一覧表（発言録ではない）

### 2. 議会中継動画（`detail.php?content=836`）

- 一般質問の動画を議員ごとに MP4 で公開
- 動画ファイル名は `{YYYYMMDD}.mp4` 形式（連番的な命名）
- 通告番号、議員名、質問内容が一覧表示される
- 令和3年〜令和8年分のアーカイブが存在
- 過去年度は年度別ページへのリンクで分類

## スクレイピング戦略

### 現状の制約

丸森町議会は **会議録本文（発言録）をオンラインで公開していない**。公開されているのは以下のみ:

1. **議決結果 PDF**: 議案名と賛否結果の一覧表（発言内容は含まれない）
2. **議会中継動画**: 一般質問の MP4 録画（テキストデータではない）

### Step 1: 議決結果 PDF の収集（実装可能）

議決結果一覧ページ `detail.php?content=1031` から PDF リンクを抽出する。

1. ページを取得し、`<a>` タグから PDF URL を抽出（`content_*.pdf` パターン）
2. リンクテキストから会議名・開催日を正規表現で抽出
3. PDF をダウンロードし、テキスト抽出（Antenna House PDF Driver で作成されたテキスト埋め込み PDF）

**会議名の抽出パターン:**

```typescript
// リンクテキストからメタ情報を抽出
const meetingPattern = /令和(\d+)年第(\d+)回丸森町議会(定例会|臨時会)（(.+?)）/;
// 例: 令和８年第１回丸森町議会臨時会（１月２７日）
//   → year=8, number=1, type="臨時会", date="１月２７日"
```

### Step 2: 議会中継動画メタ情報の収集（実装可能）

過去の議会中継ページ `detail.php?content=836` から一般質問の情報を抽出する。

1. ページを取得し、議員名・質問内容・動画 URL を抽出
2. 年度別アーカイブページも順次取得

### Step 3: 会議録本文の取得（現時点では不可）

会議録本文はオンライン公開されていないため、スクレイピングでの取得は不可能。

---

## 注意事項

- **会議録本文なし**: 発言録のテキストデータはオンラインで公開されていない。議決結果（議案の賛否）と議会中継動画のみが利用可能
- **PDF は議決結果のみ**: PDF 内容は議案名と議決結果の一覧表であり、発言者・発言内容は含まれない
- **動画からの文字起こし**: 一般質問の内容を取得するには動画（MP4）からの音声文字起こしが必要だが、現行のスクレイピング基盤の対象外
- **PDF URL のタイムスタンプ**: PDF ファイル名のタイムスタンプはアップロード日時であり、会議開催日とは一致しない。開催日はリンクテキストから取得する
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **議決結果 PDF の収集を優先**: 1 ページに全年度分のリンクが掲載されているため、効率的に全量収集が可能
2. **会議録本文は対象外**: 現時点ではオンライン公開がないため、スクレイピング対象から除外する
3. **動画メタ情報の収集**: 一般質問の議員名・質問テーマは HTML から取得可能なため、メタ情報として保存する
4. **将来的な会議録公開への対応**: 丸森町が会議録検索システムを導入した場合に再調査する
