# 知内町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.shiriuchi.hokkaido.jp/chosei/gikai/kaigiroku/
- 分類: PDF 形式で公式サイトに直接公開（検索システムなし）
- 文字コード: UTF-8
- 特記: 年度別ページに定例会・臨時会の会議録 PDF がリンクされている。YouTube 動画リンクも付随。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録トップ（年度一覧） | `https://www.town.shiriuchi.hokkaido.jp/chosei/gikai/kaigiroku/` |
| 年度別ページ | `https://www.town.shiriuchi.hokkaido.jp/chosei/gikai/kaigiroku/{年号コード}/` |
| PDF ファイル | `https://www.town.shiriuchi.hokkaido.jp/files/{ディレクトリID1}/{ディレクトリID2}/{ファイル名}.pdf` |

### 年号コードの対応

| 年号コード | 年度 |
| --- | --- |
| `h24` | 平成24年 |
| `h25` | 平成25年 |
| ... | ... |
| `h31` | 平成31年 |
| `r01` | 令和元年 |
| `r02` | 令和2年 |
| ... | ... |
| `r07` | 令和7年 |

## PDF ファイルの URL パターン

PDF ファイルの URL 構造は年度によって異なる。

### 初期（平成24年頃）

ファイル名に会議情報がエンコードされている:

```
/files/00004700/00004722/H24.1定-１.pdf   → 第1回定例会 1日目
/files/00004700/00004722/H24.2定.pdf      → 第2回定例会
/files/00004700/00004722/H24.1臨.pdf      → 第1回臨時会
/files/00004700/00004722/H24予算-１.pdf    → 予算審査 1日目
/files/00004700/00004722/H23決算-１.pdf    → 決算審査 1日目
```

### 近年（令和6年以降）

ファイル名がタイムスタンプ形式（`YYYYMMDDHHmmss`）に変更:

```
/files/00007700/00007787/20240610132320.pdf  → 令和6年
/files/00008200/00008203/20250718112546.pdf  → 令和7年
```

### ディレクトリ ID

年度ごとにディレクトリ ID が異なる:

| 年度 | ディレクトリパス |
| --- | --- |
| 平成24年 | `/files/00004700/00004722/` |
| 令和6年 | `/files/00007700/00007787/` |
| 令和7年 | `/files/00008200/00008203/` |

## ページ構造

### 会議録トップページ

- 年度別リンクのリスト形式
- ページネーションなし（全年度が1ページに表示）
- 各リンクは `/chosei/gikai/kaigiroku/{年号コード}/` へ遷移

### 年度別ページ

テーブル形式で以下のカテゴリに分類:

- **定例会**（年4回: 3月・6月・9月・12月）
- **臨時会**（随時開催）
- **特別委員会**（予算審査・決算審査、定例会に付随）

各会議のテーブルカラム:

| カラム | 内容 |
| --- | --- |
| 開会日 | 会議の開催日 |
| 区分 | 1日目、2日目、3日目など |
| 議案 | 議案 PDF へのリンク |
| 会議録 | 会議録 PDF へのリンク |
| 審議結果 | 審議結果 PDF へのリンク |
| 動画 | YouTube 動画リンク |

---

## 会議種別

### 定例会

- 年4回開催（3月・6月・9月・12月）
- 複数日にわたる場合あり（例: 3月5日〜8日で3日間）
- 各日の会議録が個別の PDF として提供

### 臨時会

- 必要に応じて随時開催（年5〜7回程度）
- 通常は単日開催
- 各回の議案と会議録が個別の PDF

### 特別委員会

- 予算審査特別委員会（3月定例会に付随）
- 決算審査特別委員会（9月定例会に付随）
- 複数日にわたる場合あり

## スクレイピング戦略

### Step 1: 年度ページ URL の収集

会議録トップページ `/chosei/gikai/kaigiroku/` から全年度ページへのリンクを抽出する。

- `<a>` タグの `href` 属性から `/chosei/gikai/kaigiroku/{年号コード}/` パターンを収集
- 対象範囲: h24 〜 r07（平成24年〜令和7年）

### Step 2: 各年度ページから PDF リンクの収集

各年度ページのテーブルから会議録 PDF へのリンクを抽出する。

**収集方法:**

1. 各年度ページの HTML を取得
2. テーブル内の `<a>` タグから `.pdf` で終わるリンクを抽出
3. 会議種別（定例会/臨時会/特別委員会）と開催日をテーブルの構造から取得
4. メタ情報（会議名、日付、区分）と PDF URL を紐付けて保存

### Step 3: PDF のダウンロードとテキスト抽出

1. 収集した PDF URL から PDF ファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 抽出したテキストから発言者・発言内容をパース

### Step 4: 会議録テキストのパース

PDF から抽出したテキストの構造は PDF の作成方法に依存するため、実際の PDF を複数サンプリングして確認する必要がある。

**確認すべき事項:**

- 発言者名のフォーマット（「○議長」「○XX番」等のパターンがあるか）
- ページヘッダー/フッターの除去方法
- 段組みの有無
- 文字化けの有無

---

## 注意事項

- PDF 形式のため、HTML スクレイピングとは異なり PDF パース処理が必要
- 初期（平成24年頃）と近年でファイル名の命名規則が異なるため、URL パターンの統一的な予測は困難 → 年度ページから実際のリンクを収集する方式が確実
- 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
- PDF のテキスト抽出品質はファイルの作成方法に依存する（スキャン PDF の場合は OCR が必要になる可能性あり）
- YouTube 動画リンクも付随しているが、会議録テキストの取得には PDF を使用する

---

## 推奨アプローチ

1. **2段階クロール**: まずトップページ → 年度ページで全 PDF URL を収集し、次に PDF をダウンロード・パース
2. **PDF サンプリング**: 実装前に各年代の PDF を数件ダウンロードし、テキスト抽出の品質を確認する
3. **メタ情報はHTMLから取得**: 会議名・開催日・会議種別は PDF からではなく、年度ページのテーブル構造から取得する方が確実
4. **レート制限**: リクエスト間に 1〜2 秒の待機時間を設ける
5. **差分更新**: 年度ページの更新を監視し、新規追加された PDF のみをダウンロードする
