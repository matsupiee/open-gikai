# 妹背牛町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.moseushi.hokkaido.jp/gikai/gijiroku/
- 分類: 自治体独自 CMS による HTML + PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: Google Analytics（UA-60397593-1）使用、Google 翻訳ウィジェット搭載

---

## サイト構造

### 2 層構造

1. **一覧ページ**: 全会議録へのリンクを単一ページにリスト表示（ページネーションなし）
2. **詳細ページ**: 各会議の情報と PDF ダウンロードリンクを掲載（会議録本文は PDF のみ）

### 詳細ページの形式

- 平成28年〜現在: HTML 中間ページ（PDF へのリンクを含む）
- 令和元年第2回〜平成29年第3回の一部: 一覧ページから直接 PDF にリンク

## スクレイピング戦略

### Step 1: 一覧ページから会議録リンクを収集

一覧ページ `https://www.town.moseushi.hokkaido.jp/gikai/gijiroku/` をスクレイピングし、全会議録へのリンクを収集する。

- 全件が単一ページに表示されるため、ページネーション処理は不要
- リンクは HTML 詳細ページ（`.html`）または直接 PDF（`.pdf`）の 2 パターン
- リンクテキストから会議名（例: `令和7年第3回定例会`）を抽出

**収集方法:**

1. 一覧ページの HTML を取得
2. 会議録セクション内の `<a>` タグを Cheerio で抽出
3. `.html` リンクと `.pdf` リンクを分類

### Step 2: 詳細ページから PDF リンクを収集

Step 1 で取得した `.html` リンク先の詳細ページにアクセスし、PDF ダウンロードリンクを取得する。

- 各詳細ページには 1〜複数の PDF リンクが含まれる
- PDF リンクは `files/` ディレクトリ配下
- リンクテキストに「第N回妹背牛町議会定例会 第N号」の形式で会議情報が含まれる
- 一部の詳細ページには開催日情報あり（例: `令和3年12月16日、17日`）

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

- PDF は議会会議録の全文を含む
- テキスト抽出には `pdf-parse` 等のライブラリを使用
- 発言者名・発言内容のパースは抽出テキストに対して行う

### Step 4: 会議録のパース

#### メタ情報

一覧ページのリンクテキストおよび詳細ページから以下を抽出:

- 会議名: リンクテキスト（例: `令和7年第3回定例会`）
- 開催日: 詳細ページの記載または PDF ファイル名から推定
- 日目: リンクテキストまたは PDF タイトル（例: `第1号`, `1日目`）

#### PDF ファイル名からの日付推定

```
files/7.9.pdf → 令和7年9月
files/6.9.1.pdf → 令和6年9月1日（推定）
files/31.3.19.pdf → 平成31年3月19日
```

---

## 注意事項

- 会議録本文は全て PDF 形式のため、HTML パースだけでは本文を取得できない
- PDF のレイアウトやフォーマットが年度により異なる可能性がある
- URL パターンが年代によって 3 種類存在するため、リンク収集時に全パターンに対応する必要がある
- 一覧ページから直接 PDF にリンクしているケース（令和元年以前の一部）は、詳細ページを経由しない

---

## 推奨アプローチ

1. **一覧ページ起点**: 単一の一覧ページに全リンクが集約されているため、ここを起点に全量を把握する
2. **PDF 直接取得を優先**: 詳細ページは PDF リンクの中間ページに過ぎないため、最終的に必要なのは PDF URL のリスト
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **ファイル名による差分管理**: PDF ファイル名に年月情報が含まれるため、既取得ファイルとの差分管理が可能
