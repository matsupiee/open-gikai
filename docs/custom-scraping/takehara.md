## PDF ファイル命名規則

PDF ファイル名は年代によって命名規則が異なる。

### 令和年間（令和元年〜）

形式: `{令和年数}-{回数}{会議種別}{月日}.pdf`

| 種別 | ファイル名例 | 意味 |

| 臨時会 | `6-1rinnji1118.pdf` | 令和6年第1回臨時会 11月18日 |

### 平成年間（平成21年〜平成30年）

形式: `{和暦年数2桁}{回数}-{月日}.pdf`

| ファイル名例 | 意味 |

## ページ構造

### 年度別インデックスページ

- 年度別のリンクが一覧表示される
- ページネーションなし（全年度が1ページに表示）
- 各年度名をクリックすると年度別会議録ページへ遷移

### 年度別会議録ページ

- 定例会・臨時会ごとにセクションが分かれている
- 各セクション内に本会議・委員会の PDF リンクが列挙される
- 構成例:
  - `令和6年第4回定例会（12月10日〜12月20日）`
    - 本会議: 各日付の PDF リンク
    - 総務文教委員会: PDF リンク
    - 民生都市建設委員会: PDF リンク
  - `令和6年第1回臨時会（11月18日）`
    - 本会議: PDF リンク

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

1. インデックスページ `index.html` を取得
2. 年度別ページへのリンク（`/gyoseijoho/takeharashigikai/kaiginokekka/2/{ページID}.html`）を全件抽出

### Step 2: PDF リンクの収集

各年度別ページにアクセスし、PDF ファイルへのリンクを抽出する。

```typescript
// PDF リンクの抽出パターン
const pdfLinks = $('a[href$=".pdf"]');
```

- PDF リンクは `/material/files/group/20/{ファイル名}.pdf` 形式
- 絶対 URL に変換して保存

### Step 3: メタ情報の取得

PDF リンクの周辺 HTML テキストから以下を抽出:

- **会議種別**: セクション見出しから「定例会」「臨時会」「総務文教委員会」等を判定
- **開催日**: リンクテキストまたは見出しに含まれる日付情報
- **年度・回数**: セクション見出しから「令和X年第X回」を抽出

```typescript
// セクション見出しからの抽出
const sessionPattern = /(令和|平成)(\d+)年第(\d+)回(定例会|臨時会)/;
// 日付の抽出
const datePattern = /(\d{1,2})月(\d{1,2})日/;
```

### Step 4: PDF テキスト抽出

ダウンロードした PDF からテキストを抽出し、発言内容をパースする。

- PDF のテキスト抽出には `pdf-parse` 等のライブラリを使用
- 発言者・発言内容の構造は PDF 内のテキストレイアウトに依存するため、PDF ごとにフォーマットを確認する必要がある

---

## 注意事項

- PDF ファイル名の命名規則に揺らぎがあるため、ファイル名からのメタ情報取得は補助的に使い、HTML ページの見出し情報を優先すること
- 平成年間と令和年間でファイル名規則が異なる
- 平成21年（2009年）〜令和7年（2025年）の約16年間をカバー
- 委員会の会議録は令和年間のみ掲載されている可能性がある（平成21年の例では本会議のみ）
- ページネーションはなく、全年度が1ページに、各年度の全会議録も1ページに表示される

---

## 推奨アプローチ

1. **全量取得を優先**: インデックスページから全年度ページ URL を取得し、各ページから全 PDF リンクを収集する
2. **2段階クロール**: インデックスページ → 年度別ページ → PDF の2段階で取得する（検索機能がないため全ページ走査が必要）
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
4. **差分更新**: 年度別ページの追加（新しいページ ID の出現）を監視し、新規年度のみを取得する差分更新が可能
5. **PDF テキスト抽出**: PDF からのテキスト抽出が必須のため、OCR 不要な検索可能 PDF であることを前提とするが、古い年度分は画像 PDF の可能性もあるため要確認
