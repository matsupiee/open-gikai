# 枕崎市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.makurazaki.lg.jp/site/gikai/5672.html
- 分類: 市公式サイト内に年度別 PDF として会議録を掲載（専用の会議録検索システムなし）
- 文字コード: UTF-8
- 特記: 会議録インデックスページから年度別サブページへリンクし、各サブページに PDF リンクが列挙される 2 階層構造

## 会議種別

### 本会議

| 種別 | 説明 |
| --- | --- |
| 定例会 | 年4回程度開催（第2・4・5・6回など）。複数日にわたる場合は1つの PDF にまとめられる |
| 臨時会 | 随時開催（第1・3回など）。1日開催が多い |

### 委員会

| 委員会名 | 備考 |
| --- | --- |
| 予算特別委員会 | 各会期ごとに審査結果 PDF と開催日別 PDF が存在する |
| 決算特別委員会 | 主に第4回または第5回定例会に付随して開催 |

※ 古い年度（平成28年以前）には委員会記録 PDF が掲載されていない場合がある。

## PDF の命名規則・URL 構造

PDF は `/uploaded/attachment/{attachment_id}.pdf` 形式で配信される。`attachment_id` は連番だが、年度や会議種別とは直接対応しない（サイト全体の連番）。

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録トップページ `https://www.city.makurazaki.lg.jp/site/gikai/5672.html` を取得し、年度別サブページへのリンク（`/site/gikai/{page_id}.html`）を Cheerio で抽出する。

- リンクテキストは `令和X年分` / `平成X年分` の形式
- 上記の年度別 page_id 一覧は固定値として事前定義しておくことも可能

### Step 2: 各年度ページから PDF リンクを収集

年度別ページを取得し、`/uploaded/attachment/*.pdf` へのリンクを Cheerio で抽出する。

- 各リンクの直前または近傍のテキスト（会議名・開催日）をメタ情報として取得する
- 同一会議の複数日分 PDF がある場合は 1 セッションとして関連付ける

**抽出すべきメタ情報:**

- 会議種別（本会議 / 予算特別委員会 / 決算特別委員会）
- 会議名（例: `第2回定例会`、`第1回臨時会`）
- 開催日または開催期間（例: `2月28日～3月26日`）
- PDF URL

### Step 3: PDF の取得とテキスト抽出

収集した PDF URL をダウンロードし、テキストを抽出する。

- PDF は本会議会議録・委員会会議録を含む
- ファイルサイズは 100KB〜2MB 程度の範囲
- テキスト抽出には `pdf-parse` 等のライブラリを使用する

---

## 注意事項

- **専用検索システムなし**: 全文検索機能は存在しないため、PDF を直接ダウンロードしてテキスト抽出する必要がある
- **PDF のみ**: HTML 形式の会議録は存在せず、すべて PDF として提供される
- **古い年度の情報不足**: 平成28年以前は委員会記録が掲載されていない可能性がある
- **attachment_id の連番**: PDF の attachment_id はサイト全体での連番であり、会議録に限定した予測は困難なため、必ず年度別ページから収集する
- **ページネーションなし**: 各年度ページは 1 ページにすべての PDF リンクが掲載される
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **2 階層クロール**: トップページ → 年度別ページ → PDF の順にクロールする
2. **年度別 page_id の事前定義**: トップページのリンク構造は安定しているが、page_id 一覧を定数として持っておくことでロバスト性が高まる
3. **メタ情報はリンク周辺テキストから取得**: PDF リンクの前後に会議名・日付のテキストが記載されているため、Cheerio でパースして構造化する
4. **差分更新**: 最終取得済み年度の最大 attachment_id を記録し、新規 PDF のみを取得する差分更新が可能
5. **PDF テキスト抽出**: `pdf-parse` 等で全文テキストを抽出してインデックス化する
