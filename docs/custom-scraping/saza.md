## サイト構造

### 3 階層構成

```
会議録トップ (list00807.html)
  └─ 年度別一覧 (list{listID}.html)
       └─ 会議詳細ページ (kiji{kijiID}/index.html)
            └─ PDF ファイル（日ごとに 1 ファイル）
```

### 会議種別

トップページおよび年度別一覧に掲載される会議の種別:

- 定例会（第1回〜第4回）
- 臨時会（第1回〜第N回）
- 全員協議会（第1回〜第N回）
- 決算審査特別委員会

### PDF ファイルの命名規則

```
{c_id}_{kijiID}_{連番}_up_{ランダム文字列}.pdf
```

例: `3_6115_9722_up_j8xommrm.pdf`

- `3`: カテゴリ ID（議会 = 3 で固定）
- `6115`: 記事 ID（kiji の数値部分）
- `9722`: 添付ファイルの連番
- `j8xommrm`: ランダム文字列

### PDF リンクのテキストパターン

リンクテキストに日付と日数が含まれる:

```
12月17日（1日目）（PDF：974.7キロバイト）
12月18日（2日目）（PDF：1010.6キロバイト）
12月19日（3日目）（PDF：480.7キロバイト）
```

## スクレイピング戦略

### Step 1: 年度別の会議一覧を収集

会議録トップページ `list00807.html` から年度別一覧ページへのリンク（`list{listID}.html`）を取得する。

**収集方法:**

1. トップページの `.classArea2` 内にある `list{listID}.html` リンクを抽出
2. 各年度ページにアクセスし、`.kijilist` 内の `kiji{kijiID}/index.html` リンクと会議名テキストを抽出

### Step 2: 各会議の PDF リンクを収集

各会議詳細ページ `kiji{kijiID}/index.html` にアクセスし、PDF ファイルへのリンクを取得する。

**収集方法:**

1. `.danraku` 内の `<a>` タグから `.pdf` で終わる `href` を抽出
2. リンクテキストから開催日（`{月}月{日}日（{N}日目）`）を正規表現で抽出

**抽出用正規表現（案）:**

```typescript
// PDF リンクの抽出
const pdfLinkPattern = /href="([^"]+\.pdf)"/g;

// リンクテキストから日付を抽出
const datePattern = /(\d+)月(\d+)日（(\d+)日目）/;
```

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

- PDF は日ごとに 1 ファイル（1 会議あたり 1〜3 ファイル程度）
- PDF からのテキスト抽出には `pdf-parse` 等のライブラリを使用
- 抽出したテキストから発言者・発言内容をパースする

### Step 4: メタ情報の組み立て

会議詳細ページのタイトルとリンクテキストからメタ情報を構成する:

- **会議名**: ページタイトル（例: `第4回定例会`）
- **開催日**: PDF リンクテキストの日付 + 年度別一覧の年情報から算出
- **自治体**: 佐々町（長崎県）

## 注意事項

- 会議録検索システム（dbsearch や gijiroku.com 等）は未導入のため、PDF ベースでの取得が必要
- listID は連番ではなく、年度が追加されるたびに新しい ID が付与される（新年度が追加された場合はトップページを再クロールする必要がある）
- PDF のリンクテキストには月日のみで年が含まれない → 年度別一覧のページから年を特定する必要がある
- PDF のファイル名にランダム文字列が含まれるため、ファイル名からの情報抽出は不可
- トップページ（`list00807.html`）には年度別一覧に加えて直近の会議リンクも表示されるが、年度別一覧を正とする

---

## 推奨アプローチ

1. **年度一覧ページの全取得を起点とする**: トップページから年度リンクを収集し、各年度ページから会議リンクを収集する 2 段階クロール
2. **PDF テキスト抽出**: HTML 上にはテキスト本文がないため、PDF ダウンロード + テキスト抽出が必須
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 最新年度の一覧ページのみを再クロールし、新しい `kijiID` がある場合のみ PDF を取得する
5. **公開範囲**: 平成28年〜現在（11年度分）が確認されている。それ以前のデータは公開されていない可能性が高い
