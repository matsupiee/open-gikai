## PDF ファイル名パターン

### 現行形式（令和6年度〜）

```
r{和暦年}-{月}t-{日数連番}-{月日}.pdf
```

- 例: `r06-12t-01-1203.pdf` → 令和6年12月会議 第1日目（12月3日）
- 例: `r07-03t-06-0317.pdf` → 令和7年3月会議 第6日目（3月17日）

### 特別委員会

```
r{和暦年}-{種別}-{日数連番}-{月日}.pdf
```

- `k` = 決算審査特別委員会（例: `r06-k-01-0909.pdf`）
- `y` = 予算審査特別委員会（例: `r07-y-01-0310.pdf`）

### 旧形式（令和元年）

```
r{和暦年}-{回数}{種別}-hyoai_{連番}.pdf
```

- `t` = 定例会（例: `r01-08t-hyoai_0001_0001_0001_0001.pdf`）
- `r` = 臨時会（例: `r01-09r-hyoai_0001.pdf`）
- `k` = 決算審査（例: `r01-k-ai_0001_0001_0001.pdf`）

### 発言の構造

発言者パターン:

```
○議長（星　喜美男君）
○町長（佐藤　仁君）
○７番（佐藤正明君）
○９番（村岡賢一君）
○6番（後藤伸太郎君）
○12番（菅原辰雄君）
```

- `○` で始まり、役職 or 議席番号 + 括弧内に氏名 + `君`
- 氏名の姓と名の間にスペースが入る場合がある（`星　喜美男` / `佐藤　仁`）
- 議席番号は全角・半角が混在（`７番` / `6番`）
- 発言内容は発言者行の後に続くテキスト（インデント付き）

### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○議長（星　喜美男君） → role="議長", name="星　喜美男"
// 例: ○７番（佐藤正明君） → role="７番", name="佐藤正明"

// 開催日の抽出（表紙・タイトルページから）
const datePattern = /(?:令和|平成)[０-９\d]+年[０-９\d]+月[０-９\d]+日/;

// 会議名の抽出
const sessionPattern = /(?:令和|平成)[０-９\d]+年度南三陸町議会(.+?)会議録/;
```

## スクレイピング戦略

### Step 1: 年度別ページの URL 収集

会議録一覧ページ `/gikai/minutes/index.html` から年度別ページへのリンクを収集する。

- 一覧はリスト形式（`<ul><li>` 構造）
- 各年度ページの URL パターンは不規則（数値 ID またはファイル名ベース）

### Step 2: PDF リンクの収集

各年度別ページから PDF ファイルへのリンクを全て抽出する。

- PDF は `/material/files/group/16/` 配下に格納
- 見出し（`<h2>`）で会議種別・期間ごとにグループ化されている
- 各 PDF リンクのテキストに開催日が含まれる（例: `3月4日（899.3KB）`）

**収集方法:**

1. 会議録一覧ページから全年度ページの URL を取得
2. 各年度ページの HTML を取得し、`.pdf` で終わるリンクを Cheerio で抽出
3. `<h2>` 見出しテキストから会議種別を特定し、PDF に紐付ける

### Step 3: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF は MS-Mincho フォント使用、FlateDecode 圧縮
- `pdf-parse` 等のライブラリでテキスト抽出可能
- 1ファイルあたり数十ページ（例: 47ページ）

### Step 4: 会議録のパース

抽出したテキストから発言者と発言内容を分離する。

1. 表紙・タイトルページからメタ情報（年度、会議種別、開催日）を抽出
2. 出席議員リスト等のメタ情報セクションをスキップ
3. `○` で始まる行を発言者の区切りとして認識
4. 発言者行から役職・氏名を正規表現で抽出
5. 次の発言者行までのテキストを発言内容として取得

---

## 注意事項

- PDF 形式のため、テキスト抽出の精度に依存する（レイアウトによっては文字化けや改行位置のズレが発生する可能性）
- 議席番号の全角・半角が混在しているため、正規化が必要
- 氏名のスペース（全角スペース）の有無が統一されていない
- 旧形式（令和元年以前）と現行形式でファイル名パターンが異なる
- 1つの PDF に複数日分の会議録が含まれるケースがある（表紙に開会〜散会の期間が記載）
- ただし各 PDF は基本的に1日分（ファイル名に日数連番と月日が含まれる）

---

## 推奨アプローチ

1. **HTML ページから PDF URL を収集**: 年度別ページを巡回し、全 PDF リンクを収集（ページ数が少ないため全量取得が容易）
2. **PDF テキスト抽出**: `pdf-parse` や `pdfjs-dist` でテキストを抽出
3. **発言者パターンの `○` を区切りに分割**: PDF テキストを `○` で始まる行で分割し、発言単位に変換
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別ページの PDF リンク数を前回取得時と比較し、新規 PDF のみをダウンロード
