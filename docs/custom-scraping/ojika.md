## スクレイピング戦略

### Step 1: 会議録一覧ページから PDF リンクの収集

会議録一覧ページ `https://www.town.ojika.lg.jp/gyosei_joho/5/5/1/373.html` から PDF へのリンクを抽出する。

**収集方法:**

1. 一覧ページの HTML を取得
2. `<a href="...pdf">` リンクを Cheerio で抽出
3. `https://www.town.ojika.lg.jp/material/files/group/9/` 配下の PDF リンクを収集
4. ページネーションがある場合は全ページを走査

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、`pdf-parse` 等のライブラリでテキストを抽出する。

- PDF は構造化タグ付き（PDF 1.7 形式）のため、テキスト抽出の精度は比較的高い
- フォントは MS-Mincho（明朝体）と Century を使用

### Step 3: メタ情報のパース

PDF テキストから以下のメタ情報を抽出する:

#### 会議名・日目

```typescript
// 会議名の抽出
const meetingPattern = /小値賀町議会(定例|臨時)?(\S+?)会議(?:（(\S+?)）)?/;
// 例: "小値賀町議会定例９月会議（１日目）" → type="定例", name="９月", day="１日目"
```

#### 開催日

```typescript
// 開催日の抽出（議事日程セクションから）
const datePattern = /令和(\d+)年(\d+)月(\d+)日（(.+?)）/;
// 例: "令和６年９月１０日（火曜日）" → year=6, month=9, day=10
```

#### 出席議員

```typescript
// 出席議員数
const attendancePattern = /出\s*席\s*議\s*員\s+(\d+)\s*名/;
// 議員名（番号 + 氏名）
const memberPattern = /(\d+)\s+番\s+(.+)/;
```

### Step 4: 発言内容のパース

PDF から抽出したテキストの発言部分をパースする。小値賀町の会議録は小規模議会のため、発言者の区切りパターンは以下が想定される:

```typescript
// 議長・議員の発言パターン（PDF テキスト抽出後）
const speakerPattern = /^[○◯]?\s*(\S+?)(?:議長|議員|町長|副町長|教育長|課長|所長|局長)?\s*[（(](.+?)[)）]/;

// 議事日程の項目
const agendaPattern = /^第\s*(\d+)\s+(.+)/;
```

## 注意事項

- 旧サイト `ojika.net/gikairoku/` は閉鎖済みのため、新サイト `town.ojika.lg.jp` のみを対象とする
- PDF ファイル名がハッシュ値の場合、ファイル名から会議内容を推測できないため、一覧ページのリンクテキストとの紐付けが必要
- PDF は Word 由来のため、テキスト抽出時にスペースの挿入や改行位置にばらつきがある可能性がある
- 議員定数 8 名の小規模議会のため、会議録の分量は比較的少ない（1 回あたり数十ページ程度）

---

## 推奨アプローチ

1. **新サイトを正とする**: 旧サイト(ojika.net)は閉鎖済みのため、`town.ojika.lg.jp` の会議録一覧ページをエントリーポイントとする
2. **PDF テキスト抽出**: `pdf-parse` や `pdfjs-dist` を使用して PDF からテキストを抽出する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 一覧ページの PDF リンクを前回取得分と比較し、新規追加分のみダウンロード・パースする
5. **ファイル名の正規化**: ハッシュ値ファイル名の PDF は、一覧ページのリンクテキスト（会議名・日付）をメタデータとして保持する
