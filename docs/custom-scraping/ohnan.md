## PDF リンクテキストのパターン

### 令和元年〜（年度別ページ）

リンクテキスト形式:

```
令和６年第１０回定例会（第５号）１２月１３日（PDF文書／380KB）
令和６年第９回臨時会　１１月１１日　（PDF文書／264KB）
```

- 年号: `令和X年` / `平成XX年`
- 回数: `第X回`
- 会議種別: `定例会` / `臨時会`
- 号数（定例会のみ）: `（第X号）`
- 開催日: `XX月XX日`（全角数字）
- ファイルサイズ: `（PDF文書／XXXKB）`

### 平成20年〜28年（過去の会議録ページ）

リンクテキスト形式:

```
平成28年第9回定例会第5日目12月16日（PDF／240KB）
平成28年第8回臨時会11月11日（PDF／223KB）
平成28年第9回定例会一般質問事項（PDF／90KB）
```

- 号数の代わりに `第X日目` 表記
- 半角数字が使われる場合がある
- 一般質問事項の PDF も含まれる

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録トップページ `/www/genre/1000100000201/index.html` から各年度ページへのリンクを取得する。

- `ul.list00` 内の `<a>` タグから `/www/contents/{ID}/index.html` 形式のリンクを抽出
- 「過去の会議録」ページも含めて収集

### Step 2: PDF リンクの収集

各年度ページから PDF ファイルへのリンクとリンクテキスト（メタ情報）を取得する。

**令和元年〜:**
- `div.download-item` 内の `ul.list00 > li > a[href$=".pdf"]` を抽出
- リンクテキストから年度・回数・会議種別・号数・開催日を正規表現で抽出

**平成20年〜28年:**
- `p > a[href$=".pdf"]` を抽出
- リンクテキストのパターンが異なるため別の正規表現が必要

### Step 3: メタ情報のパース

リンクテキストから以下を抽出:

```typescript
// 令和元年〜のパターン
const metaPattern = /^(令和|平成)([\d１２３４５６７８９０]+)年第([\d１２３４５６７８９０]+)回(定例会|臨時会)(?:（第([\d１２３４５６７８９０]+)号）)?[　\s]*([\d１２３４５６７８９０]+)月([\d１２３４５６７８９０]+)日/;

// 平成20年〜28年のパターン
const pastMetaPattern = /^(平成)(\d+)年第(\d+)回(定例会|臨時会)(?:第(\d+)日目)?(\d+)月(\d+)日/;
```

### Step 4: PDF のダウンロードとテキスト抽出

- PDF ファイルをダウンロード
- PDF パーサー（pdf-parse 等）でテキストを抽出
- PDF 内の発言者・発言内容のパースは PDF のレイアウトに依存するため、実際の PDF 内容を確認して対応が必要

---

## 注意事項

- 会議録は PDF のみで HTML 本文は存在しない。PDF からのテキスト抽出が必須
- PDF ファイル名の命名規則に表記揺れが多い（`kaigiroku` / `gijiroku` / `kaigirou` など）。ファイル名からのメタ情報抽出は不安定なため、リンクテキストからの抽出を推奨
- 「※ここに掲載している会議録は町議会の公式の記録ではありません。」という注記がある
- 一般質問事項の PDF（`bouchou.pdf` / `boutyou.pdf`）は会議録本文ではないためスキップ対象
- ページネーションなし。各年度ページに当該年の全 PDF が一覧表示される

---

## 推奨アプローチ

1. **2段階クロール**: トップページ → 年度ページ → PDF リンク収集の2段階で全 PDF URL を取得
2. **リンクテキストからメタ情報抽出**: ファイル名ではなくリンクテキストを正規表現でパースして会議種別・開催日を取得
3. **PDF テキスト抽出**: `pdf-parse` 等のライブラリで PDF からテキストを抽出し、発言者・発言内容をパース
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度ページの `最終更新日` を確認し、更新があった年度のみ再取得する
