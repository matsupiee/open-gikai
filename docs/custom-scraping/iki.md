# 壱岐市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.iki.nagasaki.jp/soshiki/gikai_jimukyoku/shigikai/kaigiroku/index.html
- 分類: SMART CMS による PDF 公開（既存アダプターでは対応不可）

## PDF ファイル名パターン

PDF ファイル名の命名規則は時期によって異なる。以下は令和6年の例:

### パターン A（令和6年 前半: 1月〜2月会議）

| 種別 | ファイル名例 |
| --- | --- |
| 目次 | `0601gmokuji.pdf` |
| 審議期間日程 | `0601gsingi.pdf` |
| 会議録（第N号） | `0601g1gou.pdf` |
| 一括ダウンロード | `0601gikkatu.pdf` |

形式: `{YY}{MM}g{種別}.pdf`（YY=年下2桁、MM=月）

### パターン B（令和6年 中盤: 5月〜9月会議）

| 種別 | ファイル名例 |
| --- | --- |
| 目次 | `mokuji0605.pdf` |
| 審議期間日程 | `singikikannittei0605.pdf` |
| 会議録（第N号） | `kaigiroku060605.pdf` / `kaigiroku06060613.pdf` |
| 資料 | `siryou0605.pdf` |
| 一括ダウンロード | `ikkatu0605.pdf` |

形式: `{種別}{YYMM}.pdf` または `{種別}{YYMMDD}.pdf`

### パターン C（令和6年 後半: 12月会議）

| 種別 | ファイル名例 |
| --- | --- |
| 目次 | `1mokuji202412.pdf` |
| 審議期間日程 | `2sinngikikann202412.pdf` |
| 会議録（第N号） | `3dai1gou202412.pdf` |
| 資料 | `9siryou202412.pdf` |
| 一括ダウンロード | `10ikkatu202412.pdf` |

形式: `{連番}{種別}{YYYYMM}.pdf`

※ ファイル名の命名規則に一貫性がないため、正規表現での推測は困難。HTML ページからリンクを直接抽出する必要がある。

## ページ構造

### 年度一覧ページ（index.html）

- 各年度へのリンクがリスト形式で掲載
- リンクテキスト: `{和暦}年本会議 会議録（PDFファイル）`
- 最新年度が先頭、平成16年まで遡及（約20年分）

### 年度別ページ（{ページID}.html）

- h3 見出しで定例会・臨時会の月別会議を区分
- 各会議の下にリスト形式で PDF リンクを掲載
- PDF の種類: 目次、審議期間日程、第N号（会議録本体）、資料、一括ダウンロード

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

年度一覧ページ `index.html` から各年度ページへのリンクを抽出する。

- `<a>` タグの `href` 属性から `/soshiki/gikai_jimukyoku/shigikai/kaigiroku/{ページID}.html` パターンを収集
- リンクテキストから年度情報（和暦）を取得

### Step 2: 会議録 PDF リンクの収集

各年度ページから会議録本体の PDF リンクを抽出する。

**収集方法:**

1. 各年度ページの HTML を取得
2. h3 見出しから会議名（定例会○月会議 等）を取得
3. 各会議セクション内の `<a>` タグから PDF リンクを抽出
4. `material/files/group/23/` を含むリンクのみを対象とする
5. リンクテキストから文書種別（目次・第N号・一括ダウンロード等）を判別

**対象の絞り込み:**

- 会議録本体: リンクテキストに「第○号」を含むもの
- 目次・審議期間日程・資料・一括ダウンロードは除外可能（必要に応じて取得）

### Step 3: PDF のダウンロードとテキスト抽出

1. 収集した PDF URL からファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 抽出したテキストからメタ情報と発言内容をパース

### Step 4: メタ情報の抽出

以下の情報を抽出:

- **開催日**: リンクテキストの括弧内（例: `第1号(6月13日)`）から取得。年度は年度ページから補完
- **会議名**: h3 見出しから取得（例: `定例会6月会議`）
- **号数**: リンクテキストから取得（例: `第1号` → 1）

---

## 注意事項

- **PDF 形式のみ**: HTML での会議録公開はなく、全て PDF ファイル。テキスト抽出には PDF パーサーが必要
- **ファイル名の不統一**: 年度・時期によってファイル名の命名規則が異なるため、URL のパターンマッチではなく HTML からのリンク抽出が必須
- **本会議のみ**: 現時点で確認できるのは本会議の会議録のみ。委員会の会議録は別ページまたは未公開の可能性がある
- **SMART CMS**: 壱岐市は SMART CMS（smart-lgov.jp）を使用。CMS の仕様変更によりページ構造が変わる可能性がある
- **PDF 内の文字エンコーディング**: 古い PDF では文字化けやテキスト抽出不能の可能性がある。事前に数件のサンプルで抽出品質を確認すること

---

## 推奨アプローチ

1. **2段階クロール**: まず index.html から年度ページ URL を収集し、次に各年度ページから PDF リンクを収集する
2. **リンクテキストからメタ情報を取得**: PDF 内のパースに頼らず、HTML 上のリンクテキストと見出しから開催日・会議名を取得する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度一覧ページの新規リンク有無を確認し、未取得の年度ページのみをクロールする
5. **PDF テキスト抽出の前処理**: PDF から抽出したテキストは改行位置が不規則になりやすいため、発言者の区切りや段落の結合に注意する
