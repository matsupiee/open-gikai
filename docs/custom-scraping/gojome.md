# 五城目町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.gojome.akita.jp/town/gikai/kaigiroku/
- 分類: 自治体 CMS による年度別ページ + PDF 公開（検索機能なし）
- 文字コード: UTF-8
- 特記: 会議録は PDF ファイルとして公開されており、HTML 上の本文テキストは存在しない

---

| ページ | URL パターン |

| 年度別一覧（令和4年） | `https://www.town.gojome.akita.jp/town/gikai/kaigiroku/2060` |

## 検索パラメータ

検索機能は提供されていない。年度別の一覧ページから手動で PDF をたどる形式。

## サイト構造

### 年度別一覧ページ

トップページに令和4年〜令和7年の年度リンクが掲載されている。各年度ページでは以下の構成で会議録が整理されている。

- `<h3>` 見出しで会議名（例: 「令和7年第4回定例会」「令和7年第3回臨時会」）
- `<ul><li>` リストで各 PDF へのリンク

### 会議の種類

| 種類 | 開催時期 | 回数 |
| --- | --- | --- |
| 定例会 | 3月、6月、9月、12月 | 年4回 |
| 臨時会 | 不定期 | 年2〜3回 |

### PDF の分類

各会議につき以下の PDF が公開される:

1. **会期日程**: 会議のスケジュール
2. **第○号（○月○日）**: 本会議の日ごとの会議録
3. **一般質問（議員別）**: 議員ごとの一般質問会議録

## PDF ファイル名の命名規則

ファイル名の命名規則は統一されておらず、年度により異なる。

### パターン 1: 日本語ファイル名（URL エンコード済み）

```
第４回定例会 会期日程.pdf
第４回定例会 第1号（12月7日）.pdf
12月定例会議事日程.pdf
```

### パターン 2: ローマ字ファイル名（一般質問）

```
1_itounobuko.pdf        → 1番目の一般質問（伊藤信子議員）
6_kudoumasahiko.pdf     → 6番目の一般質問（工藤政彦議員）
```

### パターン 3: 日本語 + 議員名

```
1_椎名志保議員.pdf
令和５年３月０１斎藤晋議員.pdf
```

## スクレイピング戦略

### Step 1: 年度別一覧ページの URL 収集

トップページ `https://www.town.gojome.akita.jp/town/gikai/kaigiroku/` から年度別一覧ページへのリンクを取得する。

- `<a>` タグの `href` 属性から `/town/gikai/kaigiroku/{ID}` パターンのリンクを抽出
- 現在は令和4年〜令和7年の4ページ

**収集方法:**

1. トップページを取得
2. `/town/gikai/kaigiroku/\d+` にマッチするリンクを抽出
3. 各年度ページの URL リストを生成

### Step 2: PDF リンクの収集

各年度別一覧ページから PDF ファイルへのリンクを収集する。

- `<a>` タグの `href` 属性から `.pdf` で終わるリンクを抽出
- リンクテキストから会議名、日付、議員名などのメタ情報を推測
- `<h3>` 見出しから所属する会議名（定例会 / 臨時会）を特定

**収集方法:**

1. 各年度ページの HTML を取得
2. `<h3>` 見出しをパースして会議名を取得
3. 各見出し配下の `<ul><li>` から PDF リンクを抽出
4. 見出しテキストと PDF リンクテキストを紐付けて保存

### Step 3: PDF のダウンロードとテキスト抽出

PDF ファイルをダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）でテキストを抽出
- 抽出したテキストから発言者・発言内容をパースする

### Step 4: 会議録のパース

#### メタ情報

`<h3>` 見出しから以下を抽出:

```
令和７年第４回定例会
令和７年第３回臨時会
```

- 年度: `令和X年` 形式
- 会議種別: `第○回定例会` or `第○回臨時会`

PDF リンクテキストから日付を抽出:

```
第1号（12月8日）
2日目（12月6日）
```

#### 発言の構造（PDF 内）

PDF 内のテキスト構造は PDF の生成方法に依存するため、実際の PDF を取得して確認が必要。一般的な議会会議録 PDF では以下のパターンが想定される:

```
○議長（○○○○）
  発言内容...

○12番（○○○○）
  発言内容...
```

---

## 注意事項

- 検索機能がないため、全量取得が前提となる
- PDF ファイル名の命名規則が年度ごとに異なるため、ファイル名からのメタ情報抽出は補助的に使用する
- PDF のテキスト抽出精度は PDF の生成方法に依存する（スキャン PDF の場合は OCR が必要）
- 年度別一覧ページのコンテンツ ID は連番ではないため、新しい年度のページが追加された場合はトップページから再取得が必要
- ページネーションはなし（各年度ページに全会議分のリンクが掲載される）

---

## 推奨アプローチ

1. **トップページ起点でクロール**: トップページから年度ページを列挙し、各年度ページから PDF リンクを収集する2段階方式
2. **`<h3>` 見出しとの紐付け**: PDF リンクだけでなく、所属する会議名（見出し）を併せて保存することでメタ情報を補完
3. **PDF テキスト抽出**: pdf-parse 等を使い PDF からテキストを抽出。スキャン PDF の場合は OCR（Tesseract 等）を検討
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別ページの PDF リンク数を前回と比較し、新規追加分のみダウンロードする
