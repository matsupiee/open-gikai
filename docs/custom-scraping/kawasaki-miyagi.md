## 公開形式の違い（年度による分岐）

### 令和3年（2021年）以降: PDF 公開

年度別ページに `<h2>` で定例会ごとの見出し、その下に PDF リンクのリストが掲載される。

```html
<h2>令和7年3月定例会</h2>
<ul>
  <li><a href="/uploaded/attachment/8930.pdf">令和7年3月5日（第2号）</a> [PDFファイル／442KB]</li>
  <li><a href="/uploaded/attachment/8931.pdf">令和7年3月6日（第3号）</a> [PDFファイル／XXkb]</li>
</ul>
```

### 令和2年（2020年）以前: HTML 直接公開

会議ごとの個別ページに、会議録本文が HTML としてそのまま掲載される。PDF リンクはない。

平成29年（2017年）〜 令和7年（2025年）

## 会議種別

定例会のみが掲載されている（臨時会の会議録は未確認）。年4回（3月・6月・9月・12月）の定例会が主体。

各定例会は複数日にわたり、日ごとに会議号数（第1号、第2号、第3号など）が付与される。

## PDF リンクテキストのフォーマット

```
令和{N}年{M}月{D}日（第{X}号）
```

例:
- `令和7年3月5日（第2号）`
- `令和4年12月7日（第2号）`

## HTML 会議録の構造（令和2年以前）

### メタ情報

ページ冒頭に以下の情報が含まれる:

- 出席議員一覧（番号、氏名）
- 説明のため出席した者（職位、氏名）
- 議事日程

### 発言者パターン

```
○議長（眞壁範幸君）
○町長（小山修作君）
○3番（佐藤昭光君）
○教育長（相原稔彦君）
○保健福祉課長（佐藤和彦君）
```

- `○` で始まり、役職 or 議員番号 + 括弧内に氏名 + `君` の敬称
- 登壇表記: `【町長　小山修作君　登壇】`

### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○3番（佐藤昭光君） → role="3番", name="佐藤昭光君"

// 登壇の検出
const podiumPattern = /^【(.+?)　(.+?)　登壇】/;

// 開催日の抽出（PDF リンクテキストから）
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;
```

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録一覧ページ `list6-25.html` から、年度別ページへのリンクをすべて収集する。

- リンクは `/site/gikai/{ページID}.html` 形式
- 全リンクが単一ページ内に存在するため、ページネーション処理は不要

### Step 2: 公開形式の判定と分岐

各年度別ページを取得し、PDF リンクの有無で処理を分岐する。

- **PDF リンクあり（令和3年以降）**: Step 3a へ
- **PDF リンクなし・HTML 本文あり（令和2年以前）**: Step 3b へ

判定方法:

```typescript
const pdfLinks = $('a[href$=".pdf"]');
if (pdfLinks.length > 0) {
  // PDF 公開パターン
} else {
  // HTML 直接公開パターン
}
```

### Step 3a: PDF の取得・テキスト抽出（令和3年以降）

1. 年度別ページから PDF の `<a href>` を収集
2. リンクテキストから開催日・会議号数を抽出
3. `<h2>` 見出しから定例会名を取得（例: `令和7年3月定例会`）
4. PDF をダウンロードし、テキスト抽出ツール（pdf-parse 等）でテキスト化

### Step 3b: HTML 本文のパース（令和2年以前）

1. ページ本文から発言者パターン `○{役職}（{氏名}）` を検出して発言を分割
2. メタ情報（出席議員、議事日程）を冒頭から抽出
3. リンクテキストまたはページ内容から開催日を取得

---

## 注意事項

- **公開形式の混在**: 令和3年を境に PDF 公開と HTML 直接公開が混在しているため、両方のパーサーを実装する必要がある
- **ページ ID の不規則性**: 年度別ページの ID（`10823.html`、`455.html` 等）に規則性がなく、URL を推測できない。必ず一覧ページからリンクを収集する
- **令和2年以前の分割**: 令和2年以前は会議ごとに個別ページとなっており、1年分を取得するには複数ページへのアクセスが必要
- **敬称の統一**: HTML 会議録では氏名に `君` の敬称が付く（例: `佐藤昭光君`）。氏名抽出時に敬称を除去する処理が必要
- **全角数字**: 議員番号やその他の数字に全角が使われる場合がある

---

## 推奨アプローチ

1. **一覧ページを起点にする**: `list6-25.html` の 1 リクエストで全年度ページの URL を収集できる
2. **公開形式で分岐**: PDF の有無を判定し、適切なパーサーに振り分ける
3. **PDF 優先**: 令和3年以降の PDF 公開分を先に対応し、HTML 直接公開分は後回しにしてもよい
4. **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒の待機時間を設ける
5. **差分更新**: 既取得 URL のリストと比較し、新規 URL のみを取得する
