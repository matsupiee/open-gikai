# 嬬恋村議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.vill.tsumagoi.gunma.jp/www/contents/1000000000443/index.html
- 分類: 自治体公式サイト上で PDF を直接公開（会議録検索システムなし）
- 文字コード: PDF（テキスト埋め込み済み）
- 特記: 平成28年（2016年）以降の定例会・臨時会の会議録を PDF で公開。1定例会/臨時会につき1ファイル（複数日分を含む）

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 議会トップ | `https://www.vill.tsumagoi.gunma.jp/www/contents/1000000000443/index.html` |
| 会議録 PDF | `https://www.vill.tsumagoi.gunma.jp/www/contents/1000000000443/simple/{ファイル名}.pdf` |

### PDF ファイル名の規則

ファイル名に統一的な命名規則はない。年度・回次・月を組み合わせた不規則なパターンが使われている。

例:
- `28.6.pdf` — 平成28年第4回定例会（6月）
- `29.3.pdf` — 平成29年第1回定例会（3月）
- `R4.1.pdf` — 令和4年第1回臨時会（2月）
- `5.3t.pdf` — 令和5年第1回定例会（3月）、`t` = 定例会
- `5.5r.pdf` — 令和5年第2回臨時会（5月）、`r` = 臨時会
- `7.3.pdf` — 令和7年第3回定例会（6月）

**注意:** ファイル名からは会議の種別・日付を正確に特定できない。HTML ページ上のリンクテキストからメタ情報を取得する必要がある。

## スクレイピング戦略

### Step 1: PDF リンクの収集

議会トップページ（`index.html`）から全 PDF リンクとそのリンクテキストを取得する。

- ページネーションなし。1ページに全年度の PDF リンクが掲載されている
- リンクテキストから会議種別（定例会/臨時会）、回次、開催月を抽出する
- リンクテキスト例: `第１回定例会（３月）`、`第２回臨時会（５月）`

**収集方法:**

1. `index.html` を取得し、Cheerio で `/simple/.*\.pdf` パターンのリンクを抽出
2. 各リンクの周辺テキスト（年度見出し + リンクテキスト）からメタ情報を構成
3. 年度見出し例: `平成28年`、`令和7年`

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF はテキスト埋め込み済み（OCR 不要）
- 1ファイルに複数日の会議録が含まれる（例: 123ページ、3日分）
- `pdf-parse` や `pdf.js` 等でテキスト抽出が可能

### Step 3: 会議録のパース

#### PDF 全体構造

1. **表紙**: 会議名、開会日・閉会日
2. **目次**: 各号（日）ごとの議事日程・セクション一覧
3. **各号の本文**: 議事日程 → 出席議員 → 発言記録

#### メタ情報

表紙から以下を抽出:

```
令和７年第３回定例会
嬬恋村議会会議録
令和７年６月３日　開会
令和７年６月13日　閉会
```

- 会期名: `{元号}年第{N}回{定例会|臨時会}`
- 開会日・閉会日: `{元号}年{月}月{日}日`

各号の冒頭から:

```
議事日程（第１号）
令和７年６月３日（火）午前10時02分開会
```

- 号数: `第{N}号`
- 開催日: `{元号}年{月}月{日}日（{曜日}）`
- 開始時刻: `午前{時}時{分}分開会`

#### 出席議員

```
出席議員（１２名）
  1番　黒岩智未君　　2番　土屋哲夫君
  3番　伊東正吾君　　4番　下谷彰一君
```

- `{番号}番　{氏名}君` の形式
- 議員定数 12名

#### 発言者パターン

```
○議長（佐藤鈴江君）
○村長（熊川　栄君）
○副村長（黒岩　彰君）
○議会運営委員長（土屋幸雄君）
○交流推進課長（小林千速君）
○３番（伊東正吾君）
```

- `○` で始まり、役職または番号 + 括弧内に氏名
- 役職パターン: `議長`、`村長`、`副村長`、`{委員会名}委員長`、`{課名}課長`
- 議員パターン: `{N}番`（半角または全角数字）

#### 登壇表記

```
〔村長　熊川　栄君登壇〕
〔３番　伊東正吾君登壇〕
〔交流推進課長　小林千速君登壇〕
〔議会運営委員長　土屋幸雄君登壇〕
```

- `〔{役職/番号}　{氏名}登壇〕` の形式

#### 議場動作の表記

```
〔「異議なし」と呼ぶ者あり〕
〔賛成者起立〕
〔発言する者なし〕
```

- `〔...〕` で括られる

#### セクション見出し

```
◎開会及び開議の宣告
◎議事日程の報告
◎一般質問
◎行政報告
```

- `◎` で始まるセクション見出し

#### 一般質問の区切り

```
◇ 伊　東　正　吾　君
```

- `◇` で始まり、氏名がスペース区切りで表示される

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○議長（佐藤鈴江君） → role="議長", name="佐藤鈴江"
// 例: ○３番（伊東正吾君） → role="３番", name="伊東正吾"

// 登壇の検出
const podiumPattern = /^〔(.+?)登壇〕/;

// セクション見出し
const sectionPattern = /^◎(.+)/;

// 一般質問者の区切り
const questionerPattern = /^◇\s*(.+?)\s*君/;

// 開催日・時刻の抽出
const dateTimePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日.+?(午前|午後)(\d+)時(\d+)分/;

// 議場動作
const actionPattern = /^〔(.+?)〕$/;
```

## 注意事項

- PDF ファイル名に統一的な命名規則がないため、ファイル名からメタ情報を推定することはできない。必ず HTML ページのリンクテキストと PDF 内の表紙情報を使う
- 1つの PDF に複数日（号）の会議録が含まれるため、号ごとに分割してパースする必要がある
- 氏名に全角スペースが含まれる場合がある（例: `熊川　栄`、`黒岩　彰`）
- 全角数字が使われている箇所がある（議員番号、日付など）
- PDF のテキスト抽出時に、ページ番号（`−1−` 形式）が混入するため除去が必要
- 定例会は年4回（3月・6月・9月・12月）、臨時会は不定期に開催

---

## 推奨アプローチ

1. **HTML ページからリンク収集を優先**: `index.html` から全 PDF リンクとリンクテキスト（年度・会議種別）を一括取得する
2. **PDF テキスト抽出**: `pdf-parse` 等でテキストを抽出し、表紙・目次・本文を分離する
3. **号ごとの分割**: 目次と `議事日程（第N号）` パターンでPDF内を号単位に分割する
4. **レート制限**: 自治体サイトのため、PDF ダウンロード時にリクエスト間に適切な待機時間（1〜2秒）を設ける
5. **差分更新**: HTML ページのリンク一覧を前回取得分と比較し、新規追加分の PDF のみダウンロードする
