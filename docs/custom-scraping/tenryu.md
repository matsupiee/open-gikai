# 天龍村議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.vill-tenryu.jp/category/notice/administrative/government_info/parliament/
- 分類: WordPress サイトで議会活動記録 PDF を公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録は PDF 形式のみで提供。HTML 本文としての議事録テキストは存在しない

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 議会一覧（カテゴリページ） | `https://www.vill-tenryu.jp/category/notice/administrative/government_info/parliament/` |
| 議会活動記録（個別ページ） | `https://www.vill-tenryu.jp/notice/administrative/{日本語タイトル（URLエンコード済み）}/` |
| 議会活動記録 PDF | `https://www.vill-tenryu.jp/wp-content/uploads/{年}/{月}/{ハッシュ}.pdf` |

### ページ種別

一覧ページには以下の 3 種類のコンテンツが混在する:

| 種別 | リンクテキスト例 | 内容 |
| --- | --- | --- |
| 議会活動記録 | `令和８年２月天龍村議会活動記録` | 月次の議会活動をまとめた PDF を掲載 |
| 定例議会・臨時議会 | `令和８年３月　第１回定例議会` | 開催案内（会期・傍聴案内等）。議案概要を HTML で表示 |
| 議会だより | `令和８年１月天龍村議会だより` | 議会だより PDF を掲載 |

## サイト構造の特徴

### 一覧ページ

- WordPress のカテゴリアーカイブページ
- ページネーションなし（全件が単一ページに表示）
- 記事リンクは `<ul>` > `<li>` > `<a>` のリスト構造
- リンク URL は日本語タイトルを URL エンコードしたスラッグ

### 個別ページ（議会活動記録）

- `<h1>` タグにタイトル（例: `令和７年２月天龍村議会活動記録`）
- PDF サムネイル画像 + PDF ダウンロードリンク
- PDF URL: `/wp-content/uploads/{年}/{月}/{ハッシュ}.pdf`
- リンクテキスト: `{タイトル}（PDF）`

### 個別ページ（定例議会・臨時議会）

- 開催案内（会期、傍聴定員等）を HTML で表示
- 議案一覧・一般質問の概要が含まれる場合あり
- PDF リンクは含まれない場合が多い

### 個別ページ（議会だより）

- 議会だより PDF のダウンロードリンクを掲載
- PDF URL: `/wp-content/uploads/{年}/{月}/{ハッシュ}.pdf`

## スクレイピング戦略

### Step 1: 一覧ページから記事リンクを収集

カテゴリページ `https://www.vill-tenryu.jp/category/notice/administrative/government_info/parliament/` から全記事リンクを取得する。

- `<a>` タグの `href` 属性からリンク URL を抽出
- リンクテキストで「議会活動記録」を含むものをフィルタリング（「議会だより」「定例議会」「臨時議会」は除外可能）
- ページネーションなしのため、1 回のリクエストで全件取得可能

**収集対象の判別:**

```typescript
// 議会活動記録のみを対象とする
const isActivityRecord = (text: string) =>
  text.includes("議会活動記録");

// 定例議会・臨時議会（開催案内）は除外
const isSessionNotice = (text: string) =>
  /第\d+回(定例|臨時)議会/.test(text);

// 議会だよりは除外
const isNewsletter = (text: string) =>
  text.includes("議会だより");
```

### Step 2: 各記事ページから PDF URL を取得

各議会活動記録ページにアクセスし、PDF ダウンロードリンクを抽出する。

- `<a>` タグの `href` 属性が `.pdf` で終わるリンクを取得
- PDF URL パターン: `/wp-content/uploads/{年}/{月}/{ハッシュ}.pdf`

### Step 3: PDF のダウンロードとテキスト抽出

取得した PDF URL から PDF ファイルをダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）でテキストを抽出
- 議会活動記録 PDF の内容構造に応じてパースロジックを実装

### メタ情報の抽出

記事ページの `<h1>` タグまたはリンクテキストからメタ情報を抽出する。

```typescript
// タイトルから年月を抽出
const titlePattern = /(?:令和|平成)(\d+)年(\d+)月天龍村議会活動記録/;
// 例: "令和７年２月天龍村議会活動記録" → 年=7, 月=2
```

## 注意事項

- 会議録本文は PDF のみで提供されており、HTML でのテキスト取得は不可
- PDF のファイル名はハッシュ値のため、URL から内容を推測できない
- 一覧ページの URL パスに日本語（URL エンコード済み）が含まれるため、全角・半角の揺れに注意が必要
- WordPress テーマ `tenryu` を使用。jQuery、Fancybox を読み込み
- 定例議会・臨時議会のページは開催案内が主で、詳細な議事録は議会活動記録 PDF に集約されている

---

## 推奨アプローチ

1. **議会活動記録に絞って取得**: 一覧ページから「議会活動記録」を含むリンクのみを対象とする
2. **PDF テキスト抽出**: HTML 本文がないため、PDF ダウンロード → テキスト抽出のパイプラインが必須
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 一覧ページのリンク数を前回取得時と比較し、新規追加分のみを取得する
