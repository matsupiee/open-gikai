# 松前町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.matsumae.hokkaido.jp/hotnews/detail/00000317.html
- 分類: PDF 形式での公開（単一ページに全年度分の PDF リンクを掲載）
- 文字コード: UTF-8
- 対象期間: 平成25年（2013年）〜令和7年（2025年）
- 特記: 検索システムなし。1ページに全会議録 PDF へのリンクが一括掲載されている

| ページ | URL パターン |

## PDF ファイル命名規則

ファイル名の命名規則は年度によって揺れがあり、統一されていない。

### 基本パターン（令和元年以降）

`{年号下2桁}_{回数}{会議種別}.pdf`

例:
- `07_1tei_kaigiroku.pdf` → 令和7年 第1回定例会
- `07_1rinji_kaigiroku.pdf` → 令和7年 第1回臨時会
- `07_toku_yosan.pdf` → 令和7年 予算審査特別委員会
- `07_toku_kessan.pdf` → 令和7年 決算審査特別委員会

### 会議種別の略称

| 略称 | 会議種別 |
| --- | --- |
| `tei` / `teirei` | 定例会 |
| `rinji` / `rinzi` / `rin` | 臨時会 |
| `toku_yosan` | 予算審査特別委員会 |
| `toku_kessan` / `kettoku` | 決算審査特別委員会 |
| `toku_gikai` | 議会運営等特別委員会 |
| `toku_byouin` | 病院関連特別委員会 |
| `toku_suido` | 水道関連特別委員会 |
| `toku_iryo` | 医療関連特別委員会 |
| `toku_kihon` | 基本構想等特別委員会 |
| `toku_sougou` / `sougou` | 総合計画等特別委員会 |

### 命名規則の揺れ（年度別）

- **令和2年〜令和7年**: `{年号下2桁}_{回数}{種別}.pdf`（比較的統一）
- **平成31年/令和元年**: 平成31年分は `31_` 、令和元年分は `01_` で始まる
- **平成29年**: 一部 `29toku_` のようにアンダースコアなしのパターンあり
- **平成28年**: 年号プレフィックスなし（`rinji_01.pdf`）と年号あり（`28.1tei.pdf`）が混在
- **平成27年**: `27teirei_01.pdf` のように年号直結パターン
- **平成26年**: `26teirei_01.pdf` 形式
- **平成25年**: `25_1tei.pdf` 形式

## スクレイピング戦略

### Step 1: PDF リンクの収集

会議録一覧ページ `https://www.town.matsumae.hokkaido.jp/hotnews/detail/00000317.html` から全 PDF リンクを抽出する。

- 単一ページに全年度の PDF リンクが掲載されているため、1回のリクエストで全リンクを取得可能
- テーブル形式で年度・月ごとに整理されている
- リンクの `href` 属性から PDF の URL を抽出する

**収集方法:**

1. 一覧ページの HTML を取得
2. `a[href$=".pdf"]` セレクタで全 PDF リンクを抽出
3. 各リンクのテキスト（会議名）とURLをペアで保存

### Step 2: PDF のダウンロード

収集した URL から PDF をダウンロードする。

- 全 PDF 数は約 150〜170 件
- 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

### Step 3: PDF からのテキスト抽出

PDF からテキストを抽出し、構造化データに変換する。

- PDF パーサー（pdf-parse 等）でテキストを抽出
- PDF の内容は議会会議録の全文
- 1つの PDF に1回の会議の全発言が含まれる

### Step 4: テキストのパース

抽出したテキストから発言者と発言内容を分離する。

#### メタ情報の抽出

PDF のファイル名およびリンクテキストから以下を推定:
- 開催年度（ファイル名のプレフィックスから）
- 会議種別（ファイル名の種別略称から）
- 回次（ファイル名の数字から）

#### 発言の構造（PDF 内テキスト）

PDF 内のテキスト構造は実際のファイルをダウンロードして確認する必要がある。一般的な議会会議録 PDF では以下のパターンが想定される:

```
○議長（氏名）
  発言内容...

○番号番（氏名）
  発言内容...
```

※ 実際のフォーマットは PDF ダウンロード後に確認・調整が必要

## 注意事項

- PDF のファイル名命名規則に年度ごとの揺れがあるため、ファイル名からのメタ情報抽出はベストエフォートとし、リンクテキストも併用する
- 一部ファイル名にタイポと思われる揺れあり（例: `07_2teii_kaigiroku.pdf` の `teii`、`07_2rinji_kagiroku.pdf` の `kagiroku`）
- PDF の文字エンコーディングやレイアウトによってはテキスト抽出が困難な場合がある（スキャン PDF の可能性）
- `rinji` と `rinzi` の表記揺れがある（同じ「臨時会」を指す）

---

## 推奨アプローチ

1. **一括リンク取得**: 単一ページから全 PDF リンクを取得できるため、クロールは非常にシンプル
2. **PDF テキスト品質の事前確認**: 数件の PDF をダウンロードしてテキスト抽出の品質を確認し、スキャン PDF かテキスト PDF かを判定する
3. **ファイル名パースの柔軟性**: 命名規則の揺れに対応するため、正規表現は緩めに設定し、マッチしない場合はリンクテキストから情報を補完する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 一覧ページの PDF リンク数を前回と比較し、新規追加分のみダウンロードする
