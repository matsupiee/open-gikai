# 共和町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.kyowa.hokkaido.jp/education/?content=91
- 分類: 自治体公式サイト上で年度別に PDF を公開（既存アダプターでは対応不可）
- 形式: PDF ファイル
- 特記: 会議録検索システムなし。CMS の content ID ベースで年度別ページが構成され、各ページに PDF リンクを掲載。掲載されているのは教育委員会の会議録

---

## サイト構造

### 会議録トップ

`https://www.town.kyowa.hokkaido.jp/education/?content=91` を起点に、年度別ページへのリンクが並ぶ。

### 年度別ページ

各年度は `?content={ID}` のクエリパラメータで管理されている。

| 年度 | URL |
| --- | --- |
| 令和7年 | `?content=1429` |
| 令和6年 | `?content=800` |
| 令和5年 | `?content=92` |
| 令和4年 | `?content=91` |
| 令和3年 | `?content=90` |
| 令和2年 | `?content=89` |
| 平成31年（令和元年） | `?content=88` |
| 平成30年 | `?content=86` |
| 平成29年 | `?content=85` |

※ content ID に完全な規則性はない（例: 平成30年が86、平成29年が85だが、令和5年は92、令和6年は800）。

---

## URL 構造

### 年度別一覧ページ

```
https://www.town.kyowa.hokkaido.jp/education/?content={content_id}
```

### PDF ファイル

```
https://www.town.kyowa.hokkaido.jp/assets/images/content/content_{YYYYMMDD}_{HHMMSS}.pdf
```

- ファイル名はアップロード日時ベース（会議開催日ではない）
- 例: `content_20240122_180527.pdf`
- ファイルサイズは 68KB〜212KB 程度

---

## ページ構造

### 年度別一覧ページ

- ページネーションなし。全会議録が単一ページに一覧表示される
- 各会議録は「第N回会議録」というテキストで表記
- 開催日が「令和X年M月D日」の形式で併記
- PDF へのリンクが `<a>` タグで配置されている
- `href` は相対パス `../../assets/images/content/content_{日時}.pdf` 形式

### 会議の記録

- 年間 10〜13 回程度の会議が開催される
- 「第1回」「第2回」...と通し番号で管理
- 定例会・臨時会の区別は明記されていない
- 対象範囲: 平成29年〜令和7年（現在）

---

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

起点ページ `?content=91` から、各年度ページへのリンクを収集する。

- ページ内の `<a>` タグから `?content=` を含むリンクを抽出
- content ID に規則性がないため、必ず起点ページからリンクを辿る

### Step 2: PDF リンクの収集

各年度ページにアクセスし、PDF ファイルへのリンクを収集する。

- `<a>` タグで `href` が `.pdf` で終わるリンクを抽出
- リンクテキストから開催回次と開催日を抽出
- 相対パスを絶対 URL に変換

**収集するメタ情報:**

```typescript
interface KyowaMinutes {
  year: string;        // "令和7年" など
  session: string;     // "第1回" など
  date: string;        // "令和7年1月28日" など
  pdfUrl: string;      // PDF の絶対 URL
  fileSizeKb: number;  // ファイルサイズ (KB)
}
```

### Step 3: PDF のダウンロードとテキスト抽出

1. 収集した PDF URL からファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 抽出したテキストから議題・発言内容をパース

### Step 4: テキストのパース（推定）

PDF 内のテキスト構造は教育委員会の会議録形式と推定される。

- 開催日・会議名はヘッダー部分に記載
- 議題・報告事項・協議事項等のセクションに分かれる可能性がある
- PDF のレイアウトに依存するため、実際のファイルを確認して正規表現を調整する必要あり

---

## 注意事項

- PDF 形式のため、HTML スクレイピングとは異なり PDF パーサーが必要
- PDF のレイアウト（段組み・ヘッダー/フッター等）によってテキスト抽出の精度が変わるため、実際の PDF を確認してパーサーの調整が必要
- content ID に規則性がないため、必ず起点ページからリンクを辿る必要がある
- 新年度が追加された際に content ID が連番とは限らない（例: 92 → 800 → 1429）
- 問い合わせ先: 教育委員会管理課総務係（電話: 0135-67-8814）

---

## 推奨アプローチ

1. **2段階クロール**: 起点ページ → 年度別ページ → PDF リンクの順に辿る
2. **PDF テキスト抽出の事前検証**: 数件の PDF を手動でダウンロードし、テキスト抽出の品質を確認する。段組み・文字化け等の問題がないか事前に検証
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度別ページ単位で管理し、過去の年度は再取得しない。最新年度のページのみ定期的に確認する
