# 七宗町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.hichiso.jp/top/gyosei/parlament/record/
- 分類: WordPress サイトで PDF を直接公開（統一された会議録検索システムなし）
- 文字コード: UTF-8（WordPress 標準）
- 特記: 令和3年以降の定例会・臨時会の議事録を PDF 形式で提供。テキストレイヤーあり（pdftotext で抽出可能）

## PDF の内部構造

### テキストレイヤー

タグ付き PDF（アクセシビリティ対応）でテキストレイヤーあり。`pdftotext` 等でテキスト抽出可能。フォントは MS-Mincho。

### 冒頭部のメタ情報

```
令和７年

七宗町議会第３回定例会会議録

令和７年９月２日

令 和 ７ 年 七 宗 町 議 会 第 ３ 回 定 例 会 会 議 録
招集年月日  令和７年９月２日
招集場所    七宗町役場 議場
開議        ９月２日 午前１０時００分

出席議員 １番 加納竜也君、２番 和田真一郎君、３番 上野和義君、
         ４番 古田好文君、５番 松山成美君、６番 中島寛直君、
         ７番 福井德一君
欠席議員 なし
```

- 開催日: `令和X年X月X日` 形式
- 定例会の区分: `第X回定例会` という表記

### 発言者パターン

```
議 長 （ 松 山 成 美 君 ）  発言内容...
局 長 （ 渡 辺 岳 志 君 ）  発言内容...
教育課長（加納雅也君 ）  発言内容...
町 長 （ 堀 部 勝 広 君 ）  発言内容...
```

- 役職名（スペース区切りあり）+ `（` + 氏名 + `君）` の形式
- 役職は議長・副議長・町長・各課長など
- 登壇表記: `（提案説明のため登壇）`・`（報告のため登壇）` など括弧内に記載

#### パース用正規表現（案）

```typescript
// 発言者行の検出（スペース区切りあり）
const speakerPattern = /^(.+?)\（(.+?君)\）/;
// 例: "議 長 （ 松 山 成 美 君 ）" → role="議 長", name="松 山 成 美 君"

// 登壇表記の検出
const podiumPattern = /（.+?のため登壇）/;

// 開催日の抽出（タイトル部）
const datePattern = /令和(\d+)年(\d+)月(\d+)日/;

// 会議種別の抽出
const sessionPattern = /第(\d+)回(定例会|臨時会)/;
```

## スクレイピング戦略

### Step 1: PDF リンクの収集

一覧ページ `https://www.hichiso.jp/top/gyosei/parlament/record/` を取得し、`/wp-content/uploads/giziroku/` を含む `<a href>` を抽出する。

```typescript
// Cheerio でのリンク抽出例
$('a[href*="/wp-content/uploads/giziroku/"]').each((_, el) => {
  const href = $(el).attr('href');
  // => "/wp-content/uploads/giziroku/令和7年9月定例会.pdf"
  const url = `https://www.hichiso.jp${href}`;
});
```

- ファイル名から年号・月・会議種別をパースできる
- 新規追加分は一覧ページを再クロールして差分検出する

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF を取得し、テキストを抽出する。

- PDF サイズは 138KB〜796KB 程度
- テキストレイヤーが存在するため `pdftotext` 等で抽出可能
- ただし、スペース区切りで文字が分散している箇所があるため正規化が必要

```typescript
// テキスト正規化の例（スペース区切りを除去）
const normalized = text.replace(/([^\x00-\x7F])\s+(?=[^\x00-\x7F])/g, '$1');
// "議 長 （ 松 山 成 美 君 ）" → "議長（松山成美君）"
```

### Step 3: テキストのパース

1. 冒頭部から開催日・会議名・出席議員を抽出
2. 発言者パターンで本文を発言単位に分割
3. 各発言に役職名・氏名・発言内容を紐付け

---

## 注意事項

- 令和3年3月以前の会議録は掲載なし
- 令和4年2月定例会・令和6年2月定例会は掲載なし（開催なし、または別ページの可能性あり）
- PDF ファイル名は日本語（URL エンコードが必要）
- スペース区切りで文字が分散されているため、発言者パターンのマッチには正規化処理が必要
- 新規 PDF は一覧ページに追加されるため、定期的に一覧ページを再クロールして差分取得する

---

## 推奨アプローチ

1. **一覧ページのクロール**: 月次で一覧ページを取得し、新規 PDF リンクを検出
2. **PDF テキスト抽出**: `pdftotext` 等でテキストレイヤーを抽出（OCR 不要）
3. **テキスト正規化**: スペース区切りの日本語文字列を結合してからパース
4. **レート制限**: リクエスト間に 1〜2 秒の待機時間を設ける
5. **ファイル名ベースの差分管理**: PDF ファイル名（年号・月・会議種別）で取得済みを管理
