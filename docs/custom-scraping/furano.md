## サイト構造

### 一覧ページ

- URL: `https://www.city.furano.hokkaido.jp/shigikai/zokusei/gijiroku/`
- 年度・定例会別に会議録へのリンクを時系列で掲載
- ページネーションあり（3ページ程度）
  - 1ページ目: `/shigikai/zokusei/gijiroku/index.html`
  - 2ページ目: `/shigikai/zokusei/gijiroku/index.p2.html`
  - 3ページ目: `/shigikai/zokusei/gijiroku/index.p3.html`
- 各リンクは `/shigikai/docs/{数字ID}.html` 形式で詳細ページへ遷移

### リンクテキストパターン

- `令和○年 第○回定例会 会議録 (YYYY年MM月DD日)`
- `令和○年第○回臨時会 会議録 (YYYY年MM月DD日)`
- `令和○年 富良野市議会会議録 (YYYY年MM月DD日)` （年間通括）

### 詳細ページ（定例会単位）

- URL: `https://www.city.furano.hokkaido.jp/shigikai/docs/{ID}.html`
- 会議の基本情報（会期、会議名）が見出しで表示
  - 例: `令和7年第2回富良野市議会定例会(会期6月10日から6月20日)`
- 会議録PDFが日付別（号数別）にリンクされている
  - 目次PDF
  - 第1号（開会日）〜 第N号（閉会日）
- 委員会報告書PDF、意見書PDFも同ページに掲載される場合あり

### PDF の URL パターン

```
https://www.city.furano.hokkaido.jp/fs/{数字}/{数字}/{数字}/{数字}/{数字}/_/{ファイル名}.pdf
```

- `/fs/` 配下にファイルが格納されている
- ファイル名は日本語を含む（URLエンコード必要）
- ファイルサイズは各リンクに `(PDF XXX KB)` 形式で記載

## スクレイピング方針

### 全体フロー

```
1. 一覧ページ（全ページ）をクロール
   ↓
2. 各定例会・臨時会の詳細ページURLを収集
   ↓
3. 詳細ページから会議録PDFのURLを収集
   ↓
4. PDFをダウンロード
   ↓
5. PDFからテキストを抽出
```

### Step 1: 一覧ページのクロール

- `/shigikai/zokusei/gijiroku/index.html` から開始
- ページネーションリンク（`index.p{N}.html`）を辿って全ページを取得
- 各ページから `/shigikai/docs/{ID}.html` 形式のリンクを抽出

### Step 2: 詳細ページの解析

- 詳細ページのHTMLを取得
- 見出しから会議名・会期を抽出
- `/fs/` で始まる `.pdf` リンクを抽出
- リンクテキストから号数・日付を解析
  - パターン: `第○号(令和○年○月○日)` または `目次`

### Step 3: PDFのダウンロードとテキスト抽出

- PDFファイルをダウンロード
- PDF解析ライブラリ（pdf-parse等）でテキストを抽出
- 発言者・発言内容の構造化が必要（PDF内のレイアウトに依存）

## 注意事項

- 最新の定例会は会議録が「準備中」の場合がある（PDFリンクが存在しない）
- PDFのファイル名に日本語が含まれるため、URLエンコード処理が必要
- 検索パラメータによる絞り込み機能はない（一覧ページを順にクロールする方式）
- 会議録PDFの内部構造（発言者の記載パターン等）はPDFごとに確認が必要
- サイトへのアクセス間隔は適切に設定すること（1秒以上のウェイト推奨）

## メタデータ抽出

| フィールド | 抽出元 | 抽出方法 |
|-----------|--------|---------|
| 会議名 | 詳細ページ見出し | `令和○年第○回...定例会` を正規表現で抽出 |
| 会期 | 詳細ページ見出し | `会期○月○日から○月○日` を正規表現で抽出 |
| 号数 | PDFリンクテキスト | `第○号` を正規表現で抽出 |
| 開催日 | PDFリンクテキスト | `令和○年○月○日` を正規表現で抽出 |
| 年度 | 一覧ページリンクテキスト | `令和○年` / `平成○年` を正規表現で抽出 |
| 会議種別 | リンクテキスト | `定例会` / `臨時会` を判定 |
