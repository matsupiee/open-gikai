## PDF ファイルの命名規則

複数パターンが混在:

### パターン1: 和暦 + 月 + 回数

`{和暦年}.{月}遊佐町{号}号（配付用）.pdf`

例: `７．１２遊佐町１号（配付用）.pdf`（令和7年12月 第1号）

### パターン2: 回数 + 会議種別

`第{回数}回{種別}会_{種類}.pdf`

例: `第582回臨時会_会議録.pdf`

### パターン3: その他

ハッシュ値やランダム文字列が含まれるファイル名も存在。

---

## リンクテキストの形式

```
第{回数}遊佐町議会{種別}会（令和{年}年{月}月{日}日開会）
```

例: `第583回遊佐町議会定例会（令和7年12月2日開会）`

## スクレイピング戦略

### Step 1: 年度別ページ URL を収集

トップページから年度別リンクを収集する。リンクは `/ou/gikai/gikai/{ページ名}.html` 形式。

### Step 2: 年度別ページから PDF URL を収集

各年度ページをパースし、`/uploads/contents/archive_0000002698_00/` 配下の PDF リンクを全て取得する。

- リンクテキストから会議名・開会日を取得
- ファイル名が日本語の場合は URL エンコード処理が必要

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- ファイル名に日本語が含まれるため URL エンコード処理が必要
- ファイル名の命名規則が統一されていないため、リンクテキストからメタ情報を取得する
- 年度別ページの URL パターンが不規則（`copy3_of_2022gikairoku.html` 等）
- 回数が通算（第582回、第583回等）で管理されている
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
