## PDF ファイルの命名規則

複数パターンが混在:

### パターン1: 月日 + 種別（新しい年度）

`{月日4桁}{種別}.pdf`

| 例 | 種別 | 説明 |
| --- | --- | --- |
| `0304ippannsitumonn.pdf` | 一般質問 | 3月4日 一般質問 |
| `0306giannsinngi.pdf` | 議案審議 | 3月6日 議案審議 |
| `0306yosan.pdf` | 予算特別委員会 | 3月6日 予算 |
| `0913saisyuubi.pdf` | 最終日 | 9月13日 最終日 |
| `0905kessann.pdf` | 決算特別委員会 | 9月5日 決算 |
| `0519rinnjikai.pdf` | 臨時会 | 5月19日 臨時会 |

種別コード:
- `ippannsitumonn`: 一般質問
- `giannsinngi`: 議案審議
- `yosan`: 予算特別委員会
- `kessann`: 決算特別委員会
- `saisyuubi`: 最終日
- `rinnjikai`: 臨時会

### パターン2: 月日のみ

`{月日4桁}.pdf`

例: `0805.pdf`（8月5日）

### パターン3: 年月日8桁（古い年度）

`{YYYYMMDD}.pdf`

例: `20240305.pdf`（2024年3月5日）

### パターン4: 和暦 + 月日（さらに古い年度）

`{和暦2桁}{月日4桁}.pdf`

例: `060131.pdf`（令和6年1月31日）

## スクレイピング戦略

### Step 1: 一覧ページから PDF URL を収集

対象ページ `R40210.html` をパースし、相対パス `./` で参照される PDF リンクを全て取得する。

- 定例会は会議種別ごとに個別 PDF に分かれている（一般質問、議案審議、予算、決算、最終日）
- 見出しテキスト（`【令和7年第3回定例会】` 等）から会議名・年度を取得

### Step 2: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- 1ページ完結型だが、PDF 数が非常に多い（定例会が種別ごとに分割されるため）
- ファイル命名規則が年度によって異なる（新しい年度はローマ字種別、古い年度は日付のみ）
- 種別名がローマ字（ヘボン式ではなく日本語ローマ字: `ippannsitumonn`, `giannsinngi` 等）
- PDF のベース URL は `/004/` ディレクトリ
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
