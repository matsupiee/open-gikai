## PDF ファイル命名規則

PDF の URL は `https://www.town.tatsuno.lg.jp/material/files/group/11/` 配下に格納されている。

### 近年（令和以降）のパターン

```
{西暦}-{月}kaikinittei.pdf      → 会期日程
{西暦}-{月}syonichi.pdf         → 定例会会議録（初日）
{西暦}-{月}heikai.pdf           → 定例会会議録（最終日）
{西暦}-{月}ippanshitumon.pdf    → 一般質問議事録
{西暦}-{月}rinjikai.pdf         → 臨時会会議録
```

例: `2024-12syonichi.pdf`, `2024-12heikai.pdf`, `2024-12ippanshitumon.pdf`

### 古い年度（平成17年頃）のパターン

```
{西暦}_{回数}_i.pdf             → 一般質問議事録
```

例: `2005_07_i.pdf`, `2005_05_i.pdf`

※ 命名規則は年度により揺れがある（`ippanshitumon` / `ippannshitumon` / `ippannsitumon` など表記揺れあり）。

## HTML 構造

### トップページ（年度一覧）

- 各年度へのリンクが `<a>` タグで列挙されている
- リンク先は `/gyosei/choseijoho/tatsunochogikai/gikaigijiroku/{ページID}.html` 形式
- テキストは「令和6年」「平成17年」など和暦表記

### 年度別ページ

- `<h2>` に会議種別（「定例会」「臨時会」）
- `<h3>` に各回の定例会名（例:「令和6年12月定例会」）
- PDF リンクは `<p class="file-link-item">` 内の `<a class="pdf">` タグ
- リンクテキストにファイルサイズ情報を含む（例: `(PDFファイル: 626.2KB)`）

```html
<h2>定例会</h2>
<h3>令和6年12月定例会</h3>
<p class="file-link-item">
  <a class="pdf" href="//www.town.tatsuno.lg.jp/material/files/group/11/2024-12heikai.pdf">
    令和6年 第7回（12月）辰野町議会定例会会議録（最終日） (PDFファイル: 626.2KB)
  </a>
</p>
```

### ページネーション

なし。各年度ページに当該年の全会議録 PDF リンクが一括掲載されている。

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

トップページ `index.html` から全年度ページへのリンクを収集する。

- `<a>` タグの `href` 属性から `/gikaigijiroku/{ページID}.html` パターンを抽出
- テキストから和暦年を取得し、西暦に変換

### Step 2: PDF URL の収集

各年度ページにアクセスし、PDF リンクを収集する。

- `<p class="file-link-item">` 内の `<a class="pdf">` タグから `href` を抽出
- `href` はプロトコル省略形式（`//www.town.tatsuno.lg.jp/...`）なので `https:` を補完する
- リンクテキストから会議名・種別・開催月を抽出

**リンクテキストのパースパターン:**

```typescript
// リンクテキスト例:
// "令和6年 第7回（12月）辰野町議会定例会会議録（最終日） (PDFファイル: 626.2KB)"
// "令和6年 第5回（9月）定例会一般質問議事録 (PDFファイル: 2.2MB)"
// "令和7年第1回（1月）臨時会会議録 (PDFファイル: 471.3KB)"

const titlePattern = /^(.+?)\s*\(PDFファイル:.+?\)$/;
const sessionPattern = /(?:令和|平成)(\d+)年\s*第(\d+)回[（(](\d+)月[）)]/;
```

### Step 3: PDF のダウンロードとテキスト抽出

- 会期日程 PDF は対象外とする（`kaikinittei` を含む URL を除外）
- PDF からテキストを抽出し、発言者・発言内容を構造化する

---

## 注意事項

- 検索システムは存在しない。全量取得にはトップページ → 年度ページ → PDF という2段階のクロールが必要
- PDF ファイル名の命名規則に表記揺れがある（`ippanshitumon` / `ippannshitumon` / `ippannsitumon`）ため、ファイル名からの種別判定はリンクテキストを優先する
- PDF の `href` がプロトコル省略形式（`//www.town.tatsuno.lg.jp/...`）のため、`https:` を補完する必要がある
- 同一月に複数の臨時会がある場合、ファイル名にサフィックスが付く（例: `2025-5-1rinjikai.pdf`, `2025-5-2rinjikai.pdf`）

---

## 推奨アプローチ

1. **2段階クロール**: トップページで年度リンクを取得 → 各年度ページで PDF リンクを収集
2. **リンクテキストからメタ情報を抽出**: PDF ファイル名よりリンクテキストの方が正確な会議情報を含む
3. **会期日程 PDF を除外**: `kaikinittei` を含む URL、またはリンクテキストに「会期日程」を含むものを除外
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
5. **差分更新**: 年度ページの URL が固定のため、最新年度ページのみを再クロールすることで差分更新が可能
