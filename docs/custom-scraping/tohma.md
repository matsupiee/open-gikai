# 当麻町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.tohma.hokkaido.jp/parliament
- 分類: Drupal CMS による PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録は「一般質問と答弁」のみ PDF で公開。全文会議録の公開は確認できず

| 議会トップ | `https://www.town.tohma.hokkaido.jp/parliament` |

| PDF ファイル | `https://www.town.tohma.hokkaido.jp/sites/default/files/user-data/gikai/pdf/{ファイル名}.pdf` |

### PDF ファイル名の命名規則

ファイル名は全角文字を含む URL エンコードされた形式:

```
令和{N}年第{M}回定例会.pdf
```

- `{N}`: 和暦の年（全角数字: ５、６、７ 等）
- `{M}`: 定例会の回数（全角数字: １、２、３、４）
- URL エンコード例: `%E4%BB%A4%E5%92%8C%EF%BC%95%E5%B9%B4%E7%AC%AC%EF%BC%91%E5%9B%9E%E5%AE%9A%E4%BE%8B%E4%BC%9A.pdf`

## PDF の構造

### メタ情報

PDF 冒頭に以下の形式でタイトルが記載:

```
令和７年第１回定例会（３月１３日召集）
```

### 発言の構造

質問と答弁が交互に記載される:

```
○質問　{議員名}議員「{質問タイトル}」
  {質問本文}

○答弁　{役職名}
  {答弁本文}
```

#### 発言者パターン

```
○質問　上杉達則議員「米の価格高騰についての対策」
○答弁　村椿哲朗町長
```

- `○質問` で始まり、議員名 + 質問タイトル（「」内）
- `○答弁` で始まり、役職名（町長、副町長、教育長 等）

#### パース用正規表現（案）

```typescript
// PDF タイトルから開催情報を抽出
const titlePattern = /^(令和\d+)年第(\d+)回定例会（(\d+月\d+日)召集）/;

// 質問者の抽出
const questionPattern = /^○質問\s+(.+?)議員[「「](.+?)[」」]/;
// 例: ○質問　上杉達則議員「米の価格高騰についての対策」
//   → name="上杉達則", title="米の価格高騰についての対策"

// 答弁者の抽出
const answerPattern = /^○答弁\s+(.+)/;
// 例: ○答弁　村椿哲朗町長 → name="村椿哲朗町長"
```

## スクレイピング戦略

### Step 1: PDF URL の収集

`/parliament/1428` ページから PDF リンクを抽出する。

**収集方法:**

1. `/parliament/1428` を GET で取得
2. HTML を Cheerio でパースし、`<a>` タグの `href` から `.pdf` で終わるリンクを抽出
3. リンクテキストから年度・回数情報を取得

```typescript
// リンクテキストからメタ情報を抽出
const linkPattern = /令和(\d+)年第(\d+)回定例会/;
```

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF を取得し、テキストを抽出する。

- PDF は比較的小さいファイルサイズ（約 130KB）
- `pdf-parse` や `pdfjs-dist` 等のライブラリでテキスト抽出が可能
- テキストは日本語で、文字化けなく抽出可能であることを確認済み

### Step 3: テキストのパース

抽出したテキストから質問・答弁の構造を解析する。

1. PDF タイトル行から会議の開催情報（年度・回数・日付）を抽出
2. `○質問` / `○答弁` をセパレーターとして発言ブロックを分割
3. 各ブロックから発言者名・役職・本文を抽出

---

## 注意事項

- 公開範囲は「一般質問と答弁」のみであり、本会議の全文会議録（議事録）は公開されていない
- 臨時会の会議録は確認できない（定例会のみ）
- PDF ファイル名に全角文字が使われているため、URL エンコードの扱いに注意が必要
- 令和5年（2023年）以前の会議録はサイト上で確認できない
- サイトは Drupal ベースのため、ページ構造が CMS のアップデートで変更される可能性がある

---

## 推奨アプローチ

1. **PDF 一覧ページからの全量取得**: `/parliament/1428` の `<ul>` 内リンクを全件取得するだけでよく、ページネーションやパラメータ操作は不要
2. **PDF テキスト抽出**: `pdf-parse` 等で PDF からテキストを抽出し、`○質問` / `○答弁` パターンで発言を分割
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: PDF ファイル名に年度・回数が含まれるため、既存ファイルとの比較で新規追加分のみを取得可能
5. **定期監視**: 年4回の定例会ごとに新しい PDF が追加されるため、四半期ごとの確認で十分
