# 深浦町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.fukaura.lg.jp/category/bunya/gikai/
- 分類: 町公式サイト（Joruri CMS ベース）で議会情報を公開。会議録本文の HTML/PDF 公開はなく、会期日程・議決結果 PDF・議会動画配信で構成
- 文字コード: UTF-8（HTML5 標準）
- 特記: 会議録の全文テキスト公開は確認できず。定例会ページには会期日程表と議決結果 PDF のみ。議会動画配信ページに各議員の一般質問等の動画ページへのリンクがあるが、動画埋め込みやテキストは確認できず

## HTML 構造

### カテゴリ一覧ページ（定例会・臨時会）

記事一覧が日付・タイトル・担当部署の組み合わせで表示される。

```
定例会・臨時会（h1）
├── 令和７年度（h2）
│   ├── 令和8年3月第143回定例会 → /doc/2026031800110/
│   ├── 第142回臨時会（令和8年1月30日） → /doc/2026012600029/
│   └── ...
├── 令和６年度（h2）
│   └── ...
└── ...
```

### 個別記事ページ（定例会）

- `<h1>` に定例会名（例: 「令和8年3月深浦町議会第143回定例会」）
- 会期日程を `<table>` で表示（日程・月日・開会時刻・区分・内容の 5 列）
- 議決結果 PDF へのリンク（`file_contents/file_{ID}.pdf`）
- 会議録本文のテキストは含まれない

### 個別記事ページ（動画配信）

- `<h1>` にタイトル（例: 「第141回定例会（令和7年12月8日）一般質問 工藤議員」）
- 公開日のみ表示され、動画埋め込みや会議録テキストは確認できず
- 問い合わせ先（議会事務局）の情報

## スクレイピング方針

### 現状の課題

深浦町議会サイトでは **会議録の全文テキスト公開が確認できない**。提供されている情報は以下に限られる:

1. **会期日程表**（HTML テーブル）: 定例会の日程・区分・内容の概要
2. **議決結果 PDF**: 議案等の採決結果（議案名・賛否のみ、発言録ではない）
3. **動画配信ページ**: 各議員の一般質問等のタイトルのみ（動画本体・テキストは未確認）

### 取得可能なデータ

| データ | 形式 | 取得方法 |
| --- | --- | --- |
| 定例会・臨時会の一覧 | HTML | カテゴリ一覧ページをクロール |
| 会期日程 | HTML テーブル | 個別記事ページの `<table>` をパース |
| 議決結果 | PDF | 個別記事ページから PDF リンクを取得しダウンロード |
| 動画配信の一覧 | HTML | 議会動画配信カテゴリをクロール |

### Step 1: 定例会・臨時会の記事 URL 収集

カテゴリ一覧ページ `/category/bunya/gikai/teireikai/` からページネーションをたどり、全記事の URL を収集する。

- 記事 URL は `/doc/{ID}/` 形式
- ページネーション: `more@docs_2.p{N}.html` を順次取得

### Step 2: 個別記事ページの解析

各記事ページから以下を抽出:

- 定例会名（`<h1>` テキスト）
- 会期日程（`<table>` の行データ）
- 議決結果 PDF のリンク（`<a href>` で PDF を指すもの）

### Step 3: 議決結果 PDF の取得

PDF をダウンロードし、pdf-parse 等でテキスト抽出。議案名と採決結果を構造化する。

---

## 注意事項

- **会議録全文は未公開**: 発言録（誰が何を発言したか）のテキストデータは公開されていない。議決結果 PDF には議案の賛否情報のみが含まれる
- **CMS**: Joruri CMS ベースと推定される（検索エンジンに Joruri Search を使用）
- **動画配信ページの実態**: 動画ページへのリンクは存在するが、ページ本体に動画埋め込みやテキストが確認できないケースがある。JavaScript による動的読み込みの可能性あり
- **レート制限**: 自治体サイトのため、リクエスト間に 1-2 秒の待機時間を設ける
- **URL の ID 体系**: 記事 URL の ID 部分（例: `2026031800110`）は日付ベース（YYYYMMDD + 連番）と推定される
