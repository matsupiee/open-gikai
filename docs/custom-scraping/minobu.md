# 身延町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.minobu.lg.jp/site/gikai/
- 分類: 町公式ウェブサイト内 PDF 直接公開（独立した会議録検索システムなし）
- 文字コード: UTF-8
- 特記: 会議録は単一の一覧ページ（`/page/1455.html`）にまとめられており、各定例会の会議録 PDF へ直接リンクされている

---

## URL 構造

| ページ | URL |
| --- | --- |
| 議会情報トップ | `https://www.town.minobu.lg.jp/site/gikai/` |
| 会議録一覧 | `https://www.town.minobu.lg.jp/page/1455.html` |
| 会議録 PDF | `https://www.town.minobu.lg.jp/uploaded/attachment/{ID}.pdf` |

---

## 会議録の掲載形式

- 会議録は全て PDF ファイルで公開されている
- 検索システムは存在せず、`/page/1455.html` の単一ページに全年分の PDF リンクが列挙されている
- ページネーションはなく、2004年（平成16年）から現在まで全ての定例会会議録が1ページに掲載されている

---

## 会議種別

掲載されているのは**定例会のみ**。各年に原則4回の定例会が開催される。

| 回 | 時期（目安） |
| --- | --- |
| 第1回定例会 | 3月 |
| 第2回定例会 | 6月 |
| 第3回定例会 | 9月 |
| 第4回定例会 | 12月 |

※ 臨時会の会議録は一覧ページに掲載されていない。

---

## PDF リンクのパターン

PDF の URL は `/uploaded/attachment/{ID}.pdf` の形式で、ID は 3026〜3108 の連番（2025年時点）。
新しい年の会議録ほど小さい ID が割り当てられており、**降順**で並んでいる。

| PDF URL | 会議 |

| `/uploaded/attachment/3107.pdf` | 平成17年第1回定例会 |

## スクレイピング戦略

### Step 1: PDF リンクの収集

`/page/1455.html` をフェッチし、`/uploaded/attachment/*.pdf` 形式の全リンクを抽出する。

- ページネーションなし、単一ページで全件取得可能
- Cheerio で `a[href*="/uploaded/attachment/"]` を選択してリンク一覧を取得
- リンクテキスト（例: `令和7年第2回定例会`）も同時に取得し、メタ情報として利用する

```typescript
// リンク抽出例
const links = $('a[href*="/uploaded/attachment/"]').map((_, el) => ({
  url: `https://www.town.minobu.lg.jp${$(el).attr('href')}`,
  label: $(el).text().trim(),
})).get();
```

### Step 2: PDF のダウンロードとテキスト抽出

収集した PDF URL を順に取得し、PDF からテキストを抽出する。

- 各 PDF が1回の定例会全体の会議録を含む
- PDF テキスト抽出には `pdf-parse` 等のライブラリを使用する

### Step 3: メタ情報の抽出

リンクテキスト（例: `令和7年第2回定例会`）から開催年・回次を抽出する。

```typescript
// メタ情報抽出の正規表現例
const labelPattern = /^(令和|平成)(\d+)年第(\d+)回定例会$/;
// 例: "令和7年第2回定例会" → era="令和", year=7, session=2
```

## 差分更新

- `/page/1455.html` の PDF リンク一覧を定期的に再取得し、前回収集済みの URL と比較して新規追加分のみをダウンロードする
- ID は連番だが、新しい会議録が追加されるたびに小さい ID が割り振られる傾向があるため、URL の完全一致で差分を判定する

---

## 注意事項

- 会議録は定例会のみ公開。臨時会の会議録は見当たらない
- PDF の ID（`/uploaded/attachment/{ID}.pdf`）は連番だが、欠番が生じる可能性があるため、HTML から直接リンクを収集する方式を採用する
- 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
- Adobe Reader の案内リンクが含まれるため、PDF リンク抽出時に `get.adobe.com` 等の外部リンクを除外する

---

## 推奨アプローチ

1. **単一ページからの一括取得**: `/page/1455.html` を1回フェッチするだけで全 PDF リンクが取得できるため、クロールコストが低い
2. **リンクテキストの活用**: 各 PDF リンクに「令和X年第Y回定例会」形式のテキストが付いており、開催年・回次を容易に取得できる
3. **差分更新**: 既取得 PDF URL のリストと比較して新規追加分のみを処理する
