# 東峰村議会 カスタムスクレイピング方針

## 概要

- サイト: https://vill.toho-info.com/50000/50400/index.html
- 分類: 公式サイト内で年度別に PDF を直接公開（外部検索システムなし）
- 文字コード: UTF-8
- 掲載範囲: H28年度（平成28年）〜 R8年度（令和8年）
- 特記: WordPress ベースの村公式サイト。会議録は1件ずつ個別ページに掲載され、PDF1ファイルをダウンロードする形式

## 会議録個別ページの構造

各会議録ページは以下を含む:

- 会議名（タイトル）
- 開催日（本文テキスト）
- PDF ダウンロードリンク（1ファイル/1会議）
- 問い合わせ先（議会事務局）

**PDF URL の形式:**

```
https://vill.toho-info.com/{YYYY}/{MM}/{DD}/{ハッシュ値}.pdf
```

例:
- `https://vill.toho-info.com/2025/12/24/0d27345e4595e4ea81d4598673001794.pdf`
- `https://vill.toho-info.com/2023/01/13/c9c35d02cc093dbb2d4140cad8eec2bc.pdf`

ハッシュ値はランダムな16進数文字列であり、ファイル名からメタ情報を推定することはできない。

## スクレイピング戦略

### Step 1: 年度別一覧ページから個別ページ URL を収集

各年度の `index.html` を取得し、会議録個別ページへのリンクを抽出する。

- リンクは `<a href="{ファイル名}.html">` 形式の相対パス
- ページネーションはなし（1ページに全件掲載）
- 年度ディレクトリの URL は固定なので、H28〜R8 の全年度を列挙して処理する

### Step 2: 個別ページから PDF URL を抽出

各個別ページを取得し、PDF ファイルへのリンクを抽出する。

- PDF リンクは `<a href=".../{ハッシュ}.pdf">` 形式
- 1ページにつき通常 1 件の PDF だが、複数日程にまたがる定例会の場合も PDF は1ファイルにまとめられている
- リンクテキストはファイル名（例: 「令和７年第７回東峰村議会定例会会議録.pdf」）

### Step 3: PDF のダウンロードと解析

- PDF を取得してテキストを抽出する
- テキスト形式の会議録 PDF であるため、PDFtoText 等でテキスト抽出可能

---

## 注意事項

- 個別ページの URL に一貫した命名規則がない（`post-{N}.html`、`{年度}{回数}{日付}.html`、`4-3.html` など混在）
- PDF の URL に含まれるハッシュ値は推測不可能なため、必ず個別ページを経由して取得する
- R8年度以降も `r9`, `r10` のようなディレクトリが追加される想定
- サイトは村公式サイト（`vill.toho-info.com`）だが、PDF の絶対パスは同ドメイン内に格納されている

---

## 推奨アプローチ

1. **年度リスト定義**: H28〜現在の年度ディレクトリ URL を静的に定義してクロール
2. **個別ページ列挙**: 各年度 `index.html` から `<a href>` リンクを Cheerio で抽出
3. **PDF URL 取得**: 個別ページから PDF リンクを抽出してダウンロード
4. **差分更新**: 既取得の個別ページ URL を記録し、新規追加分のみ処理
5. **レート制限**: 村サーバーへの負荷を抑えるため、リクエスト間に 1〜2 秒の待機を設ける
