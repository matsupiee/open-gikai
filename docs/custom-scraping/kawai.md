# 河合町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.kawai.nara.jp/10/1_1/1/2/index.html
- 分類: 自治体公式サイト上で年度別に PDF を公開（既存アダプターでは対応不可）
- 形式: PDF ファイル
- 特記: 会議録検索システムなし。静的 HTML ページに PDF リンクを掲載

---

## サイト構造

### 議会報告トップ

`https://www.town.kawai.nara.jp/10/1_1/index.html` から以下のカテゴリに分岐する。

| カテゴリ | URL |
| --- | --- |
| 定例会 | `/10/1_1/1/index.html` |
| 臨時会 | `/10/1_1/2/index.html` |
| 全員協議会 | `/10/1_1/zeninkyougikai/index.html` |
| 特別委員会 | `/10/1_1/2_1/index.html` |

### 定例会 会議録（メインターゲット）

`https://www.town.kawai.nara.jp/10/1_1/1/2/index.html`

年度別の一覧ページへのリンクが並ぶ。対象範囲は平成23年〜令和7年（現在）。

### 特別委員会

以下の 3 種類の委員会会議録が存在する。

- ごみ処理施策検討特別委員会
- 決算審査特別委員会
- 予算審査特別委員会

### PDF ファイル

```
https://www.town.kawai.nara.jp/material/files/group/9/{ファイルID}.pdf
```

PDF ファイル名の規則:

- 古い年度: `kaigiroku{和暦年}-{月}-{日}.pdf`（例: `kaigiroku23-12-7.pdf` → 平成23年12月7日）
- 新しい年度: 数字のみの ID（例: `6961.pdf`, `69102.pdf`）。月日と連番の組み合わせと推測されるが、厳密な規則性はない

## ページ構造

### インデックスページ（年度別一覧）

各年度ページは以下の構造を持つ。

- `<h3>` 見出しで「第N回（M月）定例会」のセクションに分割
- 各セクション内に `<a>` タグで PDF へのリンクが並ぶ
- リンクテキスト: 「（M月D日 内容）(PDFファイル: サイズ)」形式

#### 会議の種類（定例会内）

各定例会は複数日にわたり、以下の種別がある。

| 種別 | 説明 |
| --- | --- |
| 初日 | 定例会の開会日 |
| 一般質問 | 議員による一般質問（通常2日間） |
| 最終日 | 定例会の閉会日 |

#### 定例会の回次

年4回の定例会が開催される。

| 回次 | 開催月 |
| --- | --- |
| 第1回 | 3月 |
| 第2回 | 6月 |
| 第3回 | 9月 |
| 第4回 | 12月 |

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

定例会会議録インデックスページ `/10/1_1/1/2/index.html` から、各年度ページへのリンクを収集する。

- `<a>` タグの `href` 属性から年度別ページの URL を抽出
- URL に規則性がないため、必ずインデックスページを起点にする

### Step 2: PDF リンクの収集

各年度ページにアクセスし、PDF ファイルへのリンクを収集する。

- `<a>` タグで `href` が `/material/files/group/9/` を含み `.pdf` で終わるリンクを抽出
- リンクテキストから開催日・会議種別（初日/一般質問/最終日）を抽出
- `<h3>` 見出しから「第N回（M月）定例会」の回次情報を抽出

**収集するメタ情報:**

```typescript
interface KawaiMinutes {
  year: string;          // "令和7年" など
  session: string;       // "第1回3月定例会" など
  date: string;          // "3月5日" など
  type: string;          // "初日" | "一般質問" | "最終日"
  pdfUrl: string;        // PDF の絶対 URL
  fileSizeKb: number;    // ファイルサイズ (KB)
}
```

### Step 3: PDF のダウンロードとテキスト抽出

1. 収集した PDF URL からファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 抽出したテキストから発言者・発言内容をパース

### Step 4: テキストのパース（推定）

PDF 内のテキスト構造は一般的な議会会議録形式と推定される。

- 開催日・会議名はヘッダー部分に記載
- 発言者は「○議長」「○N番」等の形式で記載される可能性がある
- PDF のレイアウトに依存するため、実際のファイルを確認して正規表現を調整する必要あり

---

## 注意事項

- PDF 形式のため、HTML スクレイピングとは異なり PDF パーサーが必要
- PDF のレイアウト（段組み・ヘッダー/フッター等）によってテキスト抽出の精度が変わるため、実際の PDF を確認してパーサーの調整が必要
- 古い年度（平成23年〜）と新しい年度で PDF ファイル名の命名規則が異なる
- URL のファイル名に規則性がないため、必ずインデックスページからリンクを辿る必要がある
- 臨時会・特別委員会・全員協議会の会議録も存在するが、まず定例会を優先して対応する

---

## 推奨アプローチ

1. **2段階クロール**: インデックスページ → 年度別ページ → PDF リンクの順に辿る
2. **PDF テキスト抽出の事前検証**: 数件の PDF を手動でダウンロードし、テキスト抽出の品質を確認する。段組み・文字化け等の問題がないか事前に検証
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度別ページ単位で管理し、過去の年度は再取得しない。最新年度のページのみ定期的に確認する
5. **段階的拡張**: 定例会 → 臨時会 → 特別委員会 → 全員協議会の順に対応範囲を広げる
