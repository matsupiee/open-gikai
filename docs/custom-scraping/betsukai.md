# 別海町議会 カスタムスクレイピング方針

## 概要

- サイト: https://betsukai.jp/gikai/kaigikekka/kaigiroku/
- 分類: 年度別 PDF 公開（HTML 会議録なし、PDF ダウンロード型）
- 文字コード: UTF-8
- 特記: 会議録はすべて PDF ファイルで提供。2010 年（平成 22 年）以降の定例会・臨時会を収録。

## ページ構造

### トップページ（年度一覧）

- 年度別リンクが箇条書きで一覧表示される
- ページネーションなし（全年度が 1 ページに表示）
- 各リンクは `/gikai/kaigikekka/kaigiroku/{年度コード}/` 形式

### 年度別ページ

- 「定例会」と「臨時会」のセクションに分かれて表示
- 各会議ごとに見出し（例: 「令和6年第4回定例会（令和6年12月9日から12月13日）」）
- 会議日ごとに PDF リンクが箇条書きで列挙される
- ページネーションなし（年度内の全会議録が 1 ページに表示）

### PDF ファイル名パターン

令和の場合:
```
R06.12_kaigiroku1.pdf   （令和6年12月・1日目）
R6.9_kaigiroku1.pdf     （令和6年9月・1日目）
R6.3_kaigiroku01.pdf    （令和6年3月・1日目、ゼロ埋めあり）
```

平成の場合:
```
H25.3.7_1.pdf           （平成25年3月7日・1日目）
```

※ ファイル名の命名規則は統一されておらず、年度・時期によって異なる。

## スクレイピング戦略

### Step 1: 年度ページ URL の収集

トップページ `https://betsukai.jp/gikai/kaigikekka/kaigiroku/` から全年度ページへのリンクを取得する。

**収集方法:**

1. トップページの HTML を取得
2. `/gikai/kaigikekka/kaigiroku/{年度コード}/` パターンのリンクを Cheerio で抽出
3. 年度コードのリストを生成

### Step 2: PDF リンクの収集

各年度ページから PDF ファイルへのリンクとメタ情報を取得する。

**収集方法:**

1. 各年度ページの HTML を取得
2. `.pdf` で終わるリンクを Cheerio で抽出
3. リンク周辺のテキストから以下のメタ情報を抽出:
   - 会議種別（定例会 / 臨時会）
   - 回次（第 X 回）
   - 開催日（令和 X 年 X 月 X 日）
   - 会議日数（X 日目）

### Step 3: PDF のダウンロードとテキスト抽出

PDF ファイルをダウンロードし、テキストを抽出する。

**処理方法:**

1. PDF ファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 抽出したテキストから発言者・発言内容をパース

## メタ情報の抽出

### 年度ページからの抽出

会議の見出しテキストからメタ情報を正規表現で抽出する:

```typescript
// 定例会・臨時会の見出しパターン
const meetingPattern = /(?:令和|平成)(\d+)年第(\d+)回(定例会|臨時会)(?:（(.+?)）)?/;
// 例: "令和6年第4回定例会（令和6年12月9日から12月13日）"

// 開催日の抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/g;
```

---

## 注意事項

- 会議録はすべて PDF 形式で提供されるため、HTML パースではなく PDF テキスト抽出が必要
- PDF ファイル名の命名規則が年度によって異なるため、ファイル名からのメタ情報抽出は信頼性が低い（ページの見出しテキストから取得すべき）
- `release` パス内の ID（`releaseId`, `contentId`）は年度・会議ごとに異なり、規則性がないためリンク先を直接取得する必要がある
- 臨時会の PDF は `kaigiroku.pdf` のような汎用名が使われることがある

---

## 推奨アプローチ

1. **2段階クロール**: まずトップページから年度一覧を取得し、次に各年度ページから PDF リンクを収集する
2. **メタ情報はページテキストから取得**: PDF ファイル名ではなく、年度ページの見出しテキストからメタ情報を抽出する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度ページ単位で管理し、最新年度のページのみを再クロールすることで差分更新を実現する
5. **PDF テキスト抽出**: pdf-parse 等のライブラリを使用し、PDF からテキストを抽出してから発言内容をパースする
