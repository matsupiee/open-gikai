## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録一覧ページ `/6/4/3/1/index.html` から各年度のリンクを収集する。

- `<a>` タグの `href` 属性から `/6/4/3/1/{ページID}.html` パターンを抽出
- ページネーションなし（全年度が 1 ページに表示）

### Step 2: PDF URL の収集

各年度ページから PDF リンクを収集する。

- `p.file-link-item > a.pdf` セレクタで PDF リンクを取得
- 直前の `h3` 見出しから会議種別（定例会・臨時会）を取得
- リンクテキストから開催日と日次を抽出

**リンクテキストのパターン:**

```
本会議(令和6年12月23日) (PDFファイル: 413.7KB)        → 臨時会（日次なし）
本会議1日目(令和6年12月3日) (PDFファイル: 810.6KB)     → 定例会1日目
本会議2日目（令和6年12月5日） (PDFファイル: 593.3KB)    → 定例会2日目
```

**抽出用正規表現:**

```typescript
// リンクテキストから日次と開催日を抽出
const linkPattern = /本会議(\d+日目)?[（(]((?:令和|平成)\d+年\d+月\d+日)[）)]/;
// 例: "本会議1日目(令和6年12月3日)" → day="1日目", date="令和6年12月3日"

// h3 見出しから会議種別を抽出
const sessionPattern = /第(\d+)回(定例会|臨時会)/;
// 例: "第4回定例会" → num="4", type="定例会"
```

### Step 3: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF は通常 10〜30 ページ程度（150KB〜850KB）
- pdf-parse 等のライブラリでテキスト抽出

### Step 4: 会議録のパース

#### メタ情報

PDF 冒頭に以下の形式でタイトルが記載される:

```
令和６年第１回大衡村議会臨時会会議録　第１号
```

- 年号: `令和X年` or `平成X年`
- 回次: `第X回`
- 会議種別: `定例会` or `臨時会`
- 号数: `第X号`

開催日時:

```
令和６年２月５日（月曜日）　午前１１時００分開議
```

#### 出席者情報

PDF 冒頭に以下のセクションが含まれる:

- `出席議員（XX名）`: 番号・氏名の一覧
- `欠席議員`: 欠席者の一覧
- `説明のため出席した者の職氏名`: 村長、副村長、各課長等の一覧
- `事務局出席職員氏名`: 次長、書記等

#### 発言の構造

発言者パターン:

```
議長（髙橋浩之君）
村長（小川ひろみ君）
税務課長（堀籠淳君）
10番（佐々木金彌君）
```

- `役職（氏名君）` の形式
- 役職は「議長」「村長」「副村長」「XX課長」「XX番」など
- 敬称は「君」が使用される
- 発言内容は発言者行の後に続くテキスト（インデント付き）

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^(.+?)（(.+?)君）/;
// 例: "議長（髙橋浩之君）" → role="議長", name="髙橋浩之"
// 例: "10番（佐々木金彌君）" → role="10番", name="佐々木金彌"

// 開催日の抽出
const datePattern = /(?:令和|平成)[０-９\d]+年[０-９\d]+月[０-９\d]+日/;

// タイトルの抽出
const titlePattern = /(?:令和|平成)[０-９\d]+年第[０-９\d]+回大衡村議会(定例会|臨時会)会議録/;

// 議事日程の区切り
const agendaPattern = /日程第[０-９\d]+/;
```

## 注意事項

- PDF はテキスト埋め込み型（画像ベースではない）であるため、テキスト抽出が可能
- 全角数字が使用されている箇所がある（タイトル、日付等）。正規表現では全角・半角両方に対応する必要がある
- 括弧は全角 `（）` が主に使用されるが、一部半角 `()` も混在している
- 会議種別は「本会議」のみ（委員会の会議録は公開されていない）
- PDF の URL に含まれるファイル ID は連番ではなく、法則性がないため、HTML ページからの抽出が必要

---

## 推奨アプローチ

1. **2段階クロール**: まず会議録一覧 → 年度別ページ → PDF URL を収集し、その後 PDF をダウンロード・パース
2. **PDF テキスト抽出**: pdf-parse 等を使用してテキストを抽出し、正規表現でパースする
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 各年度ページの PDF リンク数を前回と比較し、新規追加分のみを処理する
