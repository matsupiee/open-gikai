# 豊前市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.buzen.lg.jp/senkyo-gikai/gikai/gijiroku.html
- 分類: 独自 CMS による HTML ページ公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 専用の全文検索システムは存在しない。会議録一覧は単一ページに全年度分が掲載されており、各会議の詳細ページに会議録 PDF へのリンクがある。Google Tag Manager（GTM）使用。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧（全年度）| `https://www.city.buzen.lg.jp/senkyo-gikai/gikai/gijiroku.html` |
| 会議詳細ページ | `https://www.city.buzen.lg.jp/senkyo-gikai/gikai/gijiroku_{年号}_{番号}.html` |

---

## 年号・番号の命名規則

URL の `{年号}` は元号略称、`{番号}` は年内の開催順を示す連番。

| 元号 | 略称 | 例 |
| --- | --- | --- |
| 令和 | `r` | `r7`（令和7年）|
| 平成 | `h`（または `heisei`）| `h19`（平成19年）|

| 番号 | 意味 |
| --- | --- |
| 奇数（1, 3, 5, ...）| 定例会・臨時会（本番） |
| 偶数（2, 4, 6, ...）| 予備番号または別会議 |

実際の例（令和7年）:

| URL | 会議名 |
| --- | --- |
| `gijiroku_r7_2.html` | 第1回定例会 |
| `gijiroku_r7_4.html` | 第2回臨時会 |
| `gijiroku_r7_6.html` | 第3回定例会 |
| `gijiroku_r7_7.html` | 第4回臨時会 |
| `gijiroku_r7_9.html` | 第5回定例会 |
| `gijiroku_r7_12.html` | 第6回定例会 |
| `gijiroku_r8_1.html` | 令和8年第1回臨時会 |

## スクレイピング戦略

### Step 1: 会議録一覧ページから詳細ページリンクを収集

`gijiroku.html` を取得し、`gijiroku_{年号}_{番号}.html` 形式の全リンクを Cheerio で抽出する。

- ページネーションなし。単一ページに全リンクが含まれる
- 全量で69件以上のリンクが存在する（平成16年〜令和8年）

### Step 2: 各会議詳細ページから PDF リンクを収集

各詳細ページ内の PDF リンクを抽出する。

- 一般質問一覧表 PDF と会議録本体 PDF が別ファイルで提供される
- PDF の URL パターンはページごとに異なる可能性があるため、リンクテキストで判別する

### Step 3: PDF の取得・解析

各会議録 PDF を取得してテキストを抽出する。

---

## 注意事項

- 会議録の全文テキストは HTML ページには含まれず、すべて PDF 内にある
- URL の番号（`_2`, `_4` 等）は開催回と一致せず、間に番号が飛ぶため、一覧ページからのリンク抽出が確実
- 一部年度では `-` 区切りが使われている可能性がある（`gijiroku_{年号}-{番号}.html`）
- 多言語対応機能あり（ページ表示には影響しない）

---

## 推奨アプローチ

1. **一覧ページから全量収集**: `gijiroku.html` の全リンクを起点とし、URL パターンのハードコードを避ける
2. **正規表現でリンク判別**: `gijiroku_[a-z]+\d+_\d+\.html` パターンでフィルタリングする
3. **PDF テキスト抽出**: pdfjs や pdfminer 相当のライブラリで PDF を解析する
4. **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒の待機時間を設ける
