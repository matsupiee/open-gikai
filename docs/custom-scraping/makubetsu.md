# 幕別町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.makubetsu.lg.jp/gikai/hongikai/gikaikaigiroku/
- 分類: 公式サイト内で会議録を PDF として提供（HTML 会議録なし）
- 文字コード: UTF-8
- 特記: 会議録はすべて PDF ファイルで提供。定例会・臨時会、常任委員会、議会運営委員会、特別委員会の 4 カテゴリに分類

## 会議種別

### 定例会・臨時会

令和 7 年（2025 年）～平成 16 年（2004 年）の会議録を掲載。年度ごとにセクション分けされ、各年度内で定例会（第 1 回～第 4 回）と臨時会が時系列順に並ぶ。

### 常任委員会

以下の 3 委員会の会議録を掲載:

- 総務文教常任委員会
- 民生常任委員会
- 産業建設常任委員会

連合審査会の記録も含む。

### 議会運営委員会

平成 26 年～平成 28 年の 3 年度分のみ掲載。

### 特別委員会

以下の委員会の会議録を掲載:

- 決算審査特別委員会（平成 15 年度～令和 6 年度）
- 予算審査特別委員会（平成 15 年度～令和 7 年度）
- 市町村合併調査特別委員会
- 総合計画基本構想審査特別委員会
- 庁舎建設に関する調査特別委員会
- その他専門特別委員会

## PDF の URL パターン

PDF ファイルの URL は時期によって 2 つのパターンが存在する。

### パターン 1: 新形式（令和 6 年頃以降）

```
https://www.town.makubetsu.lg.jp/assets/images/makubetsu/{ファイル名}.pdf
```

例:

- `/assets/images/makubetsu/◎20250116-第１回臨時会【会議録】.pdf`
- `/assets/images/makubetsu/第１回定例会.pdf`
- `/assets/images/makubetsu/令和7年第3回定例会.pdf`
- `/assets/images/makubetsu/20250903-会議録（陳情審査）HP.pdf`

ファイル名の命名規則は統一されておらず、日本語ファイル名が使われる。

### パターン 2: 旧形式（令和 5 年頃以前）

カテゴリごとにサブディレクトリが異なる:

| カテゴリ | ベースパス |
| --- | --- |
| 定例会・臨時会 | `/chosei/gikai/gikai_kaigiroku/teireirinjikaigiroku/files/` |
| 常任委員会 | `/chosei/gikai/gikai_kaigiroku/jyoninkaigoroku/files/` |
| 議会運営委員会 | `/chosei/gikai/gikai_kaigiroku/jyoninkaigoroku/files/` |
| 特別委員会 | `/chosei/gikai/gikai_kaigiroku/tokubetsukaigiroku/files/` |

ファイル名の例:

- 定例会: `2024-1t.pdf`（年-回次 + t）
- 常任委員会: `20240624so2.pdf`（日付 + 委員会略称 + 番号、so=総務文教、mi=民生、sa=産業建設）
- 議会運営委員会: `gi280223.pdf`（gi + 和暦年月日）
- 特別委員会: `20240918-19.pdf`、`202309-kettoku.pdf`

## スクレイピング戦略

### Step 1: 一覧ページから PDF リンクを収集

4 つの一覧ページ（1898.html、1897.html、1896.html、12568.html）にアクセスし、すべての PDF リンクを抽出する。

**収集方法:**

1. 各一覧ページの HTML を取得
2. `<a>` タグから `.pdf` で終わるリンクを Cheerio で抽出
3. リンクテキストから会議名・開催日などのメタ情報を取得
4. ページネーションは存在しないため、各ページ 1 回のリクエストで全リンクを取得可能

### Step 2: PDF のダウンロード

収集した PDF URL からファイルをダウンロードする。

**注意点:**

- ファイル名に日本語・全角文字・特殊文字（`◎` など）が含まれるため、URL エンコーディングに注意
- 新形式と旧形式で URL パスが異なるが、いずれも絶対パスに変換して取得

### Step 3: PDF からのテキスト抽出

PDF ファイルからテキストを抽出し、会議録データとして構造化する。

#### メタ情報の抽出

リンクテキストから以下を抽出:

- 会議種別（定例会、臨時会、常任委員会名、特別委員会名）
- 開催日（単日または期間）
- 回次（第 N 回）

リンクテキストの例:

```
第1回臨時会【1月16日開催】
第１回定例会 【3月4日～3月21日開催】
6月12日開催（付託：陳情審査ほか）
```

#### 発言の構造

PDF 内のテキスト構造は PDF の生成元（Word 等）に依存するため、実際の PDF を取得して解析する必要がある。

---

## 注意事項

- 会議録はすべて PDF 形式であり、HTML の会議録ページは存在しない
- PDF のファイル名命名規則が時期・カテゴリによって大きく異なるため、リンクテキストからのメタ情報抽出が重要
- 新形式の URL にはファイル名に日本語が含まれるため、URL エンコーディングの処理が必要
- 一覧ページにページネーションはなく、全会議録が 1 ページに表示される

---

## 推奨アプローチ

1. **一覧ページの定期クロール**: 4 つの一覧ページを取得し、PDF リンクを差分チェック
2. **PDF テキスト抽出ライブラリの活用**: `pdf-parse` 等を使って PDF からテキストを抽出
3. **メタ情報はリンクテキストから取得**: PDF 内部のパースに頼らず、一覧ページのリンクテキストから会議名・日付を抽出する方が確実
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **URL エンコーディング**: 日本語ファイル名を含む URL は適切にエンコードして取得する
