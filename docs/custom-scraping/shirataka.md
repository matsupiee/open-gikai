# 白鷹町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.shirataka.lg.jp/1138.htm
- 分類: PDF 公開（自治体公式サイト）
- 文字コード: UTF-8

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧 | `https://www.town.shirataka.lg.jp/1138.htm` |
| PDF ファイル | `https://www.town.shirataka.lg.jp/secure/{ID}/{ファイル名}.pdf` |

---

## ページ階層

```
トップ > 町議会 > 議会会議録
├── 令和8年
│   ├── 第2回臨時会（会議録R8-2回（R8.2.16）.pdf）
│   └── ...
├── 令和7年
│   ├── 第6回定例会 12月（r07-06t-01-1204.pdf 等）
│   ├── 第5回定例会 9月
│   └── ...
└── 過去年度
```

1ページに全年度の会議録が表形式で掲載される。

---

## PDF ファイルの命名規則

複数パターンが混在:

### パターン1: 英数字コード形式

`r{和暦2桁}-{回数2桁}{種別}-{号2桁}-{月日4桁}.pdf`

| 例 | 説明 |
| --- | --- |
| `r07-06t-01-1204.pdf` | 令和7年 第6回定例会 第1号 12月4日 |
| `r07-y-01-0606.pdf` | 令和7年 予算委員会 第1号 6月6日 |
| `r07-k-01-0911.pdf` | 令和7年 決算委員会 第1号 9月11日 |

種別コード:
- `t`: 定例会
- `y`: 予算委員会
- `k`: 決算委員会

### パターン2: 日本語ファイル名形式

`会議録R{年}-{回}回（R{年}.{月}.{日}）.pdf`

例: `会議録R8-2回（R8.2.16）.pdf`

---

## スクレイピング戦略

### Step 1: 一覧ページから PDF URL を収集

対象ページ `1138.htm` をパースし、`/secure/{ID}/` 配下の PDF リンクを全て取得する。

- 表形式で日付ごとに PDF がリストされている
- 各 PDF は `<a>` タグで直接リンクされている

### Step 2: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- 1ページ完結型だが、PDF 数が多いためパース対象の `<a>` タグが多い
- PDF パスに含まれる `{ID}` は `/secure/` 配下のディレクトリ ID
- ファイル名に日本語が含まれる場合がある（URL エンコード処理が必要）
- 定例会は日付ごとに個別の PDF に分かれている
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
