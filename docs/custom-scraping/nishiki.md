## ページ構造

### 一覧ページ（list00253.html）

- 年度ごとのリンクが並ぶシンプルな一覧
- 各エントリは「錦町議会会議録（定例会・臨時会）」という統一タイトルで、リンク先が年度別ページ
- jQuery の `autopager` プラグインによる「もっと見る」形式の遅延読み込みあり
- 最低4年分（令和5年〜令和8年）が確認されている

### 年度別ページ（kiji{記事ID}/index.html）

- その年の定例会・臨時会の会議録 PDF へのリンクが列挙される
- 各 PDF は1会期につき1ファイル
- ページ末尾に「掲載されていない会議録につきましては、議会事務局へお問い合わせください」との注記あり

---

## 会議録の提供形式

- **形式**: PDF（1会期 = 1ファイル）
- **会議種別**: 定例会（年4回）・臨時会（年複数回）
- **PDF ファイル名パターン**: `3_{記事ID}_{ファイルID}_up_{ランダムハッシュ}.pdf`
  - ハッシュ部分は推測不可能なためクロールで取得する必要がある

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

一覧ページ `list00253.html` から年度別ページの URL を取得する。

- `autopager` による動的読み込みがあるため、JavaScript 実行が必要な場合がある
- リンクは `kiji{記事ID}/index.html` の形式で埋め込まれている
- 静的 HTML に初期表示分が含まれる可能性があるため、まず静的取得を試みる

### Step 2: PDF リンクの収集

各年度ページ（`kiji{記事ID}/index.html`）から PDF ファイルへの直接リンクを抽出する。

- PDF リンクは `<a href="...pdf">` タグで直接記述されている
- Cheerio 等で `.pdf` 拡張子を持つリンクを抽出する
- ファイルサイズや会議名のテキストも同時に取得できる

### Step 3: PDF のダウンロードとメタ情報の抽出

各 PDF をダウンロードし、テキストを抽出する。

- リンクテキストから会議名・開催期間を取得する（例: `令和6年第1回議会定例会（3月5日～3月12日）`）
- PDF からのテキスト抽出には `pdf-parse` 等を使用する

---

## 注意事項

- PDF ファイル名のハッシュ部分（`up_{ハッシュ}`）は推測不可能なため、必ず HTML ページをクロールして URL を取得する
- 令和5年より前の年度については「議会事務局へ問い合わせ」との記載があり、ウェブ上での公開は限定的と考えられる
- 記事ID の付番規則が一定でない（`003742`, `003905`, `0031024`, `0031174` など桁数が混在）
- `autopager` による追加読み込みが機能している場合、静的 HTML では全件取得できない可能性がある

---

## 推奨アプローチ

1. **一覧ページの静的取得を試みる**: まず `list00253.html` を静的取得し、`kiji*/index.html` へのリンクを抽出する。足りない場合は `autopager` の追加ページも取得する
2. **各年度ページから PDF リンクを抽出**: Cheerio で `.pdf` リンクを収集し、リンクテキストから会議名・開催期間を取得する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **増分更新**: 一覧ページの更新日を確認し、変更があった年度のみ再クロールする差分更新を検討する
