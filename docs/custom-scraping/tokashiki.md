# 渡嘉敷村議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.vill.tokashiki.okinawa.jp/gyoseijoho/tokashikisongikai/
- 分類: 独自 CMS による年度別 PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8（HTML ページ）、PDF 内は MS-Mincho フォント使用
- 特記: 会議録は PDF ファイルのみで公開。HTML 本文での会議録公開なし

## サイト構造

### 階層

```
議会トップ
├── 議会・定例会・臨時会一覧
│   ├── 令和8年一覧
│   │   └── 各会議ページ → PDF リンク
│   ├── 令和7年一覧
│   │   └── 各会議ページ → PDF リンク
│   ├── ...
│   └── 令和元年以前一覧
│       └── 各会議ページ → PDF リンク
├── 議会組織図・議員名簿
├── 議会だより
└── 子ども議会
```

### 年度別一覧ページ

- `<ul>/<li>` のリスト構造で会議リンクを表示
- 定例会（3月・6月・9月・12月）と臨時会のリンクが混在
- 会議録・議決案・会期日程など複数種類のページが含まれる
- ページネーションなし

### 各会議の個別ページ

- HTML ページは案内ページのみ（本文なし）
- PDF ファイルへのダウンロードリンクを 1 つ含む
- リンク形式: `//www.vill.tokashiki.okinawa.jp/material/files/group/7/{ファイル名}.pdf`
- ファイルサイズの表示あり
- 更新日の表示あり

## PDF 会議録の構造

### メタ情報（表紙）

```
令和７年
渡嘉敷村議会会議録

第７回定例会（12月10日〜12日）　3日間

渡嘉敷村議会
```

### 目次

- 会期日程、出席議員、議事日程、各日程の内容とページ番号
- 日程番号と議案番号の対応表

### 出席議員（テーブル形式）

```
議員の出欠別
議席番号 | 氏名 | 出欠別
1 | 與那嶺 雅晴 | 出
2 |     欠 員    |
3 | 玉城 保弘   | 出
...
出席議員５名
```

- 議員定数 6 名の小規模議会
- 議席番号・氏名・出欠が表形式

### 発言者パターン

```
○　當山清彦議長
○　新里武広村長
○　３番　玉城保弘議員
○　金城　満教育長
○　新垣　聡総務課長
```

- `○` + 全角スペース + 氏名 + 役職（議長・村長・議員等）
- 議員の場合: `○` + 全角スペース + 議席番号 + `番` + 全角スペース + 氏名 + `議員`
- 役職者の場合: `○` + 全角スペース + 氏名 + 役職名
- 発言内容は発言者行の次の行から続く

### 会議の流れ

1. 開会宣告（議長）
2. 会議録署名議員の指名
3. 会期の決定
4. 議長諸般の報告
5. 村長行政報告
6. 一般質問
7. 議案審議

## スクレイピング戦略

### Step 1: 年度別一覧ページから個別ページ URL を収集

年度別一覧ページ（`/1/{年度パス}/index.html`）をクロールし、各会議ページの URL を収集する。

**収集方法:**

1. 年度パスのリストをハードコード（`reiwa_8`, `reiwa7`, `reiwa6nen`, `5`, `4`, `3`, `2`, `1`）
2. 各年度一覧ページの `<li><a>` から個別ページ URL を抽出
3. リンクテキストから「会議録」を含むページのみをフィルタリング（議決案・会期日程ページを除外）

### Step 2: 個別ページから PDF URL を収集

各個別ページ（`/{ID}.html`）にアクセスし、PDF ファイルへのリンクを抽出する。

**収集方法:**

1. ページ内の `<a>` タグから `.pdf` で終わるリンクを抽出
2. プロトコル相対 URL（`//www.vill.tokashiki.okinawa.jp/...`）を `https:` 付きに変換
3. ページタイトルやリンクテキストから会議名・開催月を抽出

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

**抽出方法:**

1. PDF パーサー（pdf-parse 等）でテキストを抽出
2. MS-Mincho フォントが埋め込まれた日本語 PDF のため、テキスト抽出可能
3. 表紙・目次・出欠表ページをスキップし、本文ページからテキストを取得

### Step 4: 会議録のパース

#### メタ情報の抽出

表紙から以下を抽出:

```
令和７年
渡嘉敷村議会会議録
第７回定例会（12月10日〜12日）
```

- 年号: `令和{N}年` or `平成{N}年`
- 会議種別: `第{N}回定例会` or `第{N}回臨時会`
- 開催日: `{M}月{D}日〜{D}日`

出席議員ページから:

- 開催日時: `令和７年12月10日(水)午前10時00分`
- 出席議員名・議席番号
- 出席説明員（村長・副村長・教育長・課長等）

#### 発言の抽出

```typescript
// 発言者の抽出（役職者）
const speakerOfficialPattern = /^○\s+(.+?)(議長|村長|副村長|教育長|課長)$/;
// 例: ○　當山清彦議長 → name="當山清彦", role="議長"

// 発言者の抽出（議員）
const speakerMemberPattern = /^○\s+(\d+)番\s+(.+?)議員$/;
// 例: ○　３番　玉城保弘議員 → number="３", name="玉城保弘"

// 開催日時の抽出
const datePattern = /(令和|平成)(\d+)年(\d+)月(\d+)日/;

// 会議種別の抽出
const sessionPattern = /第(\d+)回(定例会|臨時会)/;
```

## 注意事項

- PDF ファイルのみで会議録が公開されているため、HTML パースではなく PDF テキスト抽出が必要
- 年度パスの命名規則に一貫性がないため、新年度追加時にパスの確認が必要
- 議員定数 6 名の小規模議会であり、データ量は少ない
- 年度一覧には「会議録」以外に「議決案」「会期日程」「要望決議」等のページも含まれるため、フィルタリングが必要
- PDF のファイル名にも一貫した命名規則がない（例: `reiwa7_12kaigiroku.pdf`, `reiwa7_3teireikai1.pdf`）
- `（「異議なし」の声あり）` などの記録表現が含まれる

---

## 推奨アプローチ

1. **年度パスのハードコード**: URL パスに規則性がないため、既知の年度パスをリストとして管理する
2. **PDF テキスト抽出**: `pdf-parse` 等のライブラリで PDF からテキストを抽出する
3. **会議録ページのフィルタリング**: リンクテキストに「会議録」を含むページのみを対象とする
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別にクロール済みの URL を記録し、新規追加分のみを取得する
