# 多賀町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.taga.lg.jp/
- 分類: 町役場ウェブサイト上での PDF 直接公開（専用検索システムなし）
- 文字コード: UTF-8
- 特記: 年度ごとに別ページが存在し、各ページに定例会・臨時会ごとの PDF リンクが列挙される

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 年度一覧（カテゴリ） | `https://www.town.taga.lg.jp/category_list.php?frmCd=4-5-0-0-0` |
| 年度別会議録一覧 | `https://www.town.taga.lg.jp/contents_detail.php?co=cat&frmId={frmId}&frmCd={frmCd}` |
| PDF ファイル | `https://www.town.taga.lg.jp/cmsfiles/contents/{prefix}/{frmId}/{ファイル名}.pdf` |

### 年度別ページ一覧

| 年度 | frmId | frmCd |
| --- | --- | --- |
| 令和7年（2025年） | 2116 | 4-5-5-0-0 |
| 令和6年（2024年） | 1997 | 4-5-4-0-0 |
| 令和5年（2023年） | 1838 | 4-5-3-0-0 |
| 令和4年（2022年） | 1754 | 4-5-2-0-0 |
| 令和3年（2021年） | 1735 | 4-5-1-0-0 |

- frmCd のパターンは `4-5-{連番}-0-0` で年度ごとに増加する
- frmId は不規則な値のため、年度一覧ページから都度取得する必要がある

### PDF ファイル名パターン

年度によってファイル名の命名規則が異なる：

| 年度 | 命名規則 | 例 |
| --- | --- | --- |
| 令和3年（一部） | `R{YY}{MM}{DD}.pdf`（和暦2桁） | `R030305.pdf`（令和3年3月5日） |
| 令和4年（一部） | `R{YY}{MM}{DD}.pdf` または `{YYYYMMDD}.pdf` | `R040304.pdf` / `20220607.pdf` |
| 令和7年以降 | `{YYYYMMDD}.pdf`（西暦8桁） | `20250304.pdf` |

- 同一年度内でも和暦形式・西暦形式が混在する場合がある（令和4年）
- PDF は `cmsfiles/contents/{prefix}/{frmId}/` 配下に格納される
- prefix は `0000001`（古い年度）または `0000002`（新しい年度）など

## 会議録の構成

### 定例会（年4回）

各定例会は複数の開催日で構成される：

- 3月定例会: 開会日・一般質問日（複数）・閉会日
- 6月定例会: 開会日・一般質問日（複数）・閉会日
- 9月定例会: 開会日・一般質問日（複数）・閉会日
- 12月定例会: 開会日・一般質問日（複数）・閉会日

### 臨時会

- 開催月・回数は年度によって異なる（年2〜5回程度）
- 各臨時会は開会日のみ記録が残る場合が多い

### ページネーション

なし。1つの年度ページに全 PDF リンクが列挙される。

## スクレイピング戦略

### Step 1: 年度一覧ページの取得

`https://www.town.taga.lg.jp/category_list.php?frmCd=4-5-0-0-0` を取得し、年度別ページの `frmId` と `frmCd` を抽出する。

- リンクテキストが「令和X年」「平成XX年」等に一致する `<a>` タグを抽出
- href から `frmId` と `frmCd` を取得する

### Step 2: 年度別ページの取得

各年度の一覧ページ（`contents_detail.php?co=cat&frmId={frmId}&frmCd={frmCd}`）を取得し、PDF リンクを抽出する。

- `href` が `./cmsfiles/contents/` で始まる `<a>` タグを全て収集
- リンクテキスト（例: 「3月5日開会」「6月7日一般質問」）も合わせて取得し、開催日・会期の情報として活用する

### Step 3: PDF の取得とメタ情報の抽出

各 PDF ファイルを取得してテキストを抽出する。

#### メタ情報の取得

リンクの配置から以下を推定できる：

- 会期（定例会・臨時会）: 会期の見出し（H2 等）を参照
- 開催月: 定例会は3・6・9・12月
- 開催日: ファイル名またはリンクテキストから抽出

#### PDF 本文のパース

多賀町の会議録 PDF は一般的な地方議会形式に従う。発言者と発言内容の抽出には以下のパターンが参考となる：

```
◯議長（氏名）
◯町長（氏名）
◯X番（氏名）
```

---

## 注意事項

- **ファイル名の命名規則が年度・会期によって混在**する。和暦形式（`R{YY}{MM}{DD}.pdf`）と西暦形式（`{YYYYMMDD}.pdf`）が混在するため、ファイル名から日付を推定する場合は両パターンに対応する必要がある。
- **PDF ストレージの prefix が変化**する。`cmsfiles/contents/0000001/` から `cmsfiles/contents/0000002/` に変わる年度がある。これはリンクタグの href から直接取得するため、URL を手動構築する必要はない。
- **年度ページは年度一覧から動的に取得**する。frmId は不規則なため、年度一覧ページを毎回クロールして正確な URL を取得すること。
- **令和3年以前の会議録は未確認**。平成年度の会議録が存在する場合、年度一覧ページに追加されている可能性があるため、カテゴリ一覧を都度確認する。

---

## 推奨アプローチ

1. **年度一覧を起点にする**: `category_list.php?frmCd=4-5-0-0-0` を基準に、年度別ページの URL を動的に収集する
2. **PDF リンクを直接収集**: 各年度ページから `cmsfiles/contents/` へのリンクを全件取得し、ファイル名と会期情報を紐づける
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度別ページの更新日時または既取得 PDF のファイル名リストを管理し、新規 PDF のみを取得する
