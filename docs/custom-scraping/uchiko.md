# 内子町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.uchiko.ehime.jp/site/kaigiroku/
- 分類: 公式ホームページによる PDF 公開（ダウンロード形式）
- 文字コード: UTF-8（XHTML 1.0 Transitional、`xml:lang="ja"`）
- 特記: テキスト検索システムなし。会議録は PDF のみで提供。インターネット議会中継は kensakusystem.jp で別途提供されているが、テキスト全文検索には対応していない。

## PDF リンクパターン

### 会議録詳細ページの構造

各会議録詳細ページ（`{ページID}.html`）には、会議録 PDF へのリンクが埋め込まれている。

```html
<a href="/uploaded/life/{ページID}_{ファイルID}_misc.pdf">...</a>
```

### 実例

| ページ | PDF URL |
| --- | --- |
| `/site/kaigiroku/145604.html`（令和7年12月定例会） | `/uploaded/life/145604_327831_misc.pdf`（1.64MB） |
| `/site/kaigiroku/145603.html`（令和7年10月臨時会） | `/uploaded/life/145603_327830_misc.pdf`（316KB） |
| `/site/kaigiroku/145423.html`（令和7年9月定例会） | `/uploaded/life/145423_326171_misc.pdf`（1.97MB） |

### ファイル命名規則

`{ページID}_{ファイルID}_misc.pdf`

- `ページID`: 詳細ページの HTML ファイル名と同じ数値
- `ファイルID`: CMS が付与するファイル管理 ID（連番）
- `_misc`: 固定サフィックス

1 会議録につき PDF は基本 1 ファイル（複数日程にわたる場合も統合 PDF として提供）。

## スクレイピング戦略

### Step 1: 全年度の会議一覧ページ URL を収集

`list591.html`（全年度一覧）にアクセスし、`list{カテゴリID}-{サブID}.html` 形式のリンクを全件抽出する。

```
対象 URL パターン: /site/kaigiroku/list\d+-\d+\.html
```

### Step 2: 各会議種別一覧ページから詳細ページ URL を収集

各 `list{カテゴリID}-{サブID}.html` にアクセスし、会議録詳細ページ（`/site/kaigiroku/{ページID}.html`）へのリンクを抽出する。

```
対象 URL パターン: /site/kaigiroku/\d+\.html
（list から始まらない数値のみのページ ID）
```

### Step 3: 各会議録詳細ページから PDF URL を収集

各詳細ページにアクセスし、`/uploaded/life/` で始まる PDF ファイルへのリンクを抽出する。

```
対象 URL パターン: /uploaded/life/\d+_\d+_misc\.pdf
```

### Step 4: PDF のダウンロードとテキスト抽出

取得した PDF を OCR またはテキスト抽出ツールで処理する。

## メタ情報の取得

詳細ページ（`{ページID}.html`）のタイトルや本文から以下のメタ情報を抽出できる。

| 情報 | 取得元 | 例 |
| --- | --- | --- |
| 会議名 | ページタイトル / h1 | 「令和７年１２月第１５２回内子町議会定例会会議録」 |
| 会議種別 | 会議名のパース | 定例会 / 臨時会 |
| 開催年度 | 会議名のパース | 令和7年 |
| 開催月 | 会議名のパース | 12月 |
| 回次 | 会議名のパース | 第152回 |

#### 会議名のパース用正規表現（案）

```typescript
// 例: 「令和７年１２月第１５２回内子町議会定例会会議録」
const pattern = /(?:令和|平成)(\d+)年(\d+)月第(\d+)回内子町議会(.+?)会議録/;
// → 年号期, 年, 月, 回次, 会議種別
```

## ページネーション

**なし。** 全年度の一覧は `list591.html` 単一ページに収録されており、ページネーションは存在しない。各会議種別一覧ページも単一ページ。

---

## 注意事項

- 会議録は **PDF のみ** の提供。HTML テキスト版は存在しない。全文テキスト検索機能もなし。
- PDF からのテキスト抽出には OCR が必要になる可能性がある（スキャン PDF の場合）。
- 令和8年1月臨時会（`list613-3417.html`）のように、年度によってカテゴリ ID が変わる（令和7年: 591、令和8年: 613）。全年度一覧ページ（`list591.html`）から動的に URL を収集することが重要。
- こども議会の会議録も一覧に含まれているため、会議種別フィルタリングが必要な場合は注意。
- リクエスト間には適切な待機時間（1〜2 秒）を設ける。

---

## 推奨アプローチ

1. **全量取得を優先**: `list591.html` を起点に全 `list{カテゴリID}-{サブID}.html` を収集し、各詳細ページの PDF URL を網羅的に取得する。
2. **差分更新**: 詳細ページの ID は増加傾向にあるため、前回取得済みの最大ページ ID を記録し、新規追加分のみを取得する差分更新が可能。
3. **PDF テキスト抽出**: PDF ファイルを pdfminer・pdfjs 等でテキスト抽出し、テキスト化できない場合は OCR（Tesseract 等）を適用する。
