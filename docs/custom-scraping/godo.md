# 神戸町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.godo.gifu.jp/contents/gikai/gikai08.html
- 分類: 独自 CMS による PDF 公開（専用の会議録検索システムなし）
- 文字コード: UTF-8
- 特記: 会議録はすべて PDF 形式で公式サイトに掲載。年度別・回数別で整理されており、専用の全文検索システムは導入されていない。令和6年（2024年）以降のみ掲載。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧 | `https://www.town.godo.gifu.jp/contents/gikai/gikai08.html` |
| 会議録 PDF | `https://www.town.godo.gifu.jp/contents/gikai/pdf/kaigiroku/{年度コード}_{回数}.pdf` |

### PDF URL の命名規則

`{年度コード}_{回数}.pdf` の形式。

- 年度コード: `r6`（令和6年）、`r7`（令和7年）のように `r` + 和暦年号2桁以内
- 回数: `1`〜`9` の数値（その年の第N回会議）

例:
| PDF ファイル名 | 対応する会議 |
| --- | --- |
| `r6_1.pdf` | 令和6年 第1回定例会（3月） |
| `r6_2.pdf` | 令和6年 第2回臨時会（4月） |
| `r6_5.pdf` | 令和6年 第5回定例会（12月） |
| `r7_1.pdf` | 令和7年 第1回臨時会（1月） |
| `r7_9.pdf` | 令和7年 第9回臨時会（12月） |

## 会議録一覧ページの構造

`gikai08.html` に全年度分を1ページで掲載（ページネーションなし）。

- 年度ごとに見出しで区切られ、各年度内に会議回数分のリンクが箇条書きで並ぶ
- 各リンクに「第N回定例会（M月）」または「第N回臨時会（M月）」の形式で表示
- リンク先は相対パス `pdf/kaigiroku/{年度コード}_{回数}.pdf`

---

## PDF の仕様

- 形式: PDF 1.6
- 用紙サイズ: A4（595.32 × 841.92 pt）
- 作成ソフト: Microsoft® Word（2016 または 2024）
- 作成者: 議会事務局 PC02_R
- ページ数: 100〜120 ページ程度（会議期間・議案数による）
- テキスト抽出: 可能（`pdftotext` 等で抽出可）
- 暗号化: なし

## PDF の内部構造

### 目次

PDF 冒頭に「令和X年第N回神戸町議会定例会会議録目次」として各号の日付・議事内容のページ番号が記載される。

```
令和７年第２回神戸町議会定例会会議録目次

第１号（３月３日）
  開会 …… ３
  会議録署名議員の指名について …… ３
  ...

第２号（３月12日）
  一般質問 …… ５６
    深 貝 仁 則 君 …… ５６
    ...
```

### メタ情報

会議録冒頭から以下を抽出できる:

```
令和 ７ 年 ３ 月 ３ 日  開会
令和 ７ 年 ３ 月 13 日  閉会

岐阜県神戸町議会
令和７年第２回神戸町議会定例会会議録目次
```

- 開催日・閉会日: `令和X年X月X日` 形式（スペース区切りで抽出される）
- 会議名: `令和X年第N回神戸町議会{定例会|臨時会}会議録`

### 出席者情報

各号の本文冒頭に出席議員・欠席議員・説明員の一覧が記載される:

```
出席議員（１０名）
議  長  飯  沼  満  君   副議長  小  川  榮  一  君
１  番  深  貝  仁  則  君   ２  番  大  場  光  晴  君
...

地方自治法第１２１条第１項の規定により説明のため会議に出席した者の職・氏名
町  長  藤  井  弘  之  君
副  長  金  指  義  樹  君
...
```

氏名は縦組み風に文字間スペースが挿入されているため、テキスト抽出後に正規化が必要。

### 発言者パターン

`○` で始まる行が発言者を示す:

```
○議長（飯沼　満君）
○副議長（小川榮一君）
○副町長（金指義樹君）
○町長（藤井弘之君）
○教育長（宇野秀宣君）
○総務部長兼総務課長兼危機管理監（河出真志君）
○産業建設部長兼企業誘致推進室長（土屋典生君）
○５番（西脇博文君）
○８番（飯沼　満君）
```

- `○` + 役職 or 議席番号 + `（` + 氏名 + `君）` の形式
- 氏名末尾は常に `君）` で終わる（男女問わず「君」敬称）
- 役職が長い場合（兼務等）はそのまま記載
- 議席番号は半角数字で「N番」形式

動議・会場反応の記述:

```
〔「異議なし」と呼ぶ者あり〕
〔挙手する者あり〕
```

- `〔` 〜 `〕` で囲まれた行が場内反応・動作の記述

## スクレイピング戦略

### Step 1: PDF URL の収集

`gikai08.html` を取得し、`pdf/kaigiroku/` 以下へのリンクを全件抽出する。

1. HTML 内の `<a href="pdf/kaigiroku/...pdf">` を Cheerio 等で抽出
2. 相対パスをベース URL `https://www.town.godo.gifu.jp/contents/gikai/` で絶対パスに変換
3. 収集した PDF URL と表示テキスト（「第N回定例会（M月）」等）をセットで管理

**注意**: ページネーションなし。一覧ページ1件のみのリクエストで全 URL を収集可能。

### Step 2: PDF のダウンロードとテキスト抽出

収集した各 PDF URL にアクセスし、テキストを抽出する。

1. PDF をダウンロード
2. `pdftotext`（poppler）または `pdf-parse`（Node.js）等でテキスト抽出
3. テキスト抽出後、氏名に挿入されたスペースを正規化する（例: `飯  沼  満  君` → `飯沼満君`）

### Step 3: 会議録のパース

抽出したテキストを以下の手順で構造化する:

1. 冒頭の開催日・閉会日・会議名を正規表現で抽出
2. `○` で始まる行を発言者として識別し、続くテキストを発言内容として紐付ける
3. `〔...〕` 行は場内反応として別途記録

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○５番（西脇博文君） → role="５番", name="西脇博文"
// 例: ○議長（飯沼　満君） → role="議長", name="飯沼　満"

// 場内反応の検出
const reactionPattern = /^〔(.+?)〕/;

// 開催日の抽出
const datePattern = /(?:令和|平成)\s*(\d+)\s*年\s*(\d+)\s*月\s*(\d+)\s*日/;

// 会議名の抽出
const titlePattern = /令和(\d+)年第(\d+)回神戸町議会(定例会|臨時会)/;
```

## 注意事項

- 現状、一覧ページには令和6年（2024年）以降のみ掲載されており、それ以前の PDF は404エラーとなる（令和5年以前は公開されていないと判断）
- PDF テキスト抽出後、氏名・役職名の漢字間にスペースが挿入されるため正規化が必要
- 年度によって回数が異なる（令和6年は5回、令和7年は9回確認）
- 同一年度内に定例会・臨時会が混在する
- 会議録は1回の会議期間（複数日）分がまとめて1つの PDF に収録される
- PDF 作成ソフトは Word 2016〜2024 と年度によって異なるが、テキスト抽出の方法は同一で問題ない
- リクエスト間に適切な待機時間（1〜2 秒）を設けること

---

## 推奨アプローチ

1. **一覧ページを起点**: `gikai08.html` の1ページから全 PDF URL を収集できる
2. **PDF 全量取得**: 一覧ページに掲載されている全 PDF をダウンロードしてテキスト抽出
3. **氏名スペースの正規化**: `pdftotext` 出力の氏名に含まれる余分なスペースを除去する前処理を実施
4. **差分更新**: 一覧ページを定期チェックし、未取得の PDF URL のみダウンロード・処理
5. **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒のウェイトを設ける
