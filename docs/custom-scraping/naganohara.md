## PDF ファイル命名規則

### 平成27年〜平成30年

```
H{和暦2桁}{月2桁}_{区分}.pdf
```

- `_T`: 定例会
- `_R`: 臨時会

例:
- `H2703_T.pdf` → 平成27年3月定例会
- `H2702_R.pdf` → 平成27年2月臨時会

### 令和元年〜令和7年

```
R{和暦2桁}{月2桁}.pdf
```

- 定例会・臨時会の区別サフィックスなし

例:
- `R0503.pdf` → 令和5年3月（定例会）
- `R0612.pdf` → 令和6年12月（定例会）

## 会議録 PDF の構造

### メタ情報（PDF 冒頭部分）

```
令和６年第４回定例会

長 野 原 町 議 会 会 議 録
令 和 ６ 年 12月３日   開会
令 和 ６ 年 12月12日   閉会
```

- 1 PDF = 1 定例会（または臨時会）の全日程分
- 目次ページに各号（日程ごと）の議事日程が記載される
- 出席議員、欠席議員、説明出席者、事務局職員の一覧が各号の冒頭にある

### 発言者パターン

```
○議長（黒岩　巧君）
○町長（萩原睦男君）
○１０番（牧山　明君）
○総務文教常任委員長（入澤信夫君）
○産業建設常任委員長（牧山　明君）
○副町長（梶野寛丈君）
○教育長（小林敦子君）
```

- `○` で始まり、役職または議席番号 + 括弧内に氏名（`君` 付き）
- 議席番号は全角数字
- 氏名中にスペース（全角空白）が入る場合がある

### 登壇表記

```
〔町長　萩原睦男君　登壇〕
〔総務文教常任委員長　入澤信夫君　登壇〕
```

- `〔` `〕` で囲まれ、役職 + 氏名 + `登壇` の形式

### その他の議場表記

```
〔「なし」と呼ぶ者あり〕
〔「異議なし」と呼ぶ者あり〕
〔発言する者なし〕
```

### セクション区切り

```
─────────────────────────────────────────────────
◎散会について
◎開議の宣告
◎議事日程の報告
```

- `◎` で始まるセクション見出し
- 罫線（`─` の連続）でセクションが区切られる

## スクレイピング戦略

### Step 1: 年度別ページから PDF URL の収集

会議録トップページから年度別ページへのリンクを取得し、各年度ページ内の PDF リンクを収集する。

**収集方法:**

1. トップページ (`/www/genre/1461123870183/index.html`) から年度別ページの URL を取得
2. 各年度ページの HTML をパースし、`<a>` タグの `href` から `.pdf` リンクを抽出
3. リンクテキストから会議名（定例会 / 臨時会）と月を取得

### Step 2: PDF のダウンロード

収集した URL から PDF をダウンロードする。

- PDF サイズは 200KB〜24MB と幅がある（定例会は大きく、臨時会は小さい）
- 全年度合計で概算 80〜100 ファイル程度

### Step 3: PDF からのテキスト抽出

PDF はテキスト埋め込み型（Microsoft Word 2013 由来）のため、`pdftotext` 等でテキスト抽出可能。

- OCR は不要
- ただし Word 由来のレイアウトにより、テキスト抽出時に不要な空白や改行が入る場合がある
- 目次ページ・表紙ページはスキップが必要

### Step 4: テキストのパース

#### メタ情報の抽出

```typescript
// 会議名・回次の抽出
const sessionPattern = /(?:令和|平成)[０-９\d]+年第[０-９\d]+回(定例会|臨時会)/;

// 開催日の抽出（各号冒頭の議事日程から）
const datePattern = /(?:令和|平成)[０-９\d]+年[０-９\d]+月[０-９\d]+日/;
```

#### 発言者の抽出

```typescript
// 発言者パターン
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○議長（黒岩　巧君） → role="議長", name="黒岩　巧君"
// 例: ○１０番（牧山　明君） → role="１０番", name="牧山　明君"

// 登壇の検出
const podiumPattern = /^〔(.+?)　(.+?)　登壇〕/;

// セクション見出し
const sectionPattern = /^◎(.+)/;
```

#### 号（日程）の分割

1 つの PDF に複数日程が含まれるため、`第１号（12月３日）` / `第２号（12月12日）` のようなパターンで分割する。

```typescript
const issuePattern = /第[０-９\d]+号[（(][０-９\d]+月[０-９\d]+日[）)]/;
```

## 注意事項

- PDF のみの提供であり、HTML 本文ページは存在しない
- PDF のテキスト抽出時、Word 由来のレイアウトにより文字間に不要なスペースが入ることがある（例: `長 野 原 町 議 会`）
- 全角数字と半角数字が混在する（表紙は全角、本文中は半角の場合あり）
- 氏名に全角スペースが含まれる場合がある（例: `黒岩　巧君`）
- 平成27〜30年は PDF ファイル名に `_T`（定例会）/ `_R`（臨時会）の区分サフィックスがあるが、令和以降はない
- ページネーションは年度別ページ・トップページともになし

---

## 推奨アプローチ

1. **2 段階クロール**: トップページ → 年度ページ → PDF URL の順で全量リストを作成
2. **PDF テキスト抽出**: `pdftotext` または `pdf-parse` 等で抽出（OCR 不要）
3. **号単位で分割**: 1 PDF 内の複数日程を号パターンで分割し、各号ごとにメタ情報・発言を抽出
4. **テキスト正規化**: 抽出テキストの不要な空白・改行を正規化してからパースする
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
6. **差分更新**: 年度ページの PDF リンク数を前回と比較し、新規追加分のみダウンロードする
