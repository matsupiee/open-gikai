# 小鹿野町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.ogano.lg.jp/menyu/gikai/sinkaigiroku/index.html
- 分類: 独自 HTML 公開（フレームセット構成）
- 文字コード: Shift_JIS
- 会議録形式: HTML テキスト形式（PDF なし）
- 対象期間: 平成 21 年（2009 年）以降
- 特記: キーワード検索機能なし、会次番号方式


| --- | --- |

- `R0703T.html` → 令和 7 年 3 月 定例会

## ページ構成（フレームセット）

各会議録は 2 フレーム構成。

```
{ファイル名}.html        ← フレームセット定義
  ├── {ファイル名}_index.html  ← 左フレーム: 目次（アンカーリンク一覧）
  └── {ファイル名}main.html   ← 右フレーム: 本文テキスト（全会議日程を 1 ファイルに収録）
```

- 本文ファイル（`main.html`）1 つに 1 回分の定例会・臨時会の全日程が含まれる
- 各開会日は `<a name="0100">`, `<a name="0200">` のようにアンカーで区切られる
- 目次フレームのリンクは `main.html#0100` のような形式でアンカーを参照する

## スクレイピング戦略

### Step 1: 会議録ファイル一覧の収集

一覧ページ `index.html` を Shift_JIS でフェッチし、`<a href="{ファイル名}.html">` のリンクを抽出する。

```typescript
// 一覧ページから全ファイル名を抽出
const baseUrl = "https://www.town.ogano.lg.jp/menyu/gikai/sinkaigiroku/";
const indexUrl = `${baseUrl}index.html`;

// Shift_JIS デコードが必要
const res = await fetch(indexUrl);
const buffer = await res.arrayBuffer();
const text = new TextDecoder("shift_jis").decode(buffer);

// リンク抽出（.html かつ index.html 以外）
const links = [...text.matchAll(/href="([A-Z][A-Z0-9]+\.html)"/gi)]
  .map(m => m[1])
  .filter(name => !name.includes("_index") && !name.includes("main"));
```

### Step 2: 本文 HTML の取得

各会議録の本文は `{ファイル名}main.html` を直接取得する（フレームセット本体は不要）。

```typescript
const mainUrl = `${baseUrl}${fileName.replace(".html", "")}main.html`;
const res = await fetch(mainUrl);
const buffer = await res.arrayBuffer();
const text = new TextDecoder("shift_jis").decode(buffer);
```

### Step 3: 本文テキストのパース

本文 HTML は monospace テキスト形式。Cheerio 等で `<body>` テキストを取得し、以下の要素を解析する。

#### アンカー区切り

```
<a name="0100"> → 第1開会日の開始
<a name="0200"> → 第2開会日の開始
<a name="0300"> → 第3開会日の開始
（以降、開会日ごとに 100 番台でインクリメント）
```

#### メタ情報

ファイル名から年月・種別を判定できる。本文冒頭にも会議名が記載される。

```
令和７年第１回（３月）小鹿野町議会定例
```

#### 発言者パターン

```html
<b>議長（氏名）</b>　発言内容...
<b>10番（氏名）</b>　発言内容...
<b>町長（氏名）</b>　発言内容...
```

- 発言者は `<b>` タグで囲まれている
- 役職または議席番号 + 括弧内に氏名の形式

#### パース用正規表現（案）

```typescript
// 発言者の抽出（<b>タグ内）
const speakerPattern = /^(.+?)（(.+?)）/;
// 例: 議長（鈴木一郎） → role="議長", name="鈴木一郎"
// 例: 10番（山田花子） → role="10番", name="山田花子"

// 開催日の抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;

// 開会時刻の抽出
const timePattern = /午前(\d+)時(\d+)分開会|午前(\d+)時(\d+)分(\d+)秒開会/;
```

## 注意事項

- 文字コードは **Shift_JIS**。`fetch` → `ArrayBuffer` → `TextDecoder("shift-jis")` の手順でデコードする
- フレームセット HTML（`{ファイル名}.html`）は構成定義のみで本文を含まない。本文は `main.html` を直接取得する
- 平成 21 年分（H2105R01.html / H2105R02.html など）は同月に複数臨時会が開催されたケースがあり、末尾に `01`, `02` の連番が付く
- 令和元年分は年度の途中で年号が切り替わるため、同一年度内に `H3103T.html`（平成 31 年 3 月）と `R0106T.html`（令和元年 6 月）が混在する
- キーワード検索機能は存在しないため、全量取得を前提とする
- レート制限: リクエスト間に 1〜2 秒の待機を設ける

---

## 推奨アプローチ

1. **全量取得を優先**: 一覧ページから全ファイル名リストを取得し、`main.html` を順次フェッチする
2. **ファイル名ベースの差分更新**: ファイル名に年月情報が含まれるため、取得済みファイル名と照合して新規分のみ取得する差分更新が可能
3. **Shift_JIS デコード**: Node.js 標準の `TextDecoder("shift-jis")` または `iconv-lite` を使用する
4. **本文の直接フェッチ**: フレームセット HTML をパースする必要はなく、`{ファイル名}main.html` を直接フェッチすれば本文全文を取得できる
