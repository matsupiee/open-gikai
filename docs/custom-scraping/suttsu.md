# 寿都町議会 カスタムスクレイピング方針

## 概要

- サイト: http://www.town.suttu.lg.jp/town/detail.php?id=63
- 分類: 議会だより（PDF）の一覧ページ（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録検索システムは未導入。議会だより「寿都湾」を PDF で公開しており、議会の審議内容・一般質問・行政報告等はこの PDF に含まれる。Incapsula（WAF）導入済み。

## PDF ファイルの URL パターン

すべての PDF は以下の形式で公開されている:

```
http://www.town.suttu.lg.jp/common/img/content/cassette_{カセット番号}_pdf{連番}_{タイムスタンプ}.pdf
```

- `cassette_{N}`: グループ番号（1〜8）。10件ずつの区切りに対応
- `pdf{NN}`: グループ内の連番（01〜10）
- `{タイムスタンプ}`: `YYYYMMDD_HHMMSS` 形式のアップロード日時

### 例

```
cassette_1_pdf01_20160913_174423.pdf  → NO.140号（平成21年2月発行）
cassette_7_pdf09_20260205_090028.pdf  → NO.208号（令和8年2月発行）
```

## HTML 構造

議会だより一覧ページは `<ul class="pdf-list">` のリスト形式で構成されている。

```html
<ul class="pdf-list" id="content-1">
  <li><a href="../common/img/content/cassette_1_pdf01_20160913_174423.pdf" target="_blank">NO.140号（平成21年2月発行）</a></li>
  ...
</ul>
```

- 各 `<ul>` はカセット番号に対応（`id="content-1"` 〜 `id="content-7"`）
- `<li>` 内の `<a>` タグに PDF への相対パスリンクとリンクテキスト（号数・発行時期）が含まれる
- 検索フォームは存在しない

## PDF の内容構造

議会だより PDF は議会事務局・広報編集委員会が編集した広報紙形式。

### 含まれる情報

- 行政報告（町長の施政方針報告）
- 審議した案件（条例改正・予算・意見書等）の一覧と議決結果
- 一般質問（質問者名・質問内容・町長等の答弁要旨）
- 委員会報告
- 議員名簿・出席状況

### 一般質問のフォーマット（PDF 内）

```
■ 質問
{議員名}　議員

{質問タイトル}

{質問本文}

● 町長
{答弁本文}

■ 再質問
{再質問本文}

● 町長
{再答弁本文}
```

- 質問者は「{氏名}　議員」の形式
- 答弁者は「● {役職}」の形式（町長、教育長 等）
- 再質問・再答弁のやり取りが続く場合がある

## スクレイピング戦略

### Step 1: PDF リンクの収集

議会だより一覧ページ `detail.php?id=63` から全 PDF リンクを抽出する。

**収集方法:**

1. 一覧ページの HTML を取得
2. `<ul class="pdf-list">` 内の `<a>` タグから `href` 属性を抽出
3. 相対パス（`../common/img/content/...`）をベース URL と結合して絶対 URL に変換
4. リンクテキストから号数・発行時期をメタ情報として抽出

**メタ情報の抽出（リンクテキストから）:**

```typescript
// リンクテキスト例: "NO.208 号（令和8年2月発行）"
const issuePattern = /[NＮ][OＯ][.．]?\s*[０-９\d]+\s*号/;
const datePattern = /(令和|平成)([０-９\d]+)年([０-９\d]+)月/;
```

### Step 2: PDF のダウンロードとテキスト抽出

1. 収集した PDF URL からファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出
3. 議会だよりは段組み・写真入りの広報紙形式のため、テキスト抽出時にレイアウト情報が失われる可能性がある

### Step 3: テキストのパース

#### メタ情報

PDF 1ページ目のヘッダーから以下を抽出:

```
寿都湾 No.208 令和8年2月
令和7年 第4回定例会
```

- 号数: `No.{数字}`
- 発行年月: `令和{N}年{M}月` or `平成{N}年{M}月`
- 定例会情報: `令和{N}年 第{M}回定例会`

#### 一般質問の抽出

```typescript
// 質問者の抽出
const questionerPattern = /(.+?)　議員/;

// 答弁者の抽出
const responderPattern = /●\s*(.+)/;
// 例: ● 町長 → "町長"
```

## 注意事項

- **会議録検索システムは未導入**: 寿都町は会議録検索システムを導入しておらず、議会だより（広報紙）の PDF のみが公開されている
- **議会だよりは会議録の要約版**: 全文の会議録ではなく、議会事務局が編集した要約であるため、発言の全文は含まれない
- **PDF は広報紙レイアウト**: 段組み・写真・図表が含まれており、テキスト抽出の精度が通常の会議録 PDF より低くなる可能性がある
- **号数の表記ゆれ**: `NO.` `ＮＯ.` `NO.` など全角・半角の混在がある
- **Incapsula（WAF）導入**: スクレイピング時にブロックされる可能性がある
- **担当部署**: 寿都町議会事務局 TEL 0136-62-2511 内線71

## 推奨アプローチ

1. **一覧ページから全 PDF リンクを一括取得**: ページネーションがないため、1回のリクエストで全号のリンクを取得可能
2. **PDF テキスト抽出の精度検証**: 広報紙レイアウトのため、pdf-parse だけでなく OCR ベースの抽出（Tesseract 等）も検討する
3. **レート制限**: Incapsula（WAF）導入のため、リクエスト間に適切な待機時間（2〜3 秒）を設ける
4. **差分更新**: 号数（NO.xxx）を基準に管理し、前回取得済みの最大号数以降の PDF のみを取得する
5. **一般質問を優先抽出**: 議会だよりの中で最も構造化しやすいのは一般質問セクションであるため、まずこの部分のパースを実装する
