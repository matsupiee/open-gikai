# 宿毛市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.sukumo.kochi.jp/01/04/02/02/
- 分類: 公式サイトで年度別に PDF を直接リンク提供（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 会議録検索システムなし。年度別一覧ページから PDF へ直リンク

定例会は年4回（3月・6月・9月・12月）、臨時会は年によって0〜6回。

| 種別 | 主な開催月 |
| --- | --- |
| 定例会（第1回） | 3月 |
| 定例会（第2回） | 6月 |

| 定例会（第4回） | 12月 |
| 臨時会 | 1月・2月・4月・5月・7月・8月・10月・11月（年度によって異なる） |

## PDF の命名規則

PDF のファイルパスは `/fs/{数字スラッシュ区切りのパス}/{ファイル名}.pdf` の形式。ファイル名は時期によって命名規則が異なり、統一されていない。

| 時期 | 命名パターン例 | 備考 |
| --- | --- | --- |
| 令和6〜7年（新形式） | `_____6__1_3____.pdf` | 全角・アンダースコア混在 |
| 令和5年（混在） | `r0503.pdf` / `__5__1_________________.pdf` | 2形式が混在 |
| 平成30年前後 | `h3001.pdf` / `h30r1.pdf` | `h{2桁年}{2桁月}.pdf` 形式 |
| 平成17年（旧形式） | `01_h17r1.pdf` | 連番プレフィックス付き |

ファイル名からメタ情報を機械的に解析するのは困難であり、各年度ページの HTML からリンクテキスト（会議名）と PDF URL を対で取得する方針が適切。

## スクレイピング戦略

### Step 1: 年度別一覧ページの PDF リンク収集

上記の年度別 URL リストを使い、各年度ページから PDF リンクとそのラベル（会議名）を抽出する。

**収集方法:**

1. 上記の年度 URL リストを順に取得
2. 各ページの `<a href="...pdf">` リンクと、そのリンクテキストを Cheerio で抽出
3. リンクテキストから会議名・年度・会議種別（定例会 or 臨時会）・回次・開催月を正規表現でパース

**抽出対象の HTML 要素例:**

```html
<a href="/fs/7/0/0/5/0/_/_____6__1____.pdf">令和6年第1回臨時会会議録(令和6年1月開催)</a>
```

### Step 2: PDF のダウンロードとテキスト抽出

取得した PDF URL から PDF をダウンロードし、テキストを抽出する。

- PDF はすべて `https://www.city.sukumo.kochi.jp` を基底 URL として絶対 URL に変換
- PDF のテキスト抽出には `pdf-parse` 等を使用

### Step 3: メタ情報のパース

リンクテキストからメタ情報を抽出する。

**リンクテキストのパターン:**

```
令和6年第1回臨時会会議録(令和6年1月開催)
令和7年第2回定例会会議録
平成17年第3回定例会
```

**パース用正規表現（案）:**

```typescript
// 年号・年・回次・種別の抽出
const titlePattern = /^(令和|平成)(\d+)年第(\d+)回(定例会|臨時会)/;
// 例: "令和6年第1回臨時会" → era="令和", year=6, session=1, type="臨時会"

// 開催月の抽出（括弧内）
const monthPattern = /(?:令和|平成)\d+年(\d+)月開催/;
// 例: "(令和6年1月開催)" → month=1
```

---

## 注意事項

- PDF ファイルパス（`/fs/...`）はサーバー内部のディレクトリ構造を反映しており、ファイル名からメタ情報を推測することは困難
- 年度ページの URL に `p` プレフィックス型・数値型・特殊形式（`P300608-2.html`）が混在するため、URL を動的に探索せず固定リストを使う
- リンクテキストに開催月が含まれない場合もあるため、その場合は会議種別・回次から推定する
- 平成17年は臨時会が6回と多く、古い年度ほど臨時会の頻度が高い傾向あり

---

## 推奨アプローチ

1. **固定 URL リストを使用**: 年度別ページは URL を動的探索せず、上記の固定リストから処理する
2. **リンクテキストを優先**: ファイル名ではなくリンクテキストからメタ情報を抽出する
3. **PDF URL の絶対化**: `/fs/...` 形式の相対 URL を `https://www.city.sukumo.kochi.jp/fs/...` に変換
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2秒）を設ける
5. **アーカイブ範囲**: 平成17年（2005年）〜令和7年（2025年）の全21年度が対象
