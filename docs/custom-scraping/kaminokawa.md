## ページ構造

### 会議録トップページ (`genre2-0-001.html`)

- 年度別リンク（令和6年〜平成28年）が一覧表示される
- 最新の会議録が「最新記事」として冒頭に表示される
- ページネーションなし

### 年度別一覧ページ (`genre3-0-001.html`)

- 各年度内の会議（定例会・臨時会）がリスト形式で表示される
- 各リンクテキスト例: 「令和6(2024)年第6回議会定例会(12月)会議録」
- ページネーションなし

### 会議詳細ページ (`info-{ページID}-0.html`)

- テーブル形式で日程ごとの情報を表示
- 列構成: 「日にち」「曜日」「審議内容」
- 「日にち」列に PDF ファイルへの直リンクが含まれる
- PDF ファイル名はランダムハッシュ値（例: `67b29378f06e1.pdf`）

| 回次 | 種別 | 月 |

| 第1回 | 臨時会 | 1月 |

- 臨時会: 年によって回数・時期は異なる

## スクレイピング戦略

### Step 1: 年度別一覧ページから会議リンクを収集

会議録トップページ `genre2-0-001.html` から年度別 URL を取得し、各年度ページから会議詳細ページの URL を収集する。

**収集方法:**

1. トップページの年度リンク（`genre3-0-001.html`）を取得
2. 各年度ページ内の `info-{ページID}-0.html` リンクをすべて抽出
3. リンクテキストから会議名・種別・月を正規表現で抽出

```typescript
// 会議名の抽出
const meetingPattern = /(.+?)年第(\d+)回議会(定例会|臨時会)\((\d+月)\)会議録/;
// 例: 令和6(2024)年第6回議会定例会(12月)会議録
//   → 年="令和6(2024)", 回次="6", 種別="定例会", 月="12月"
```

### Step 2: 会議詳細ページから PDF リンクを収集

各会議詳細ページ（`info-{ページID}-0.html`）のテーブルから PDF リンクを抽出する。

**収集方法:**

1. テーブル内の `<a>` タグから `manage/contents/upload/*.pdf` パターンの href を抽出
2. 同じ行の「日にち」「審議内容」をメタ情報として保持

```typescript
// PDF URL パターン
const pdfUrlPattern = /manage\/contents\/upload\/[a-f0-9]+\.pdf/;
```

### Step 3: PDF のダウンロードとテキスト抽出

PDF ファイルをダウンロードし、テキストを抽出する。

**注意点:**

- PDF ファイル名はランダムハッシュ値のため、URL から会議日を特定できない
- テーブルの行情報（日にち・審議内容）と PDF を紐づけて管理する必要がある
- PDF のテキスト抽出には `pdf-parse` 等のライブラリを使用

---

## 注意事項

- PDF ファイル名がランダムハッシュ値のため、ファイル名からメタ情報を推測できない
- 年度別のジャンル ID に規則性がないため、トップページからの動的取得が必要
- 定例会は複数日にわたり開催され、日ごとに個別の PDF が公開される
- 臨時会は通常1日のみで PDF も1ファイル
- 平成28年(2016年)以前の会議録は公開されていない

---

## 推奨アプローチ

1. **3段階クロール**: トップページ → 年度別一覧 → 会議詳細 → PDF の順で段階的に URL を収集
2. **メタ情報の紐づけ**: PDF ファイル名に意味がないため、会議詳細ページのテーブル情報を必ず PDF と紐づけて保存
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度別一覧ページの会議数を前回取得時と比較し、新規追加分のみを取得する
5. **PDF テキスト抽出**: PDF からのテキスト抽出品質に注意。レイアウトが複雑な場合は OCR の併用も検討
