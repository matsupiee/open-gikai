# 北島町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.kitajima.lg.jp/docs/402721.html
- 分類: PDF ダウンロード提供（専用検索システムなし）
- 文字コード: UTF-8
- 特記: ssp.kaigiroku.net・dbsr.jp 等の専用検索システムの導入なし。PDF ファイルを直接公開する形式。

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧 | `https://www.town.kitajima.lg.jp/docs/402721.html` |
| PDF ファイル | `https://www.town.kitajima.lg.jp/fs/{階層パス}/_/{ファイル名}.pdf` |

PDF の URL は `/fs/` 配下に 6 階層の数値ディレクトリが続き、`_/` の後にファイル名が来る構造。ファイル名は URL エンコードされた日本語を含む場合がある。

```
例: /fs/6/1/2/9/0/7/_/%E4%BC%9A%E8%AD%B0%E9%8C%B2HP%E7%94%A8%20R07-4.pdf
```

## ページ構造

### 一覧ページ

- ページネーションなし（全年度が 1 ページに収まる）
- 年度ごとに `<h2>` 見出し（例: `<h2>令和7年</h2>`）で区切られる
- 各 PDF へのリンクは `<p>` タグ内の `<a>` 要素として配置
- リンクテキストの先頭に全角丸「○」記号が付く

```html
<h2>令和7年</h2>
<p>　○<a href="/fs/6/1/2/9/0/7/_/...pdf">第4回定例会 (PDF 614KB)</a></p>
<p>　○<a href="/fs/6/1/1/7/8/8/_/...pdf">第4回町長諸報告 (PDF 22.8KB)</a></p>
```

## 公開データの種別と年度範囲

### 年度範囲

平成29年（2017年）〜 現在（令和8年時点で令和8年分まで確認）

### 各年度に含まれるファイル種別

| 種別 | リンクテキスト例 |
| --- | --- |
| 定例会会議録 | `第1回定例会 (PDF xxxKB)` |
| 町長諸報告 | `第1回町長諸報告 (PDF xxxKB)` |
| 所信表明 | `令和○年町長所信表明 (PDF xxxKB)` |

各年度に定例会が第1〜4回、対応する町長諸報告が付随する。

## スクレイピング戦略

### Step 1: PDF リンクの収集

一覧ページ `https://www.town.kitajima.lg.jp/docs/402721.html` から全 PDF リンクを取得する。

1. ページを取得し、`<h2>` タグから年度（元号・年）を抽出
2. 各 `<h2>` 配下の `<p>` タグ内 `<a href="...pdf">` を取得
3. リンクテキストから種別（定例会/町長諸報告/所信表明）と回次を判定
4. 会議録本体（定例会）のみを対象とし、町長諸報告・所信表明は除外する

**会議録本体の判定:**

```typescript
// リンクテキストに「定例会」が含まれるものを会議録本体とする
const isMinutes = linkText.includes("定例会");
```

### Step 2: 年度・回次のパース

リンクテキストから年度と回次を抽出する。

```typescript
// 例: "第4回定例会 (PDF 614KB)"
const sessionPattern = /第(\d+)回定例会/;

// 年度は親の <h2> テキストから取得
// 例: "令和7年" → 2025年
const yearPattern = /(?:(令和|平成|昭和)(\d+)年)/;
```

元号から西暦への変換:
- 令和: 2018 + N（令和1年 = 2019年）
- 平成: 1988 + N（平成29年 = 2017年）

### Step 3: PDF のダウンロードと本文抽出

各 PDF をダウンロードし、テキスト抽出ツール（pdftotext 等）で本文を得る。

- PDF の文字コードは UTF-8 ベース（テキスト埋め込み型 PDF）
- 発言者パターン等の構造化は PDF 本文の内容に依存するため、実際の PDF を確認して判断する

---

## 注意事項

- 専用の検索システムがないため、差分更新の仕組みはない。一覧ページを定期的にクロールして新規 PDF を検出する方式が必要
- PDF ファイルの URL は `/fs/` 配下の数値ディレクトリ構造で、連番規則性はない（推測による直接アクセスは不可）
- 町長諸報告・所信表明は会議録本体ではないため、取り込み対象から除外する
- ファイルサイズが大きいもの（600KB 超）は 1 回の定例会の全発言を含む本会議録、小さいもの（20〜30KB）は町長報告のみの可能性が高い

---

## 推奨アプローチ

1. **一覧ページのポーリング**: 差分検出のため、定期的に一覧ページを取得して新規リンクを検出する
2. **種別フィルタ**: リンクテキストで「定例会」を含むもののみを対象とし、町長諸報告・所信表明は除外する
3. **PDF テキスト抽出**: ダウンロードした PDF からテキストを抽出し、発言者・発言内容を構造化する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
