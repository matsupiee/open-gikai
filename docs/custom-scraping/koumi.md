# 小海町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.koumi-town.jp/office2/archives/gikai/
- 分類: 独自 CMS による PDF 公開（会議録は PDF ファイルで掲載）
- 文字コード: UTF-8
- 特記: 令和8年〜平成25年の会議録を PDF で公開。平成25年以前は議会事務局での閲覧のみ

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 議会トップ | `https://www.koumi-town.jp/office2/archives/gikai/` |
| 会議録一覧 | `https://www.koumi-town.jp/office2/archives/gikai/post-167.html` |

### PDF の URL パターン

PDF リンクには 2 つの形式が存在する:

| パターン | URL 形式 | 時期 |
| --- | --- | --- |
| ハッシュ値ベース | `https://www.koumi-town.jp/office2/archives/{40文字16進数ハッシュ}.pdf` | 令和5年以降（新しい会議録） |
| ファイル名ベース | `https://www.koumi-town.jp/office2/archives/files/pdf/{ファイル名}.pdf` | 平成25年〜平成30年頃 |

## ページ構造

### 会議録一覧ページ（post-167.html）

- 1 ページに全年度・全会議の PDF リンクが掲載されている（ページネーションなし）
- 各会議は `<h3>` 見出しで区切られている
- 見出しの直下に PDF への `<a>` タグが配置される

### 見出しの形式

```
令和８年第１回臨時会
令和７年第４回定例会
令和７年第４回臨時会
...
平成２５年第１回臨時会
```

### 会議種別

| 種別 | 説明 |
| --- | --- |
| 定例会 | 年 4 回程度開催。複数日程にわたる |
| 臨時会 | 必要に応じて開催。単一または少数日程 |

## PDF ファイルの構成

### 新しい会議録（令和5年以降）

1 会議あたり 1〜2 個の統合 PDF ファイル。

例: `令和８年第１回臨時会　会議録.pdf`

### 古い会議録（平成30年頃以前）

1 会議あたり複数の分割 PDF ファイル:

| セクション | ファイル名パターン例 |
| --- | --- |
| 目次 | `mokuzi{年}.{回}teirei.pdf` |
| 日程 | `nittei{年}.{回}teirei.pdf` |
| 招集日 | `syousyuubi{年}.{回}teirei.pdf` |
| 議案質疑 | `situgi{年}.{回}teirei.pdf` |
| 一般質問 | `ippann{年}.{回}teirei.pdf` |
| 最終日 | `saisyuubi{年}.{回}teirei.pdf` |

## スクレイピング戦略

### Step 1: PDF リンクの収集

会議録一覧ページ `post-167.html` から全 PDF リンクを一括で抽出する。

- ページネーションがないため、1 回のリクエストで全リンクを取得可能
- `<h3>` 見出しをパースして会議名（年度・回次・種別）を取得
- 見出し配下の `<a>` タグから PDF URL を抽出

**収集方法:**

1. `post-167.html` を取得
2. Cheerio で `<h3>` 見出しを順に走査
3. 各 `<h3>` の後続要素から PDF リンク（`href` が `.pdf` で終わるもの）を抽出
4. 見出しテキストから会議メタ情報（年号・年・回次・種別）を正規表現で抽出

### Step 2: PDF のダウンロード

収集した URL から PDF ファイルをダウンロードする。

- 2 つの URL パターン（ハッシュ値ベース、ファイル名ベース）の両方に対応
- レート制限として 1〜2 秒のインターバルを設ける

### Step 3: PDF からテキスト抽出

PDF ファイルからテキストを抽出し、会議録データとして整形する。

- `pdf-parse` 等のライブラリで PDF からテキストを抽出
- 古い会議録は複数 PDF に分割されているため、同一会議のファイルを結合して処理

---

## パース用正規表現（案）

```typescript
// h3 見出しから会議情報を抽出
const meetingPattern = /^(令和|平成)([\d０-９]+)年第([\d０-９]+)回(定例会|臨時会)/;
// 例: 令和８年第１回臨時会 → era="令和", year="８", num="１", type="臨時会"

// PDF リンクテキストからセクション情報を抽出
const sectionPattern = /(目次|日程|招集日|議案質疑|一般質問|最終日|会議録)/;
```

---

## 注意事項

- 平成25年以前の会議録は PDF 未公開（議会事務局での閲覧のみ）
- PDF ファイルのため、テキスト抽出の精度は PDF の作成方法に依存する
- 古い会議録はセクション分割されているため、同一会議の PDF を正しくグルーピングする必要がある
- 見出しの数字は全角（例: `２５年第４回`）で記載されている

---

## 推奨アプローチ

1. **一括取得**: 一覧ページにページネーションがないため、1 リクエストで全 PDF リンクを収集可能
2. **PDF テキスト抽出**: HTML ではなく PDF 公開のため、PDF パーサーが必須
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **分割 PDF の統合**: 古い会議録は複数ファイルに分かれているため、会議単位でグルーピングして処理する
5. **差分更新**: 一覧ページの `<h3>` 見出しをチェックし、新規追加分のみダウンロードする
