# 佐川町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.sakawa.lg.jp/life/dtl.php?hdnKey=1076
- 分類: 独自 CMS による PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 議事録は PDF 形式で提供。年度ごとに詳細ページが分かれている

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 議事録一覧（年度索引） | `https://www.town.sakawa.lg.jp/life/dtl.php?hdnKey=1076` |
| 年度別詳細ページ | `https://www.town.sakawa.lg.jp/life/dtl.php?hdnKey={ページID}` |
| PDF ダウンロード | `https://www.town.sakawa.lg.jp/file/?t=LD&id={ページID}&fid={ファイルID}` |

### 年度別詳細ページ

各年度ページには、定例会・臨時会ごとの議事録 PDF へのリンクが掲載されている。

**会議の種類:**

- 定例会（3月・6月・9月・12月）
- 臨時会（不定期、年1〜3回程度）

**リンクの表記例:**

```
3月定例会1日目（R7.3.7）  [PDF 457KB]
3月定例会4日目（R7.3.10） [PDF 690KB]
第1回臨時会（R7.1.29）    [PDF 371KB]
```

- 定例会は日目（1日目、4日目、5日目等）で分割されている
- 括弧内に開催日が `R{和暦年}.{月}.{日}` 形式で記載
- 各リンクは `/file/?t=LD&id={ページID}&fid={ファイルID}` 形式の PDF ダウンロード URL

---

## PDF ダウンロード URL のパラメータ

| パラメータ | 説明 |
| --- | --- |
| `t` | 固定値 `LD`（ダウンロード種別） |
| `id` | 年度別詳細ページの hdnKey と同じ値 |
| `fid` | 各 PDF ファイルの一意識別子（数値連番、増加傾向） |

**fid の範囲例:**

- 令和6年: 15795〜17812
- 令和7年: 17914〜20166
- 令和8年: 20381〜

## スクレイピング戦略

### Step 1: 年度別ページ ID の収集

議事録一覧ページ `dtl.php?hdnKey=1076` から全年度のリンクを抽出する。

- 各年度リンクの `hdnKey` パラメータを収集
- リンクは `<a href="https://www.town.sakawa.lg.jp/life/dtl.php?hdnKey={ID}">` 形式

**収集方法:**

1. 一覧ページの HTML を取得
2. `dtl.php?hdnKey=` を含むリンクを Cheerio で抽出
3. hdnKey の値をリスト化

### Step 2: PDF リンクの収集

各年度ページ `dtl.php?hdnKey={年度ID}` から PDF ダウンロードリンクを抽出する。

- `/file/?t=LD&id={ID}&fid={FID}` 形式のリンクを全て収集
- リンクテキストから会議名・開催日を抽出

**抽出対象:**

```
href="/file/?t=LD&id=2925&fid=18403"
→ ページID: 2925, ファイルID: 18403
```

**メタ情報の抽出（リンクテキストから）:**

```
3月定例会1日目（R7.3.7）
→ 会議種別: 定例会, 月: 3月, 日目: 1日目, 開催日: 令和7年3月7日
```

### Step 3: PDF のダウンロードとテキスト抽出

各 PDF を `https://www.town.sakawa.lg.jp/file/?t=LD&id={ID}&fid={FID}` からダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）でテキストを抽出
- PDF のサイズは 200KB〜2MB 程度

---

## 注意事項

- PDF 形式のため、HTML パースではなく PDF テキスト抽出が必要
- 定例会は複数日にわたり、日ごとに別 PDF として提供される
- 開催日の表記は和暦略式（`R7.3.7` = 令和7年3月7日）
- 一覧ページ（hdnKey=1076）に掲載されていない年度ページが存在する可能性があるが、確認した限り平成24年〜令和8年の範囲
- Adobe Reader が必要との記載があり、標準的な PDF 形式と推定される

---

## 推奨アプローチ

1. **2段階クロール**: まず一覧ページから年度別ページ ID を収集し、次に各年度ページから PDF リンクを収集する
2. **メタ情報はリンクテキストから抽出**: PDF 内のテキスト解析前に、HTML 上のリンクテキストから会議名・開催日を取得できる
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: fid は数値連番で増加傾向にあるため、前回取得済みの最大 fid 以降のみを取得する差分更新が可能
5. **PDF テキスト抽出**: pdf-parse 等のライブラリを使用し、PDF からテキストを抽出する
