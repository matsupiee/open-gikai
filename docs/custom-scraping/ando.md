## 会議の種別と構造

各年度ページには `div.mol_attachfileblock` が会議ごとにブロックとして並ぶ。

- **タイトル**: `p.mol_attachfileblock_title` に会議名（例: `第1回定例会`、`第1回臨時会`）
- **PDF リスト**: `ul > li > a` に各開催日の PDF リンク（例: `第1日（3月4日）`）

## PDF ファイル名の命名規則

ファイル名は年度によって表記が統一されていない。以下のパターンが混在する。

| パターン例 | 形式 |
| --- | --- |
| `R6.3.4esturan.pdf` | `{元号略称}{年}.{月}.{日}{内容略称}.pdf` |
| `20240305.pdf` | `{YYYYMMDD}.pdf` |
| `R6.9.4ippanshistumon.pdf` | 日付 + 内容種別（一般質問等） |

ファイル名からメタ情報を推測することは困難なため、HTML の `a` タグのリンクテキスト（例: `第1日（3月4日）`）から日付と開催順を取得する。

## スクレイピング戦略

### Step 1: 年度ページの URL 収集

会議録トップページ（`category/10-6-3-0-0-0-0-0-0-0.html`）をフェッチし、`ul.category_end` 内の `a` タグから年度別記事 ID を抽出する。

```typescript
// 年度ページ URL の抽出
const yearLinks = $("ul.category_end li a");
// href 例: "https://www.town.ando.nara.jp/0000003757.html"
// 記事 ID: 7桁ゼロ埋め文字列から抽出 → 3757
```

### Step 2: 年度ページから PDF リンクを収集

各年度ページをフェッチし、`div.mol_attachfileblock` ブロックを走査して会議名と PDF リンクを収集する。

```typescript
$("div.mol_attachfileblock").each((_, block) => {
  const meetingName = $(block).find("p.mol_attachfileblock_title").text();
  // 例: "第1回定例会"

  $(block).find("ul li a").each((_, link) => {
    const dayLabel = $(link).text().trim();
    // 例: "第1日（3月4日）"
    const pdfRelUrl = $(link).attr("href");
    // 例: "./cmsfiles/contents/0000003/3757/R6.3.4esturan.pdf"
  });
});
```

PDF の絶対 URL は `https://www.town.ando.nara.jp/` をベースに解決する。

### Step 3: PDF のダウンロードとテキスト抽出

PDF ファイルをダウンロードし、PDF パーサー（`pdf-parse` 等）でテキストを抽出する。

- 1 PDF = 1 開催日分の会議録
- キーワード検索不可のため、全 PDF を取得してローカルで全文検索・インデックス化する
- 差分更新: トップページの更新日付（`span.date`）を監視し、変化があった年度ページのみ再クロールする

---

## 注意事項

- PDF のファイル名命名規則が統一されていないため、ファイル名からのメタ情報抽出は不可。HTML のリンクテキストを正として扱う
- 「閲覧にあたっての注意事項」ページ（記事 ID: 3862）は会議録ではないため除外する
- トップページの年度リストは最新年度が追加されるため、定期的に再取得が必要
- レート制限: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **トップページから年度リストを取得**: `ul.category_end` をパースして全年度の記事 ID を収集
2. **年度ページから PDF リンクを収集**: `div.mol_attachfileblock` 単位で会議名・開催日・PDF URL をセットで取得
3. **全 PDF を取得**: キーワード検索不可のため全量取得し、ローカルでインデックス化
4. **差分更新**: トップページの `span.date`（更新日）を前回取得値と比較し、変化のある年度のみ再取得
