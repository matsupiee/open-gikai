## 会議種別と構成

### 定例会

年4回開催（3月・6月・9月・12月）。各定例会につき以下のファイルが提供される。

| ファイル | 内容 |
| --- | --- |
| 日程表 | 会議日程の PDF |
| 目次 | 会議録目次の PDF |
| 1日目〜N日目 | 日別の会議録 PDF（会議日数分） |

会議録の日数は定例会により異なる（例: 3月定例会は5〜7日分、6月・9月定例会は3〜4日分、12月定例会は4〜5日分）。

### 臨時会

開催月・回数は年度により異なる。各臨時会につき日程表・目次・1日目（1〜2日分）の PDF が提供される。

### 決算審査特別委員会

通常の年度別ページとは別に専用ページ（`/chosei/_1010/_1414/_1454.html`）が存在する。平成16年度決算分から現在まで、各年度3日分の会議録 PDF が提供される。

## PDF ファイルパスのパターン

会議録 PDF のパスは以下の形式に統一されている。

```
/var/rev0/{グループ番号}/{シリアル番号}/{ファイル名}.pdf
```

- **グループ番号**: 4桁のゼロ埋め整数（例: `0001`, `0011`, `0019`）。年代が新しいほど番号が大きい傾向がある
- **シリアル番号**: 4桁以上の整数（例: `3209`, `9914`）
- **ファイル名**: 以下の2形式が混在している

| 形式 | 例 | 使用時期 |
| --- | --- | --- |
| タイムスタンプ型（新） | `126219181612.pdf` | 令和以降 |
| 和暦記号型（旧） | `h1812_kaigiroku_01.pdf` | 平成期の一部 |

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

会議録トップページ `https://www.town.tara.lg.jp/chosei/_1010/_1414.html` から、年度別ページへのリンクを Cheerio で抽出する。

- 抽出対象: `/chosei/_1010/_1414/` 配下の `.html` リンク
- 決算審査特別委員会ページ（`_1454.html`）も別途収集する

### Step 2: 年度別ページから PDF URL を収集

各年度別ページから会議録 PDF へのリンクを抽出する。

- 抽出対象: `/var/rev0/` で始まる `.pdf` リンク
- `href` に含まれるファイル名から会議種別・日付を推定する（旧形式の場合）
- 日程表・目次・本文（日別）の3種類が混在するため、リンクテキストで種別を判断する

**収集が必要な情報:**

| 項目 | 取得元 |
| --- | --- |
| 会議種別（定例会・臨時会等） | リンク前後のテキスト（例: `3月定例会`、`5月臨時会`） |
| 開催年度 | 年度別ページの URL またはページタイトル |
| 資料種別（日程表・目次・本文） | リンクテキスト |
| PDF URL | `href` 属性 |

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

- PDF はすべてスキャン画像ではなくテキスト埋め込み形式と想定されるが、古い年度（平成18年前後）はスキャン PDF の可能性がある
- テキスト抽出には `pdfjs-dist` または `pdf-parse` を使用する
- 抽出したテキストから発言者・発言内容を構造化する

---

## 注意事項

- 専用の検索システムが存在しないため、**全量クロールが必要**（差分更新は PDF URL の変化で検知する）
- ページネーションは存在しない（各年度ページに全 PDF リンクが列挙されている）
- PDF のファイル名・パスは CMS の内部 ID に依存しており、年度やページ内容から直接推定できない
- 年度別ページの URL（`_XXXX.html` の番号）は連番でなくランダムなため、トップページからのリンク収集が必須
- 決算審査特別委員会は通常の年度別ページには含まれず、専用ページから別途収集する必要がある
- 古い年度（平成18年前後）はファイル名が `h18MM_kaigiroku_NN.pdf` 形式で、比較的推測しやすい

---

## 推奨アプローチ

1. **トップページから全年度 URL を収集**: `_1414.html` および `_1454.html`（決算特別委）の2ページを起点にする
2. **年度別ページを全取得**: 21年度分（令和7年〜平成18年）+ 決算審査特別委員会ページ
3. **PDF URL を一括収集**: 各年度ページから `/var/rev0/` 配下の PDF リンクをすべて抽出し、リンクテキストと合わせて保存
4. **PDF のテキスト抽出**: 収集した PDF を順次ダウンロード・テキスト抽出し、発言内容を構造化
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
