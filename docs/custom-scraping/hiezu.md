## 概要

- サイト: https://www.hiezu.jp/list/gikai/y446/

- 特記: 検索機能なし、会議録本文は HTML 上に表示されず PDF ダウンロードのみ

## スクレイピング戦略

### Step 1: 年度ページ URL の収集

会議録トップ `/list/gikai/y446/` にアクセスし、年度別ページへのリンクを収集する。

- 年度 ID は不規則な英数字のため、トップページから動的に取得する必要がある
- 対象範囲: 平成28年〜現在（10年度分）

### Step 2: 会議一覧の収集

各年度ページ `/list/gikai/y446/{年度ID}/` にアクセスし、会議（定例会・臨時会）へのリンクを収集する。

- 各年度に定例会 4 回（3月・6月・9月・12月）＋臨時会（不定回数）
- リンクテキストから会議名と開催月を抽出可能
  - 例: `令和6年第４回定例会（12月）`

### Step 3: 議事区分ページの収集

各会議ページ `/list/gikai/y446/{年度ID}/{会議ID}/` にアクセスし、議事区分（提案説明、一般質問、議案質疑、討論・採決）へのリンクを収集する。

- 議事区分は会議によって異なる場合がある

### Step 4: PDF ファイルの取得

各議事区分ページから PDF リンクを抽出し、ダウンロードする。

**PDF URL パターン:**

```
/user/filer_public/{2文字hash}/{2文字hash}/{ファイル名}.pdf
```

**確認済み PDF URL 例:**

| 議事区分 | PDF URL |
| --- | --- |
| 提案説明 | `/user/filer_public/f3/2a/f32aa4ca-e325-4c82-a627-5932f6193661/ling-he-6nian-12yue-3ri-chu-ri.pdf` |
| 一般質問 | `/user/filer_public/4e/d5/4ed54bba-79d9-4b6b-9bef-b2133bed5f06/ri-ji-jin-di-04hui-ding-li-r06nian-12yue-04ri.pdf` |
| 議案質疑 | `/user/filer_public/ac/2b/ac2b037b-17fa-4d19-aab3-7f1b3b568cab/ling-he-6nian-12yue-9ri-yi-an-zhi-yi.pdf` |

- PDF ファイル名は日本語をローマ字化した命名規則（CMS 自動生成と推定）
- ファイルサイズは 400KB〜850KB 程度

### Step 5: PDF のパース

PDF からテキストを抽出し、以下のメタ情報と発言内容を取得する。

#### メタ情報の抽出

- 開催日: PDF ファイル名やリンクテキストから抽出（例: `令和６年１２月４日`）
- 会議名: 階層のリンクテキストから取得（例: `令和6年第４回定例会`）
- 議事区分: 提案説明、一般質問、議案質疑、討論・採決

#### 発言内容のパース

PDF 内のテキスト構造は PDF を実際にダウンロードして確認する必要がある。一般的な自治体議会録 PDF では以下のパターンが想定される:

- 発言者名（役職＋氏名）の後に発言内容が続く
- `○` や `◎` 等の記号で発言者が示される場合がある

---

## 注意事項

- URL の ID 部分（年度 ID、会議 ID、区分 ID）は不規則な英数字であり、ルールベースでの URL 生成は不可能。必ずページをクロールしてリンクを収集する必要がある
- PDF 形式のため、テキスト抽出の精度は PDF の生成方法に依存する（スキャン画像 PDF の場合は OCR が必要）
- 検索機能がないため、全量取得が前提となる
- ページネーションは確認されていない（各階層ページで全件表示）

---

## 推奨アプローチ

1. **階層クロールで全 PDF URL を収集**: トップ → 年度 → 会議 → 議事区分の 4 階層を順にクロールし、全 PDF リンクを収集する
2. **PDF ダウンロード後にテキスト抽出**: `pdf-parse` 等のライブラリで PDF からテキストを抽出する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 年度ページの最新年度のみを再クロールし、新規会議録を検出する方式が効率的
5. **PDF 品質の事前確認**: 最初に数件の PDF をダウンロードし、テキスト抽出可能か（スキャン画像でないか）を確認する
