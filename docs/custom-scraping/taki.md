# 多気町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.taki.mie.jp/life/soshiki/gikai_jimukyoku/
- 分類: 独自 CMS による PDF 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: 独立した会議録検索システムを持たず、公式ホームページ内で会議種別・年度別に PDF を掲載している

## 会議種別と掲載内容

| 会議種別 | 掲載内容 | 議事録本文の有無 |
| --- | --- | --- |
| 一般質問会議録 | 各定例会の一般質問議事録（PDF） | **あり**（スクレイピング対象） |
| 定例会 | 会期・議事日程表、議案一覧、一般質問通告要旨 | なし（別ページに誘導） |
| 臨時会 | 会期・議事日程表、議案一覧、結果報告 | なし |
| その他の会議（全員協議会） | 開催予定・議題一覧（HTML テキスト） | なし |

議事録本文 PDF を取得できるのは **「一般質問会議録」のみ**。定例会・臨時会ページは議事日程や議案一覧に限られ、議事録本文は掲載されていない。

## PDF の命名規則

PDF のファイル名に一定の規則性はなく、年月・回数を組み合わせた独自形式が使われている。

例:
- `hpR712.pdf` → 令和7年第2回（推定）
- `HPR79ippan.pdf` → 令和7年第9月一般質問（推定）
- `R612ippan.pdf` → 令和6年12月一般質問
- `giziroku060611.pdf` → 令和6年6月11日議事録
- `giziroku0603.pdf` → 令和6年3月議事録
- `202303.pdf` → 2023年3月

ファイル名からのメタ情報抽出は困難なため、**リンクテキストを正規表現で解析してメタ情報を得る**ことを推奨する。

---

## リンクテキストのパターン

一般質問会議録ページでは、各 PDF リンクのテキストが以下の形式で統一されている。

```
令和{元号年}年第{回数}回 多気町議会定例会一般質問
令和{元号年}年第{回数}回多気町議会定例会（一般質問）
```

例:
- `令和6年第4回定例会一般質問`
- `令和5年第1回 多気町議会定例会一般質問`

#### パース用正規表現（案）

```typescript
// 会議名からの年・回数抽出
const titlePattern = /令和(\d+)年第(\d+)回\s*多気町議会定例会(?:\(|（)?一般質問(?:\)|）)?/;
// グループ1: 令和年数, グループ2: 回数
```

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

一般質問会議録インデックスページ（`kaigiroku/index.html`）から、年度別ページへのリンク（`kaigiroku/{ID}.html`）を抽出する。

- リンクは年度ごとに 1 件ずつ列挙されている
- ページネーションなし
- 現時点では令和5〜7年の 3 年度分が公開

### Step 2: 各年度ページから PDF リンクを収集

各年度ページ（`kaigiroku/{ID}.html`）を取得し、PDF リンク一覧を抽出する。

- 1 ページにつき定例会 4 回分（第1〜4回）の PDF リンクが掲載される
- 各リンクは `//www.town.taki.mie.jp/material/files/group/14/{ファイル名}.pdf` の形式
- リンクテキストから会議名（年・回数）を抽出する

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL から PDF をダウンロードし、テキストを抽出する。

- PDF は議事録本文形式で、一般質問の質疑応答が記載されている
- メタ情報（年度・回数）はリンクテキストから取得し、PDF のファイル名には依存しない
- PDF のテキスト抽出には `pdf-parse` 等のライブラリを使用する

---

## 注意事項

- 議事録本文が公開されているのは**令和5年以降**のみ。それ以前の取得は問い合わせ対応が必要
- 定例会・臨時会ページには議事録本文 PDF が存在しないため、スクレイピング対象外
- PDF のファイル名に規則性がないため、ファイル名からのメタ情報抽出は行わない
- `//` で始まるプロトコル相対 URL は `https:` を付加して絶対 URL に変換する

---

## 推奨アプローチ

1. **インデックスページから年度 ID を収集**: `kaigiroku/index.html` を取得し、リンクを列挙
2. **年度ページから PDF リンクを収集**: 各 `kaigiroku/{ID}.html` を取得し、PDF URL とリンクテキストをペアで収集
3. **リンクテキストからメタ情報を抽出**: 正規表現で令和年・回数を取得
4. **PDF をダウンロードしてテキスト抽出**: テキスト抽出後に発言者・発言内容を解析
5. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
