### 議事運営日程・上程議案・一般質問通告

令和5年以降を確認した範囲では以下のパターン:

| 変数 | 値の例 |
| --- | --- |
| `{年}` | `R5`（令和5年）、`R6`、`R7`、`R8` |
| `{回}` | `1`（第1回）、`2`、`3`、`4` |
| `{種別}` | `teirei`（定例会）、`rinji`（臨時会）、省略される場合もあり |

命名に表記揺れがあり（例: `R6_2_teirei.pdf` と `R7_2_teirei.pdf` が異なる種別を指す場合がある）、ファイル名だけから内容を確定することが難しいため、ページ上のリンクテキストと組み合わせて判断する必要がある。

## スクレイピング戦略

### 会議録の発言全文取得は不可

現時点では、白川村議会の発言全文テキストをウェブ上から取得する手段がない。

**推奨対応:** 村議会事務局（05769-6-1311）に会議録のテキストデータ提供または会議録の掲載予定について問い合わせる。

### 議会だより・通告資料の取得（参考情報として）

発言全文の代替として、一般質問の概要情報を取得する場合のアプローチ:

#### Step 1: リンク収集

議会トップページとページネーションリンクを順次取得し、PDFリンクを収集する。

```
GET https://www.vill.shirakawa.lg.jp/1098.htm
GET https://www.vill.shirakawa.lg.jp/dd.aspx?moduleid=2797&pfromid=4
GET https://www.vill.shirakawa.lg.jp/dd.aspx?moduleid=2797&pfromid=8
（pfromid を 4 ずつ増やして次のページへ遷移。「次の一覧へ」リンクがなくなるまで繰り返す）
```

各ページから `/secure/` を含む `<a>` タグの `href` 属性を抽出し、ファイル名から種別（nittei/gian/ippan/gikai）を分類する。

#### Step 2: PDF のダウンロードとテキスト抽出

収集したURLからPDFをダウンロードし、`pdftotext` 等でテキスト抽出を行う。

- **議事運営日程** (`nittei`): 開催日・場所の確認用
- **一般質問通告** (`ippan`/`shitsumon`): 質問者氏名・質問項目の把握
- **議会だより** (`gikai`): 一般質問の要約・議会活動の概要

## 注意事項

- **発言全文なし**: テキスト形式の発言録は公開されておらず、スクレイピングだけでは会議録データベースへの収録は困難。
- **ssp.kaigiroku.net/tenant/shirakawa/ は別自治体**: このテナントは白川市（福島県）のシステムであり、白川村（岐阜県）とは無関係。混同しないこと。
- **PDFのみ**: テキスト抽出には `pdftotext` 等が必要。議会だよりはスキャン画像を含む可能性があり、OCRが必要になる場合がある。
- **ファイル名の表記揺れ**: PDF命名規則が一貫しておらず、ページ上のリンクテキストと照合して種別を判断する必要がある。
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける。
