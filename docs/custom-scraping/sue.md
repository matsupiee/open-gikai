## スクレイピング戦略

### Step 1: トップページから年度別ページ ID を収集

`index.html` を取得し、`/gyosei/gikai/1/{ページID}.html` 形式の全リンクを抽出する。

- ページネーションなし。単一ページに全年度のリンクが含まれる
- 対象範囲: 平成23年（2011年）〜令和7年（2025年）

### Step 2: 年度別ページから PDF リンクを収集

各年度ページから PDF リンクを抽出する。

- 定例会は複数開催日に分割されており、日ごとに別 PDF が存在する
- 臨時会は1〜2回/年

### Step 3: PDF の取得・解析

各会議録 PDF を取得してテキストを抽出する。

---

## 注意事項

- ページ ID は連番ではなく、年度ごとにランダムな数値が割り当てられている
- そのため、年度 ID を推測する手法は使えず、必ず `index.html` からリンクを収集する
- 各年度ページから取得できるのは PDF リンクのみ。HTML 本文での全文掲載はない
- 問い合わせ先: 議会事務局（監査委員事務局）092-932-1151

---

## 推奨アプローチ

1. **インデックスから動的取得**: `index.html` のリンクを起点に年度ページ ID を収集する
2. **PDF テキスト抽出**: pdfjs や pdfminer 相当のライブラリで PDF を解析する
3. **レート制限**: 自治体サイトのため、リクエスト間に 1〜2 秒の待機時間を設ける
4. **差分更新**: 各年度ページの最終更新日を記録し、変化がない年度の再クロールを省略する
