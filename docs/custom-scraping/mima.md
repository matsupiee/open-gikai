## 概要

- サイト: https://www.city.mima.lg.jp/gyosei/shisei/gikai/kaigiroku/
- 分類: 市公式サイト上で PDF 形式の会議録を直接公開（独自ドキュメント管理システム）

- 特記: 会議録は PDF ファイルとして公開。検索システムなし。一覧ページから年度別ページへ遷移し、各 PDF を直接ダウンロードする方式。

## ページ構造

### 一覧トップページ

- ページネーションあり（2 ページ構成）
- 各年度を1行のリンクとして掲載（例: 「令和7年市議会会議録」）
- リンク先は `/gyosei/docs/{ID}.html` 形式の年度別ページ

### 年度別ページ（`/gyosei/docs/{ID}.html`）

- 見出し（`<h2>`）で定例会・臨時会を区別
  - 例: 「令和7年3月定例会」「令和7年第1回臨時会」
- 各見出しの下に PDF ファイルへのリンクをリスト形式で掲載
- 各 PDF リンクのテキスト形式:
  - 定例会: `令和X年Y月美馬市議会定例会(第N号)(開催日YYYY年MM月DD日)`
  - 臨時会: `令和X年第N回美馬市議会臨時会(開催日YYYY年MM月DD日)`

### 会議の開催パターン

- **定例会**: 年4回（3月・6月・9月・12月）、各定例会につき複数号（通常 3〜4 号）
- **臨時会**: 年1〜2回程度（5月前後に多い）

---

## PDF URL パターン

PDF のパスは `/fs/` から始まり、数字ディレクトリが階層化された形式:

```
/fs/{d1}/{d2}/{d3}/{d4}/{d5}/{d6}/_/{ファイル名}.pdf
```

ファイル名は日本語を含む場合とラテン文字のみの場合が混在:

- 旧形式（日本語混在）: `___7_3___________1______2025_2_18__.pdf`
- 新形式（英数字のみ）: `20250618____2_____-cleaned.pdf` / `20250909___________.pdf`

ファイル名からのパターン推定は困難なため、**年度別ページのHTMLをパースしてリンクを収集する**方式を採用する。

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

一覧トップページおよび 2 ページ目から、`/gyosei/docs/` へのリンクを全て抽出する。

- URL: `https://www.city.mima.lg.jp/gyosei/shisei/gikai/kaigiroku/`
- URL: `https://www.city.mima.lg.jp/gyosei/shisei/gikai/kaigiroku/index.p2.html`
- `<a href="/gyosei/docs/{ID}.html">` の形式でリンクが存在する
- ドキュメント ID と年度のマッピングは上記テーブルを参照（新年度追加時は一覧を再クロール）

### Step 2: 年度別ページから PDF リンクの収集

各年度別ページ（`/gyosei/docs/{ID}.html`）をフェッチし、`/fs/` で始まる PDF リンクを抽出する。

**収集情報:**

- PDF の完全 URL: `https://www.city.mima.lg.jp` + `/fs/.../*.pdf`
- 会議名: リンクテキスト（例: `令和7年3月美馬市議会定例会(第1号)(開催日2025年2月18日)`）
- 会議種別: リンクの直前の `<h2>` 見出しテキスト（「3月定例会」「臨時会」等）

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL から PDF をダウンロードし、テキストを抽出する。

- PDF からのテキスト抽出には pdf-parse 等のライブラリを使用
- 発言者・発言内容のパースは PDF テキスト抽出後に行う

---

## メタ情報の抽出

### リンクテキストからのパース

リンクテキストを正規表現でパースしてメタ情報を取得:

```typescript
// 定例会: 「令和7年3月美馬市議会定例会(第1号)(開催日2025年2月18日)」
const regularPattern = /(?:令和|平成)(\d+)年(\d+)月美馬市議会定例会\(第(\d+)号\)\(開催日(\d{4})年(\d{1,2})月(\d{1,2})日\)/;

// 臨時会: 「令和7年第1回美馬市議会臨時会(開催日2025年5月16日)」
const specialPattern = /(?:令和|平成)(\d+)年第(\d+)回美馬市議会臨時会\(開催日(\d{4})年(\d{1,2})月(\d{1,2})日\)/;
```

### h2 見出しからの会議種別取得

各 PDF リンクが属する `<h2>` 見出しを参照:

```typescript
// 例: 「令和7年3月定例会」「令和7年第1回臨時会」「令和7年6月定例会」
const sectionPattern = /(?:令和|平成)\d+年(\d+月定例会|第\d+回臨時会)/;
```

## 注意事項

- PDF ファイルの内容（発言者・発言テキスト）は PDF テキスト抽出後の処理が必要
- 最古の会議録は平成18年（2006年）から
- ファイル名に日本語が含まれる場合があるため、URL エンコードに注意
- 令和8年（2026年）のページは「現在、作成中です。」となっている場合がある（年度開始直後）
- レート制限: 市公式サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **一覧ページの全量クロール**: 2 ページ分の一覧から年度別ページ URL を全量収集
2. **年度別ページの HTML パース**: `<h2>` 見出しと `/fs/` リンクの対応関係を Cheerio で抽出
3. **差分更新**: 新年度ページの追加は一覧トップを定期クロールして検出
4. **PDF ダウンロード**: 収集した PDF URL を順次ダウンロード（1〜2 秒の間隔を設ける）
