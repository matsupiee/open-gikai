## サイト構造

### 一覧ページ（トップ）

- 全年度のリンクが単一ページにリスト表示（ページネーションなし）
- 各リンクは `goto_chosei_full.php?eid={eid}&r=4&wcid=l00002x4` 形式

### 年度別ページ

- 該当年度に開催された本会議（定例会・臨時会）の一覧を表示
- 各会議ごとに PDF ダウンロードリンクが付与されている
- 会議の種類は **本会議のみ**（委員会の会議録は掲載されていない）

### PDF ダウンロード

- URL パターン: `cmd/dlfile.php?entryname=benri&entryid={eid}&fileid={fileid}&/{filename}.pdf`
- `fileid` は年度ページ内で連番（例: `00000016`〜`00000024`）
- ファイルサイズは 0.56MB〜3.95MB 程度

## スクレイピング戦略

### Step 1: 年度別ページの URL 収集

会議録一覧ページ `goto_chosei.php?wcid=l00002x4` から全年度のリンクを取得する。

- 全年度（平成18年〜令和8年）が単一ページに表示されている
- `goto_chosei_full.php?eid={eid}` のリンクを Cheerio で抽出
- eid は上記一覧の通り固定値だが、将来の年度追加に対応するためページから動的に取得する

### Step 2: 各年度の PDF リンク収集

年度別ページ `goto_chosei_full.php?eid={eid}&r=4&wcid=l00002x4` から PDF ダウンロードリンクを取得する。

- `cmd/dlfile.php` へのリンクを Cheerio で抽出
- 各リンクから以下のメタ情報を取得:
  - `entryid`（年度の eid）
  - `fileid`（ファイル固有 ID）
  - ファイル名（定例会/臨時会の区分、開催日の推定に使用）
- リンク周辺のテキストから会議名・開催日も併せて取得

**収集方法:**

1. 一覧ページから全 eid を取得
2. 各年度ページにアクセスし、PDF リンクと会議情報を抽出
3. 会議名（第X回定例会/臨時会）と開催日をテキストから抽出

### Step 3: PDF のダウンロードと本文抽出

PDF ファイルをダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）を使用してテキストを抽出
- PDF のため HTML パースは不要

### Step 4: 会議録のパース

#### メタ情報

年度別ページのテキストおよび PDF ファイル名から以下を抽出:

- 開催日: ページ内テキストに「X月X日」「X月X日〜X日」形式で記載
- 会議名: 「第X回定例会」「第X回臨時会」
- 年度: eid に対応する年度（令和/平成）

#### 本文

- PDF からテキスト抽出後、発言者・発言内容を分離
- 発言者パターンは PDF 内のフォーマットに依存するため、実際の PDF を解析して正規表現を決定する必要がある

---

## 注意事項

- 会議録は **PDF のみ** で提供されており、HTML 形式の本文ページは存在しない
- PDF ファイル名の命名規則が年度によって異なるため、ファイル名からのメタ情報抽出には注意が必要
- 委員会の会議録は掲載されておらず、**本会議のみ** が対象
- 検索機能は提供されていない
- 平成31年の表記で令和元年分も含まれる可能性がある

---

## 推奨アプローチ

1. **2段階クロール**: まず一覧ページから eid を収集し、次に各年度ページから PDF リンクを収集する
2. **PDF テキスト抽出**: `pdf-parse` や `pdfjs-dist` を使用して PDF からテキストを抽出
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: eid が年度ごとに固定のため、新年度の eid が追加されたかを一覧ページで確認し、未取得の年度のみを処理する
5. **ファイル名非依存**: ファイル名の命名規則が不統一のため、メタ情報はページ内テキストから取得することを優先する
