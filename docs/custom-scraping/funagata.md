- 文字コード: UTF-8

## PDF ファイルの命名規則

`r{和暦2桁}-{回数2桁}{種別}-{月}.{日}.pdf`

| パターン | 例 | 説明 |
| --- | --- | --- |
| 臨時会 | `r08-01rinji-1.21.pdf` | 令和8年 第1回臨時会 1月21日 |
| 定例会 | `r07-06teirei-12.3.pdf` | 令和7年 第6回定例会 12月3日 |

命名要素:
- `r{年}`: 和暦年号（2桁）
- `{回数}`: 会議の回数（2桁）
- `{種別}`: `rinji`（臨時会）/ `teirei`（定例会）
- `{月}.{日}`: 開催日

---

## スクレイピング戦略

### Step 1: 年度別ページ URL を収集

トップページから年度別リンクを収集する。リンクは `/s027/gikai/010/{カテゴリID}/{タイムスタンプ}.html` 形式。

### Step 2: 年度別ページから PDF URL を収集

各年度ページをパースし、相対パス `./` で参照される PDF リンクを全て取得する。ベース URL は年度ページの URL から算出する。

### Step 3: PDF のダウンロードとテキスト抽出

収集した PDF URL からファイルをダウンロードし、テキストを抽出する。

---

## 注意事項

- PDF は年度ページと同じディレクトリに格納されている（相対パス `./`）
- 年度ページの URL にはタイムスタンプが使われている
- 平成23年〜令和8年の範囲で公開されている
- レート制限: リクエスト間に適切な待機時間（1〜2 秒）を設ける
