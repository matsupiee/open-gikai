## ページ構造

### 会議録一覧ページ

`http://www.vill.doshi.lg.jp/ka/info.php?if_id=719&ka_id=7` は単一ページで、**ページネーションなし**。平成23年以降の全会議録 PDF へのリンクが 1 つの HTML テーブルにまとめて掲載されている。

テーブルのカラム構成:

```
| 会議録（PDF リンク） | 会期（開催日） |
```

各行の例:

```html
<tr>
  <td><a href="/upload/file/gikai/H30-/kaigiroku/R7/道志村R07年12月定例会.pdf">令和7年第6回定例会会議録（PDF）</a></td>
  <td>令和7年12月9日～12日</td>
</tr>
```

---

## 会議の種別

道志村議会の会議は本会議のみ（委員会会議録の公開なし）。会議種別は以下の 2 種類:

| 種別 | 説明 |
| --- | --- |
| 定例会 | 年複数回開催（例: 3月・6月・9月・12月の定例会） |
| 臨時会 | 必要に応じて随時開催 |

年間の回次は定例会と臨時会を通算して付番されており、第1回〜第8回程度が存在する（年によって異なる）。

## スクレイピング戦略

### Step 1: PDF リンクの収集

会議録一覧ページ（`info.php?if_id=719&ka_id=7`）をフェッチし、テーブル内のすべての PDF リンクと会議名・開催日を抽出する。

- ページネーションはないため、1 リクエストで全件取得できる
- PDF リンクのテキストから会議名・回次・種別を取得する
- 隣接する `<td>` から開催日を取得する

**Cheerio を使ったリンク収集例:**

```typescript
const $ = load(html);
const records: { title: string; date: string; pdfUrl: string }[] = [];

$("table tr").each((_, row) => {
  const cells = $(row).find("td");
  if (cells.length < 2) return;

  const link = $(cells[0]).find("a");
  const href = link.attr("href");
  if (!href || !href.endsWith(".pdf")) return;

  records.push({
    title: link.text().trim(),       // 例: "令和7年第6回定例会会議録（PDF）"
    date: $(cells[1]).text().trim(), // 例: "令和7年12月9日～12日"
    pdfUrl: `http://www.vill.doshi.lg.jp${href}`,
  });
});
```

### Step 2: メタ情報のパース

リンクテキストから会議名・回次・種別を正規表現で抽出する。

```typescript
// 会議名のパース
const titlePattern = /^(令和|平成)(\d+)年第(\d+)回(定例会|臨時会)会議録/;
// 例: "令和7年第6回定例会会議録（PDF）"
// → era="令和", year="7", kai="6", type="定例会"

// 開催日のパース（単日）
const singleDatePattern = /(令和|平成|昭和)(\d+)年(\d+)月(\d+)日$/;

// 開催日のパース（期間）
const rangeDatePattern = /(令和|平成|昭和)(\d+)年(\d+)月(\d+)日[〜～](\d+)日/;
// 複数月にまたがる場合: /(令和|平成|昭和)(\d+)年(\d+)月(\d+)日[〜～](\d+)月(\d+)日/
```

### Step 3: PDF の取得とテキスト抽出

各 PDF ファイルをダウンロードし、PDF テキスト抽出ライブラリ（`pdf-parse` 等）でテキストを取得する。

- PDF は村役場サーバーから直接ダウンロードする
- 会議録のテキスト構造は PDF ごとに若干異なる可能性があるため、柔軟なパースが必要

---

## 注意事項

- **ページネーションなし**: 一覧は単一ページに全件掲載されている
- **PDF ファイル名は非統一**: ファイル名の命名規則がバラバラであるため、ファイル名からメタ情報を抽出することは困難。リンクテキストと隣接セルのテキストから取得すること
- **一部 PDF のパスが例外的**: 大半は `/upload/file/gikai/H30-/kaigiroku/{年号}/` 配下だが、一部は `/upload/file/` 直下など異なるパスに存在する（例: 令和3年第5回定例会 `/upload/file/道志村R03年９月定例会.pdf`）。ファイルパスを推測せず、必ず一覧ページのリンクから取得すること
- **会議録の追加**: 新しい会議が開催されると一覧ページに PDF リンクが追加されるため、定期的に差分チェックが必要
- **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける

---

## 推奨アプローチ

1. **一覧ページ 1 リクエストで全件収集**: ページネーションがないため、単一 GET で全 PDF リンクを収集する
2. **リンクテキストからメタ情報を取得**: ファイル名ではなく `<a>` タグのテキストと隣接セルのテキストを利用する
3. **PDF ダウンロードは順次処理**: 同時ダウンロードを避け、1 件ずつ順次取得する
4. **差分更新**: 既存の PDF URL リストと照合し、新規追加分のみ取得する
