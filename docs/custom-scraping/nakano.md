# 中野区議会 カスタムスクレイピング方針

## 概要

- サイト: https://kugikai-nakano.jp/
- 分類: 独自 CMS による HTML 公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: Google Tag Manager（GTM-MNVSWTVJ）使用

## スクレイピング戦略

### Step 1: 会議録 ID の収集

会議一覧ページ `search.html?s2={会議ID}&s1={年}&flg=check` から `gijiroku_id` を抽出する。

- 検索結果はテーブル形式で表示される
- 各行に `view.html?gijiroku_id={ID}` へのリンクが含まれる
- 結果は 20 件ずつページネーション（`全（N）件` 表示あり）
- 全会議種別 × 全年を組み合わせてクロールする

**収集方法:**

1. `s2` の全会議 ID と `s1` の年範囲（2004〜現在）を組み合わせて URL を生成
2. 各ページの検索結果テーブルから `gijiroku_id` を正規表現 or Cheerio で抽出
3. ページネーションがある場合は次ページも取得

### Step 2: 会議録本文の取得

`view.html?gijiroku_id={ID}&flg=etc` を使い、本文のみの簡易 HTML を取得する。

- `flg=etc` を付けるとナビゲーション等が省略され、本文テキストのみが返る
- パース負荷が軽減される

### Step 3: 会議録のパース

#### メタ情報

ページ冒頭から以下を抽出:

```
令和６年１２月１２日中野区議会本会議（第４回定例会）の会議録
```

- 開催日: 全角数字で `令和X年XX月XX日` 形式
- 会議名: `中野区議会{会議種別}（{定例会/臨時会名}）`

#### 発言の構造

発言者パターン:

```
○議長（酒井たくや）
○副議長（木村広一）
○４１番（森たかゆき）
○区長（酒井直人）
○教育長（入野貴美子）
```

- `○` で始まり、役職 or 番号 + 括弧内に氏名
- 登壇表記: `〔森たかゆき議員登壇〕`
- 発言内容は発言者行の後に続くテキスト

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○４１番（森たかゆき） → role="４１番", name="森たかゆき"

// 登壇の検出
const podiumPattern = /^〔(.+?)議員登壇〕/;

// 開催日の抽出
const datePattern = /(?:令和|平成)(\d+)年(\d+)月(\d+)日/;
```

## 注意事項

- 英数字検索は完全一致（全角・半角・大文字・小文字を区別）
- 「出席議員」「出席説明員」は検索対象外と明記されている
- 発言議員の検索は平成23年5月24日以降のみ対応
- HTML は MS Word 由来のスタイル（`MsoNormal` 等）が含まれる → Cheerio でテキスト抽出時にスタイル要素を無視する必要あり

---

## 推奨アプローチ

1. **全量取得を優先**: 会議種別 × 年のマトリクスでまず `gijiroku_id` の全量リストを作成
2. **`flg=etc` を活用**: 本文のみ HTML で取得し、不要なナビゲーション要素のパースを回避
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: `gijiroku_id` は数値連番のため、前回取得済みの最大 ID 以降のみを取得する差分更新が可能
