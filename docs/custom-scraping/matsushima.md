# 松島町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.miyagi-matsushima.lg.jp/index.cfm/15,html
- 分類: 町独自のカスタムシステム（ColdFusion ベースの CMS）、PDF 公開
- 文字コード: UTF-8
- 特記: 会議録は PDF 形式で公開。検索システムなし。1 ページに全年度分が掲載されている

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 議会トップ | `https://www.town.miyagi-matsushima.lg.jp/index.cfm/15,html` |
| 定例会・臨時会（会議録一覧） | `https://www.town.miyagi-matsushima.lg.jp/index.cfm/15,12059,27,html` |
| 会議録 PDF | `https://www.town.miyagi-matsushima.lg.jp/index.cfm/15,12059,c,html/12059/{ファイル名}.pdf` |

### PDF ファイル名パターン

PDF ファイル名には 2 種類のパターンがある:

1. **タイムスタンプ形式**: `{YYYYMMDD}-{HHMMSS}.pdf`（例: `20260317-134052.pdf`）
2. **セッション形式**: `{元号略}{回数}{種別}-{連番}-{MMDD}.pdf`（例: `r06-02t-01-0606.pdf`）

## 会議録一覧ページの構造

定例会・臨時会ページ（`/index.cfm/15,12059,27,html`）に全年度の会議情報が 1 ページにまとめて掲載されている。

### 年度範囲

平成22年（2010年）〜 令和8年（2026年）現在

### 年度ごとの構成

各年度は `<h4>` タグで区切られ、その下に `<table>` で定例会・臨時会ごとの情報が並ぶ。

```
<h4>令和8年(2026年)</h4>
<table>
  <tr>
    <td><strong>第1回定例会(3月3日から17日)</strong></td>
    <td><a href="...pdf">付議案件</a></td>
    <td><a href="...pdf">会期日程</a></td>
    <td><a href="...pdf">一般質問</a></td>
    <td><a href="...pdf">3月3日会議録</a> <a href="...pdf">3月4日会議録</a>...</td>
  </tr>
  ...
</table>
```

### 会議録リンクの識別

- リンクテキストが `{M月D日}会議録` の形式（例: `12月15日会議録`、`3月3日会議録`）
- `title` 属性にも同じテキストが設定されている
- 付議案件・会期日程・一般質問のリンクは会議録ではないため除外が必要

### ページネーション

なし。全データが 1 ページに掲載されている。

## PDF の構造

### 表紙（1 ページ目）

```
令和７年１２月１５日（月曜日）

第４回松島町議会定例会会議録

（第１日目）
```

### メタ情報（2 ページ目〜）

```
令和７年第４回松島町議会定例会会議録（第１号）
```

続いて以下の情報が記載:

- 出席議員（議席番号、氏名）
- 欠席議員
- 説明のため出席した者（役職、氏名）
- 事務局職員出席者
- 議事日程

### 発言の構造

発言者パターン:

```
○議会事務局長（千葉浩司君）
○臨時議長（片山正弘君）
○１番（赤間幸夫君）
○議長（髙橋利典君）
○町長（櫻井公一君）
```

- `○` で始まり、役職 or 議席番号 + 括弧内に氏名 + `君`
- 発言内容は発言者行の後に続くテキスト（インデント付き）

その他の記載パターン:

```
［「異議なし」と呼ぶ者あり］
午前１０時０２分　休　憩
（休　憩　中）
```

### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○臨時議長（片山正弘君） → role="臨時議長", name="片山正弘"
// 例: ○１番（赤間幸夫君） → role="１番", name="赤間幸夫"

// 開催日の抽出（表紙から）
const datePattern = /(?:令和|平成)[０-９]+年[０-９]+月[０-９]+日/;

// 会議名の抽出（表紙から）
const sessionPattern = /第[０-９]+回松島町議会(定例会|臨時会)会議録/;

// 休憩の検出
const recessPattern = /[午前後]+[０-９]+時[０-９]+分\s*休\s*憩/;
```

## スクレイピング戦略

### Step 1: 会議録 PDF URL の収集

定例会・臨時会ページ（`/index.cfm/15,12059,27,html`）を取得し、会議録 PDF へのリンクを抽出する。

**収集方法:**

1. ページ全体の HTML を取得（1 ページに全年度分が掲載されている）
2. `<a>` タグのうち、テキストが `会議録` を含むリンクのみを抽出
3. 各リンクの前後から年度・定例会回数・開催日を特定

**会議録リンクの判定:**

```typescript
// リンクテキストに「会議録」を含むものが対象
const isMinutesLink = (text: string) => text.includes("会議録");

// 付議案件・会期日程・一般質問・議決結果は除外
```

### Step 2: PDF のダウンロードとテキスト抽出

- PDF をダウンロードし、テキストを抽出する
- PDF からのテキスト抽出には `pdf-parse` 等のライブラリを使用
- 全角数字が多用されているため、正規表現では全角数字に対応する必要がある

### Step 3: メタ情報の抽出

PDF テキストから以下を抽出:

- **開催日**: 表紙の `令和X年X月X日` から抽出（全角数字）
- **会議名**: 表紙の `第X回松島町議会定例会会議録` から抽出
- **日目**: 表紙の `（第X日目）` から抽出

### Step 4: 発言のパース

- `○` で始まる行を発言者として検出
- 括弧内の氏名（`君` を除去）と役職を抽出
- 次の `○` 行までのテキストを発言内容として取得
- `［...］` で囲まれた行は議場の状況記述として分離

---

## 注意事項

- 会議録は個人情報の保護のため一部編集されており、正式な会議録とは多少異なる旨の注記がある
- 平成23年第1回定例会の一般質問は東日本大震災により全員取り下げ
- PDF ファイル名に規則性がないため、ファイル名からメタ情報を抽出することは困難。PDF 本文からの抽出が必須
- 全角数字（`０`〜`９`）が多用されている
- ColdFusion ベースの CMS（`index.cfm` パス）のため、URL 構造が独特

---

## 推奨アプローチ

1. **一括取得**: 1 ページに全年度分が掲載されているため、1 回の HTML 取得で全会議録 PDF の URL リストを構築できる
2. **PDF テキスト抽出**: `pdf-parse` 等で PDF からテキストを抽出し、発言者・発言内容をパースする
3. **レート制限**: 自治体サイトのため、PDF ダウンロード間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: ページ内の会議録リンク一覧と既存データを比較し、新規分のみをダウンロード・パースする
