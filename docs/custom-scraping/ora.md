## PDF ファイルの命名パターン

PDF ファイル名は年代・時期によって命名規則が異なる。

### 近年（令和6年〜7年）

```
kaigiroku-r7-6-day1.pdf     → 令和7年6月 日程第1号
kaigiroku-r7-6-day2.pdf     → 令和7年6月 日程第2号
r7-12-day1.pdf               → 令和7年12月 日程第1号
kaigiroku-r7-5rinji-day1.pdf → 令和7年5月臨時会 日程第1号
```

### 平成17年

```
teirei03-2.pdf  → 第3回定例会 日程第2号
rinji05.pdf     → 第5回臨時会
```

※ 命名規則が統一されていないため、HTML のリンクテキストからメタ情報を抽出する必要がある。

## スクレイピング戦略

### Step 1: 年別一覧ページの URL 収集

トップページ `kaigiroku.html` の `<div class="txtbox">` 内の `<a>` タグから、全年別一覧ページの URL を抽出する。

- 各 `<li>` 内に `<a href="./XXX/YYY.html">` 形式でリンクが存在
- 20年分（平成17年〜令和7年）のリンクを取得

### Step 2: PDF リンクの収集

各年別一覧ページの `<div class="txtbox">` から PDF へのリンクを抽出する。

- `<h2>` タグに「令和X年第Y回定例会（Z月）」等の会議名が記載
- `<ul>` 内の `<li>` に PDF リンクが配置
- リンクテキストに「MM月DD日 日程第N号（PDF：XXXKB）」形式で日付・日程番号が記載

**抽出すべきメタ情報:**

| 情報 | 抽出元 |
| --- | --- |
| 年 | 年別一覧ページの `<h1>` タグ（例: `議会会議録（令和7年分）`） |
| 定例会/臨時会名 | `<h2>` タグ（例: `令和7年第2回定例会（6月）`） |
| 開催日 | リンクテキスト（例: `6月10日 日程第2号`） |
| 日程番号 | リンクテキスト（例: `日程第2号`） |
| PDF URL | `<a>` タグの `href` 属性（相対パス） |

### Step 3: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、`pdftotext` 等でテキストを抽出する。

### Step 4: 会議録のパース

#### メタ情報

PDF の冒頭に以下の形式でメタ情報が記載されている。

```
令和７年第２回邑楽町議会定例会議事日程第２号

令和７年６月10日（火曜日）

午前１０時開議
邑楽町議会議場
```

- 会議名: `{元号}年第{N}回邑楽町議会{定例会|臨時会}議事日程第{M}号`
- 開催日: `{元号}年{M}月{D}日（{曜日}）`（全角数字）
- 開議時刻: `午前{H}時{MM}分開議`

#### 出席議員リスト

```
〇出席議員（１３名）
１番　山本裕子議員
２番　三ツ村由紀議員
```

- `〇出席議員` / `〇欠席議員` の見出しで区切られる
- 議員名は全角空白で区切られた名前 + `議員` の形式（PDF のテキスト抽出時に空白が入る）

#### 発言の構造

発言者パターン:

```
〇松島茂喜議長
〇14番　松村　潤議員
〇川島隆史学校教育課長
〇橋本光規町長
〇小林淳一教育長
```

- `〇` で始まり、役職名または番号 + 氏名 + 肩書き
- 議長: `〇{氏名}議長`
- 議員: `〇{番号}番　{氏名}議員`（番号は全角）
- 執行部: `〇{氏名}{役職名}`（例: `町長`, `学校教育課長`）

登壇表記:

```
〔14番　松村　潤議員登壇〕
〔川島隆史学校教育課長登壇〕
```

セクション見出し:

```
◎開議の宣告
◎一般質問
◇　松村　潤　議員
```

- `◎` は議事進行上のセクション区切り
- `◇` は一般質問の質問者名

#### パース用正規表現（案）

```typescript
// 発言者の抽出（議長）
const chairPattern = /^〇(.+?)(議長|副議長)/;
// 例: 〇松島茂喜議長 → name="松島茂喜", role="議長"

// 発言者の抽出（議員）
const memberPattern = /^〇(\d+)番\s*(.+?)議員/;
// 例: 〇14番　松村　潤議員 → number="14", name="松村　潤"

// 発言者の抽出（執行部）
const executivePattern = /^〇(.+?)(町長|副町長|教育長|.+課長|.+局長|会計管理者)/;
// 例: 〇川島隆史学校教育課長 → name="川島隆史", role="学校教育課長"

// 登壇の検出
const podiumPattern = /^〔(.+?)登壇〕/;

// 開催日の抽出
const datePattern = /(?:令和|平成)([\d０-９]+)年([\d０-９]+)月([\d０-９]+)日/;

// セクション見出し
const sectionPattern = /^◎(.+)/;

// 一般質問の質問者
const questionerPattern = /^◇\s*(.+?)\s*議\s*員/;
```

## 注意事項

- 会議録はすべて PDF 形式で公開されており、HTML 形式の本文は存在しない
- PDF のテキスト抽出時に、出席議員リスト部分で全角空白による名前の分割が発生する（例: `山　本　裕　子`）
- PDF ファイル名の命名規則が年代によって異なるため、HTML のリンクから URL を取得する必要がある
- 年別一覧ページのサブディレクトリ名にも一貫性がないため、トップページからのリンク辿りが必須
- ページ番号（`- 15 -` 等）がテキスト中に含まれるため、パース時に除去が必要
- 平成17年〜令和7年で約20年分のデータがあり、定例会（年4回）+ 臨時会で各会議複数日程があるため、PDF 総数は数百件規模
- 平成17年当時は議員19名、令和7年では13名と定数が変化している

---

## 推奨アプローチ

1. **トップページ起点の2段階クロール**: トップページ → 年別一覧ページ → PDF URL の順に辿り、全 PDF の URL リストを構築する
2. **HTML リンクテキストからメタ情報を取得**: PDF ファイル名からではなく、HTML のリンクテキストと `<h2>` 見出しから会議名・開催日・日程番号を抽出する（命名規則が不統一のため）
3. **PDF テキスト抽出**: `pdftotext` でテキストを抽出し、発言者パターンで分割する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年別一覧ページの「更新日」（`<p class="right">更新日：YYYY年M月D日</p>`）を監視し、更新があった年のみ再取得する
