# 青ヶ島村議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.vill.aogashima.tokyo.jp/
- 分類: 広報誌（PDF）内に議決一覧を掲載（会議録検索システムなし）
- 文字コード: UTF-8
- 形式: PDF ファイル（広報あおがしま）
- 特記: 日本最少人口の村（約155人）。議会の全文会議録はオンライン非公開。広報誌内に「議決一覧」として議案番号・議決結果のみ掲載。一般質問は録画し村内 CATV で放映。

---

## サイト構造

### 広報あおがしま

広報誌ポータル: `https://www.vill.aogashima.tokyo.jp/press/index.html`

年度選択メニューは PHP で動的生成される。

| エンドポイント | 用途 |
| --- | --- |
| `/php/pressmenu.php` | 年度一覧メニュー（2006年〜現在） |
| `/php/press.php?year={年}` | 指定年の広報誌 PDF リンク一覧 |
| `/press/koho{YYMM}.pdf` | 広報誌 PDF 本体 |

### 議会専用ページ

存在しない。サイトマップにも「議会」「会議録」の項目はない。

## URL 構造

### 広報誌 PDF

```
https://www.vill.aogashima.tokyo.jp/press/koho{YYMM}.pdf
```

- `{YY}`: 西暦下2桁（例: `25` → 2025年）
- `{MM}`: 月（ゼロ埋め2桁、例: `01` → 1月）
- 例: `koho2501.pdf` → 2025年1月号（No.414）

### 年度別一覧 API

```
https://www.vill.aogashima.tokyo.jp/php/press.php?year={年}
```

- `{年}`: 西暦4桁（例: `2024`）
- 対応範囲: 2006年〜現在
- レスポンス: `<ul><li>` 形式の HTML フラグメント（各月号の PDF リンクとファイルサイズ）

## 議会情報の掲載パターン

広報誌 PDF 内に以下の形式で議会関連情報が散在する。

### 1. 議決一覧（定例会後の号に掲載）

定例会終了後の翌月号に「議決一覧」が掲載される。

```
令和6年青ヶ島村議会第1回定例会議決一覧
3月7日
議案第 1 号　原案可決　青ヶ島村長等の給料等に関する条例の一部を改正する条例について
議案第 2 号　原案可決　...
...
3月28日
議案第16号　原案可決　青ヶ島村介護保険条例の一部を改正する条例について
議案第17号　同意　　　青ヶ島村監査員の選任の同意について
議案第18号　原案可決　令和5年度一般会計補正予算（第7号）について
```

- 掲載場所: 通常2〜3ページ目
- 内容: 議案番号、議決結果（原案可決/同意）、議案名
- 発言内容・質疑応答は含まれない

### 2. 村長公務（毎号掲載）

```
村長公務　令和6年12月
12月 5日（木）令和6年第4回青ヶ島村議会 定例会
```

- 定例会の開催日がわかる

### 3. 定例会開催案内（定例会前の号に掲載）

```
令和7年青ヶ島村議会第1回定例会
日時：3月6日（木）午前9時〜
場所：青ヶ島村役場2階 議会室
※傍聴を希望される方は、8時55分までに受付を済ませてから議会室へお入りください。
※一般質問を録画し、後日村内CATVにて放映します。
```

### 4. 教育委員会定例会（毎号掲載）

```
令和6年度 青ヶ島村教育委員会第9回定例会　令和6年12月26日（木）
議案第16号　可決　「青ヶ島村出身高校生の奨学金支給規則」の一部改正について
```

### 定例会の開催頻度

年4回の定例会が開催される。

| 回次 | 開催月 | 議決一覧の掲載号 |
| --- | --- | --- |
| 第1回 | 3月 | 4月号 |
| 第2回 | 6月 | 7月号 |
| 第3回 | 9月 | 10月号 |
| 第4回 | 12月 | 1月号 |

## スクレイピング戦略

### Step 1: 広報誌 PDF リンクの収集

`/php/press.php?year={年}` に 2006〜現在の各年を指定してリクエストし、PDF の URL 一覧を取得する。

- レスポンスは `<ul><li><a href="kohoYYMM.pdf">` 形式
- 各リンクの `href` を `https://www.vill.aogashima.tokyo.jp/press/` と結合して絶対 URL を生成

### Step 2: PDF のダウンロードとテキスト抽出

1. 収集した全 PDF URL からファイルをダウンロード
2. PDF パーサー（pdf-parse 等）でテキストを抽出

### Step 3: 議決一覧のパース

PDF テキストから議決一覧セクションを検出・パースする。

#### 議決一覧の検出

```typescript
// 議決一覧セクションの検出
const sessionPattern = /(?:令和|平成)\d+年青ヶ島村議会第\d+回定例会議決一覧/;
```

#### 議案のパース

```typescript
// 議案行の抽出
const billPattern = /議案第\s*(\d+)\s*号\s*(原案可決|同意|否決|撤回)\s+(.+)/;
// 例: "議案第 1 号　原案可決　青ヶ島村長等の..." → number="1", result="原案可決", title="青ヶ島村長等の..."

// 開催日の抽出
const datePattern = /(\d+)月(\d+)日/;
```

#### 村長公務からの開催日抽出

```typescript
// 定例会の開催日
const sessionDatePattern = /(\d+)月\s*(\d+)日.+青ヶ島村議会.+第(\d+)回定例会/;
```

## 注意事項

- **全文会議録は非公開**: 青ヶ島村議会は全文の会議録をオンラインで公開していない。取得できるのは議案番号・議決結果・議案名のみ
- **発言者情報なし**: 広報誌には個々の議員の発言内容や質疑応答は掲載されない
- **一般質問は CATV のみ**: 一般質問は録画され村内 CATV で放映されるが、テキストとしては公開されない
- **PDF のレイアウト**: 広報誌は多様なレイアウト（写真・表・図など）を含むため、テキスト抽出時にノイズが混入する可能性がある
- **議決一覧の掲載位置は不定**: 号によって掲載ページが異なるため、全ページのテキストを対象にパターンマッチする必要がある
- **教育委員会の議決一覧も存在**: 村議会とは別に教育委員会の定例会議決一覧も掲載されている。混同しないよう「青ヶ島村議会」を含むセクションのみを対象にする

---

## 推奨アプローチ

1. **議決一覧の抽出に特化**: 全文会議録が存在しないため、議案番号・議決結果・議案名の構造化データ収集を目標とする
2. **対象号の絞り込み**: 全号を処理するのではなく、定例会後の号（1月・4月・7月・10月号）を優先的に処理する
3. **PDF テキスト抽出の事前検証**: 数件の PDF を手動で確認し、議決一覧部分のテキスト抽出品質を検証する
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: PDF ファイル名が `kohoYYMM.pdf` の規則的な命名のため、取得済みの号を管理し新規号のみダウンロードする
6. **将来的な拡張**: 議会事務局への問い合わせにより全文会議録の提供可否を確認することを推奨
