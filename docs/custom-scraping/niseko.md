# ニセコ町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.niseko.lg.jp/chosei/gikai/kaigi
- 分類: 自治体公式サイトに PDF を直接掲載（会議録検索システムなし）
- 文字コード: UTF-8
- 会議録形式: PDF（テキスト埋め込み型）

## 検索パラメータ

検索フォームやクエリパラメータによる絞り込み機能は**なし**。年度別ページに直接アクセスして PDF リンクを取得する方式。

## 会議種別

年度別ページ内で h3 見出しにより以下の会議種別が区分されている:

- **定例会**（3月、6月、9月、12月）
- **臨時会**（不定期、年に複数回）
- **予算特別委員会**（3月定例会に併設）

見出しの表記例:
- `令和6年(2024年)第4回ニセコ町議会定例会`
- `令和6年(2024年)第1回ニセコ町議会臨時会`
- `令和6年度予算特別委員会`

## スクレイピング戦略

### Step 1: PDF URL の収集

年度別一覧ページ（`/chosei/gikai/kaigi/{年号}/`）から PDF リンクを抽出する。

**収集方法:**

1. トップページ `/chosei/gikai/kaigi` から全年度のリンク（`r07/` 〜 `h27/`）を取得
2. 各年度ページの HTML を取得し、PDF へのリンク（`/resources/output/contents/file/release/.../*.pdf`）を Cheerio で抽出
3. リンクテキスト（日付）と所属する h3 見出し（会議種別）を併せて記録

**ページネーション:** なし（各年度ページに全 PDF リンクが一括表示）

### Step 2: PDF のダウンロードとテキスト抽出

各 PDF をダウンロードし、テキストを抽出する。

- PDF はテキスト埋め込み型のため、`pdf-parse` 等のライブラリでテキスト抽出可能
- 画像 PDF（スキャン）ではないことを確認済み

### Step 3: 会議録のパース

#### メタ情報の抽出

ページ冒頭から以下を抽出:

- 会議名: 1行目（例: `令和6年(2024年)第4回ニセコ町議会定例会`）
- 開催日: 2行目（例: `令和6年(2024年)6月19日（水曜日）`）

#### 発言の構造

発言者パターン:

```
○議長（青羽雄士君）
○町長（片山健也君）
○3番（高木直良君）
```

- `○` で始まり、役職 or 番号 + 括弧内に氏名（`君` 敬称付き）

#### パース用正規表現（案）

```typescript
// 発言者の抽出
const speakerPattern = /^○(.+?)（(.+?)君）/;
// 例: ○議長（青羽雄士君） → role="議長", name="青羽雄士"
// 例: ○3番（高木直良君） → role="3番", name="高木直良"

// 開催日の抽出
const datePattern = /令和(\d+)年.*?(\d+)月(\d+)日/;

// 議事日程の区切り
const agendaPattern = /^◎(.+)/;
// 例: ◎日程第1　会議録署名議員の指名

// 議場の注記
const notePattern = /^（.+）$/;
```

## 注意事項

- 会議録は PDF 形式のみで提供されており、HTML 版は存在しない
- PDF のファイル名規則が年度によって異なる（`YYYYMMDD.pdf`、`RYYMMDD.pdf`、`hYYMMDD.pdf` など）
- `release_id` が年度によって変わる（近年: `10778`、古い年度: `852` など）
- 予算特別委員会の PDF はファイル名に `yosan` サフィックスが付く
- 出席議員は 10 名（小規模議会）

---

## 推奨アプローチ

1. **年度ページのクロールで PDF URL を一括収集**: 11 年度分のページを巡回するだけで全 PDF の URL が取得できる
2. **PDF テキスト抽出**: `pdf-parse` や `pdf.js` を使用してテキストを抽出
3. **メタ情報は年度ページの HTML から取得**: PDF をパースせずとも、リンクテキスト（日付）と h3 見出し（会議種別）から基本メタ情報を取得可能
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度ページの PDF リンク数を前回と比較し、新規追加分のみダウンロードする
