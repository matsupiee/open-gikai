# 茂原市議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.city.mobara.chiba.jp/category/6-4-3-0-0-0-0-0-0-0.html
- 分類: 自治体 CMS による PDF 公開（年度別ページに PDF リンクを掲載）
- 文字コード: UTF-8
- 特記: インターネット中継システム（jfit.co.jp）で映像配信と簡易検索が可能

## PDF ファイル命名規則

PDF ファイル名は `{和暦年下2桁}.{月}.{日}.pdf` の形式。

| 例 | 意味 |
| --- | --- |
| `25.11.26.pdf` | 令和7年（2025年）11月26日 |
| `24.11.27.pdf` | 令和6年（2024年）11月27日 |
| `23.9.01.pdf` | 令和5年（2023年）9月1日 |
| `23.02.22.1.pdf` | 令和5年（2023年）2月22日（同日複数の場合に末尾連番） |

- 月・日のゼロ埋めは不統一（`9.01` のように混在する場合あり）
- 同日に複数の会議録がある場合、末尾に `.1`, `.2` 等の連番が付く場合がある

## ページ構造

### 一覧ページ（トップ）

- 年度別のリンクが降順で並ぶ（令和7年〜平成20年）
- ページネーションなし（単一ページに全年度を表示）
- 各リンクのテキスト: `令和X年会議録`

### 年度別ページ

- `<h2>` / `<h3>` で定例会・臨時会のセクションを区分
  - 例: `<h3>12月定例会　会議録</h3>`
- 各会議録は `<ul>/<li>` のリスト形式で表示
- リスト項目に PDF アイコン画像（`pdf.gif`）付きのリンク
- リンクテキスト例: `第1号　令和7年11月26日　（PDF形式、534.27KB）`

### 会議の種類

各年度ページには以下の会期が含まれる（年度により異なる）:

- 3月定例会（通常5号まで）
- 6月定例会（通常5号まで）
- 9月定例会（通常5号まで）
- 12月定例会（通常5号まで）
- 臨時会（不定期、通常1号のみ）

## スクレイピング戦略

### Step 1: 年度別ページ URL の収集

一覧ページ `category/6-4-3-0-0-0-0-0-0-0.html` から全年度ページへのリンクを取得する。

**収集方法:**

1. 一覧ページの HTML を取得
2. `0000XXXXXX.html` パターンのリンクを Cheerio で抽出
3. 各リンクのテキストから年度情報を取得

### Step 2: PDF リンクの収集

各年度別ページから PDF ファイルへのリンクを取得する。

**収集方法:**

1. 年度別ページの HTML を取得
2. `cmsfiles/contents/` 配下の `.pdf` リンクを抽出
3. リンクテキストから以下のメタ情報をパース:
   - 号数（第X号）
   - 開催日（令和X年X月X日）
   - 会期名（所属する `<h3>` セクションから取得: 「12月定例会」等）

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

**方法:**

1. PDF をダウンロードしてローカルに保存
2. `pdf-parse` 等のライブラリでテキストを抽出
3. テキストから発言者・発言内容をパース

### Step 4: 会議録のパース

#### メタ情報

PDF リンクのテキストおよびセクション見出しから以下を抽出:

- 開催日: リンクテキストの `令和X年X月X日` 部分
- 会議名: `<h3>` セクション見出し（例: `12月定例会`）
- 号数: リンクテキストの `第X号` 部分

#### 発言の構造（PDF 内）

PDF のテキストから発言者と発言内容を分離する。茂原市議会の会議録 PDF は一般的な議会会議録のフォーマットに従い、以下のパターンが想定される:

```typescript
// 発言者の抽出（一般的な議会会議録パターン）
const speakerPattern = /^○(.+?)（(.+?)）/;
// 例: ○議長（山田太郎） → role="議長", name="山田太郎"

// 番号付き発言者
const numberedSpeakerPattern = /^○(\d+番)\s*（(.+?)）/;

// 開催日の抽出
const datePattern = /令和(\d+)年(\d+)月(\d+)日/;
```

※ PDF からのテキスト抽出結果により、実際のパターンは調整が必要。

## 注意事項

- 会議録は PDF 形式のみで公開されており、HTML 版は存在しない
- PDF ファイル名の月・日のゼロ埋めが不統一なため、ファイル名からの日付パースには注意が必要
- PDF のレイアウト（段組み、ヘッダー/フッター等）によりテキスト抽出の精度が変動する可能性がある
- 平成20年（2008年）まで遡及可能
- インターネット中継システム（jfit.co.jp）の簡易検索は映像配信用であり、テキストデータの取得には不向き

---

## 推奨アプローチ

1. **一覧ページから全量リスト作成**: まず一覧ページ → 年度別ページの2段階で全 PDF の URL リストを作成
2. **PDF ダウンロード → テキスト抽出の2段階処理**: ダウンロードとパースを分離し、再実行コストを削減
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **PDF テキスト抽出の品質確認**: 初回は数件の PDF を手動確認し、テキスト抽出の品質と発言者パターンを検証する
5. **差分更新**: 年度別ページ ID は単調増加のため、新しいページ ID のみを取得する差分更新が可能
