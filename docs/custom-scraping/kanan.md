## PDF ファイル名パターン

PDF のファイル名は年代によって命名規則が異なる。

### 令和3年〜現在（新パターン）

```
R{和暦2桁}{月2桁}_teireikaigi.pdf   （定例会議）
R{和暦2桁}{月2桁}_rinjikaigi.pdf    （臨時会議）
```

例:
- `R0703_teireikaigi.pdf` → 令和7年3月定例会議
- `R0701_rinjikaigi.pdf` → 令和7年1月臨時会議
- `R03_06teireikaigi.pdf` → 令和3年6月定例会議（アンダースコア位置が異なる変種あり）

### 平成28年（旧パターン）

```
{西暦下2桁}{月2桁}kaigiroku.pdf      （定例会議）
{西暦下2桁}{月2桁}rinjikaigiroku.pdf  （臨時会議）
```

例:
- `2803kaigiroku.pdf` → 平成28年3月会議録
- `2810rinjikaigiroku.pdf` → 平成28年10月臨時会議録

### グループ ID の変遷

- 令和6年〜: `group/22`
- 令和3年〜令和4年: `group/1`
- 平成28年: `group/22`

※ 年度によってグループ ID が異なるため、PDF URL を直接組み立てるのは困難。各年度一覧ページから PDF リンクを抽出する方式を推奨。

## PDF の格納形式

- 各会議（定例会議・臨時会議）ごとに 1 つの PDF ファイル
- PDF には会議全体の会議録が含まれる（発言者ごとの分割なし）
- 予算・決算常任委員会は議題別・日ごとに分冊化されている（例: 「補正予算及び当初予算（1日目）」）

## スクレイピング戦略

### Step 1: 年度別一覧ページの URL 収集

トップページ `index.html` から年度別一覧ページへのリンクを収集する。

- `<a>` タグの `href` 属性から `/gyoseijoho/gikai/1/{ページID}.html` パターンを抽出
- 定例会議と予算・決算常任委員会の両方を対象とする

### Step 2: PDF リンクの収集

各年度別一覧ページから PDF ファイルへのリンクを収集する。

- `<a>` タグの `href` 属性から `material/files/group/{グループID}/{ファイル名}.pdf` パターンを抽出
- リンクテキストから会議名（「令和6年12月定例会議会議録」等）を取得

### Step 3: PDF のダウンロードとテキスト抽出

PDF をダウンロードし、テキストを抽出する。

- PDF パーサー（pdf-parse 等）を使用してテキストを抽出
- 会議録全体が 1 ファイルに含まれるため、発言者ごとの分割が必要

### Step 4: 会議録のパース

#### メタ情報

PDF のリンクテキストから抽出:

```
令和6年12月定例会議会議録
令和7年1月臨時会議会議録
```

- 開催年月: リンクテキストまたはファイル名から取得
- 会議種別: 「定例会議」「臨時会議」「予算・決算常任委員会」

#### 発言の構造

PDF 内のテキストから発言者と発言内容を分離する必要がある。具体的なフォーマットは PDF の内容に依存するが、一般的な議会会議録では以下のパターンが想定される:

```
○議長（氏名）
○XX番（氏名）
○町長（氏名）
```

※ 実際の PDF を確認してパースルールを確定する必要がある。

## 注意事項

- 会議録検索システムが未導入のため、全文検索は不可。PDF 単位での取得が必要
- PDF ファイル名の命名規則が年代によって異なるため、URL を直接生成するのではなく HTML ページからリンクを抽出する方式が必須
- グループ ID（`group/1` or `group/22`）も年度によって変わる
- PDF のファイルサイズは 271KB〜3.3MB と幅がある
- 予算・決算常任委員会は令和5年・6年のみ掲載（それ以前は定例会議に含まれている可能性あり）

---

## 推奨アプローチ

1. **2段階クロール**: まずトップページ → 年度別一覧ページ → PDF リンク、の 2 段階でリンクを収集
2. **PDF テキスト抽出**: `pdf-parse` 等のライブラリで PDF からテキストを抽出
3. **発言者パース**: PDF テキストから発言者パターンを検出し、発言単位に分割
4. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
5. **差分更新**: 年度別一覧ページの PDF リンク数を前回と比較し、新規追加分のみ取得する
