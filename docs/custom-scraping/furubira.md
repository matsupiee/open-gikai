# 古平町議会 カスタムスクレイピング方針

## 概要

- サイト: https://www.town.furubira.lg.jp/town/detail.php?id=59
- 分類: 公式ウェブサイト上で PDF 形式の会議録を年別に公開（既存アダプターでは対応不可）
- 文字コード: UTF-8
- 特記: Google Tag Manager（GTM-T7Z8969）使用

---

## URL 構造

| ページ | URL パターン |
| --- | --- |
| 会議録一覧 | `https://www.town.furubira.lg.jp/town/detail.php?id=59` |
| 会議録 PDF | `https://www.town.furubira.lg.jp/common/img/content/{ファイル名}.pdf` |

---

## PDF ファイル命名規則

PDF ファイル名は `cassette_{カセット番号}_pdf{連番}_{アップロード日時}.pdf` の形式。

- 例: `cassette_2_pdf02_20250520_093945.pdf`
- 例: `cassette_40_pdf01_20250218_112505.pdf`
- 例: `cassette_42_pdf01_20260311_091722.pdf`

ファイル名から会議種別や開催日を推定することはできないため、リンクテキストからメタ情報を取得する必要がある。

---

## ページ構造

### 一覧ページの HTML 構造

- 年度ごとに `<h2 class="sec-title06"><span>令和X年</span></h2>` の見出しで区切り
- 各年の会議録 PDF リンクは `<ul class="pdf-list">` 内の `<li><a href="...pdf" target="_blank">会議名</a></li>` として列挙
- PDF リンクのパスは相対パス `../common/img/content/xxx.pdf` 形式
- ページネーションなし（全年度の会議録を 1 ページに掲載）
- 1 つの年度に対して `<ul class="pdf-list">` が複数存在する場合がある（令和5年、令和6年 等）

### 掲載範囲

- 平成 24 年（2012 年）〜 現在（令和 8 年）
- 年度表記の揺れあり: 「令和5年」「令和４年分」「令和元年（平成31年）分」「平成30年分」等

### 会議種別

| 種別 | 説明 |
| --- | --- |
| 定例会 | 年 3〜4 回開催、複数号数に分かれる場合あり（例: 第１回定例会（第１号）、第１回定例会（第２号）） |
| 臨時会 | 年 1〜5 回程度開催 |
| 予算審査特別委員会 | 定例会に付随、複数号数に分かれる場合あり |
| 決算審査特別委員会 | 定例会に付随、複数号数に分かれる場合あり |

## スクレイピング戦略

### Step 1: PDF リンクの収集

一覧ページ `https://www.town.furubira.lg.jp/town/detail.php?id=59` から全 PDF リンクを抽出する。

- ページネーションがないため、1 リクエストで全データを取得可能
- `<ul class="pdf-list">` 内の `a[href$=".pdf"]` セレクタでリンクを抽出
- 直前の `<h2 class="sec-title06">` から年度を取得

**収集方法:**

1. 一覧ページの HTML を取得
2. Cheerio で `h2.sec-title06` と `ul.pdf-list` を順に走査
3. 各 `h2` に対応する直後の `ul.pdf-list` 内の PDF リンクを紐付け
4. リンクテキスト（例: 「第１回定例会（第１号）」）から会議種別・回数・号数を抽出
5. 相対パス `../common/img/content/xxx.pdf` をベース URL と結合してフル URL を生成

### Step 2: メタ情報の抽出

#### 年度の抽出（見出しから）

```typescript
// 年度パターン（複数の表記揺れに対応）
const yearPatterns = [
  /令和(\d+)年/, // 令和8年、令和5年
  /令和元年/, // 令和元年（平成31年）分
  /平成(\d+)年/, // 平成30年分、平成24年分
];
```

#### 会議名の抽出（リンクテキストから）

```typescript
// リンクテキストパターン
const meetingPattern = /^(第[０-９\d]+回)?(定例会|臨時会|予算審査特別委員会|決算審査特別委員会)(?:（(第[０-９\d]+号)）)?$/;
// 例: "第１回定例会（第１号）" → 回=1, 種別=定例会, 号=1
// 例: "第２回臨時会" → 回=2, 種別=臨時会
// 例: "予算審査特別委員会（第１号）" → 種別=予算審査特別委員会, 号=1
```

### Step 3: PDF のダウンロード

収集した PDF URL に対してファイルをダウンロードする。

- ベース URL: `https://www.town.furubira.lg.jp/town/detail.php?id=59`
- 相対パス `../common/img/content/xxx.pdf` を解決して `https://www.town.furubira.lg.jp/common/img/content/xxx.pdf` を生成

### Step 4: PDF からのテキスト抽出

ダウンロードした PDF から会議録本文を抽出する。

- PDF パーサー（pdf-parse 等）を使用してテキストを抽出
- PDF 内のレイアウトに依存するため、抽出精度の検証が必要

---

## 注意事項

- PDF 形式のため、テキスト抽出の精度は PDF の作成方法に依存する（画像 PDF の場合は OCR が必要）
- ファイル名に会議種別や開催日の情報が含まれないため、メタ情報はリンクテキストと見出しから取得する必要がある
- 年度表記に揺れがある（「令和5年」vs「令和４年分」、全角・半角混在）
- 1 つの年度に対して `<ul class="pdf-list">` が複数に分割されている場合があるため、見出しとリストの対応関係に注意が必要
- 開催日の情報はページ上に掲載されていないため、PDF 本文内から抽出する必要がある

---

## 推奨アプローチ

1. **1 ページ完結**: 一覧ページが 1 ページで完結しているため、クロール処理はシンプルに実装可能
2. **PDF パース**: HTML ではなく PDF からのテキスト抽出が主要な課題。まず数件の PDF をダウンロードしてテキスト抽出の品質を検証する
3. **レート制限**: 自治体サイトのため、リクエスト間に適切な待機時間（1〜2 秒）を設ける
4. **差分更新**: 一覧ページの PDF リンク数を前回取得時と比較し、新規リンクのみをダウンロードする差分更新が可能
5. **見出しとリストの紐付け**: `h2.sec-title06` と `ul.pdf-list` の DOM 上の順序関係で年度を特定する（1 つの年度に複数リストがある場合に注意）
